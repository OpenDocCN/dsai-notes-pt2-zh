## 树的基本概念 🌳

树是一种层次化的数据结构，通常用于以易于导航和搜索的方式组织数据。它本质上是由节点和边组成的层次集合。

以下是构成树的基本元素：
*   **节点**：树中的每个圆圈代表一个节点。
*   **边**：连接两个节点的有向箭头称为边。一棵有 `n` 个节点的树，有 `n-1` 条边。

根据节点在树中的位置，我们可以对其进行分类：
*   **根节点**：没有父节点的节点，通常位于树的顶部。
*   **叶节点**：没有子节点的节点，位于树的末端。
*   **内部节点**：既不是根节点也不是叶节点的节点。
*   **兄弟节点**：位于同一层级的节点。

此外，从父节点到子节点的分支数量称为**分支因子**。我们目前看到的树分支因子为2，这是一种特殊情况，称为**二叉树**。在更一般的情况下，树可以有任意数量的分支。

## 树的遍历 🚶‍♂️

了解了树是什么之后，我们来看看如何遍历树。主要有两种方法：广度优先搜索和深度优先搜索。

### 广度优先搜索

广度优先搜索的思路很直接：按层级遍历树。想象你在一栋建筑里，你会先访问第一层的所有房间，然后再去第二层，依此类推。

在树中，这意味着：
1.  从根节点（例如节点5）开始。
2.  访问第二层的所有节点（节点3和8）。
3.  访问第三层的所有节点（节点1, 4, 7, 10）。

在列出节点顺序时，通常左子节点优先于右子节点。因此，对这棵树的BFS遍历顺序是：**5, 3, 8, 1, 4, 7, 10**。

### 深度优先搜索

上一节我们按层级遍历了树，本节中我们来看看另一种策略：深度优先搜索。DFS会沿着一条路径尽可能深入，然后再回溯并探索其他路径。

对于DFS，有三种不同的遍历顺序，我们通过一个简单的子树（根节点5，左子节点3，右子节点8）来说明：

以下是三种遍历方式：
*   **前序遍历**：顺序为 **节点(N)，左子树(L)，右子树(R)**。
    *   访问顺序：先访问当前节点5，然后遍历左子树（节点3），最后遍历右子树（节点8）。结果为：**5, 3, 8**。
*   **中序遍历**：顺序为 **左子树(L)，节点(N)，右子树(R)**。
    *   访问顺序：先遍历左子树（节点3），然后访问当前节点5，最后遍历右子树（节点8）。结果为：**3, 5, 8**。
*   **后序遍历**：顺序为 **左子树(L)，右子树(R)，节点(N)**。
    *   访问顺序：先遍历左子树（节点3），然后遍历右子树（节点8），最后访问当前节点5。结果为：**3, 8, 5**。

名称“前序”、“中序”、“后序”指示了在遍历过程中**访问当前节点的时机**。

现在，让我们将这些规则应用到之前更大的例树上：
*   **前序遍历**：5, 3, 1, 4, 8, 7, 10
*   **中序遍历**：1, 3, 4, 5, 7, 8, 10
*   **后序遍历**：1, 4, 3, 7, 10, 8, 5

**所有这些遍历方法（BFS和三种DFS）的时间复杂度都是 O(n)**，其中 `n` 是树中节点的数量，因为每个节点都会被访问一次。

## 二叉搜索树 🔍

之前我们讨论了普通的二叉树，现在我们来学习一种带有额外约束的特殊二叉树——二叉搜索树。

BST的约束条件是：对于树中的任意节点，其值**严格大于**其左子树中所有节点的值，并且**小于或等于**其右子树中所有节点的值。用公式表示，对于节点 `N`，其左子节点 `L` 和右子节点 `R`，满足：`L.value < N.value <= R.value`。

### BST插入操作

了解了BST的定义后，我们来实现一个BST的插入方法。给定一个BST类，其构造函数初始化节点值，我们需要完成 `insert` 函数。

```python
class BST:
    def __init__(self, value):
        self.value = value
        self.left = None
        self.right = None

    def insert(self, value):
        if value < self.value:          # 空白 A：当插入值小于当前节点值时，应进入左子树
            if self.left is None:       # 空白 F：如果左子节点为空
                self.left = BST(value)  # 空白 B：则创建一个新的BST节点作为左子节点
            else:
                self.left.insert(value) # 空白 C：否则，递归地在左子树中插入
        else:                           # 空白 D：当插入值大于等于当前节点值时，进入右子树
            if self.right is None:      # 空白 L：如果右子节点为空
                self.right = BST(value) # 空白 E：则创建一个新的BST节点作为右子节点
            else:
                self.right.insert(value)# 空白 M：否则，递归地在右子树中插入
        return self
```

逻辑解析：
1.  **比较值**：将待插入值与当前节点值比较。
2.  **向左插入**：如果值更小，则尝试插入左子树。
    *   若左子节点为空，则直接创建新节点。
    *   若左子节点已存在，则递归调用左子节点的 `insert` 方法。
3.  **向右插入**：如果值大于或等于当前值，则尝试插入右子树，逻辑与向左插入对称。

### 构建最小高度的BST

现在，我们面临一个更综合的问题：如何从一个已排序的列表 `[1, 2, 5, 7, 10, 13, 14, 15, 22]` 构建一棵**高度最小**的二叉搜索树。

关键思路是**平衡**。为了得到最小高度，我们需要让树的左右子树尽可能包含相同数量的节点。这可以通过始终选择列表的**中间元素**作为（子）树的根节点来实现。

以下是构建最小高度BST的递归函数：

```python
def minHeightBst(array):
    # 空白 A 和 B：递归基
    if len(array) == 0:
        return None                     # 空白 A：如果数组为空，返回None
    if len(array) == 1:
        return BST(array[0])            # 空白 B：如果数组只有一个元素，直接创建节点返回

    # 找到中间索引和中间值
    mid_idx = len(array) // 2           # 空白 C：计算中间索引（整数除法）
    root_value = array[mid_idx]         # 空白 D：获取中间值作为根节点值
    root = BST(root_value)              # 空白 E：创建根节点

    # 分割左右子数组
    left_subarray = array[:mid_idx]     # 空白 F：左子数组（中间元素之前的部分）
    right_subarray = array[mid_idx+1:]  # 空白 G：右子数组（中间元素之后的部分）

    # 递归构建左右子树
    root.left = minHeightBst(left_subarray)   # 空白 H：递归构建左子树
    root.right = minHeightBst(right_subarray) # 空白 I：递归构建右子树

    return root
```

构建过程示例（以列表 `[1, 2, 5, 7, 10, 13, 14, 15, 22]` 为例）：
1.  中间元素是 `10`，成为根节点。
2.  左子数组 `[1, 2, 5, 7]` 递归构建左子树。
    *   其中间元素 `5` 成为左子树的根。
    *   左子数组 `[1, 2]` 递归构建，中间元素 `2` 成为节点，`1` 成为其左子节点。
    *   右子数组 `[7]` 直接成为节点5的右子节点。
3.  右子数组 `[13, 14, 15, 22]` 递归构建右子树，过程类似。

最终得到的BST高度最小，且满足BST的性质。
