## 无限假设空间的复杂性

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_16.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_18.png)

上一节我们讨论了有限假设空间下的PAC学习理论。本节中，我们来看看当假设空间H是无限时，如何衡量其复杂性。这是机器学习中更常见的情况。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_29.png)

为了处理无限假设空间，我们需要引入一个新的复杂性度量——VC维。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_31.png)

### VC维的定义

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_33.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_35.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_36.png)

给定一个假设空间H和一个包含M个数据点的数据集S，我们可以将H中的每个假设h应用于S，得到一组对S的标记（即预测标签的向量）。所有不同标记的集合记为Π_H(S)。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_38.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_39.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_40.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_41.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_42.png)

如果假设空间H能够对数据集S产生所有可能的标记（即2^M种），那么我们称H**打散**了S。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_44.png)

VC维的定义是：假设空间H的VC维，是能被H打散的最大集合S的大小。如果H可以打散任意大的集合，则其VC维为无穷大。

### 计算VC维的步骤

以下是计算一个假设空间VC维的一般步骤：
1.  找到一个大小为d的数据集S，证明H可以打散S。
2.  证明不存在任何大小为d+1的数据集能被H打散。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_46.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_47.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_48.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_49.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_50.png)

### VC维计算示例

让我们通过一个例子来理解这个过程。考虑二维空间中的所有线性分类器（即所有可能的直线）。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_52.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_54.png)

*   **1个点**：可以打散。通过调整直线的位置和方向，可以得到该点为+1或-1的两种标记。
*   **2个点**：可以打散。对于任意两个点，总可以找到一条直线将它们按任意方式（++, +-, -+, --）分开。
*   **3个点**：**存在**一个能被H打散的3点集（例如，三个不共线的点）。可以找到8条不同的直线实现所有8种标记组合。
*   **4个点**：**不存在**能被H打散的4点集。无论四个点如何排列（例如，构成凸四边形的顶点，或一个点在另外三个点构成的三角形内部），总有一些标记组合无法用一条直线实现。

因此，二维线性分类器的VC维是3。直观上，这与模型参数数量（斜率和截距，共3个自由度）有关，但并非总是如此。

## 基于VC维的学习理论

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_56.png)

现在，我们可以用VC维来替代之前有限假设空间定理中的|H|，从而得到适用于无限假设空间的泛化界。

### 可实现情况下的VC界

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_58.png)

如果真实目标函数c* ∈ H（可实现情况），那么只要训练样本数M满足以下条件，就能以高概率保证所有训练误差为0的假设，其真实误差R(h) ≤ ε。

**公式**：
M ≥ O( (1/ε) ( VC(H) + log(1/δ) ) )

由此可以推导出统计学习理论的推论：给定M个样本，假设h的真实误差满足：

**公式**：
R(h) ≤ R̂(h) + O( √( (VC(H) + log(1/δ)) / M ) )

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_60.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_62.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_63.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_65.png)

### 不可知情况下的VC界

在更一般的不可知情况下（不假设c* ∈ H），相应的界为：

**公式**：
M ≥ O( (1/ε²) ( VC(H) + log(1/δ) ) )

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_67.png)

其统计学习理论推论为：

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_69.png)

**公式**：
R(h) ≤ R̂(h) + O( √( (VC(H) + log(1/δ)) / M ) )

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_71.png)

### 近似-泛化权衡

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_73.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_74.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_75.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_77.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_78.png)

观察不可知情况下的推论，我们可以发现一个机器学习中的基本权衡：

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_80.png)

**公式**：
R(h) ≤ (训练误差 R̂(h)) + (泛化惩罚项 O(√(VC(H)/M)) )

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_82.png)

*   **训练误差**：衡量假设h在训练数据上对目标函数的近似程度。通常，更复杂的模型（VC维更高）能更好地拟合训练数据，降低此项。
*   **泛化惩罚项**：衡量模型复杂性带来的过拟合风险。VC维越高，此项越大。

这导致了**近似-泛化权衡**：为了更好拟合数据，我们希望使用更复杂的模型（高VC维），但这会增加过拟合风险，损害泛化能力。在实践中，我们通过模型选择（如正则化、验证集调参）来寻找最佳平衡点，使真实误差最小化。

## 从频率主义到贝叶斯：参数估计

现在，我们将话题转向概率学习。之前我们假设存在一个确定性的目标函数c*。更现实的设定是存在一个目标分布P*，给定输入x，输出y有一定的概率。我们的目标是学习一个能近似P*的概率模型。

### 最大似然估计

学习概率模型参数θ的一种经典方法是**最大似然估计**。其思想是：选择能使观测到当前数据D的概率最大的参数值。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_84.png)

**公式**：
θ_MLE = argmax_θ P(D | θ)

由于数据点通常独立同分布，联合概率可分解为乘积。为方便优化，我们通常最大化**对数似然**：

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_86.png)

**公式**：
θ_MLE = argmax_θ Σ_{i=1}^N log P(x_i | θ)

#### MLE示例：伯努利分布

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_88.png)

假设我们有一个服从伯努利分布（如抛硬币）的数据集，参数φ表示正面朝上的概率。给定观测数据（一系列正面/反面），其对数似然为：

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_90.png)

**代码/公式**：
LL(φ) = n_head * log(φ) + n_tail * log(1 - φ)

通过求导并令其为零，可得到MLE估计：

**公式**：
φ_MLE = n_head / (n_head + n_tail)

这直观地告诉我们，最好的估计就是观测到的正面频率。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_92.png)

### 最大后验估计

MLE完全依赖于观测数据。然而，我们通常对参数有一些先验知识或信念（例如，认为硬币大致均匀）。**最大后验估计**允许我们将这些先验知识融入参数估计。

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_94.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_96.png)

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_98.png)

MAP估计旨在最大化给定数据后参数的后验概率。根据贝叶斯定理：

**公式**：
P(θ | D) = [P(D | θ) * P(θ)] / P(D)

由于P(D)与θ无关，在优化中可以忽略。因此：

![](img/67da55c34ebbf0a4f5cf2f2803dff61d_100.png)

**公式**：
θ_MAP = argmax_θ P(D | θ) * P(θ) = argmax_θ [似然 * 先验]

这里，P(θ)是参数的**先验分布**，代表了我们在看到数据之前对θ的信念。MAP估计在似然的基础上，用先验对参数估计进行“平滑”或修正，避免像MLE那样因数据量小而产生极端估计（例如，5次抛掷均正面朝上就估计φ=1）。

## 总结

本节课中我们一起学习了：
1.  **无限假设空间的复杂性度量**：引入了VC维的概念，它描述了假设空间打散数据集的能力，用于替代有限假设空间的大小|H|。
2.  **基于VC维的泛化理论**：得到了适用于无限假设空间的PAC学习界，并理解了其统计学习理论推论。
3.  **近似-泛化权衡**：从理论界中揭示了模型复杂性与泛化能力之间的基本权衡，这指导了模型选择和实践。
4.  **参数估计方法**：回顾了**最大似然估计**，它寻找使观测数据概率最大的参数；并引入了**最大后验估计**，它在MLE的基础上结合了参数的先验分布，从而能融入领域知识，得到更稳健的估计。
