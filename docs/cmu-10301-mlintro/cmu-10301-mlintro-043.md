在本节课中，我们将要学习矩阵微积分、前向传播和反向传播的核心概念。这些内容是理解神经网络如何训练和优化的基础。我们将从矩阵微积分的布局约定开始，逐步深入到神经网络的前向计算和反向梯度计算。

## 矩阵微积分

上一节我们介绍了课程的整体结构，本节中我们来看看矩阵微积分的基础知识。矩阵微积分是处理向量和矩阵导数的工具，在机器学习优化问题中至关重要。

我们使用的布局约定称为分母布局。在这种约定下，导数的形状由分母的形状决定。例如，对于向量 **y** ∈ ℝʳ 和 **x** ∈ ℝᵖ，导数 **dy/dx** 是一个 **p × r** 的矩阵。

### 链式法则与形状匹配

在标量微积分中，链式法则允许我们交换导数的顺序，因为标量乘法是可交换的。然而，在矩阵微积分中，乘法的顺序至关重要。

假设我们有 **z = g(y)** 和 **y = f(x)**，其中 **x** ∈ ℝᵖ, **y** ∈ ℝʳ, **z** ∈ ℝⁿ。根据分母布局，导数的形状如下：
*   **dy/dx** 是 **p × r** 矩阵。
*   **dz/dy** 是 **r × n** 矩阵。
*   **dz/dx** 是 **p × n** 矩阵。

为了通过链式法则得到 **dz/dx**，我们必须以正确的顺序相乘：
**dz/dx = (dy/dx) (dz/dy)**
只有这样，矩阵的维度 (**p × r**) × (**r × n**) = (**p × n**) 才能匹配。

### 单元素法与向量形式

有时，通过计算单个矩阵元素来推导向量形式的导数会更简单。考虑一个 **n × n** 的矩阵 **A** 和一个 **n** 维向量 **x**。我们想求 **d(Ax)/dx**。

首先，我们计算这个 **n × n** 矩阵的第 *ij* 个元素。根据分母布局，这等于输出向量第 *j* 个元素对输入向量第 *i* 个元素的导数：
**(d(Ax)/dx)ᵢⱼ = d(Ax)ⱼ / dxᵢ**

展开 **(Ax)ⱼ = Σₖ Aⱼₖ xₖ**，并对 **xᵢ** 求导，结果是 **Aⱼᵢ**。因此，整个导数矩阵的第 *ij* 个元素是 **Aⱼᵢ**，这意味着：
**d(Ax)/dx = Aᵀ**

这个例子表明，仅靠形状匹配（结果都是 **n × n**）不足以确定正确答案，我们必须仔细推导。

### 应用于线性回归

让我们将矩阵微积分规则应用于线性回归的闭式解推导。损失函数是：
**L(θ) = ½ ||y - Xθ||²**
其中 **y** ∈ ℝⁿ, **X** ∈ ℝⁿˣᵈ, **θ** ∈ ℝᵈ。我们的目标是求 **dL/dθ** 并令其为零。

首先，展开损失函数：
**L(θ) = ½ (yᵀy - 2yᵀXθ + θᵀXᵀXθ)**

现在，逐项对 **θ** 求导：
1.  **d(yᵀy)/dθ = 0**
2.  **d(-2yᵀXθ)/dθ = -2Xᵀy** （应用规则 **d(Bθ)/dθ = Bᵀ**，其中 **B = -2yᵀX**）
3.  **d(θᵀXᵀXθ)/dθ = 2XᵀXθ** （应用类似标量平方的链式法则规则）

![](img/2d1474c72c360abfdb0b1f9a54b9f318_7.png)

![](img/2d1474c72c360abfdb0b1f9a54b9f318_8.png)

将各项相加：
**dL/dθ = -2Xᵀy + 2XᵀXθ**

令导数为零并求解 **θ**：
**0 = -2Xᵀy + 2XᵀXθ**
**XᵀXθ = Xᵀy**
**θ = (XᵀX)⁻¹Xᵀy**

这就是线性回归的闭式解。我们仅使用了少数几条矩阵微积分规则就完成了推导，无需降维到标量计算。

## 前向传播

上一节我们介绍了如何使用矩阵微积分进行优化，本节中我们来看看神经网络中的前向传播。前向传播是根据输入数据、权重和激活函数计算损失函数值的过程。

考虑一个具有一个隐藏层的简单神经网络：
*   输入层：两个特征 **x = [x1, x2]ᵀ**，我们添加一个截距项 **x0 = 1**，所以输入变为 **x̂ = [1, x1, x2]ᵀ**。
*   隐藏层：两个节点，输出 **z = [z1, z2]ᵀ**。我们同样添加截距 **z0 = 1**，得到 **ẑ = [1, z1, z2]ᵀ**。
*   输出层：一个标量预测值 **ŷ**。
*   权重：从输入到隐藏层的权重矩阵 **α**，从隐藏层到输出的权重向量 **β**。
*   激活函数：隐藏层使用 ReLU 函数，输出层不使用激活函数。

以下是前向传播的步骤：

1.  **计算隐藏层线性组合**：**a = α x̂**，其中 **a = [a1, a2]ᵀ**。
2.  **应用激活函数**：**z = ReLU(a)**，即 **zᵢ = max(0, aᵢ)**。
3.  **添加隐藏层截距**：**ẑ = [1, z1, z2]ᵀ**。
4.  **计算输出**：**ŷ = β ẑ**。
5.  **计算损失**：假设使用均方误差，**L = ½ (ŷ - y)²**，其中 **y** 是真实值。

### 为什么需要截距项和激活函数？

以下是关于网络设计的两个关键点：

*   **截距项**：与线性回归类似，截距项允许模型更好地拟合数据，通过平移决策边界或超平面。
*   **非线性激活函数**：如果所有层都使用线性激活函数，那么整个网络将等价于一个线性回归模型。非线性激活函数（如 ReLU）引入了非线性，使网络能够学习更复杂的关系。

## 计算图

上一节我们描述了前向传播的步骤，本节中我们使用计算图来形式化表示这些计算。计算图以图形方式表示一系列数学运算，每个节点代表一个基本操作或中间变量。

构建计算图有助于理解数据流和依赖关系，对于高效实现反向传播至关重要，因为它指明了哪些中间值需要在正向传递中保存以供后续梯度计算使用。

对于上述神经网络，其计算图包含以下节点和连接：
1.  输入：**x**（以及隐含的常数 1）。
2.  操作：将 **x** 与常数 1 连接，形成 **x̂**。
3.  操作：矩阵乘法 **a = α x̂**。
4.  操作：逐元素应用 ReLU 函数，得到 **z**。
5.  操作：将 **z** 与常数 1 连接，形成 **ẑ**。
6.  操作：矩阵乘法 **ŷ = β ẑ**。
7.  操作：计算损失 **L = ½ (ŷ - y)²**（输入还包括真实值 **y**）。

重要的是，在计算图中，我们不仅包含输入数据 **x** 和 **y**，还必须包含模型参数 **α** 和 **β**，因为反向传播的目标正是计算相对于这些参数的梯度。

## 反向传播

上一节我们通过计算图描述了信息的前向流动，本节中我们来看看反向传播，它用于计算损失函数相对于网络权重的梯度。之所以称为“反向”，是因为我们从最后一层开始计算梯度，然后向后传播到第一层。

对于一个简单的单隐藏层网络，我们需要计算两个梯度：**∂L/∂αᵢⱼ** 和 **∂L/∂βᵢ**。我们使用链式法则递归地计算这些梯度。

### 梯度计算示例

假设我们有一些具体的值，并想计算 **∂L/∂β₁** 和 **∂L/∂α₁₁**。

首先，计算 **∂L/∂β₁**：
**∂L/∂β₁ = (∂L/∂ŷ) * (∂ŷ/∂β₁)**
根据损失函数 **L = ½ (ŷ - y)²**，有 **∂L/∂ŷ = (ŷ - y)**。
根据输出公式 **ŷ = Σᵢ βᵢ zᵢ**，有 **∂ŷ/∂β₁ = z₁**。
因此，**∂L/∂β₁ = (ŷ - y) * z₁**。代入具体数值即可得到结果。

其次，计算 **∂L/∂α₁₁**，这需要应用多层链式法则：
**∂L/∂α₁₁ = (∂L/∂ŷ) * (∂ŷ/∂z₁) * (∂z₁/∂a₁) * (∂a₁/∂α₁₁)**
1.  **∂L/∂ŷ = (ŷ - y)** （已知）
2.  **∂ŷ/∂z₁ = β₁** （从 **ŷ = Σᵢ βᵢ zᵢ** 得出）
3.  **∂z₁/∂a₁**：由于 **z₁ = ReLU(a₁)**，当 **a₁ > 0** 时导数为 1，否则为 0。
4.  **∂a₁/∂α₁₁ = x₁** （从 **a₁ = Σⱼ α₁ⱼ xⱼ** 得出）

将所有这些导数相乘，并代入具体数值，即可得到 **∂L/∂α₁₁**。

### 向量形式梯度

在实际实现中，使用向量或矩阵形式进行计算更高效。我们的目标是计算 **∂L/∂β** 和 **∂L/∂α**。

对于 **∂L/∂β**：
**∂L/∂β = (∂L/∂ŷ) * (∂ŷ/∂β) = (ŷ - y) * ẑᵀ**
注意，为了确保梯度 **∂L/∂β** 的形状与参数 **β** 相同（一个行向量），我们需要对 **ẑ** 进行转置。

对于 **∂L/∂α**，计算更复杂一些：
**∂L/∂α = (∂L/∂a) * (∂a/∂α)**
其中，
**∂L/∂a = (∂L/∂ŷ) * (∂ŷ/∂z) * (∂z/∂a)**
*   **∂L/∂ŷ = (ŷ - y)**
*   **∂ŷ/∂z = β**（这里 **β** 是包含偏置的权重向量）
*   **∂z/∂a** 是一个对角矩阵，因为 **zᵢ** 只依赖于 **aᵢ**。当 **aᵢ > 0** 时，对角线元素为 1，否则为 0。

计算得到 **∂L/∂a** 后（确保其形状与 **a** 相同），最终计算：
**∂L/∂α = (∂L/∂a) * x̂ᵀ**
这利用了 **a = α x̂** 的关系，且 **∂a/∂α = x̂ᵀ**。

一个重要的检查点是：梯度 **∂L/∂θ** 必须与参数 **θ** 本身具有相同的形状。这确保了我们可以直接使用该梯度来更新参数，例如使用梯度下降更新规则：**θ ← θ - η * (∂L/∂θ)**。
