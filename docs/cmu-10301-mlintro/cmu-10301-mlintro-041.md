## 决策树：构建与剪枝

![](img/3cd47ee8bea92fc84c530ca184d83cfa_1.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_3.png)

上一节我们介绍了机器学习的基本概念，本节中我们来看看决策树算法。决策树通过一系列基于特征的判断来对数据进行分类或回归。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_5.png)

### 问题一：构建零训练误差的决策树

![](img/3cd47ee8bea92fc84c530ca184d83cfa_7.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_9.png)

首先，我们讨论一个分类问题。目标是构建一棵决策树，将图中的三个数据簇（绿色、紫色、黄色）完美分开，使得训练误差为零。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_11.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_13.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_15.png)

以下是构建步骤：
1.  首先，我们选择一个特征（例如 `x1`）和一个分割点（例如 `4`）。在 `x1=4` 处画一条垂直线，可以将绿色簇与其他两个簇分开。
2.  接着，在剩下的区域（紫色和黄色簇）中，我们选择另一个特征（例如 `x2`）和分割点（例如 `4`）。在 `x2=4` 处画一条水平线，可以将紫色和黄色簇分开。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_17.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_18.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_20.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_22.png)

因此，我们得到的决策树规则是：
*   如果 `x1 < 4`，则进入左子树，进一步判断 `x2`。
    *   如果 `x2 < 4`，则预测为黄色。
    *   如果 `x2 >= 4`，则预测为紫色。
*   如果 `x1 >= 4`，则直接预测为绿色。

这里的参数 `alpha` 和 `beta` 都是 `4`。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_24.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_25.png)

### 问题二：最小化均方误差的回归树

现在，我们考虑一个回归问题。数据点具有连续的目标值（显示在点上方，如7.6, 6.3等）。目标不再是完美分类，而是构建一棵回归树，以最小化预测值的**均方误差**。

分割方式与问题一相同，因为这是最能区分数据的分割。但是，每个叶节点（最终分区）的预测值不再是类别标签，而是一个具体的数值。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_27.png)

为了最小化一个区域内的均方误差，理论推导表明，最优的预测值是该区域内所有数据点目标值的**均值**。

以下是计算过程：
1.  对于 `x1 >= 4` 的区域（绿色簇），计算该簇内所有点目标值的平均值。
2.  对于 `x1 < 4` 且 `x2 < 4` 的区域（黄色簇），计算其目标值的平均值。
3.  对于 `x1 < 4` 且 `x2 >= 4` 的区域（紫色簇），计算其目标值的平均值。

这些计算出的均值（例如6.9, 2.5, 9.2）就是对应叶节点的预测值。

### 问题三：树的深度与过拟合

接下来，我们讨论决策树深度的影响。如果增加树的最大深度，在预测未见过的数据时会发生什么？我们更倾向于较浅的树还是很深的树？

增加树的深度意味着创建更精细、更复杂的分区。虽然这可以更好地拟合训练数据（甚至达到零误差），但可能导致**过拟合**。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_29.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_31.png)

过拟合是指模型过度学习了训练数据中的噪声和细节，以至于在新数据上表现不佳。一个非常深的树可能会为个别训练样本创建特定的规则，这些规则无法推广到更广泛的总体数据。

因此，对于预测未见数据，我们通常更倾向于深度较浅的树，因为它可能具有更好的**泛化能力**。通过限制深度（剪枝），我们鼓励模型学习更通用、更简单的模式，从而在测试数据上获得更稳定的性能。

## K近邻算法：分类与回归

![](img/3cd47ee8bea92fc84c530ca184d83cfa_33.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_35.png)

上一节我们探讨了决策树，本节中我们来看看另一种简单而强大的非参数方法：K近邻算法。KNN可用于分类和回归任务。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_37.png)

### KNN分类示例

![](img/3cd47ee8bea92fc84c530ca184d83cfa_39.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_40.png)

假设我们有一个数据集，包含蓝色和红色两类点，一个新来的绿色点需要被分类。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_42.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_43.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_44.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_46.png)

以下是不同K值下的分类逻辑：
*   **K=3**：查看绿色点的3个最近邻。假设其中有2个红色点和1个蓝色点，根据多数投票原则，该点被分类为红色。
*   **K=5**：查看5个最近邻。假设其中有3个蓝色点和2个红色点，则该点被分类为蓝色。
*   **K=4**：如果最近邻中恰好有2个蓝色和2个红色，则出现平局。处理平局的方法有：
    1.  随机选择一个类别。
    2.  考虑邻居的加权距离（更近的邻居权重更大）。
    3.  调整K值（例如改用K=3或K=5）来打破平局。

### KNN回归示例

![](img/3cd47ee8bea92fc84c530ca184d83cfa_48.png)

在回归任务中，我们预测连续值。假设在一维空间中有一些数据点，所有点在X轴上等距分布。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_50.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_52.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_54.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_55.png)

以下是不同设置下的预测结果：
*   **K=1（最近邻回归）**：对于任意查询点，其预测值就是最近的那个数据点的Y值。这会产生一个阶梯状的预测曲线，在每个数据点中点处发生跳跃。
*   **K=2（未加权）**：预测值是两个最近邻数据点Y值的简单**平均值**。这会产生一个分段线性的预测曲线。
*   **K=2（距离加权）**：预测值是两个最近邻数据点Y值的**加权平均**，权重与距离成反比（越近的点权重越大）。公式可以表示为：`y_pred = (w1*y1 + w2*y2) / (w1+w2)`，其中 `w1 = 1/d1`, `w2 = 1/d2`。这会产生一条平滑的、在两点间线性变化的预测曲线。对于远离所有数据点的查询点，其预测值会趋向于所有数据点Y值的全局均值。

### KNN的决策边界（K=1）

![](img/3cd47ee8bea92fc84c530ca184d83cfa_57.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_59.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_61.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_63.png)

对于K=1的情况，决策边界是分类发生变化的区域。对于两个属于不同类别的点，其决策边界是连接这两点线段的**垂直平分线**。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_65.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_67.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_68.png)

在一个多点的数据集中，最终的决策边界由所有“点对”的垂直平分线中相关的线段组成，形成一个由直线段构成的**Voronoi图**。每个区域内的点都会被打上该区域内那个唯一训练样本的标签。

## 线性回归与梯度下降

![](img/3cd47ee8bea92fc84c530ca184d83cfa_74.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_76.png)

上一节我们介绍了KNN，本节中我们来看看线性回归，一种用于预测连续值的参数化方法。其目标是找到一组参数（权重），使得预测值与真实值之间的误差最小。

### 基础概念

线性回归模型表示为：`y_hat = θ^T · x`，其中 `x` 是特征向量（通常增加一个值为1的截距项 `x0`），`θ` 是待学习的权重向量。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_78.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_80.png)

我们需要一个**目标函数**来衡量模型的好坏。常用的目标是**均方误差**：
`J(θ) = (1/n) * Σ (y_i - y_hat_i)^2`
其中 `y_i` 是真实值，`y_hat_i` 是预测值。

理想的优化目标函数应是**可微的**（便于求梯度）和**凸的**（能保证找到全局最优解）。均方误差满足这些条件。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_82.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_83.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_85.png)

### 梯度下降算法

![](img/3cd47ee8bea92fc84c530ca184d83cfa_87.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_89.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_91.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_93.png)

梯度下降是一种迭代优化算法，用于寻找最小化目标函数的参数 `θ`。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_95.png)

算法步骤如下：
1.  **初始化**：随机选择初始参数 `θ`（例如全零向量）。
2.  **循环迭代**直到满足停止条件（如梯度接近零）：
    a. **计算梯度**：计算目标函数 `J(θ)` 对参数 `θ` 的梯度 `∇J(θ)`。对于MSE，梯度为 `∇J(θ) = (-2/n) * Σ (y_i - θ^T·x_i) * x_i`。
    b. **更新参数**：`θ_new = θ_old - η * ∇J(θ)`。其中 `η` 是**学习率（步长）**，控制每次更新的幅度。
3.  **返回**优化后的参数 `θ`。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_97.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_98.png)

**第一次迭代：**
1.  **计算梯度**：根据梯度公式，代入 `θ=0` 简化计算。
    *   `∂J/∂θ0 = (-2/5) * Σ (y_i)`
    *   `∂J/∂θ1 = (-2/5) * Σ (x1_i * y_i)`
    *   `∂J/∂θ2 = (-2/5) * Σ (x2_i * y_i)`
    计算求和项后，得到梯度向量 `∇J(θ) ≈ [-12.8, -47.2, 10.4]^T`。
2.  **更新参数**：`θ_new = [0,0,0] - 0.1 * [-12.8, -47.2, 10.4] = [1.28, 4.72, -1.04]`。

### 定义与更新规则

![](img/3cd47ee8bea92fc84c530ca184d83cfa_163.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_164.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_165.png)

假设数据点 `x` 和标签 `y ∈ {+1, -1}`。感知机算法维护一个权重向量 `θ`（定义决策超平面）。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_169.png)

更新规则：如果对样本 `(x_i, y_i)` 分类错误（即 `y_i * (θ^T · x_i) <= 0`），则更新权重：
`θ_new = θ_old + y_i * x_i`

![](img/3cd47ee8bea92fc84c530ca184d83cfa_171.png)

当所有训练样本都被正确分类时，算法收敛。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_173.png)

### 错误界限定理

![](img/3cd47ee8bea92fc84c530ca184d83cfa_177.png)

对于一个线性可分的数据集，如果所有数据点都位于一个半径为 `R`、以原点为中心的球内，并且存在一个最优分离超平面 `θ*`，其与最近数据点的**间隔**为 `γ`，那么感知机算法在训练过程中的**错误更新次数**有一个上界：
`错误次数 ≤ (R / γ)^2`

![](img/3cd47ee8bea92fc84c530ca184d83cfa_179.png)

### 计算示例

![](img/3cd47ee8bea92fc84c530ca184d83cfa_181.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_183.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_185.png)

给定三个点：正类 `(1, -1)`, `(4, 0)`；负类 `(0, 2)`。最优分离超平面为 `θ* = [-1, 1]^T`。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_187.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_189.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_190.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_192.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_193.png)

1.  **计算半径 R**：找出所有点中到原点距离的最大值。点 `(4,0)` 的距离为4，所以 `R = 4`。
2.  **计算间隔 γ**：计算每个数据点到超平面 `θ*` 的距离，取最小值。点到超平面的距离公式为 `|θ*^T · x| / ||θ*||`。
    *   对于点 `(1, -1)`：距离 = `|(-1*1) + (1*-1)| / √((-1)^2+1^2) = | -2 | / √2 = √2`。
    *   其他点距离更远。因此，`γ = √2`。
3.  **计算错误上界**：`(R / γ)^2 = (4 / √2)^2 = 16 / 2 = 8`。

![](img/3cd47ee8bea92fc84c530ca184d83cfa_195.png)

![](img/3cd47ee8bea92fc84c530ca184d83cfa_197.png)

这意味着，在这个数据集上训练感知机，保证最多经过8次权重更新（即犯8次错误）后，算法就能收敛到一个完美的分类器。

2.  **K近邻算法**在分类（多数投票）和回归（平均/加权平均）中的应用，及其决策边界的形态。
3.  **线性回归**的均方误差目标函数，以及使用**梯度下降**算法求解权重的详细步骤，包括学习率的重要性。
4.  **感知机**算法的错误更新规则，以及在数据线性可分条件下，其错误次数的理论上界计算方法 `(R/γ)^2`。
