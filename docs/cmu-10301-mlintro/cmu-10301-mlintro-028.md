## 1.1 感知器树

上一节我们介绍了课程的基本情况，本节中我们来看看第一个综合概念：感知器树。它结合了决策树的结构与感知器的分类能力。

### 1.1a 计算预测值

给定一个数据样本 `X = [A=1, B=1, C=0, D=1, E=0, F=1]` 和一个感知器树结构，我们需要计算其预测值。结构首先根据特征A和D的值进行判断，最后将剩余特征 `[B, C, E, F]` 输入一个感知器。

感知器的权重向量为 `W = [1, 0, -1, 1]`，并带有一个偏置项 `b = 1`。输入向量为 `[B=1, C=0, E=0, F=1]`。

计算过程如下：
1.  计算点积：`W · X = 1*1 + 0*0 + (-1)*0 + 1*1 = 2`
2.  加上偏置：`2 + 1 = 3`
3.  由于结果为正，预测值为 `+1`。

### 1.1b 决策边界是否总是线性？

感知器树的决策边界是否总是线性的？答案是否定的。虽然每个叶子节点处的感知器产生一个线性边界，但决策树结构本身会引入非线性分割。最终的决策边界是由多个线性边界组合而成的复杂区域，例如矩形或多边形，因此整体上是非线性的。

### 1.1c 小深度下的欠拟合倾向

对于较小的 `max_depth`（最大深度）值，感知器树比普通感知器更容易欠拟合吗？是的。这是因为感知器树在浅层就停止了基于特征的划分，转而依赖一个线性模型（感知器）来做最终决策。如果数据本身具有复杂的非线性关系，一个简单的线性模型可能无法捕捉其模式，从而导致欠拟合。

## 1.2 模型选择与假设

以下是选择模型时需要考虑的一些关键点：

*   **感知器与线性回归**：这些模型假设数据在某种程度上是线性可分的。如果数据不满足这个强假设，它们的性能会很差。
*   **K近邻与决策树**：这些模型参数化程度较低，对数据分布的假设也较少。K近邻基于局部相似性，决策树可以学习轴平行的复杂边界，因此它们对非线性数据的适应性更强。

在选择模型前，理解其内在假设至关重要。

---

## 1.3 ID3算法与最优性

ID3算法能保证找到最优的决策树吗？不能。ID3是一种**贪心算法**。它在每个节点选择能带来最大互信息增益的特征进行分裂，而**不考虑**这个选择对后续节点可能产生的影响。这种局部最优的选择策略，可能导致最终生成的树并非全局最优（即训练误差最小的树）。

---

## 1.4 决策树与过拟合

“决策树的一个优点是不容易过拟合。”这个说法正确吗？错误。恰恰相反，如果不对决策树的生长加以限制（例如设置最大深度、最小叶子样本数或进行剪枝），它非常容易过拟合。无限深度的决策树可以一直分裂，直到完美分类每一个训练样本，这会导致模型学习到噪声和训练数据中的特定细节，从而泛化能力很差。

## 1.5 熵的计算

纳鲁瓦尔抛掷两枚公平的硬币，令Y表示正面朝上的硬币数量。Y的熵是1吗？我们来计算一下。

熵的公式为：
`H(Y) = - Σ P(y) * log₂(P(y))`

Y的可能取值及其概率为：
*   `P(Y=0) = P(反反) = 1/4`
*   `P(Y=1) = P(正反或反正) = 1/2`
*   `P(Y=2) = P(正正) = 1/4`

代入公式计算：
`H(Y) = - [ (1/4)*log₂(1/4) + (1/2)*log₂(1/2) + (1/4)*log₂(1/4) ]`
`= - [ (1/4)*(-2) + (1/2)*(-1) + (1/4)*(-2) ]`
`= - [ -0.5 - 0.5 - 0.5 ]`
`= 1.5`

计算结果为1.5，不等于1，因此原命题为假。

## 2.4 K近邻回归

现在我们来看一个关于K近邻回归的实例。我们有一个一维实数训练集 `(x, y)`，算法通过寻找查询点最近的k个邻居，并取其y值的平均值来进行预测。

以下是针对具体问题的解答：

**当 k=1 时，训练集上的均方误差是多少？**
0。因为每个训练点都是其自身的最近邻，预测值总是等于其真实值，所以训练误差为0。

**当 k=2 时，在点 A (x=4) 的预测值是多少？**
点4的两个最近邻是 `(4,4)` 和 `(2,4)` 或 `(6,4)`。无论哪种组合，邻居的y值都是4。因此预测值为 `(4+4)/2 = 4`。

**当 k=2 时，在点 B (x=5) 的预测值是多少？**
点5的两个最近邻是 `(4,4)` 和 `(6,6)`。预测值为 `(4+6)/2 = 5`。

**当 k=3 时，在点 C (x=7) 的预测值是多少？**
点7的三个最近邻是 `(6,6)`, `(8,7)`, `(9,8)`。预测值为 `(6+7+8)/3 = 7`。

**当 k=8 时，在点 C (x=7) 的预测值是多少？**
由于整个训练集只有8个点，k=8意味着使用所有点的y值求平均。计算所有y值的平均值即可得到预测值。

## 3. 交叉验证与超参数搜索

我们有一个模型，包含 `gamma` 和 `omega` 两个超参数。训练一个模型固定需要1小时。我们计划进行随机子采样交叉验证：总共有1000个数据点，进行4轮 (`k=4`)，每轮随机抽取25% (`alpha=0.25`) 的数据作为验证集，其余作为训练集。我们总共有100小时的预算。

**问题：** 如果我们想测试 `omega` 的8个不同值，那么最多能测试 `gamma` 的多少个离散值？

**计算过程：**
1.  每次交叉验证运行（1个训练/验证集划分）需要训练多个模型，因为我们要测试不同的超参数组合。
2.  对于 `omega` 的8个值，在**一次**运行中就需要训练8个模型。
3.  我们有4次运行，所以总共需要训练的模型数量为：`4 次运行 * 8 个omega值 * N 个gamma值 = 32N` 个模型。
4.  每个模型训练耗时1小时，总时间预算为100小时。因此：`32N <= 100`。
5.  解得 `N <= 3.125`。所以，最多能测试 `gamma` 的 **3** 个离散值。

**增加 alpha 值（验证集比例）的一个优势是什么？**
增加验证集比例意味着用于评估模型性能的数据更多。这能提供对模型**泛化能力**更可靠、方差更低的估计，帮助我们更准确地判断哪些超参数组合在未知数据上可能表现更好。

## 4.1 感知器与数据缩放

给定一个线性可分数据集 `S`，和另一个数据集 `S‘`，其中 `S‘` 的所有样本 `x` 都乘以了一个常数 `c`。

**a) S‘ 的感知器错误界比 S 大吗？**
错误界公式为 `(R/γ)²`，其中 `R` 是数据半径（到原点最远点的距离），`γ` 是间隔。将数据缩放 `c` 倍后，半径 `R‘ = cR`，间隔 `γ‘ = cγ`。因此错误界变为 `(cR / cγ)² = (R/γ)²`，保持不变。所以该说法错误。

**b) 在 S 和 S‘ 上运行的感知器算法会返回模常数因子相同的分类器吗？**
是的。在训练过程中，每次权重更新 `θ` 都会乘以因子 `c`。最终得到的权重向量 `θ‘` 与 `θ` 仅相差一个常数因子 `c`，它们定义的是同一个决策超平面（因为 `sign(cθ·x) = sign(θ·x)`）。所以该说法正确。

**c) 感知器算法在 S‘ 上会收敛吗？**
是的。既然 `S` 线性可分，感知器在 `S` 上收敛。根据 (b)，在 `S‘` 上训练得到的分类器只是 `S` 上分类器的常数倍，因此算法同样会在 `S‘` 上收敛。所以该说法正确。

---

## 4.2 收敛时间与样本量

对于线性可分数据集，感知器算法的收敛时间（或错误界）是否依赖于样本量 `n`？不直接依赖。错误界 `(R/γ)²` 依赖于数据集的几何性质（最远点距离 `R` 和间隔 `γ`），而与数据点的数量 `n` 没有直接的函数关系。你可以用很少的点构造出很大的 `R`，也可以用很多的点聚集在很小的区域。因此该说法错误。

## 4.6 感知器算法实例

我们通过一个实例来一步步运行感知器算法。初始参数：`θ = [0, 0]`, `b = 0`。

**迭代1 (样本1: x=[-1, 2], y=-1):**
*   计算：`θ·x + b = 0 + 0 = 0`。预测 `+1`。
*   错误：真实标签为 `-1`。
*   更新：`θ = [0,0] - (-1)*[-1,2] = [-1, 2]`；`b = 0 - (-1) = 1`。
*   更新后：`θ = [-1, 2]`, `b = 1`。

**迭代2 (样本2: x=[-2, -2], y=+1):**
*   计算：`(-1)*(-2) + 2*(-2) + 1 = 2 - 4 + 1 = -1`。预测 `-1`。
*   错误：真实标签为 `+1`。
*   更新：`θ = [-1,2] + 1*[-2,-2] = [-3, 0]`；`b = 1 + 1 = 2`。
*   更新后：`θ = [-3, 0]`, `b = 2`。

**迭代3 (样本3: x=[1, -1], y=+1):**
*   计算：`(-3)*1 + 0*(-1) + 2 = -3 + 0 + 2 = -1`。预测 `-1`。
*   错误：真实标签为 `+1`。
*   更新：`θ = [-3,0] + 1*[1,-1] = [-2, -1]`；`b = 2 + 1 = 3`。
*   更新后：`θ = [-2, -1]`, `b = 3`。

**迭代4 (样本4: x=[3, 1], y=-1):**
*   计算：`(-2)*3 + (-1)*1 + 3 = -6 -1 + 3 = -4`。预测 `-1`。
*   正确：预测与真实标签一致。
*   无需更新。

## 4.7 特征映射与线性可分性

原始数据集 `(x, y)` 是非线性可分的，但存在一个特征映射函数 `φ(x)`，使得映射后的数据集 `(φ(x), y)` 是线性可分的。那么，将原始 `x` 替换为 `φ(x)` 后，运行标准的感知器算法是否正确？是的。感知器算法的核心是处理线性可分数据。只要我们在新的特征空间 `φ(x)` 中运行算法，它就能正常工作并找到一个线性分类器。该说法正确。

## 5.2 线性回归的维度分析

给定线性回归的闭式解公式，我们需要检查其中矩阵和向量的维度。假设有 `n=35` 个训练样本，`m=5` 个特征，并且已将偏置项并入特征中。

*   `X`: 设计矩阵。每行一个样本，每列一个特征（包含偏置项）。维度为 `(n, m+1) = (35, 6)`。
*   `y`: 标签向量。每个样本对应一个标签。维度为 `(n, 1) = (35, 1)`。
*   `θ`: 参数向量。每个特征（包含偏置项）对应一个参数。维度为 `(m+1, 1) = (6, 1)`。

---

## 5.3 均方误差的向量表达

均方误差公式为 `(1/n) * ||预测值 - 真实值||²`。我们需要判断哪个选项中的向量运算维度是合法的。

给定 `X` 维度为 `(n, m+1)`，`θ` 维度为 `(m+1, 1)`，`y` 维度为 `(n, 1)`。

*   `Xθ` 的结果维度是 `(n, 1)`。
*   `y` 的维度是 `(n, 1)`。

因此，合法的表达式应为 `||Xθ - y||²`，因为 `Xθ` 和 `y` 维度相同，可以相减。其他选项如 `θX` 或 `Xθ - yᵀ` 都存在维度不匹配的问题。
