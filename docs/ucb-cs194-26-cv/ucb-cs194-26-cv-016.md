# 16：卷积神经网络

在本节课中，我们将要学习卷积神经网络（CNN）的核心概念、工作原理以及实际应用。我们将从全连接层的问题出发，逐步引入卷积操作，并探讨CNN中的关键组件，如池化、归一化以及现代网络架构。课程内容旨在让初学者能够理解并掌握CNN的基本原理。

---

## 回顾：全连接层的问题

上一节我们介绍了全连接神经网络。在这种网络中，每个神经元都与输入图像中的每一个像素相连接。

这种全局连接的方式存在一个主要问题：它非常不理想。例如，如果我们想学习一个垂直边缘检测器，问题在于信息是全局的。这意味着网络必须学习图像中所有位置的垂直边缘，相当于使用一个与图像尺寸相同的巨大卷积核，这并非高效的做法。

那么，有什么改进建议呢？

## 局部连接层

一个更好的步骤是使用**局部连接层**。这与多层感知机（MLP）类似，但每个神经元的感受野（或连接范围）仅限于输入图像的特定区域。

这比全连接更好，因为每个神经元可以专注于一个较小的区域（例如10x10的区域）。然而，这仍然不是卷积操作。这更像是为每个特定位置（例如左上角的5x5区域）分配一个专用的神经元。

这种方法也存在问题。为了捕捉相同的特征（如垂直边缘），网络必须在所有不同位置学习相同的滤波器。这会导致严重的过拟合。例如，一个位于右上角的神经元可能只关注该区域是否为蓝色，但我们不希望边缘检测依赖于像素的具体位置。

那么，下一个更好的方案是什么？

## 引入卷积操作

答案是**卷积**。还记得在项目1中实现的卷积操作吗？卷积就是一个在图像上滑动的滤波器。

这种方式很好，因为一旦学习到一个滤波器（例如垂直边缘检测器），它就可以在整个图像上运行，无论特征出现在哪里，它都能被激活。这被称为**平移等变性**，意味着如果输入中的垂直边缘发生平移，输出也会相应地平移。

在CNN中，我们将学习这个卷积核。它是一个可学习的检测器，能够学习对当前任务有用的特征。

假设输入图像是RGB三通道，尺寸为32x32。滤波器（卷积核）尺寸为5x5。由于输入通道是3（RGB），所以卷积核的尺寸实际上是3x5x5。

卷积操作后，会生成一个**激活图**。输入是RGB（32x32x3），输出激活图的尺寸会略小（例如28x28），因为卷积操作会缩小尺寸。我们可以通过填充（padding）来控制输出尺寸。

在CNN中，一个“神经元”的输出不再是一个单一数值，而是一整张激活图（一张图像），因为它被应用到了整个空间位置。

## 多滤波器与通道

我们通常希望网络学习多种不同的特征。因此，我们会使用多个滤波器。

例如，我们可以有6个不同的3x5x5滤波器。每个滤波器学习不同的特征（如垂直边缘、水平边缘、圆形检测器等）。每个滤波器都会在输入图像上进行卷积，产生一张激活图。然后，我们将这6张激活图堆叠起来，形成一个6x28x28的输出张量。

这6个滤波器中的每一个都对应一个“神经元”，它们观察输入图像的相同区域，但寻找不同的模式。

## 批处理与张量表示

在实际训练中，我们通常使用**批处理**。这意味着我们一次性处理多个图像（一个批次），以提高计算效率。

输入张量的维度变为：`[批次大小 (N), 输入通道数 (C_in), 高度 (H), 宽度 (W)]`。例如，`[2, 3, 32, 32]` 表示一个包含2张RGB图像（32x32）的批次。

卷积操作会独立应用于批次中的每张图像，但以并行方式实现。输出张量的维度变为：`[批次大小 (N), 输出通道数 (C_out), 输出高度 (H_out), 输出宽度 (W_out)]`，例如 `[2, 6, 28, 28]`。

在PyTorch等框架中，卷积层的权重 `W` 是一个四维张量：`[输出通道数 (C_out), 输入通道数 (C_in), 核高度 (K_h), 核宽度 (K_w)]`。偏置 `b` 通常是一个长度为 `C_out` 的向量，每个输出通道有一个偏置值。

## 卷积核的可视化与学习

训练后，我们可以可视化第一层学习到的卷积核。例如，在AlexNet的第一层（可能有64个11x11的滤波器）中，这些滤波器看起来像各种边缘和纹理检测器，有些类似于垂直梯度检测器。

滤波器是彩色的，因为输入是RGB图像。每个滤波器是一个3x5x5的张量，我们可以将其可视化为三个分别对应R、G、B通道的5x5图像。

网络初始化时，滤波器是随机的噪声。通过梯度下降和反向传播，根据损失函数的监督信号，滤波器逐渐更新，最终收敛到有意义的特征检测器。网络没有机制保证每个滤波器学习不同的特征，可能会出现冗余或“死亡”的滤波器。但一个优化良好的网络倾向于学习多样化的滤波器，以最好地最小化损失。

## 输出尺寸计算与填充

卷积操作会改变特征图的尺寸。如果输入宽度为 `W_in`，卷积核大小为 `K`，步幅为 `S`，填充为 `P`，则输出宽度 `W_out` 的计算公式为：

**`W_out = (W_in - K + 2P) / S + 1`**

如果不使用填充（`P=0`）且步幅为1（`S=1`），输出尺寸就会缩小，例如从32x32缩小到28x28（当`K=5`时）。为了保持空间尺寸不变，我们通常使用零填充。

## 感受野

**感受野**是指网络中层的一个元素（像素）在输入图像上所能“看到”的区域大小。

对于一个`K x K`的卷积核，它直接查看输入中`K x K`的区域。当叠加多个卷积层时，感受野会复合增长。第二层的一个神经元，其卷积核作用于第一层的输出上，而第一层的每个输出已经对应了输入中`K x K`的区域。因此，第二层的神经元间接地看到了输入图像中更大的区域。

随着网络加深，单个输出单元的感受野会变得非常大，最终可能覆盖整个输入图像。这是我们所期望的，因为高层的决策（如图像分类）需要全局信息。

## 下采样：步幅与池化

为了减少计算量、内存消耗并获得更抽象的特征，我们需要在网络中**下采样**（缩小特征图尺寸）。主要有两种方式：

1.  **步幅卷积**：在卷积时，不是每次滑动一个像素，而是滑动多个像素（步幅 > 1）。例如，步幅为2会使输出尺寸减半。
2.  **池化层**：在卷积后，使用一个聚合操作。
    *   **最大池化**：在一个小窗口（如2x2）内取最大值。
    *   **平均池化**：在一个小窗口内取平均值。

池化层没有可学习的参数。它通过聚合局部信息，提供了一定的平移不变性，并降低了特征图尺寸。最大池化在实践中更常用，因为它能保留最显著的特征。

## 归一化

归一化对于稳定和加速网络训练至关重要。其核心思想是调整每一层输入的分布，使其具有稳定的均值和方差，避免内部协变量偏移问题。

最常见的归一化方法是**批归一化**。对于一个批次的数据，它沿着批次维度计算每个特征通道的均值和方差，然后用其来归一化该通道的所有激活值。公式如下：

**对于输入批次 `x`，计算：**
`mean = mean(x, dim=batch)`
`var = variance(x, dim=batch)`
`x_norm = (x - mean) / sqrt(var + epsilon)`
`out = gamma * x_norm + beta`

其中 `gamma` 和 `beta` 是可学习的缩放和偏移参数，用于恢复网络的表达能力。批归一化通常插入在卷积层/全连接层之后，非线性激活函数（如ReLU）之前。

其他归一化方法包括：
*   **实例归一化**：对每个样本的每个通道单独计算均值和方差。常用于风格迁移。
*   **层归一化**：对每个样本的所有通道和空间位置计算均值和方差。常用于循环神经网络。

## 经典网络架构案例研究

以下是两个里程碑式的CNN架构：

1.  **LeNet-5 (1998)**：用于手写数字识别。
    *   结构：卷积 -> 池化 -> 卷积 -> 池化 -> 全连接层。
    *   它展示了卷积、池化、全连接组合的基本模式。

2.  **AlexNet (2012)**：在ImageNet竞赛中取得突破。
    *   更深更大：5个卷积层，3个全连接层。
    *   使用了ReLU激活函数、Dropout和数据增强。
    *   其结构体现了随着网络深入，空间尺寸减小、通道数增加的模式。

## 残差网络：解决深度网络训练难题

实验发现，并非网络越深性能越好。过深的网络可能难以优化，训练误差甚至比浅层网络更高。

**残差网络** 通过引入“快捷连接”或“恒等映射”巧妙地解决了这个问题。在一个残差块中，不再直接学习目标映射 `H(x)`，而是学习残差映射 `F(x) = H(x) - x`，然后将原始输入加回去：`Output = F(x) + x`。

**`Output = F(x, {W_i}) + x`**

这样，如果最优映射是恒等映射（`H(x) = x`），那么将残差 `F(x)` 推向0比学习一个恒等矩阵要容易得多。这使得训练极深的网络（如ResNet-50, ResNet-101）成为可能，并显著提升了性能。

## 训练CNN：工作流程与技巧

一个典型的CNN工作流程如下：
1.  **准备数据**：获取图像和标签（如类别、关键点坐标）。
2.  **设计网络**：选择架构（如ResNet）。
3.  **定义损失函数**：如用于分类的交叉熵损失，用于回归的L2损失。
4.  **训练**：使用优化器（如SGD或Adam）更新网络权重。

**重要技巧：**
*   **学习率调整**：观察训练损失曲线。损失震荡不降可能意味着学习率太高或代码有bug。损失下降过快然后停滞，可能意味着学习率太高陷入了局部最小。平滑下降至较低值是理想情况。
*   **数据增强**：通过对训练图像进行随机变换（如翻转、旋转、裁剪、颜色抖动）来人工增加数据多样性，防止过拟合。**注意**：进行空间变换（如水平翻转）时，必须同步修正对应的标签（如左右关键点）。
*   **合理性检查**：在完整数据集上训练前，尝试用**单个样本**训练你的网络。网络应该能够快速地将训练损失降到接近零（即“记住”这个样本）。如果做不到，说明网络实现、损失函数或数据管道可能存在错误。

## 应用于关键点检测

在项目5中，你们将用CNN进行人脸关键点检测。
*   **回归方法**：让网络直接输出关键点的`(x, y)`坐标，并使用L2损失。这种方法简单，但有时效果有限，因为模型需要为每个点做出绝对的位置决策。
*   **热图方法**：这是更先进的方法。网络为每个关键点输出一个概率热图，峰值位置表示关键点最可能出现的位置。这为模型提供了更丰富的空间信息和更柔性的监督信号。这将在后续课程中详细讨论。
