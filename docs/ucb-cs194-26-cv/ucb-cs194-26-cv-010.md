# 10：相机模型

## 概述

在本节课中，我们将学习相机的基本工作原理。我们将从最简单的相机模型开始，理解图像是如何形成的，并探讨如何通过数学公式描述三维世界到二维图像的投影过程。

## 针孔相机模型

上一节我们介绍了图像处理的基础，本节中我们来看看图像是如何被捕捉的。相机是连接真实世界与二维图像的桥梁。成像过程始于光源发出光子，光子照射到场景中的物体上，然后部分光子反射并进入相机，最终相机生成图像。

那么，我们如何捕捉世界？如何构建一台相机？让我们进行第一次尝试设计相机。

以下是第一次尝试的步骤：
1.  取出一张感光相纸。
2.  将其暴露在场景（例如观众）前。
3.  然后将其收回。

这种方法能产生一张照片吗？为什么这不是一种好的摄影方式？如果将其冲洗出来，照片上会是什么？

实际上，照片上会是一片均匀的灰色。这是因为场景中的每个点都向所有方向发射光子。从相纸的角度看，每个点都接收到了来自整个场景的所有光子的总和，即整个场景的平均亮度。因此，这不会是一张好照片。

那么，我们如何让它真正产生一张照片（例如观众的照片）？关键在于，当前的问题是太多光线相互干扰并平均化。解决方案是阻止大多数光线照射到我们的胶片上。

我们引入一个带有小孔的屏障。现在，场景中的点（例如树顶）仍然向所有方向发射光子，但大多数光子被屏障阻挡。只有恰好穿过小孔的那一束光线才能到达胶片。对于场景中的另一个点也是如此。最终，我们在胶片上会得到一个倒立的树像。

女士们先生们，这就是摄影的核心思想。这个微小的孔被称为光圈。这基本上是所有眼睛和相机捕捉视觉世界的基本方式。

这种原始设计设备被称为暗箱。暗箱是一个黑暗的房间，只在外部开有一个小孔。如果外部条件合适，你将在房间内观察到外部世界的倒立影像。

你甚至可以自己制作一个。例如，一位艺术家用酒店房间制作暗箱。他封住窗户，只留下一个小开口，然后进行长时间曝光拍摄。

针孔无处不在。例如，在校园里，树冠之间的缝隙会在地面上形成小圆点。这些圆点实际上是太阳的图像。在日偏食期间，这些圆点会变成新月形，因为它们确实是在拍摄太阳的图像。

即使你不做任何特殊处理，你房间的窗户本身也像一个巨大的“针孔”，你仍然能获得一些关于外部世界的信息，尽管图像质量可能不佳。

因此，针孔相机是思考相机的主要模型。它非常简单：一个带孔的盒子，观察外部环境，在盒子背面形成倒立的图像。

## 相机几何与投影

针孔相机捕捉的是通过一个单点的所有光线集合，这个集合称为光束。这个点被称为投影中心。每台相机都有一个唯一的投影中心，它捕捉所有通过该点的光线。

盒子的后壁被称为成像平面，图像在此形成。盒子的深度被称为有效焦距，有时记作 D 或 F，它是投影中心到成像平面的距离。

思考一下，如果增加盒子的深度（即焦距变长），图像会如何变化？图像会变大，但看到的场景范围会变小，这类似于放大。如果缩短盒子深度，图像会变小，但看到的场景范围更广，这类似于缩小。因此，盒子的宽度（焦距）类似于变焦功能。

现在，让我们看看从三维世界投影到二维图像的实际模型。这里有一个技巧：在示意图中，图像通常被“翻转”到场景的前方。实际上，图像应该在相机后方且是倒立的。但为了方便起见（例如让图像正立，且所有坐标在正侧），我们将其虚拟地翻转到前方。你可以想象成像平面是透明的。从根本上说，这与原始模型是等价的。

现在，我们可以清楚地看到投影过程。我们有一个三维世界中的点 (X, Y, Z)，它被投影到成像平面上的某个位置 (x‘, y’)。由于成像平面是一个平面，其深度坐标始终是固定的（即焦距 F）。因此，我们将一个三维点 (X, Y, Z) 投影到了一个二维点 (x‘, y’)。

如果我们要求你写出 x‘ 和 y’ 的方程，你能做到吗？根据相似三角形原理，我们可以推导出投影公式。

考虑 X 坐标。大三角形的边长为 Z 和 X，小三角形的边长为 F 和 x‘。根据相似三角形，比例关系为：x‘ / X = F / Z。

因此，我们得到投影公式：
**x‘ = F * (X / Z)**
**y‘ = F * (Y / Z)**

这就是投影的核心。你通过深度 Z 进行归一化，然后乘以焦距 F。第三个坐标我们不需要，因为它总是固定的。

## 投影的维度缩减与透镜引入

现在，我们有一个有趣的观察。投影本质上是一个降维机器。观察者身处三维世界，但在他视网膜上形成的图像是二维的。我们正从三维世界走向二维图像。

这里有一个关于整个事情奇怪的地方：仅从一张二维图片中，你能判断出桌子和窗户哪个更近吗？你能说出桌面的形状吗？实际上，单眼视觉（或单张图片）丢失了深度信息，这给理解三维场景带来了挑战。

到目前为止，我们还没有讨论对焦。因为对焦这个概念在纯针孔相机中并不存在。对焦只在你拥有透镜时才出现。透镜帮助我们汇聚光线。

那么，对焦是什么意思？现在，我们有了替代设备（透镜）。我们仍然有主光线（即如果没有透镜，会穿过针孔的那条光线）。但现在，我们要说，我将添加额外的“辅助”光线，并使用我的透镜来添加这些辅助线。透镜将来自物体同一点的多条光线汇聚到成像平面的同一点上。

透镜解决了针孔相机进光量小的问题，但它也引入了像差（如色差）。高质量的光学器件通过打磨透镜来减少这些缺陷。现代手机（如Pixel）甚至可以通过软件校准来校正这些像差。

## 总结

本节课中，我们一起学习了相机的基本原理。我们从最简单的针孔相机模型开始，理解了图像形成的过程，并推导了从三维世界到二维图像投影的核心公式 **x‘ = F * (X / Z)** 和 **y‘ = F * (Y / Z)**。我们认识到投影是一个降维过程，并简要介绍了透镜的作用是汇聚光线以实现对焦，同时也可能引入像差。掌握这些基础概念是理解更复杂计算机视觉和计算摄影技术的起点。
