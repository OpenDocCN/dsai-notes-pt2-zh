# 19：生成模型

在本节课中，我们将学习生成模型的核心思想，特别是如何通过匹配图像的统计特征来合成新的纹理和图像。我们将从经典的纹理合成方法开始，逐步过渡到基于深度学习的现代生成技术，如风格迁移和扩散模型。

## 纹理合成：从统计特征出发

上一节我们回顾了纹理的基本概念。本节中，我们来看看如何利用统计特征来合成新的纹理。

纹理可以被视为从某个分布中抽取的样本。尽管不同的纹理样本在像素层面差异很大，但它们共享着某些统计特性。我们的目标就是找到并匹配这些统计特征，从而生成新的、符合该纹理分布的图像。

### 参数化纹理合成：Heeger & Bergen 方法

一种经典的方法是参数化纹理合成。其核心思想是：如果我们能提取出代表目标纹理的一组统计参数，那么我们就可以通过强制一个随机噪声图像匹配这些参数，来生成新的纹理图像。

以下是该算法的关键步骤：

1.  **特征提取**：使用一个滤波器组（如方向性滤波器）对输入纹理图像进行卷积，得到一个多尺度、多方向的“金字塔”特征表示。
2.  **初始化**：生成一个完全随机的“白噪声”图像作为输出图像的起点。
3.  **统计匹配**：计算输入纹理图像和噪声图像在每一个特征通道上的直方图。然后，通过**直方图匹配**技术，修改噪声图像的特征，使其每个通道的直方图都与输入纹理对应通道的直方图尽可能相似。
    *   公式：`histogram_matching(noise_feature_map, target_histogram)`
4.  **图像重建**：将修改后的特征金字塔“折叠”回图像空间，得到更新后的噪声图像。此时，噪声图像已开始呈现输入纹理的结构。
5.  **迭代优化**：重复步骤1-4多次（例如5-10次）。每次迭代都迫使噪声图像的特征统计更接近目标纹理，最终将其“投影”到由该纹理定义的图像流形上。

这个过程的妙处在于，它并不要求像素级匹配，而是通过匹配高阶统计特征（滤波器响应的分布），让图像“自然而然”地呈现出目标纹理的视觉效果。

## 从传统滤波器到深度特征：Gram矩阵

上一节我们介绍了基于手工设计滤波器的纹理合成。本节中我们来看看如何利用深度卷积神经网络（CNN）提取更强大的特征来进行类似的操作。

CNN的中间层特征包含了从边缘到复杂模式的丰富信息。我们可以将这些特征作为新的“统计单元”。那么，如何用这些特征来定义纹理的统计量呢？研究者提出了**Gram矩阵**。

Gram矩阵的计算方法如下：
*   对于CNN某一层的特征图，假设其形状为 `[C, H, W]`（通道数，高度，宽度）。
*   我们将每个空间位置（H x W）的特征向量视为一个样本。
*   Gram矩阵 `G` 是一个 `C x C` 的矩阵，其中元素 `G[i, j]` 是第 `i` 个特征通道与第 `j` 个特征通道在所有空间位置上内积的平均值。
*   公式：`G[i, j] = mean( F[i, :, :] * F[j, :, :] )`，其中 `F` 是特征图。

Gram矩阵捕获了不同特征通道之间的相关性，它是一种**二阶统计量**，忽略了特征的具体空间位置，但保留了它们之间如何共同出现的模式信息。这非常适合于描述纹理。

### 神经风格迁移

基于Gram矩阵，一个革命性的应用是神经风格迁移。其目标是将一幅内容图像（C）的结构与另一幅风格图像（S）的纹理相结合，生成新图像（G）。

以下是实现过程：

1.  **特征提取**：使用一个预训练的CNN（如VGG）分别提取内容图像、风格图像和生成图像的多层特征。
2.  **定义损失函数**：总损失由两部分加权组成。
    *   **内容损失**：确保生成图像在某一较深层的特征与内容图像相似。这通常使用特征图之间的均方误差（MSE）。
        *   公式：`L_content = MSE(phi_l(G), phi_l(C))`，其中 `phi_l` 表示第 `l` 层的特征。
    *   **风格损失**：确保生成图像在多个层上的Gram矩阵与风格图像的Gram矩阵相似。
        *   公式：`L_style = sum_l ( w_l * MSE(Gram(phi_l(G)), Gram(phi_l(S))) )`，其中 `w_l` 是各层的权重。
3.  **优化图像**：初始化生成图像为白噪声（或内容图像）。不训练网络权重，而是通过梯度下降**直接优化生成图像的像素值**，以最小化总损失 `L_total = alpha * L_content + beta * L_style`。通过调整 `alpha` 和 `beta` 的比例，可以控制内容与风格的权衡。

这种方法成功地将纹理合成的统计匹配思想，与CNN的层次化特征表示结合起来，实现了令人惊叹的艺术效果。

## 迈向图像生成：扩散模型

前面两节我们学习了通过匹配现有图像的统计特征来生成新样本。本节中我们来看看一种更通用的、能够从零生成复杂图像的强大方法——扩散模型。

扩散模型的核心思想是一个“先破坏再重建”的过程。想象一下，将一张清晰的图片逐步添加噪声，直到它变成完全随机的白噪声，这个过程非常容易。扩散模型的神奇之处在于，它学习如何**逆转这个过程**，即从噪声中重建出清晰的图像。

### 训练阶段：学习去噪

1.  **前向过程（加噪）**：对训练数据集中的每张真实图像 `x0`，按照一系列步骤 `t=1...T` 逐步添加高斯噪声，最终得到纯噪声 `xT`。这定义了一个固定的噪声添加轨迹。
2.  **学习去噪**：模型（通常是一个U-Net结构的神经网络）被训练来预测给定噪声图像 `xt` 和噪声步数 `t` 时，所添加的噪声 `ε`，或者直接预测去噪后的图像。其目标是让预测值接近真实值。

### 生成阶段（推理）：从噪声到图像

1.  **起点**：从一个纯高斯噪声样本 `xT` 开始。
2.  **迭代去噪**：对于 `t=T, T-1, ..., 1`：
    *   模型根据当前噪声图像 `xt` 和步数 `t`，预测出如何从 `xt` 中去除一部分噪声，得到 `x_{t-1}`。
    *   通常在此过程中会重新注入少量噪声以保持随机性。
3.  **结果**：经过多次迭代，最终得到一张清晰的、位于自然图像流形上的高质量图像 `x0`。

### 条件生成：用语言引导

基本的扩散模型生成的是随机图像。为了生成特定内容的图像（如“一个戴着红色围巾的男人”），我们需要**条件扩散模型**。

*   在训练时，我们不仅给模型提供噪声图像 `xt` 和步数 `t`，还提供对应的文本描述 `y`。
*   模型学习在给定文本描述 `y` 的条件下，如何对 `xt` 进行去噪。
*   在生成时，我们提供目标文本描述 `y`，扩散模型就会沿着指向该描述的轨迹进行去噪，从而生成符合文本描述的图像。

现代系统如DALL-E 2、Stable Diffusion等，都结合了强大的文本编码器（如CLIP）和扩散模型，实现了文生图的惊人能力。它们的成功很大程度上依赖于**海量的图文配对数据**，使得模型能够学习到视觉概念与语言描述之间复杂的统计关联。

## 总结

本节课中我们一起学习了生成模型的发展脉络：
1.  我们从**纹理合成**出发，理解了通过匹配统计特征（如滤波器响应的直方图）可以生成新的视觉模式。
2.  接着，我们看到了如何利用深度神经网络的特征和**Gram矩阵**来捕获更丰富的纹理统计，并实现了**神经风格迁移**。
3.  最后，我们探讨了强大的**扩散模型**。它通过学习和逆转一个逐步加噪的过程，能够从纯噪声生成高度复杂和逼真的图像。当与语言模型结合进行条件控制时，便开启了文生图的新时代。

这些方法的共同核心是：**学习数据分布中的关键统计规律，并通过某种优化过程，使新样本满足这些规律，从而生成符合期望的数据**。从直方图匹配到Gram矩阵，再到扩散模型中的迭代去噪，这一思想贯穿始终，并随着数据和计算能力的增长而不断焕发新的生命力。
