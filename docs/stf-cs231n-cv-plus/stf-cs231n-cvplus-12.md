上一节我们介绍了前馈神经网络，它处理的是单一的输入和输出。本节中，我们来看看循环神经网络，它专门设计用于处理序列数据。

![](img/bef0ca1f166cdce497bf1e84c24ada10_33.png)

![](img/bef0ca1f166cdce497bf1e84c24ada10_35.png)

![](img/bef0ca1f166cdce497bf1e84c24ada10_37.png)

循环神经网络的核心思想是引入“隐藏状态”，该状态在时间步之间传递信息，使得网络能够记住之前的输入信息。以下是循环神经网络的基本结构：

- **输入序列**：`x_1, x_2, ..., x_T`
- **隐藏状态**：`h_t`，在时间步 `t` 更新
- **输出序列**：`y_1, y_2, ..., y_T`

![](img/bef0ca1f166cdce497bf1e84c24ada10_39.png)

隐藏状态的更新遵循一个递归公式，该公式在**每个时间步使用相同的权重矩阵**，这使得网络能够处理任意长度的序列。

![](img/bef0ca1f166cdce497bf1e84c24ada10_41.png)

**公式**：
```
h_t = f_W(h_{t-1}, x_t)
```
其中 `f_W` 是一个带有可学习权重 `W` 的函数。

## 🧠 循环神经网络的应用类型

循环神经网络可以应用于多种序列任务，根据输入和输出的结构，主要分为以下几类：

![](img/bef0ca1f166cdce497bf1e84c24ada10_43.png)

以下是几种常见的应用类型：

1.  **一对一**：单一输入，单一输出（例如图像分类）。
2.  **一对多**：单一输入，序列输出（例如图像描述生成）。
3.  **多对一**：序列输入，单一输出（例如视频分类）。
4.  **多对多**：序列输入，序列输出（例如机器翻译、视频帧分类）。

## 🏗️ 循环神经网络的基本结构：Vanilla RNN

最简单的循环神经网络称为 Vanilla RNN 或 Elman RNN。它的隐藏状态更新公式如下：

**公式**：
```
h_t = tanh(W_{hh} * h_{t-1} + W_{xh} * x_t + b)
y_t = W_{hy} * h_t
```
其中：
- `W_{hh}` 和 `W_{xh}` 是权重矩阵。
- `b` 是偏置项。
- `tanh` 是激活函数。

**代码描述（伪代码）**：
```python
# 初始化隐藏状态
h = zeros(hidden_size)

![](img/bef0ca1f166cdce497bf1e84c24ada10_47.png)

for t in range(sequence_length):
    # 更新隐藏状态
    h = tanh(dot(W_hh, h) + dot(W_xh, x[t]) + b)
    # 计算输出
    y[t] = dot(W_hy, h)
```

## 📊 循环神经网络的训练：随时间反向传播

训练循环神经网络需要使用**随时间反向传播**算法。该算法通过展开循环神经网络的计算图，将序列处理转化为一个深度前馈网络，然后使用标准反向传播计算梯度。

![](img/bef0ca1f166cdce497bf1e84c24ada10_53.png)

然而，对于非常长的序列，完全展开计算图会占用大量内存。为了解决这个问题，我们使用**截断随时间反向传播**算法：

![](img/bef0ca1f166cdce497bf1e84c24ada10_55.png)

![](img/bef0ca1f166cdce497bf1e84c24ada10_57.png)

以下是截断随时间反向传播的步骤：

![](img/bef0ca1f166cdce497bf1e84c24ada10_59.png)

1.  将长序列分割成较短的块。
2.  对每个块执行前向传播，并记录最后一个时间步的隐藏状态。
3.  计算该块的损失，并执行反向传播（仅在该块内）。
4.  将记录的隐藏状态作为下一个块的初始隐藏状态。
5.  重复此过程，直到处理完整个序列。

## 🧩 循环神经网络的变体：长短期记忆网络

![](img/bef0ca1f166cdce497bf1e84c24ada10_63.png)

Vanilla RNN 在处理长序列时容易遇到梯度消失或爆炸的问题。为了解决这个问题，我们引入了**长短期记忆网络**。

LSTM 通过引入“细胞状态”和三个门控机制（输入门、遗忘门、输出门）来更好地控制信息的流动和记忆。

**公式**：
```
i_t = σ(W_i * [h_{t-1}, x_t] + b_i)   # 输入门
f_t = σ(W_f * [h_{t-1}, x_t] + b_f)   # 遗忘门
o_t = σ(W_o * [h_{t-1}, x_t] + b_o)   # 输出门
g_t = tanh(W_g * [h_{t-1}, x_t] + b_g) # 候选细胞状态

c_t = f_t ⊙ c_{t-1} + i_t ⊙ g_t       # 更新细胞状态
h_t = o_t ⊙ tanh(c_t)                 # 更新隐藏状态
```
其中 `σ` 是 sigmoid 函数，`⊙` 表示逐元素乘法。

## 🖼️ 循环神经网络在计算机视觉中的应用：图像描述生成

循环神经网络可以与卷积神经网络结合，用于图像描述生成任务。这是一个“一对多”问题的典型例子。

以下是图像描述生成模型的基本流程：

1.  使用预训练的 CNN（如 ResNet）提取图像特征。
2.  将图像特征作为循环神经网络（如 LSTM）的额外输入。
3.  LSTM 以序列方式生成描述图像的单词。
4.  训练时，使用交叉熵损失函数；推理时，通过采样生成单词序列。

![](img/bef0ca1f166cdce497bf1e84c24ada10_65.png)

**模型结构简化表示**：
```
图像 -> CNN -> 特征向量 -> LSTM -> 单词序列
```
