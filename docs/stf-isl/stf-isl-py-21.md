# Python 版 21：分类问题导论 📊

![](img/aeeea9d9515bb7793e2513c98be4467b_0.png)

在本节课中，我们将学习分类问题的基本概念。分类是统计学习与机器学习中的核心领域，其目标是预测一个具有两个或更多类别的定性响应变量。我们将探讨分类与回归的区别，并介绍为何线性回归在处理分类问题时存在局限性。

---

## 从回归到分类 🔄

上一节我们讨论了响应变量为定量变量的回归问题。本节中，我们来看看当响应变量是定性变量（即分类变量）时的情况。分类问题在实际中非常常见，例如判断电子邮件是否为垃圾邮件，或预测患者疾病结局。

分类器的数学描述如下：给定一个特征向量 **X**，分类任务是构建一个函数 **C(X)**，其输出属于一个离散值集合 **C**。用公式表示为：

![](img/aeeea9d9515bb7793e2513c98be4467b_6.png)

C(X) \in \mathcal{C}
\]

## 线性回归用于分类？🤔

一个自然的问题是：能否使用线性回归解决分类问题？对于二分类问题，我们可以将响应变量编码为0和1（例如，0=未违约，1=违约），然后进行线性回归，并设定一个阈值（如0.5）进行分类。

![](img/aeeea9d9515bb7793e2513c98be4467b_53.png)

理论上，在总体中，**Y** 的条件期望 **E(Y|X)** 恰好等于 **Y=1** 的条件概率 **P(Y=1|X)**。因此，线性回归似乎适合此任务。

![](img/aeeea9d9515bb7793e2513c98be4467b_55.png)

![](img/aeeea9d9515bb7793e2513c98be4467b_57.png)

然而，线性回归存在一个明显缺陷：其预测值可能超出合理的概率范围 [0, 1]。下图展示了用线性回归拟合“违约概率”与“余额”关系时，预测值在低余额区域小于0的情况。

![](img/aeeea9d9515bb7793e2513c98be4467b_59.png)

![](img/aeeea9d9515bb7793e2513c98be4467b_60.png)

相比之下，逻辑回归的预测值则被限制在0到1之间，更适用于概率估计。

## 多分类问题的挑战 🎯

![](img/aeeea9d9515bb7793e2513c98be4467b_67.png)

当响应变量类别超过两个时（例如，患者诊断结果：中风、药物过量、癫痫发作），使用线性回归会面临更大问题。

![](img/aeeea9d9515bb7793e2513c98be4467b_69.png)

![](img/aeeea9d9515bb7793e2513c98be4467b_70.png)

![](img/aeeea9d9515bb7793e2513c98be4467b_72.png)

如果简单地将类别编码为1、2、3，会强加一个不存在的顺序，并隐含类别间差异相等的错误假设。这对于线性回归模型是不合适的。

![](img/aeeea9d9515bb7793e2513c98be4467b_74.png)

![](img/aeeea9d9515bb7793e2513c98be4467b_76.png)

![](img/aeeea9d9515bb7793e2513c98be4467b_77.png)

对于多分类问题，我们将采用更合适的方法，如多类逻辑回归或判别分析，这些将在后续课程中讨论。

## 总结 ✨

![](img/aeeea9d9515bb7793e2513c98be4467b_88.png)

![](img/aeeea9d9515bb7793e2513c98be4467b_90.png)

![](img/aeeea9d9515bb7793e2513c98be4467b_91.png)

本节课我们一起学习了分类问题的基本概念。我们了解到：
1.  分类预测的是定性响应变量。
2.  估计类别概率通常比直接输出分类标签更有价值。
3.  线性回归可用于二分类，但其预测概率可能超出[0,1]范围，因此存在局限性。
4.  对于多分类问题，线性回归通常不适用。
