# R 版 93：K均值聚类 📊

在本节课中，我们将要学习一种非常实用的数据聚类方法——K均值聚类。这是一种可以在任意维度数据中寻找簇结构的工具。为了直观演示，我们将在二维数据上进行操作，以便观察聚类效果。

## 概述：什么是K均值聚类？

K均值聚类是一种通过迭代计算来寻找数据中`K`个簇的方法。其核心思想是：将数据点分配到`K`个簇中，使得每个点到其所属簇中心的距离平方和最小。

## 第一步：生成演示数据

为了演示，我们首先需要生成一些包含已知簇结构的模拟数据。以下是生成数据的步骤。

以下是生成模拟数据的R代码：

```r
# 设置随机种子以保证结果可重现
set.seed(101)

# 生成一个100行2列的矩阵，数据来自标准正态分布
x <- matrix(rnorm(100*2), ncol=2)

# 生成4个簇的中心点（均值）
# 每个中心点是一个二维向量，其值来自标准差为4的正态分布
xmean <- matrix(rnorm(8, sd=4), ncol=2)

# 随机为100个数据点分配簇标签（1到4）
which <- sample(1:4, 100, replace=TRUE)

# 根据分配，将对应的簇中心加到原始数据点上，形成4个分离的簇
x <- x + xmean[which, ]
```

现在，我们已经有了一个包含4个清晰簇的二维数据集。

## 第二步：可视化原始数据

在应用聚类算法之前，我们先看看数据的原始样貌。我们可以用不同颜色代表不同的真实簇。

以下是绘制原始数据的R代码：

```r
# 绘制数据点，颜色根据其真实簇标签（which变量）决定
plot(x, col=which, pch=19)
```

生成的图形会显示四个用不同颜色（黑、红、绿、蓝）标记的簇，这是我们预先设定的“标准答案”。

## 第三步：应用K均值聚类算法

上一节我们生成了数据，本节中我们来看看如何让K均值算法自动发现这些簇。我们将使用`kmeans`函数，并指定寻找4个簇。

以下是执行K均值聚类的R代码：

```r
# 执行K均值聚类，指定簇数K=4，并设置15次随机初始值以寻找最优解
km <- kmeans(x, 4, nstart=15)

# 查看聚类结果摘要
km
```

运行`km`会输出以下关键信息：
*   **簇中心**：算法找到的4个簇的中心坐标。
*   **簇分配向量**：每个数据点被分配到的簇编号（1到4）。
*   **簇内平方和**：每个簇内所有点到其簇中心的距离平方和。
*   **组间/总平方和比值**：这类似于回归中的R²，表示簇中心能解释的数据方差比例。值越高，说明聚类效果越好。

## 第四步：可视化聚类结果

算法运行完毕后，我们需要评估其效果。最直观的方法是将算法找到的簇与真实的簇进行对比。

以下是可视化对比的R代码：

```r
# 绘制大圆圈，颜色代表K均值算法找到的簇标签
plot(x, col=km$cluster, cex=2, pch=1, lwd=2)

# 在大圆圈内部绘制小点，颜色代表数据真实的原始簇标签
points(x, col=which, pch=19)

# 由于算法分配的簇编号顺序是任意的，可能与原始编号不匹配
# 我们可以重新调整颜色映射以便于对比
plot(x, col=c(4,3,2,1)[km$cluster], cex=2, pch=1, lwd=2)
points(x, col=which, pch=19)
```

在最终的对比图中：
*   **外圈颜色**代表K均值算法的聚类结果。
*   **内部点的颜色**代表数据的真实归属。

通过对比，你可以发现一些被“错分”的点。例如，一个原本属于某个簇的点，因为距离另一个簇的中心更近，而被算法分配到了后者。这直观地展示了K均值聚类基于距离进行分配的原理。

## 总结

本节课中我们一起学习了K均值聚类的基本流程：
1.  **生成与理解数据**：我们创建了具有明确簇结构的模拟数据。
2.  **运行算法**：使用`kmeans()`函数，指定簇数`K`和多次随机初始值`nstart`来获得稳定结果。
3.  **评估结果**：通过检查簇内平方和与组间方差解释度来量化评估，并通过可视化对比算法结果与真实情况来直观理解算法的表现。

K均值聚类的核心在于通过迭代优化，将数据点划分到`K`个簇中，使得**所有点到其所属簇中心的距离平方和最小**。当数据维度较低时，可视化是理解和诊断聚类效果的强大工具。
