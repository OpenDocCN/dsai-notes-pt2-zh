## 从线性到多项式：引入样条

上一节我们介绍了线性函数和多项式拟合。本节中，我们来看看样条。样条比多项式更灵活，为我们提供了更广泛的可能性。

R语言中有许多内置工具可以拟合样条。我们将重点观察三次样条。首先需要加载名为 `splines` 的包。

```r
library(splines)
```

现在我们可以开始操作了。

我们将使用之前用过的 `wage` 数据，拟合一个线性模型。然后，我们将在年龄为25、40和60处设置节点，拟合一个三次样条。

```r
# 假设数据已加载
fit <- lm(wage ~ bs(age, knots=c(25,40,60)), data=wage_data)
```

接着，我们生成包含节点的拟合图。

```r
plot(age, wage, col="gray")
lines(age.grid, predict(fit, newdata=list(age=age.grid)), col="red", lwd=2)
abline(v=c(25,40,60), lty=2, col="darkgreen")
```

让我解释一下这个图的含义。样条本质上是分段的三次多项式。默认的 `bs()` 函数会生成三次样条的基函数。节点是函数不连续的点，但这里的不连续性仅存在于三阶导数。这意味着在每个区间内，函数都是三次多项式，但在节点处被约束为连续，并且具有连续的一阶和二阶导数，这使得曲线非常平滑。实际上，平滑到人眼无法察觉节点处的变化。

样条的思想在于它们比多项式更具局部性。它们没有多项式那样严重的“摆动尾部”，是拟合灵活函数的一种好方法。在R中，我们指定了节点的位置，操作非常简单。`BS` 函数用于生成基函数，有多种方式可以指定灵活性和节点位置。

## 平滑样条与自由度

固定节点回归样条或平滑样条是更进一步的工具。平滑样条很方便，因为它们本质上在每个数据点处都有节点，并通过粗糙度惩罚来控制函数的平滑程度。

在R中，我们使用 `smooth.spline` 函数。它不使用公式语言，我们直接指定预测变量是 `age`，响应变量是 `wage`，并设定所需的自由度。

```r
fit_smooth <- smooth.spline(age, wage, df=16)
```

我们将拟合的函数添加到图中。

```r
lines(fit_smooth, col="blue", lwd=2)
```

16个自由度相当多，你可以看到它捕捉到了一些波动。这是平滑样条的有效自由度，允许相当程度的摆动，在这个案例中可能有点过于波动了。

或者，使用 `smooth.spline` 时，我们可以利用留一法交叉验证来确定平滑参数。在教材中，平滑样条有一个控制函数粗糙度的参数。在第一次调用中，我们通过指定有效自由度来控制它，粗糙度惩罚相应地被选择。现在，我们将告诉函数使用交叉验证来选择调优参数或粗糙度参数。

```r
fit_smooth_cv <- smooth.spline(age, wage, cv=TRUE)
lines(fit_smooth_cv, col="purple", lwd=2)
```

所有曲线看起来都很相似，需要放大才能看到差异。使用交叉验证的平滑样条产生的有效自由度更少，大约是6.79。记住，有效自由度不一定是整数，它是函数粗糙程度的一种启发式度量。在这个案例中，大约是6到7个自由度。

对于一个固定节点的回归样条，让我们看看它有多少自由度。我们将重新拟合并查看摘要。

```r
fit_refit <- lm(wage ~ bs(age, knots=c(25,40,60)), data=wage_data)
summary(fit_refit)
```

我们可以数一下参数数量，有6个参数，所以它有6个自由度。有四个节点，最终得到六个自由度。而平滑样条使用略微不同的方法，最终得到略高于六的自由度。

## 广义可加模型 (GAMs)

我们通常希望拟合包含多个预测变量的模型，而不仅仅是单个预测变量。因此，我们需要一种方法将这些变量组合在一起，并生成漂亮的图表。`gam` 包将帮助我们做到这一点。

首先加载 `gam` 包。

```r
library(gam)
```

我们将再次使用 `wage` 数据，并拟合一个广义可加模型。我们调用 `gam` 函数，指定响应变量是 `wage`。我们希望在 `age` 上有一个平滑项（4个自由度），在 `year` 上有一个平滑项（4个自由度）。`education` 是一个因子变量，所以我们将其作为线性项加入，它会为每个水平创建虚拟变量并拟合常数。

```r
gam_fit <- gam(wage ~ s(age, 4) + s(year, 4) + education, data=wage)
```

这里的 `s()` 是 `gam` 包认识的特殊函数，它告诉 `gam` 我们希望在变量 `age` 上使用具有4个自由度的平滑样条，对 `year` 项也是如此。这两个项将被联合拟合，我们将在生成图表时看到结果。

在绘图之前，我们将设置一个包含3列的绘图网格。

```r
par(mfrow=c(1,3))
plot(gam_fit, se=TRUE, col="blue")
```

`plot` 函数为 `gam` 模型中的每一项生成一个图表，并绘制出这些项及其标准误。这是我们之前在单变量图表中看到的 `age` 变量的形状，但现在我们也包含了 `year` 和 `education`。我们可以看到，薪水似乎随着年份增长而增加，在2005年左右似乎有一个下降，但随后继续上升。毫不意外，薪水随着教育水平单调增加。

由于绘图空间有限，对于像 `education` 这样的因子变量，通常会在图表顶部标明每个水平，但这里没有空间。所以它只标出了最低的水平，但这些是教育水平的级别，并且它们随着教育程度的增加而单调递增。

这是一种非常方便的方法。这是使用 `plot.gam` 完成的。这是一种在多个变量中拟合非线性函数并生成漂亮图表的便捷方式。

`gam` 也适用于逻辑回归和其他类型的广义线性模型。我们可以使用二值化的 `wage` 变量，在 `age` 和 `year` 上拟合平滑项。

```r
gam_logit <- gam(I(wage>250) ~ s(age, 4) + s(year, 4) + education, family=binomial, data=wage)
plot(gam_logit, se=FALSE) # 这里关闭了标准误，因为有些区间很宽
```

这个图表绘制了对数几率概率的贡献，由于模型在这些项上是可加的，所以每个都是单独的函数。

使用 `gam`，你可以进行一些检验。例如，我们可以检验变量 `year` 是否需要非线性项。

```r
gam_linear_year <- gam(wage ~ s(age, 4) + year + education, data=wage)
anova(gam_fit, gam_linear_year, test="Chisq")
```

它给出了两个模型并进行了检验。卡方检验的P值为0.82，这表明我们确实不需要 `year` 的非线性项，线性项就足够了。事实上，你还可以进一步检验是否完全需要 `year` 项。

## 使用GAM方法绘制其他模型

我们已经看到 `gam` 知道如何漂亮地绘制函数。事实上，你甚至可以使用 `gam` 的绘图方法来绘制由 `lm` 和 `glm` 拟合的模型。

我们现在来演示一下。我们将再次以 `wage` 作为响应变量拟合一个线性模型，但这次不使用 `gam`，而是使用 `lm`，并使用另一个样条函数 `ns()` 生成样条基，它生成自然样条的基函数。

```r
lm_ns_fit <- lm(wage ~ ns(age, df=4) + ns(year, df=4) + education, data=wage)
```

我们想要一个 `age` 的自然样条，这里我们指定的是有效自由度，而不是节点。这看起来与调用 `gam` 非常相似，只是我们使用了一个固定的基函数集（缩减的基函数）来表示样条。对 `year` 也是如此，然后加上 `education` 的线性项。

我们拟合这个模型，现在即使它是一个 `lm` 对象，我们也可以使用 `gam` 的绘图方法。由于 `lm` 不属于 `gam` 类，我们需要写出 `plot.gam` 来指定我们想使用那个特定的绘图方法。

```r
plot.gam(lm_ns_fit, se=TRUE, col="red")
```

它做了与为合法的 `gam` 对象所做类似的事情：绘制拟合函数并包含了标准误。

这只是 `gam` 函数能力的一瞥。可以查看 `gam` 的帮助文件，有关于 `gam` 的文档。实际上，Rob和我有一本1991年关于广义可加模型的书，你可以找到更多细节。也有各种关于GAMs的论文，你也可以查阅《统计学习导论》中的相关章节，了解更多关于GAMs的知识及其功能。
