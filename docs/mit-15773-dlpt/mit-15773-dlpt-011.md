## 第一部分：如何生成图像

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_35.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_36.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_38.png)

首先，我们探讨如何构建一个模型，使其能够从一类潜在图像中随机生成不同的图像。例如，我们希望模型能生成各种“庄严的大学建筑”的图片。

### 训练数据与核心挑战

以下是构建此类模型的第一步：收集训练数据。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_40.png)

*   我们收集了大量“庄严的大学建筑”的图片作为训练集。
*   目标是：每次向模型请求时，它都能生成一个不同的、但符合“庄严大学建筑”类别的全新图像。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_42.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_43.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_45.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_47.png)

那么，如何让模型每次生成不同的结果呢？答案是：我们给它不同的**噪声**作为起点。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_49.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_51.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_53.png)

### 噪声作为起点

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_55.png)

噪声可以理解为一张所有像素值都是随机数的图像。每次我们给模型一个不同的随机噪声图像，它就会从这个不同的起点开始，生成一张不同的图片。由于模型是在“庄严大学建筑”的图片上训练的，因此它生成的图片也会属于这个类别。

### 关键问题：如何训练？

一个直接的问题是：如何训练一个模型，让它能从纯噪声生成一张具体的图像（比如麻省理工学院的基利安庭院）？这听起来很荒谬。

这里运用了一个巧妙的问题解决启发式方法：**将问题反转**。既然从图像生成噪声很容易，那我们就先学习这个反向过程。

**正向过程（加噪）**：给定一张清晰图像，我们可以很容易地通过多次添加随机噪声，使其逐渐变成一张完全随机的噪声图。

以下是为图像添加噪声的代码示例：
```python
import numpy as np
from PIL import Image

# 读取并归一化图像
image = Image.open(“kiliand_court.png”)
image_array = np.array(image) / 255.0

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_57.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_58.png)

# 生成与图像形状相同的随机噪声（均值为0，标准差为0.3）
noise = np.random.normal(0, 0.3, image_array.shape)

# 将噪声添加到原图
noisy_image = image_array + noise

# 将像素值裁剪回[0, 1]范围
noisy_image = np.clip(noisy_image, 0, 1)

# 显示噪声图像
```
通过逐步增大噪声的强度，我们可以得到从清晰图像到纯噪声的一系列图片。

### 构建训练数据对

既然正向过程（加噪）很容易，我们就可以用它来创建训练数据。核心思想是：我们不直接学习从噪声到图像的大跳跃，而是学习一系列微小的“去噪”步骤。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_60.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_61.png)

具体方法如下：
*   对于训练集中的每张图像，我们生成其不同程度的噪声版本。
*   我们将**噪声较多的版本**作为输入 `X`。
*   我们将**噪声稍少的版本**作为目标输出 `Y`。
*   这样，我们就得到了许多 `(X, Y)` 数据对，其中 `Y` 是比 `X` **略微清晰一点**的图像。

这种“略微”的差异至关重要，它使得学习任务变得可行。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_63.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_64.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_65.png)

### 训练去噪网络

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_67.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_69.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_71.png)

有了这些 `(X, Y)` 数据对，我们就可以训练一个神经网络（例如U-Net），其任务是：给定一个带噪声的图像，预测出一个噪声更少的版本。这本质上是一个**去噪网络**。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_73.png)

通过标准的随机梯度下降等方法，我们可以训练这个网络学会去除图像中的少量噪声。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_75.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_76.png)

### 生成图像：逆向过程

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_78.png)

一旦训练好去噪模型，我们就可以用它来从噪声生成图像了。过程如下：
1.  从纯噪声图像开始。
2.  将当前噪声图像输入训练好的去噪模型，得到一个噪声稍少的图像。
3.  将上一步的输出作为新的输入，重复步骤2。
4.  经过多次迭代（如50或100步），一张清晰的图像就会从“迷雾”中逐渐显现。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_80.png)

这个过程被称为**扩散模型**的逆向过程。初始的扩散模型论文对此有清晰阐述。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_82.png)

### 改进：预测噪声

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_84.png)

在实践中，一个有效的改进是：不让模型直接预测去噪后的图像，而是让它预测**添加到输入图像中的噪声**。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_86.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_88.png)

*   **原始方法**：输入 `X`（噪声图），输出 `Y`（更清晰的图）。
*   **改进方法**：输入 `X`（噪声图），输出预测的噪声 `ε`。然后，通过 `X - ε` 来得到更清晰的图。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_90.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_92.png)

虽然数学上近似等价，但预测噪声的方法在实践中能产生质量更高的图像。现代扩散模型大多采用这种“迭代减去预测噪声”的方式。

## 第二部分：如何用文本控制图像生成

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_100.png)

上一节我们介绍了如何生成特定类别的图像。本节中，我们来看看如何通过文本提示来精确控制生成的内容，例如根据“一只可爱的拉布拉多犬”生成对应的图片。

### 核心直觉：在嵌入空间中进行引导

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_109.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_110.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_111.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_113.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_115.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_117.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_119.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_120.png)

关键思路是：如果我们能为文本提示计算一个**嵌入向量**，并且这个向量与所有符合该描述的图像的嵌入向量在同一个向量空间中非常接近，那么我们就可以用这个文本嵌入来引导去噪过程。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_122.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_124.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_126.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_128.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_130.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_132.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_134.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_135.png)

具体来说，我们希望：
1.  有一个文本编码器，能将任何文本（如“可爱的拉布拉多犬”）映射为一个嵌入向量。
2.  有一个图像编码器，能将任何图像映射为另一个嵌入向量。
3.  确保**匹配的文本和图像**的嵌入向量彼此接近，而**不匹配的**则相距甚远。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_137.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_139.png)

如果实现了这一点，那么给定一个文本提示，它的嵌入就会自然靠近所有相关图像的嵌入区域。在去噪过程中，将这个文本嵌入作为额外条件输入模型，就能引导生成过程朝着符合文本描述的方向进行。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_141.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_142.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_144.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_146.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_147.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_149.png)

### CLIP模型：对齐文本与图像

OpenAI提出的**CLIP模型**完美解决了上述问题。CLIP代表“对比语言-图像预训练”。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_151.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_153.png)

**CLIP的工作原理如下：**
1.  **架构**：它包含一个**文本编码器**（基于Transformer）和一个**图像编码器**（基于ResNet）。
2.  **训练数据**：使用海量（如4亿个）从互联网收集的（图像，文本描述）对。
3.  **训练目标（对比学习）**：
    *   对于一个批次中的图像-文本对，分别用两个编码器得到它们的嵌入向量。
    *   计算所有图像嵌入和文本嵌入之间的**余弦相似度矩阵**。
    *   **目标1**：最大化**对角线**上的相似度（匹配的图文对）。
    *   **目标2**：最小化**非对角线**上的相似度（不匹配的图文对）。
4.  **损失函数**：结合以上两点，形成一个对比损失函数，防止模型将所有嵌入都压缩成同一个点（模型坍塌）。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_155.png)

通过这种训练，CLIP学会了将语义上相关的文本和图像映射到嵌入空间中相近的位置。

### 零样本图像分类

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_157.png)

一个CLIP的直接应用是**零样本图像分类**。传统分类器需要针对固定类别进行训练。而使用CLIP，你可以动态提供任意一组类别标签（文本），CLIP通过计算图像嵌入与每个标签文本嵌入的相似度，来预测最可能的类别，无需针对这些特定类别进行重新训练。

### 构建条件扩散模型

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_165.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_166.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_168.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_170.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_171.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_173.png)

现在，我们可以将CLIP与扩散模型结合起来，构建**文本条件扩散模型**。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_175.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_177.png)

**训练阶段：**
1.  对于训练集中的每张图像及其文本描述，我们用CLIP的文本编码器得到文本嵌入。
2.  我们将这个**文本嵌入**与**带噪声的图像**一起，作为去噪网络的输入 `X`。
3.  去噪网络的目标 `Y` 仍然是噪声更少的图像（或需要预测的噪声）。
4.  这样，模型在学习去噪的同时，也学会了如何根据文本嵌入来调整生成内容。

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_179.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_181.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_183.png)

**推理/生成阶段：**
1.  给定一个新的文本提示（如“麻省理工学院春天的基利安庭院”），用CLIP文本编码器得到其嵌入向量。
2.  从一个随机噪声图像开始。
3.  在每一步去噪迭代中，将当前噪声图像**与文本嵌入一起**输入条件去噪模型。
4.  经过多步迭代，最终生成符合文本描述的图像。

### 潜在扩散模型：加速生成

原始扩散模型在像素空间操作，计算量很大。**潜在扩散模型**（如Stable Diffusion）进行了关键优化以加速生成：
1.  **在潜在空间操作**：首先，用一个图像编码器（如VAE的编码器）将图像压缩到一个更低维的**潜在空间**，得到潜在编码。
2.  **在潜在空间扩散**：扩散（加噪/去噪）过程在这个紧凑的潜在编码上进行，而非高维的像素空间，大大减少了计算量。
3.  **解码**：去噪完成后，再用解码器将潜在编码转换回像素图像。

此外，文本条件（CLIP嵌入）通常通过**交叉注意力机制**被集成到U-Net结构的去噪网络中。

### Hugging Face Diffusers库

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_200.png)

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_201.png)

我们可以使用Hugging Face的`diffusers`库轻松运行扩散模型。以下是一个使用Stable Diffusion模型生成图像的示例：
```python
from diffusers import StableDiffusionPipeline
import torch

![](img/dd968c99ed5f15b5abdf94d26b17ce2a_203.png)

# 加载预训练管道
pipe = StableDiffusionPipeline.from_pretrained(“CompVis/stable-diffusion-v1-4”, torch_dtype=torch.float16)
pipe = pipe.to(“cuda”)

# 定义提示词并生成图像
prompt = “a photograph of an astronaut riding a horse”
image = pipe(prompt).images[0]
image.save(“astronaut_rides_horse.png”)
```

### 高级控制：负向提示

`diffusers`库支持高级功能，如**负向提示**。你可以指定不希望出现在图像中的元素。
```python
prompt = “a labrador in the style of Vermeer”
negative_prompt = “blue” # 不希望出现蓝色
image = pipe(prompt, negative_prompt=negative_prompt).images[0]
```

### Hugging Face Hub的视觉模型

Hugging Face Hub也提供了丰富的预训练计算机视觉模型，可用于图像分类、目标检测、图像分割等任务，使用方式与NLP模型类似，通过`pipeline`函数即可快速调用。

本节课中我们一起学习了扩散模型的核心原理。我们从如何通过逐步去噪从随机噪声生成图像开始，深入探讨了扩散模型的正向和逆向过程。接着，我们学习了如何利用CLIP模型将文本和图像的语义在嵌入空间中对齐，从而实现了通过文本提示来精确控制图像生成。最后，我们介绍了潜在扩散模型等加速技术，并展示了如何使用Hugging Face等工具库来实践这些强大的生成能力。扩散模型不仅在图像生成领域成果斐然，其思想也正被拓展到视频、3D结构、生物科学等更多领域，展现出巨大的潜力。
