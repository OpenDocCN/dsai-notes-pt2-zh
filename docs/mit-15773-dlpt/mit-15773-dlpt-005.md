NLP的发展经历了几个阶段：

1.  **基于规则的方法**：早期依赖语言学家手工编写语法规则。
2.  **统计机器学习方法**：通过统计词频、共现等特征来构建模型，效果往往优于复杂的规则系统。
3.  **深度学习方法**：自2012年起，循环神经网络等模型推动了NLP的进步。
4.  **Transformer时代**：2017年Transformer架构的提出，彻底改变了NLP领域，成为当前所有先进模型（包括大语言模型）的基础。

## 文本处理基础：向量化

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_57.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_59.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_60.png)

神经网络处理的是数值数据，因此我们需要将文本转换为向量。这个过程称为**文本向量化**，通常包含四个步骤，我们可以用缩写 **S-T-I-E** 来记忆。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_62.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_64.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_66.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_67.png)

### 第一步：标准化 (Standardization)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_69.png)

标准化是对原始文本进行清洗和规范化的过程。以下是常见的操作：

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_71.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_73.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_75.png)

*   **总是进行**：将文本转换为小写，并移除标点符号和重音符号。目的是告诉模型，单词的大小写形式在语义上是相同的。
*   **经常进行**：移除停用词（如 “the”, “a”, “is”）。这些词出现频率高但通常对预测任务帮助不大。
*   **有时进行**：词干提取或词形还原。例如，将 “running”, “ran”, “runs” 都归约为 “run”。目的是将语义相近的不同形式视为同一词元。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_77.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_78.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_79.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_80.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_82.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_83.png)

**示例**：
原始句子：`Hello, what do you picture when you think of travel to Mexico?`
标准化后：`hello what do you picture when you think of travel to mexico`

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_85.png)

### 第二步：分词 (Tokenization)

分词是将标准化后的文本序列切分成更小单元（称为词元）的过程。最常用的默认方法是**按空白字符（空格、换行等）分割**，每个词作为一个词元。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_87.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_89.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_91.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_93.png)

**示例**：
标准化句子：`hello what do you picture when you think of travel to mexico`
分词后：`[‘hello’, ‘what’, ‘do’, ‘you’, ‘picture’, ‘when’, ‘you’, ‘think’, ‘of’, ‘travel’, ‘to’, ‘mexico’]`

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_95.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_97.png)

然而，简单的按空格分词有其局限性，例如无法处理复合词（如 “New York”）、没有空格分隔的语言（如中文）或超长单词。现代大语言模型使用更复杂的分词方法，如**字节对编码**，我们将在后续课程中介绍。

### 第三步：建立索引 (Indexing)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_99.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_101.png)

在对整个训练语料库进行标准化和分词后，我们会得到一个包含所有**唯一词元**的列表，称为**词汇表**。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_103.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_104.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_106.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_107.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_109.png)

建立索引就是为词汇表中的每个唯一词元分配一个唯一的整数ID。通常，我们会添加一个特殊的词元 `[UNK]`（代表未知词），并为其分配ID 0。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_110.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_112.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_114.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_116.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_118.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_120.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_122.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_124.png)

**示例**：
假设词汇表为：`[‘[UNK]‘, ‘the’, ‘cat’, ‘sat’, ‘on’, ‘mat’, …]`
则索引映射为：`{‘[UNK]‘: 0, ‘the’: 1, ‘cat’: 2, ‘sat’: 3, ‘on’: 4, ‘mat’: 5, …}`

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_126.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_128.png)

### 第四步：编码 (Encoding)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_130.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_131.png)

编码是将索引（整数）转换为向量的过程。最简单直接的方法是**独热编码**。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_133.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_135.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_137.png)

对于一个大小为 `V` 的词汇表（包含 `[UNK]`），每个词元对应一个长度为 `V` 的向量。该向量中，只有对应词元索引位置为1，其余位置均为0。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_139.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_141.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_142.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_144.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_146.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_148.png)

**公式**：
如果词汇表 `V = [‘[UNK]‘, ‘a’, ‘apple’, …]`，词元 `’apple’` 的索引是2，那么它的独热编码向量是：`[0, 0, 1, 0, 0, …, 0]`

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_150.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_151.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_153.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_155.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_156.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_158.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_160.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_162.png)

**示例**：
句子 “the cat sat on the mat” 经过 S-T-I-E 处理后，每个词都变成了一个独热向量。最终，整个句子表示为一个矩阵（行数为词数，列数为词汇表大小V）。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_164.png)

## 词袋模型

上一节我们介绍了如何将句子表示为矩阵，但神经网络通常接受固定长度的向量输入。我们需要一种方法将这个矩阵“聚合”成一个向量。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_166.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_167.png)

以下是两种常见的聚合方法：

*   **多重独热编码**：查看词汇表中每个词是否在句子中出现过，出现则为1，否则为0。这相当于对矩阵的每一列做“或”操作。
*   **计数编码**：统计词汇表中每个词在句子中出现的次数。这相当于对矩阵的每一列求和。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_169.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_171.png)

这种忽略单词顺序，只关注单词是否出现或出现频率的模型，称为**词袋模型**。

**词袋模型的优缺点**：
*   **优点**：简单有效，对于许多分类任务（如情感分析、主题分类）已经足够好。
*   **缺点**：
    1.  **丢失词序信息**： “猫坐在垫子上” 和 “垫子坐在猫上” 的表示完全相同。
    2.  **向量高维稀疏**：词汇表可能很大（如数万），但每个句子只包含其中少数几个词，导致向量中绝大部分是0。
    3.  **固定长度**：无论句子长短，输出向量长度固定为词汇表大小，可能造成计算浪费。
    4.  **未登录词问题**：对于训练词汇表中没有的词，只能统一用 `[UNK]` 代替，丢失了信息。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_173.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_175.png)

实践中，我们通常只保留最高频的若干词（如5000个）来构建词汇表，以控制向量维度和模型复杂度。

## 实战：歌曲流派分类

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_177.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_179.png)

现在，让我们应用词袋模型来解决一个实际问题：根据歌词预测歌曲的流派（摇滚、嘻哈或流行）。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_181.png)

### 1. 构建文本向量化层

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_183.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_185.png)

我们将使用 Keras 的 `TextVectorization` 层来一站式完成 S-T-I-E 过程。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_187.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_188.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_190.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_192.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_194.png)

```python
from tensorflow import keras
from tensorflow.keras import layers

# 设置参数
VOCAB_SIZE = 5000  # 只保留最高频的5000个词
MAX_SEQUENCE_LENGTH = None  # 不限制序列长度，因为我们会用词袋模型

# 创建文本向量化层，使用多重独热编码
vectorize_layer = layers.TextVectorization(
    max_tokens=VOCAB_SIZE,
    output_mode=’multi_hot’, # 输出多重独热编码向量
    standardize=’lower_and_strip_punctuation’, # 标准化：小写并移除标点
    split=’whitespace’ # 分词：按空白字符分割
)
```

### 2. 适配数据并构建词汇表

我们需要用训练数据来“适配”这个层，使其基于我们的数据构建词汇表。

```python
# 假设 `train_texts` 是训练集歌词列表
vectorize_layer.adapt(train_texts)

# 查看词汇表大小和前几个词
print(“词汇表大小:”, len(vectorize_layer.get_vocabulary()))
print(“前10个词:”, vectorize_layer.get_vocabulary()[:10])
```

### 3. 构建神经网络模型

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_196.png)

我们将构建一个简单的全连接神经网络。

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_198.png)

```python
# 模型架构
inputs = keras.Input(shape=(VOCAB_SIZE,)) # 输入是5000维的向量
x = layers.Dense(16, activation=’relu’)(inputs) # 一个16个神经元的隐藏层
x = layers.Dropout(0.5)(x) # 添加Dropout层防止过拟合
outputs = layers.Dense(3, activation=’softmax’)(x) # 输出层：3个类别（摇滚、嘻哈、流行）

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_200.png)

model = keras.Model(inputs=inputs, outputs=outputs)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_202.png)

# 编译模型
model.compile(optimizer=’adam’,
              loss=’categorical_crossentropy’, # 多分类交叉熵损失
              metrics=[‘accuracy’])
```

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_204.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_205.png)

**Dropout 简介**：Dropout 是一种正则化技术，在训练过程中随机“丢弃”（置零）一部分神经元的输出。这可以防止神经元之间复杂的共适应，有效减轻过拟合。可以将其理解为随机让一部分神经元“失效”，迫使网络学习更鲁棒的特征。

### 4. 训练与评估

将歌词数据通过向量化层转换为特征向量，然后训练模型。

```python
# 将文本数据向量化
X_train = vectorize_layer(train_texts).numpy()
X_val = vectorize_layer(val_texts).numpy()

# 训练模型
history = model.fit(X_train, y_train_one_hot, # y_train_one_hot 是独热编码后的标签
                    epochs=10,
                    batch_size=32,
                    validation_data=(X_val, y_val_one_hot))

# 在测试集上评估
test_loss, test_acc = model.evaluate(X_test, y_test_one_hot)
print(f’测试集准确率: {test_acc:.3f}’)
```

### 5. 提升：使用 N-gram 模型

为了捕捉一些上下文信息，我们可以使用 **N-gram**。例如，二元语法不仅考虑单个词，还考虑相邻的词对。

```python
# 创建包含二元语法的向量化层
vectorize_layer_bigram = layers.TextVectorization(
    max_tokens=20000, # 二元语法组合更多，所以增大词汇表容量
    output_mode=’multi_hot’,
    ngrams=2, # 关键参数：使用二元语法
    standardize=’lower_and_strip_punctuation’,
    split=’whitespace’
)
vectorize_layer_bigram.adapt(train_texts)
```

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_209.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_211.png)

![](img/bbeb11715dc50114ee2ccc8e4c0ead3b_213.png)

使用二元语法特征重新训练模型，通常能获得比仅使用单词（unigram）更好的性能，因为它捕获了部分局部词序信息。

## 总结

本节课中我们一起学习了自然语言处理的基础知识：

1.  **文本向量化**：我们掌握了将文本转换为数值向量的标准流程 **S-T-I-E**（标准化、分词、索引、编码）。
2.  **词袋模型**：我们学习了如何通过聚合（如多重独热编码或计数编码）将句子表示为一个固定长度的向量，并理解了其简单有效但会丢失词序信息的特点。
3.  **实战应用**：我们构建了一个基于词袋模型和简单神经网络的歌曲流派分类器，并尝试通过引入 N-gram 特征来提升模型性能。
4.  **正则化技术**：我们简要介绍了 Dropout 层的作用，它通过随机丢弃神经元来防止模型过拟合。
