# 17：图模型与贝叶斯网络 🧠

在本节课中，我们将要学习一种新的主题——图模型，特别是其中的贝叶斯网络。这是一种用于表示联合概率分布、建模事物之间关系的方法。它不仅能用于监督学习，还能进行推断，即推测未观测变量的值。本节课我们将介绍图模型的基本概念、贝叶斯网络的结构、条件概率表，以及如何利用它们进行高效的参数学习和推断。

---

## 什么是图模型？🤔

到目前为止，我们已经学习了监督学习（约占课程的40%）和无监督学习（如聚类和降维）。现在，我们转向一个将花费数周时间的新主题：图模型。图模型是一个方法家族，用于表示我们对世界的假设和信念，并利用这些进行推断或分类。

图模型的核心思想是填补两种极端方法之间的空白：
1.  **完全依赖模型（联合概率分布）**：例如，计算三个二元变量 A、B、C 的联合概率 `P(A, B, C)`。这需要 `2^3 - 1 = 7` 个参数。对于 n 个二元变量，则需要 `2^n - 1` 个参数，数量巨大。
2.  **完全独立模型**：例如朴素贝叶斯中的假设，联合概率为 `P(A) * P(B) * P(C)`。这只需要 3 个参数，但假设过强，通常会牺牲准确性。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_1.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_3.png)

图模型则是一种折中方案。它允许我们引入关于变量间关系的先验信念，从而在不牺牲太多准确性的前提下，显著减少所需参数的数量。这是一种在“参数过多难以学习”和“假设过强不够准确”之间架起桥梁的方法。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_5.png)

## 贝叶斯网络：图结构 📊

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_9.png)

贝叶斯网络是图模型家族中的重要成员。它使用有向无环图来表示变量间的依赖关系。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_11.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_13.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_15.png)

以下是贝叶斯网络的核心组件：

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_17.png)

*   **节点**：代表模型中的随机变量。例如，一个关于“我是否在电影中睡着”的模型，可能包含变量：电影长度 `L`、我是否喜欢它 `I`、我是否睡着 `S`。
*   **有向边**：代表变量间的依赖关系。边的存在表示（条件）依赖，缺乏边则表示（在给定其他变量的条件下）条件独立。
*   **条件概率表**：每个节点都附有一个 CPT，定义了该节点在其父节点取不同值时的条件概率分布。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_19.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_21.png)

贝叶斯网络必须是有向无环图，即图中不能存在循环路径。这个特性对于后续的高效推断至关重要。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_23.png)

## 联合概率与参数效率 ⚙️

一个贝叶斯网络定义了一个联合概率分布。其公式是：网络中所有节点的条件概率的乘积。

对于一个具有节点 `X1, X2, ..., Xn` 的网络，其联合概率为：
`P(X1, X2, ..., Xn) = Π_i P(X_i | Parents(X_i))`

其中，`Parents(X_i)` 表示节点 `X_i` 的所有父节点。如果一个节点没有父节点，则使用其先验概率 `P(X_i)`。

**示例**：在之前的电影例子中（`L -> S`, `L -> I`），联合概率为：
`P(L, I, S) = P(L) * P(I | L) * P(S | L)`

**参数数量分析**：
*   完全依赖模型需要 7 个参数。
*   完全独立模型需要 3 个参数。
*   这个贝叶斯网络只需要 5 个参数（`P(L)` 需1个，`P(I|L)` 和 `P(S|L)` 各需2个）。

通过利用变量间条件独立的领域知识（例如，已知电影长度 `L` 时，`I` 和 `S` 独立），我们能用更少的参数准确地表示联合分布，这意味着我们需要更少的数据进行学习。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_27.png)

## 构建贝叶斯网络 🛠️

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_29.png)

构建一个贝叶斯网络通常包含以下步骤：

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_31.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_33.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_35.png)

1.  **识别变量**：确定问题领域中的所有相关随机变量。
2.  **确定网络结构（变量顺序与依赖关系）**：这可以通过领域知识手动完成，或通过算法从数据中学习。本课程中，我们主要关注手动构建。
    *   节点的添加顺序至关重要。不同的顺序会导致网络结构不同，从而影响参数效率。
    *   通常，我们以“根本原因”变量开始，然后添加受其直接影响的变量。
3.  **估计条件概率表**：一旦结构确定，CPT 可以通过在训练数据上计算相应的条件概率来轻松估计（例如，使用频率估计）。

**经典示例：报警问题**
变量：入室行窃 `B`、地震 `E`、警报响 `A`、约翰打电话 `J`、玛丽打电话 `M`。
基于领域知识的合理网络结构是：`B` 和 `E` 是 `A` 的父节点；`A` 是 `J` 和 `M` 的父节点。这个结构仅需10个参数，而完全连接的网络需要31个参数，却能表示相同的联合分布。

## 条件独立与 D-分离 🔗

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_37.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_39.png)

在贝叶斯网络中，仅从图结构（无需知道 CPT）就可以判断变量间是否条件独立，这通过 **D-分离** 准则来实现。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_41.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_43.png)

如果两个变量 `X` 和 `Y` 在给定变量集 `Z` 的条件下是 D-分离的，那么它们在给定 `Z` 时条件独立。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_45.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_47.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_49.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_51.png)

**D-分离的三条规则：**

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_53.png)

1.  **当 Z 为空集时**：如果 `X` 和 `Y` 之间的所有路径都包含“碰撞节点”（即具有两个入边的节点，形状如 `-> Node <-`），则 `X` 和 `Y` 是 D-分离的（独立）。否则，它们相连（可能依赖）。
2.  **当 Z 非空，且路径不含碰撞节点**：如果所有连接 `X` 和 `Y` 的路径上都至少有一个节点在 `Z` 中，则 `X` 和 `Y` 是 D-分离的。
3.  **当 Z 非空，且路径包含碰撞节点**：如果一条路径上有碰撞节点或其子孙节点**不在** `Z` 中，则该路径被“阻断”。只有当所有路径都被阻断时，`X` 和 `Y` 才是 D-分离的。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_55.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_57.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_59.png)

**马尔可夫毯**：一个节点的马尔可夫毯包括其父节点、子节点以及子节点的其他父节点。给定一个节点的马尔可夫毯，该节点与网络中所有其他节点条件独立。

## 推断：贝叶斯网络的核心任务 🎯

贝叶斯网络的主要用途之一是进行**概率推断**：在观察到部分变量取值的情况下，计算其他未观测变量的概率分布。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_61.png)

**示例**：在报警网络中，如果观察到约翰打电话了而玛丽没有 `(J=1, M=0)`，我们想计算入室行窃发生的概率 `P(B=1 | J=1, M=0)`。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_63.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_65.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_67.png)

**基本方法：枚举法**
根据条件概率定义和贝叶斯网络联合概率公式：
`P(B | J, M) = P(B, J, M) / P(J, M)`
其中，`P(B, J, M) = Σ_A Σ_E P(B, E, A, J, M)`。我们需要对未观测变量 `E` 和 `A` 的所有可能取值组合进行求和。对于小网络，这是可行的。

**复杂性挑战**
然而，对于大型网络，枚举所有未观测变量的组合是指数级复杂的。如果有100个变量，只观测2个，要计算另外3个的联合概率，需要对剩下的95个变量求和，有 `2^95` 种组合，这无法计算。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_69.png)

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_71.png)

**推断的复杂性**
事实上，在一般的贝叶斯网络中进行精确概率推断是一个 **NP-难问题**。我们可以通过将著名的 NP-难问题（如3-SAT问题）规约到贝叶斯网络推断问题来证明这一点。这意味着，不存在已知的多项式时间算法能解决所有贝叶斯网络的精确推断问题。

尽管如此，对于许多实际网络结构（如树状结构），存在高效的精确推断算法（如信念传播）。对于一般网络，则常使用近似推断算法（如采样方法）。

## 总结 📝

本节课我们一起学习了图模型与贝叶斯网络的基础知识。

![](img/62b9eb4c23c8d84d35d4dc7658a6306a_73.png)

*   我们了解了图模型如何作为完全依赖和完全独立模型之间的桥梁，以更少的参数表示复杂的联合概率分布。
*   我们学习了贝叶斯网络的构成：代表变量的**节点**、表示依赖关系的**有向边**，以及定义概率关系的**条件概率表**。
*   我们掌握了贝叶斯网络的**联合概率公式**，并理解了其参数高效的原因。
*   我们探讨了如何基于领域知识**构建网络**，并认识到变量顺序的重要性。
*   我们引入了 **D-分离** 准则，用于从图结构判断变量的条件独立性。
*   最后，我们明确了贝叶斯网络的**核心任务——概率推断**，认识了其基本方法（枚举法）以及所面临的**计算复杂性挑战**（NP-难问题）。

贝叶斯网络是一种强大的框架，不仅可用于分类，更能在存在部分观测的情况下进行复杂的概率推理，广泛应用于医疗诊断、计算生物学等多个领域。
