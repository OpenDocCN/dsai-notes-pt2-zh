# 18：两种泛化界证明方法

在本节课中，我们将学习两种用于证明算法泛化性能界限的方法：压缩界和算法稳定性。这两种方法旨在分析算法在训练集上的表现（经验风险）与其在未知数据上表现（期望风险）之间的差距，并探讨在何种条件下这个差距会很小。

上一节我们讨论了局部方法和偏差-方差分解，本节中我们来看看两种新的分析框架。

## 压缩界方法

压缩界方法的核心思想是：如果一个算法的输出仅依赖于训练数据中的一个小子集，那么该算法的泛化性能将主要取决于这个小子集，从而可以推导出泛化差距的界限。

### 核心概念与假设

假设存在一个压缩函数（或选择函数）**C**，它接收整个数据集 **S** 作为输入，并输出一个大小最多为 **K** 的子集 **C_K(S)**。

我们对算法做出如下假设：算法 **G_hat_n**（基于数据集 **S** 训练得到）的输出，与仅基于子集 **C_K(S)** 训练得到的算法 **G_hat_K** 的输出完全相同。用公式表示即：

**G_hat_n(S) = G_hat_K(C_K(S))**

这意味着算法的行为完全由这 **K** 个数据点决定，其余数据点可以被“遗忘”而不影响最终结果。

以下是几个满足压缩假设的算法示例：
*   **在矩形内可分的感知器**：决策边界仅由最靠近边界的少数几个点（支持向量）决定。
*   **支持向量机**：分离超平面仅由支持向量定义。
*   **一维阈值分类器**：在可分情况下，最优阈值仅由分界点两侧最近的两个数据点决定。

### 泛化差距分析

在上述假设下，我们可以分析期望泛化差距 **E_S[ E_Z[G_hat_n(Z)] - (1/n) Σ_i G_hat_n(Z_i) ]**。

分析的关键在于，由于算法只依赖于 **K** 个点，当评估一个不在该子集中的新数据点 **Z** 时，其行为近乎独立。通过将子集中 **K** 个“有问题”的数据点替换为新鲜样本，并利用损失函数有界（假设在 **[A, B]** 区间内）的条件，可以推导出泛化差距的上界。

最终，在忽略常数因子的情况下，我们得到泛化差距的量级为：

**O( (K * (B-A)) / n + sqrt( log( C(n, K) ) / n ) )**

其中 **C(n, K)** 是从 **n** 个点中选取 **K** 个子集的数量，其对数项约为 **K log n**。因此，该界限大致为 **O( K log n / n )**。这与基于VC维的经典泛化界有相似之处，但压缩界直接源于算法的特性。

## 算法稳定性方法

算法稳定性从另一个角度衡量算法对数据扰动的敏感度。其核心思想是：如果一个算法对数据集中任何单个数据点的微小变化不敏感，那么它的经验风险就很可能接近其期望风险。

### 稳定性与泛化差距的等价关系

通过巧妙的变量重命名，我们可以将期望泛化差距等价地重写为：

**E_{S, Z}[ G_hat_n(Z, Z_2,..., Z_n)(Z_1) - G_hat_n(Z_1, Z_2,..., Z_n)(Z_1) ]**

这个等式的含义是：泛化差距的大小，等于在平均意义下，考察当第一个训练数据点 **Z_1** 被一个独立的新样本 **Z** 替换后，算法在该点 **Z_1** 上的预测值变化有多大。如果变化很小，则经验风险接近期望风险。

### 一致稳定性

基于上述等价关系，我们引入一个更强、更易于分析的条件：一致 **β** 稳定性。

一个算法被称为是**一致 β 稳定**的，如果对于任何数据集 **S**、任何索引 **i**、以及任何可能的替代数据点 **Z‘**，以下不等式对所有 **Z** 成立：

**| G_hat_n(..., Z_i, ...)(Z) - G_hat_n(..., Z‘_i, ...)(Z) | ≤ β**

这意味着，改变任何一个训练数据点，所得到的两个算法在任何点上的输出差异都不会超过 **β**。显然，一致稳定性意味着前述泛化差距很小。

### 核岭回归的稳定性

核岭回归是一个具有一致稳定性的经典例子。其目标函数为：

**F_hat_n = argmin_{f∈H} (1/n) Σ_i (f(X_i) - Y_i)^2 + λ ||f||_H^2**

其中 **H** 是再生核希尔伯特空间，**λ** 是正则化参数。假设核函数的界为 **κ^2**，即 **K(x, x) ≤ κ^2**。

可以证明，核岭回归算法是**一致 β 稳定**的，且 **β = O( 1 / (λ n) )**。

证明思路利用了目标函数的强凸性。定义原始目标函数 **Φ(f)** 和替换一个数据点后的目标函数 **Φ‘(f)**。由于 **λ||f||^2** 项的存在，两个函数都是 **2λ** 强凸的。

设 **f** 和 **f‘** 分别是 **Φ** 和 **Φ‘** 的最小值点。根据强凸性定义和两个目标函数仅在一点损失项上不同的事实，可以推导出 **||f - f‘||_H** 的界。再利用 RKHS 中函数无穷范数受其范数约束的性质（**||f||_∞ ≤ κ ||f||_H**），最终可得点态误差界，从而证明稳定性。

这个结果表明，正则化项不仅防止过拟合，还直接赋予了算法稳定性，保证了经验风险是期望风险的良好指示。

## 总结

本节课中我们一起学习了两种证明泛化界的方法。
*   **压缩界**方法要求算法输出仅依赖于一个小规模的数据子集，从而限制了算法对数据的依赖程度，使得泛化差距可控。
*   **算法稳定性**方法则要求算法对任何单个数据点的变化不敏感，通过分析算法在数据点替换前后的输出变化来界定泛化差距。我们以核岭回归为例，展示了其强凸的正则化目标函数如何带来一致性稳定，进而保证良好的泛化性能。

这两种方法为我们提供了不同于均匀收敛的分析工具，有助于理解不同算法家族泛化行为背后的机制。
