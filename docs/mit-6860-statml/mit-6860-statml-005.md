# 5：逻辑回归与支持向量机

在本节课中，我们将学习两种重要的分类算法：逻辑回归和支持向量机。我们将从最小二乘法的视角出发，探讨它们之间的类比与差异，并引入“次梯度”这一凸优化工具来处理非光滑问题。课程的核心是理解如何计算这些模型的解，以及如何从线性模型推广到非线性模型（通过特征映射或核方法）。

---

## 概述与背景

上一节我们讨论了从线性模型到非线性模型的扩展。本节中，我们将系统性地应用类似思路，主要研究线性模型，并指出如何通过特征或核技巧进行非线性扩展。

我们主要关注分类问题。与回归问题中“差值”有意义不同，分类问题中“乘积” `y * f(x)` 是关键量。当其为正时，预测正确；为负时，预测错误。因此，我们将研究依赖于该乘积的损失函数。

我们将考虑以下形式的正则化经验风险最小化问题：
```
min_{w ∈ R^d} (1/n) * Σ_{i=1}^{n} L(y_i, w^T x_i) + λ * ||w||^2
```
其中 `λ ≥ 0` 是正则化参数。我们将这个最小化解记为 `w_hat_λ`。我们的目标是理解如何针对不同的损失函数 `L` 求解此问题。

---

## 损失函数与最大间隔

我们首先介绍两种常用的分类损失函数：

以下是两种常见的用于分类的替代损失函数（或称为间隔损失函数）：
*   **逻辑损失**：`L(y, f) = log(1 + exp(-y * f))`
*   **合页损失**：`L(y, f) = max(1 - y * f, 0)`

逻辑损失是处处可微的，而合页损失在 `y*f=1` 处不可微。合页损失是0-1损失的一个上界。

一个有趣的问题是：当正则化参数 `λ` 趋近于0时，上述正则化问题的解会趋向于什么？对于逻辑损失和合页损失，在数据**线性可分**的假设下，解的方向会收敛于所谓的**最大间隔**解。

最大间隔问题可以表述为：
```
max_{||w||=1} min_{i} [ y_i * (w^T x_i) ]
```
其思想是：在保证分类正确（乘积为正）的所有直线中，选择那个使得“最不确定”的点（乘积最小的点）的置信度最大的直线。为了问题有意义，我们需要约束 `w` 的范数（如欧几里得范数为1），否则解可以无限放大而变得平凡。

等价地，我们可以考虑**硬间隔支持向量机**问题：
```
min ||w||^2
s.t. y_i * (w^T x_i) ≥ 1, for all i
```
这个问题的解，在归一化后，与最大间隔解是相同的。这与我们在最小二乘法中看到的“最小范数插值解”形成了平行对比：在回归中，我们要求严格等式 `w^T x_i = y_i`；在分类中，我们要求不等式 `y_i * (w^T x_i) ≥ 1`。

正则化的作用在此显而易见：如果数据完全可分且无噪声，让 `λ → 0` 追求最大间隔是合理的。但如果数据有噪声或不可分，我们就需要在允许错误和追求简单解（小范数）之间进行权衡，此时 `λ` 应取一个大于0的值，通常通过交叉验证来选择。

## 逻辑回归

现在，我们来看逻辑回归的求解。逻辑回归对应的优化问题是凸且可微的，因此解存在且唯一。

### 梯度计算与梯度下降

我们通过求梯度并令其为零来寻找最小化点。梯度表达式如下：
```
∇_w [ (1/n)Σ log(1+exp(-y_i w^T x_i)) + λ||w||^2 ] = (1/n)Σ [ -y_i x_i / (1+exp(y_i w^T x_i)) ] + 2λw
```
由于梯度方程不是线性的，我们无法直接解析求解。因此，**梯度下降**是一个自然的选择。

梯度下降的迭代公式为：
```
w_{t+1} = w_t - γ * [ (1/n)Σ (-y_i x_i / (1+exp(y_i w_t^T x_i))) + 2λ w_t ]
```
其中 `γ` 是步长。对于光滑的凸问题，我们可以选择一个恒定的步长，其大小与损失函数海森矩阵特征值的上界有关。

每次迭代的计算成本约为 `O(n*d)`，即需要对每个样本进行一次 `d` 维的向量操作。对于大规模数据，随机梯度下降（每次迭代只用一个样本）或小批量梯度下降更为常用。

### 从线性到非线性与核化

与最小二乘法类似，我们可以证明逻辑回归的梯度下降迭代解始终可以表示为输入样本的线性组合。即，对于任意迭代步 `t`，存在系数 `c_i^t` 使得：
```
w_t = Σ_{i=1}^{n} c_i^t * x_i
```
相应的预测函数为：
```
f_t(x) = Σ_{i=1}^{n} c_i^t * (x_i^T x)
```
通过归纳法可以证明这一点，并得到系数 `c_i^t` 的更新规则。这一表示定理的直接好处是，我们可以将内积 `x_i^T x` 替换为核函数 `K(x_i, x)`，从而实现**核逻辑回归**。在核化版本中，我们直接更新系数 `c_i^t`，而不需要显式处理可能非常高维的特征空间向量。

## 支持向量机与次梯度方法

接下来我们讨论支持向量机。其对应的合页损失函数是凸的，但不可微。为此，我们需要引入**次梯度**的概念。

### 次梯度简介

对于一维凸函数，次梯度在某点就是其左导数和右导数构成的区间。对于可微点，该区间退化为一个点（即导数）。对于多维凸函数，次梯度 `∂f(w)` 是一个集合，其中的每个向量 `v` 都满足对于所有 `u`，有 `f(u) ≥ f(w) + v^T (u-w)`。几何上，次梯度给出了在 `w` 点处位于函数图像下方的一系列支撑超平面。

我们有以下有用的规则：
*   **和规则**：`∂(f+g)(w) = ∂f(w) + ∂g(w)`（在适当条件下）。
*   **链式法则**：可用于计算复合函数的次梯度。

对于最小化一个不可微的凸函数，我们可以使用**次梯度下降**：
```
w_{t+1} = w_t - γ_t * g_t, 其中 g_t ∈ ∂f(w_t)
```
与梯度下降的关键区别有两点：
1.  `g_t` 是次梯度集合中的一个元素（我们需选择一个，例如左导数）。
2.  步长 `γ_t` 通常需要衰减，例如按 `O(1/√t)` 衰减，以保证收敛。次梯度下降不是严格的下降法。

### SVM的次梯度下降求解

对于合页损失正则化问题，我们计算其次梯度。考虑单个样本的损失 `L = max(1 - y_i w^T x_i, 0)`。其次梯度（选择左导数）为：
```
∂_w L = 
    { -y_i x_i }， 如果 y_i w^T x_i < 1
    { 0 }，        如果 y_i w^T x_i > 1
    { -α y_i x_i | α ∈ [0,1] }， 如果 y_i w^T x_i = 1
```
在实践中，我们通常做一个方便的选择，例如当 `y_i w^T x_i = 1` 时取0。

那么，完整目标函数的次梯度为：
```
g_t ∈ (1/n) Σ_{i=1}^{n} [ 指示函数(y_i w_t^T x_i < 1) * (-y_i x_i) ] + 2λ w_t
```
其中指示函数在条件为真时取1，否则取0。

因此，SVM的次梯度下降更新为：
```
w_{t+1} = w_t - γ_t * [ (1/n) Σ_{i: y_i w_t^T x_i < 1} (-y_i x_i) + 2λ w_t ]
```
这个算法非常直观：在每一步，我们找出那些分类间隔小于1的样本（即犯错误或位于间隔内的样本），然后沿着这些样本的梯度方向的反方向，结合权重衰减项，更新参数。

### SVM的核化

与逻辑回归完全类似，我们可以证明SVM的次梯度下降迭代解也具有表示定理的形式：
```
w_t = Σ_{i=1}^{n} c_i^t * x_i
```
通过归纳法，我们可以得到系数 `c_i^t` 的更新规则。这允许我们直接将算法核化：在更新和判断条件时，我们使用核函数 `K(x_i, x_j)` 来计算 `f_t(x) = Σ c_j^t K(x_j, x)`。核化SVM的每次迭代成本约为 `O(n^2)`，因为需要计算所有样本对之间的核函数值。

关于“支持向量”名称的由来，传统的SVM推导（通过对偶问题及KKT条件）会显示，最终解中只有一部分系数 `c_i` 非零，这些样本即称为“支持向量”。在上述次梯度下降的原始形式中，这种稀疏性并不明显，但通过分析最优性条件（`0 ∈ ∂f(w)`）可以恢复出类似的框约束，从而与经典解释联系起来。

## 总结

本节课我们一起学习了逻辑回归和支持向量机这两种核心分类算法。

我们首先从正则化经验风险最小化的统一框架出发，比较了逻辑损失与合页损失，并探讨了当正则化强度趋近于零时，它们与最大间隔分类器的联系。

接着，我们详细研究了逻辑回归的求解，因其光滑可微，我们使用梯度下降法，并展示了其解可表示为样本的线性组合，从而自然引出了核逻辑回归的实现。

对于支持向量机，由于其损失函数不可微，我们引入了次梯度这一凸分析工具。我们介绍了次梯度的定义和基本规则，并给出了使用次梯度下降求解SVM的具体算法。该算法直观且易于实现，特别适合大规模数据。最后，我们也说明了SVM同样可以核化。

通过本课，我们不仅掌握了两种重要算法，还学习了处理非光滑优化问题的基本工具，并再次实践了通过表示定理将线性模型推广到非线性模型的技术。
