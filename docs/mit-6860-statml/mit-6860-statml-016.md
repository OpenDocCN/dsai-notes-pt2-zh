# 17：局部方法

在本节课中，我们将学习一种与之前不同的方法——局部方法。首先，我们将回顾上一讲中关于线性函数（超平面）的拉德马赫复杂度分析，并对比两种不同的泛化误差分析方法。接着，我们将聚焦于平方损失函数，推导其特有的偏差-方差分解，并以此为基础，分析K近邻算法的性能。

## 回顾：线性函数的拉德马赫复杂度

上一讲我们介绍了拉德马赫复杂度的概念。本节中，我们来看看一个具体的例子：线性函数类。

考虑函数类 **F**，它由所有线性函数组成，其权重向量 **w** 的 L2 范数以 **B** 为界。同时，我们假设输入数据 **x** 也位于单位 L2 球内。在这种情况下，可以证明该函数类的拉德马赫复杂度以 **B / √n** 为界。

以下是具体的计算过程：

**R̂ₙ(F) = E_ε [ sup_{‖w‖₂ ≤ B} (1/n) Σ_{i=1}^n ε_i ⟨w, x_i⟩ ]**

利用线性性质，可以将 **w** 提取出来：

**= E_ε [ sup_{‖w‖₂ ≤ B} ⟨w, (1/n) Σ_{i=1}^n ε_i x_i⟩ ]**

根据对偶范数的定义，这等于 **B** 乘以随机向量平均的范数期望：

**= B * E_ε [ ‖ (1/n) Σ_{i=1}^n ε_i x_i ‖₂ ]**

利用詹森不等式和 **ε_i** 的独立性（交叉项期望为零），最终得到：

**≤ B / √n**

这个结果与之前分析单纯 L2 球时的界相同，但这是一种巧合。值得注意的是，如果我们分析的是函数符号类 **sign(F)**，其拉德马赫复杂度会依赖于维度 **d**，形式为 **√(d/n)**。当维度 **d** 高于样本数 **n** 时，这个界是平凡的（无意义的）。然而，通过引入间隔（margin）分析，我们可以得到一个依赖于 **1/γ**（间隔的倒数）和函数类本身拉德马赫复杂度的界，后者是维度无关的（例如 **B/√n**）。这种分析对于理解高维空间中的核方法和神经网络至关重要。

## 平方损失下的偏差-方差分解

现在，我们将目标转向回归问题，并采用平方损失函数：**L(f(x), y) = (f(x) - y)²**。我们的目标是找到一个算法，使得其期望样本外性能 **E[L(f̂)]** 接近最优可能性能 **L(f*)**，其中 **f*(x) = E[y | x]** 是回归函数。

对于平方损失，预测误差可以等价地表示为估计误差：

**E[L(f̂)] - L(f*) = E[ (f̂(x) - f*(x))² ]**

这意味着，在平方损失下，预测问题与在 **L₂(Pₓ)** 范数下估计回归函数 **f*** 是同一个问题。

接下来，我们进行另一种形式的偏差-方差分解。固定训练集 **S** 中的 **x₁, ..., xₙ**，考虑期望仅对 **y** 变量取。定义 **f̂** 为基于数据 **S** 学得的函数。

**E_y[ (f̂(x) - f*(x))² ] = (E_y[f̂(x)] - f*(x))² + E_y[ (f̂(x) - E_y[f̂(x)])² ]**

*   **第一项 (E_y[f̂(x)] - f*(x))²** 称为**偏差平方**。它衡量了算法平均输出与真实回归函数之间的差距。
*   **第二项 E_y[ (f̂(x) - E_y[f̂(x)])² ]** 称为**方差**。它衡量了由于 **y** 的随机性导致的算法输出围绕其均值的波动。

这是一个恒等式，为我们分析局部方法（如K近邻）提供了框架。

## K近邻算法分析

K近邻是一种典型的局部方法。对于一个新的查询点 **x**，算法步骤如下：
1.  找出训练集中与 **x** 距离最近的 **K** 个点。
2.  将这 **K** 个点的 **y** 值取平均，作为 **f̂(x)** 的预测值。

我们将利用上述偏差-方差分解来分析其性能。假设真实的回归函数 **f*** 是 Lipschitz 连续的（即平滑的），并且数据域是紧致的。

以下是分析的核心步骤：

**方差项分析：**
固定查询点 **x** 和训练集的 **x₁, ..., xₙ**。**f̂(x)** 是 **K** 个独立（条件独立）的 **y** 值的平均。每个 **y** 的方差是常数（假设 **y** 有界）。因此，方差项的行为类似于 **1/K**。

**偏差项分析：**
偏差项涉及 **E_y[f̂(x)] = (1/K) Σ_{i=1}^K f*(x_(i))**，其中 **x_(i)** 是第 **i** 近邻。由于 **f*** 是 Lipschitz 的，**|f*(x_(i)) - f*(x)|** 可由 **‖x_(i)} - x‖** 控制。通过一些几何概率论证（例如将数据分桶），可以证明，在 **d** 维空间中，最近邻的期望平方距离 behaves like **(K/n)^{2/d}**。因此，偏差平方项的行为类似于 **(K/n)^{2/d}**。

**平衡偏差与方差：**
总误差（偏差平方+方差）大致为：
**误差 ≈ (K/n)^{2/d} + 1/K**

*   当 **K** 增大时，方差项 **1/K** 减小，但偏差项 **(K/n)^{2/d}** 增大。
*   当 **K** 减小时，情况相反。

通过优化 **K** 来平衡这两项，令两项数量级相等，解得最优 **K ~ n^{2/(2+d)}**，对应的最小误差率为 **n^{-2/(2+d)}**。

## 总结

本节课中我们一起学习了以下内容：
1.  回顾了线性函数类的拉德马赫复杂度分析，并对比了基于函数类符号的复杂度（依赖维度）与基于函数值本身的复杂度（可能维度无关）之间的区别，后者可通过间隔分析实现。
2.  针对平方损失函数，推导了预测误差与回归函数估计误差的等价性，并得到了条件于输入 **x** 的偏差-方差分解恒等式。
3.  将偏差-方差分解应用于K近邻算法。在假设回归函数 Lipschitz 连续的前提下，我们分析了方差项（~ **1/K**）和偏差项（~ **(K/n)^{2/d}**）的行为。
4.  通过平衡偏差和方差，得到了K近邻算法的最优收敛速率 **n^{-2/(2+d)}**。这个速率在数据维度 **d** 很高时会变得很慢，这体现了“维数灾难”，但该方法在非常弱的假设下是一致的。

这种基于局部平滑性的分析与之前基于全局函数类一致收敛的分析范式不同，为理解非参数方法提供了重要工具。
