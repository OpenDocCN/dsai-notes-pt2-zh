# 2：监督学习问题定义 🎯

在本节课中，我们将系统地介绍监督学习问题的数学定义。我们将逐一拆解定义中的每个组成部分，包括输入/输出空间、概率分布、损失函数以及期望风险，并探讨它们背后的假设和含义。理解这些基础概念是后续学习各类高级算法的重要前提。

## 概述 📋

监督学习的目标是，基于有限的、带有标签的示例数据，学习一个能够对未来未知数据做出良好预测的模型。我们将从一个简洁的数学定义出发，逐步展开，理解其背后的每一个符号和假设。

## 问题定义与核心要素

监督学习问题仅由两个核心要素定义：

1.  两个空间 **X** 和 **Y**，以及其上的一个概率分布 **P**。
2.  一个损失函数 **L**。

基于此，问题的目标是找到一个函数 **f**，使其期望风险（即平均损失）最小。

### 输入与输出空间

上一节我们介绍了问题的整体框架，本节中我们来看看构成问题的第一个要素：输入空间 **X** 和输出空间 **Y**。

*   **X** 通常被称为输入空间或特征空间。
*   **Y** 通常被称为输出空间或标签空间。

它们本质上是两个集合。一个重要的区分在于这些空间是否具有线性（向量）结构。

以下是具有线性结构的空间示例：
*   **实数集 R** 或 **R^D**（D维欧几里得空间）。
*   **矩阵空间**（可视为向量化处理）。
*   **函数空间**（如平方可积函数空间 L^2）。

这些空间允许我们进行加法、数乘、定义范数（长度）和内积等操作，便于进行数学分析和计算。

以下是不具有线性结构的空间示例：
*   **二进制字符串**：两个字符串相加可能无意义。
*   **离散概率分布**（和为1的正数序列）：两个分布相加不再是一个概率分布。
*   **图结构**。

当输出空间 **Y** 不具备线性结构时（如分类标签、字符串、图），这类问题常被称为**结构化输出问题**。

根据输出空间的不同，我们可以对问题进行分类：

以下是回归类问题的示例：
*   **Y ⊆ R**：标准回归问题。
*   **Y ⊆ R^T**：多变量回归。
*   **Y** 是某个域上的 **L^2 函数空间**：函数回归。

以下是分类类问题的示例：
*   **Y = {-1, 1}** 或 **{0, 1}**：二分类。
*   **Y = {1, 2, ..., T}**：多类别分类。
*   **Y** 是标签的子集：多标签分类。

### 概率分布

我们假设数据 **(x, y)** 服从一个固定的、未知的联合概率分布 **P(x, y)**。我们只能被动地获得从这个分布中独立同分布（i.i.d.）采样得到的数据集 **{(x_i, y_i)}_{i=1}^n**。

一个关键的分解是将联合分布写为边际分布与条件分布的乘积：
```
P(x, y) = P(x) * P(y | x)
```
*   **P(x)**：输入 **x** 的分布，反映了我们在未来会遇到哪些问题（输入）以及它们的频率。它代表了**采样不确定性**。
*   **P(y | x)**：给定输入 **x** 时输出 **y** 的条件分布，反映了对于同一个问题，正确答案可能存在的**噪声或变异性**。

#### 经典模型示例

1.  **回归模型**：`y = f*(x) + ε`，其中 **ε** 是噪声（例如高斯噪声）。这对应了 `P(y | x)` 是一个以 `f*(x)` 为中心的分布。
2.  **分类模型**：`P(y | x)` 简化为一个概率向量。例如在二分类中，就是 `P(y=1 | x)` 和 `P(y=-1 | x)`。这两个概率的重叠程度定义了问题的内在难度（类似于信噪比）。

**技术备注**：我们常使用概率密度符号 `p(x)`，但这隐含了分布相对于空间体积测度（如勒贝格测度）绝对连续。在高维空间中，数据往往只占据一个低维流形（如图像、文本），其体积测度可能为零。因此，更严谨的做法是考虑相对于适当测度的密度。

### 损失函数与期望风险

损失函数 **L: Y × G → R^+** 衡量预测值 `g ∈ G`（来自预测空间，有时 `G=Y`）与真实值 `y ∈ Y` 之间的差异。

#### 常见损失函数示例

以下是回归问题中常用的损失函数：
*   **平方损失**：`L(y, g) = (y - g)^2`
*   **绝对损失**：`L(y, g) = |y - g|`
*   **ε-不敏感损失**：`L(y, g) = max(0, |y - g| - ε)`，用于支持向量回归。

在分类问题中，我们通常令 `G = R`，并基于 `y * g`（符号一致性）设计损失函数：
*   **0-1损失（误分类损失）**：`L(y, g) = I_{y * g < 0}`，非凸、不连续，难以优化。
*   **合页损失**：`L(y, g) = max(0, 1 - y*g)`，用于支持向量机（SVM），是0-1损失的凸上界。
*   **逻辑损失**：`L(y, g) = log(1 + exp(-y*g))`，用于逻辑回归，处处可微。
*   **指数损失**：`L(y, g) = exp(-y*g)`，用于AdaBoost。
*   **平方损失（用于分类）**：`L(y, g) = (y - g)^2 = (1 - y*g)^2`（因为 `y^2=1`）。其图形虽显“奇怪”，但从理论分析看仍有其合理性。

对于结构化输出等问题，设计合适的损失函数本身就是一个研究课题。

损失函数定义了我们的**期望风险（或泛化误差）**：
```
R(f) = E_{(x,y)~P} [ L(y, f(x)) ]
```
这表示函数 **f** 在所有可能数据（包括未来数据）上的平均损失，是我们的终极优化目标。

### 目标函数

在已知分布 **P** 的理想情况下，我们寻找能最小化期望风险的最优函数，称为**目标函数**或**贝叶斯最优决策函数**：
```
f_P = argmin_f R(f)
```
由于直接在函数空间优化困难，我们利用概率分解，定义**条件风险**：
```
L_x(a) = E_{y~P(·|x)} [ L(y, a) ]
```
其中 `a = f(x)` 是一个实数。那么，最小化总风险 `R(f)` 等价于对每个 **x** 分别最小化 `L_x(a)`：
```
f_P(x) = argmin_a L_x(a)
```
这样，我们将一个泛函优化问题转化为了逐点的实值函数优化问题。

#### 不同损失函数对应的目标函数

通过求解 `argmin_a L_x(a)`，我们可以得到：

对于回归损失：
*   **平方损失**：`f_P(x) = E[y | x]`（条件期望，即回归函数）。
*   **绝对损失**：`f_P(x) = Median[y | x]`（条件中位数）。

对于分类损失（假设二分类 `y ∈ {-1, 1}`）：
*   **0-1损失**：`f_P(x) = sign( P(y=1|x) - P(y=-1|x) )`，即**贝叶斯最优分类器**。
*   **逻辑损失**：`f_P(x) = log( P(y=1|x) / P(y=-1|x) )`，输出是概率比的对数。
*   **合页损失**：有趣的是，其最优解也是 `f_P(x) = sign( P(y=1|x) - P(y=-1|x) )`，与0-1损失相同。
*   **平方损失**：`f_P(x) = P(y=1|x) - P(y=-1|x)`，其符号与贝叶斯最优分类器一致。

这表明，许多用于分类的凸替代损失函数，虽然优化的是不同的目标，但其最优解的方向与最小化分类错误的目标是一致的。然而，像SVM（合页损失）这样的方法通常不直接提供概率估计 `P(y|x)`。

## 从问题定义到学习算法 🛠️

到目前为止，我们只是定义了一个理想化的问题。在现实中，我们既不知道真实分布 **P**，也无法在所有可能的函数中搜索。因此，我们需要构建切实可行的学习算法。

核心思路是进行两次近似：
1.  **用经验风险近似期望风险**：由于我们只有数据，用经验均值（训练集上的平均损失）代替期望。
    ```
    R_emp(f) = (1/n) * Σ_{i=1}^n L(y_i, f(x_i))
    ```
2.  **用假设空间 H 近似函数空间**：限制搜索范围在一个可控的函数集合 **H** 内（如线性函数、神经网络、核方法等）。

从而，实际的学习算法旨在解决：
```
f_n = argmin_{f ∈ H} R_emp(f)
```

当然，这不是唯一途径（例如随机梯度下降法有不同的视角），但这是最直观的框架之一。本课程后续将深入探讨如何选择 **H**、如何有效地优化 `R_emp(f)`，以及如何从理论和计算上保证 `f_n` 的性能接近真正的目标 `f_P`。

## 总结 📝

本节课中，我们一起学习了监督学习问题的严格数学定义。我们详细剖析了定义中的四个核心组件：输入/输出空间、数据概率分布、损失函数以及由此导出的期望风险和目标函数。我们了解到，不同性质的输出空间定义了不同类型的学习任务（回归、分类、结构化预测），而不同的损失函数则引导我们寻找不同的最优解（如条件期望或条件中位数）。最后，我们指出了理想定义与现实算法之间的鸿沟——未知的分布和无限大的函数空间，并引出了通过**经验风险最小化**在**假设空间**中寻找近似解的基本学习范式。这是理解所有后续高级机器学习算法的基础。
