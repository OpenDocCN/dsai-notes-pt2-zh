# 15：样本复杂度与Rademacher平均值

在本节课中，我们将要学习样本复杂度的概念，以及一个被称为Rademacher平均值的工具。这个工具能极大地简化我们对函数类复杂度的理解。

上一节我们介绍了一种特定的算法——经验风险最小化。人们通常会选择一个架构或函数类F（例如神经网络），然后最小化其在数据上的经验拟合。我们称之为在类F上的经验风险最小化。一个关键问题是：这种做法何时是好的？何时是可行的？

## 从经验风险最小化到一致偏差

我们之前看到过一个简单的不等式序列。经验风险最小化器的样本外性能期望值，与类中最优函数的性能期望值之差，存在一个上界。这个上界有时被称为泛化误差，即经验风险最小化器的样本外性能期望值减去其样本内性能期望值。

为了研究这个上界，我们转而分析一个更一般化的对象：函数类F上经验损失与期望损失的最大偏差。我们将其称为**一致偏差**，因为它衡量了经验值与期望值在所有函数上的最坏偏差。

今天的目标是，为这个一致偏差对象提供一个进一步的上界，这个上界将由Rademacher平均值给出。重要的是，这个上界在常数因子范围内是紧的，这意味着我们不会像之前的不等式那样可能损失很多信息。随后，Rademacher平均值本身又可以通过其他技术（如覆盖数、VC维等）进一步上界。因此，可以将此视为一个线性递进：我们从一个目标出发，通过一系列越来越宽松但更易于分析的上界，最终证明其值很小。

当然，由此得到的样本复杂度估计通常是悲观的，但它至少提供了一些保证。

## 经验过程的视角

为了简化表述，让我们将损失函数与函数f的复合视为一个新函数g。那么，当f在类F中变化时，就定义了一个函数类G。经验过程的观点如下：固定数据分布P后，每个函数g都有一个期望损失。我们期望找到使期望损失最小的最优函数g*。

然而，当我们只有数据时，我们看到的是一条“扰动”的经验曲线。经验风险最小化的风险在于，如果我们选取的经验曲线最小值点所对应的函数，其真实期望损失（我们看不到）与经验损失相差甚远，那么我们就严重过拟合了。

因此，整个问题的关键在于理解：何时可以安全地追寻经验曲线的最小值？我们通过选择函数类F（即G）来解决这个问题。我们希望这个类满足两个性质：
1.  类中最优函数的性能（期望损失）接近全局最优g*的性能。
2.  经验曲线不会严重误导我们，即经验值与期望值在整个类上保持一致。

第二个性质正是我们今天要研究的：如何定义“不太大”，以确保经验曲线上的凹陷不会将我们引向一个完全不同的函数。

## 衡量集合的复杂度

现在，我们暂时不讨论函数类，而是考虑一个N维空间中的子集G，它是超立方体{-1, 1}^N的一个子集。我们想理解这个集合的“大小”或复杂度。

以下是几种可能的度量方式：
*   **计数**：如果G有限，可以计算其元素个数|G|。但G可能是无限的。
*   **体积**：测量其N维体积。但当一个集合维度降低时（例如变为N-1维），其N维体积为零，而N-1维体积可能非零，因此体积度量在某些情况下不令人满意。
*   **平均宽度**：从原点出发，随机选取一个方向（单位球面上的均匀分布），测量集合G在该方向上的投影的最大长度（即最大内积），然后对所有方向取平均。这就像用X光从各个角度照射物体并测量其宽度。
*   **高斯平均宽度**：与平均宽度类似，但方向向量改为从均值为零、协方差矩阵为单位矩阵的高斯分布中抽取。
*   **Rademacher平均值**：不取所有方向，而是仅取超立方体的顶点方向（即由独立、 unbiased的±1硬币抛掷序列定义的方向），并测量投影宽度。我们将主要使用这个概念。

这些度量彼此相关，在常数或对数因子范围内可以相互界定。

## Rademacher平均值的定义与性质

**Rademacher平均值** 对于一个集合G ⊆ ℝ^N，其（经验）Rademacher平均值定义为：
```
R̂(G) = (1/N) * E_ε [ max_{g∈G} ∑_{i=1}^{N} ε_i * g_i ]
```
其中，ε = (ε_1, ..., ε_N) 是一个向量，其分量是独立的随机变量，满足 P(ε_i = +1) = P(ε_i = -1) = 1/2。这些ε_i被称为Rademacher随机变量。

让我们看几个例子来建立直觉：
*   **单点集**：R̂({g}) = 0。
*   **整个超立方体{-1, 1}^N**：R̂ = 1。这意味着模型可以“解释”任何情况，容易过拟合。
*   **L1范数单位球**：R̂ ≈ 1/N。这个集合非常“小”。
*   **L2范数单位球（欧几里得球）**：R̂ ≈ 1/√N。这与仅有两个对跖点的情况复杂度相同！
*   **两个对跖点{1, 1,...,1}, {-1,-1,...,-1}**：R̂ ≈ 1/√N。值得注意的是，这个度量无法区分一个球体和两个点，这与体积度量截然不同。
*   **有限集**：如果|G|有限，则 R̂(G) ≤ O(√(log|G|/N))。这是一个上界，但可能很宽松。

Rademacher平均值具有一些重要性质，便于我们处理：
1.  **单调性**：如果 F ⊆ G，则 R̂(F) ≤ R̂(G)。
2.  **缩放性**：对于常数c，有 R̂(c · G) = |c| · R̂(G)。
3.  **凸包不变性**：R̂(conv(G)) = R̂(G)，因为最大值总是在极点处取得。
4.  **收缩原理**：如果 φ: ℝ → ℝ 是一个1-Lipschitz函数（即 |φ(a)-φ(b)| ≤ |a-b|），且 φ(0)=0，那么对于由φ逐点作用于G中向量得到的集合 φ∘G，有 R̂(φ∘G) ≤ R̂(G)。这个性质非常有用，例如可以帮助我们“去掉”损失函数（如铰链损失）。

## 连接一致偏差与Rademacher平均值

现在，我们回到最初的函数类G。关键的联系由以下**对称化引理**给出：

设 G 是一个从空间 Z 到 ℝ 的函数类。那么，一致偏差的期望值满足：
```
E[ sup_{g∈G} ( E[g(Z)] - (1/N)∑_{i=1}^{N} g(Z_i) ) ] ≤ 2 * E_{Z, ε}[ R̂( G|_{Z_1,...,Z_N} ) ]
```
其中，G|_{Z_1,...,Z_N} 表示将函数类G限制在数据点 Z_1, ..., Z_N 上得到的向量集合，即 { (g(Z_1), ..., g(Z_N)) : g ∈ G } ⊆ ℝ^N。

这个引理的意义在于：为了理解难以处理的一致偏差（涉及未知的期望E[g(Z)]），我们只需研究一个完全由已知数据定义的量——Rademacher平均值。除了一个常数因子2，我们没有损失任何信息。实际上，在另一个常数因子范围内，这也构成一个下界。因此，Rademacher平均值为我们控制一致偏差提供了紧致的工具。

更重要的是，右边的对象在原则上是可计算（尽管计算量可能很大）且依赖于数据的。接下来的策略是：固定数据，然后利用Rademacher平均值的性质，分析这个数据依赖度量的复杂度。通过这种方式，我们通常只需几行推导，就能得到关于问题样本复杂度的结果，而该结果仅依赖于模型的性质。

## 总结

本节课中，我们一起学习了样本复杂度的分析框架。我们从经验风险最小化出发，引出了一致偏差的概念。为了度量函数类的复杂度并控制一致偏差，我们引入了Rademacher平均值这一强大工具。我们看到了它的定义、计算示例、重要性质，并通过对称化引理将其与核心的理论目标——一致偏差的期望——紧密联系起来。这为我们后续使用覆盖数、VC维等工具进一步分析具体模型（如神经网络）的样本复杂度奠定了坚实的基础。下一讲，我们将通过具体例子来演示如何应用这些工具。
