# 13：感知机算法

在本节课中，我们将学习感知机算法。我们将完成上次课程中关于感知机的讨论，探讨感知机在独立同分布数据下的两个重要推论，并讨论偏差-方差权衡问题。

## 感知机算法回顾

上一节我们介绍了统计学习的基本框架，本节中我们来看看一个最基础的算法——感知机。

感知机算法操作于一个数据序列 `(X1, Y1), ..., (XT, YT)`。在每一个时间步 `t`，算法会查看新的数据点及其标签，并做出预测 `Y_hat_t`。如果预测正确，算法不进行任何操作；如果预测错误，则执行更新。

**算法更新公式**：
```
如果 sign(WT * XT) != YT:
    W_{T+1} = W_T + Y_T * X_T
否则:
    W_{T+1} = W_T
```

## 间隔的定义与性质

为了分析感知机的性能，我们需要引入“间隔”的概念。间隔衡量了数据被一个超平面分离的难易程度。

**间隔的数学定义**：
```
γ = max_{||W*||=1} min_i (Y_i * (W* · X_i) ∧ 0)
```

这里的 `∧` 表示取最大值操作。间隔 `γ` 可以被理解为：在所有可能的单位向量 `W*` 中，找到那个能使所有数据点 `(X_i, Y_i)` 的投影 `Y_i * (W* · X_i)` 都至少为 `γ` 的最大值。本质上，它衡量了数据点距离最佳分隔超平面的最小距离。

一个重要的观察是：**最大化间隔等价于寻找一个具有最小范数的分隔器**。如果我们允许分隔器 `W*` 的范数为 `D`，那么间隔的界限将变为 `D^2 / γ^2`。

## 感知机的错误界限定理

感知机算法有一个非常简洁而强大的理论保证：对于任意数据序列，其犯错误的总次数有一个明确的上限。

**定理**：对于任意数据序列，感知机算法犯错误的总次数 `M` 满足：
```
M ≤ D^2 / γ^2
```
其中 `D` 是数据点 `X_i` 的直径（最大范数），`γ` 是该数据序列的间隔。

以下是该定理的简要证明思路：

1.  假设算法在 `T` 轮运行中总共犯了 `M` 次错误。
2.  在每次犯错误的轮次 `t`，权重向量的平方范数增长满足：
    `||W_{t+1}||^2 ≤ ||W_t||^2 + 1` （假设 `||X_t|| ≤ 1`）。
3.  因此，经过 `M` 次错误后，最终权重向量的范数满足：`||W_T||^2 ≤ M`。
4.  另一方面，根据间隔 `γ` 的定义，对于能达到最大间隔的单位向量 `W*`，在每次犯错误的轮次有：`W* · (W_{t+1} - W_t) ≥ γ`。
5.  将上述不等式对所有错误轮次求和并利用柯西-施瓦茨不等式，最终可得 `M ≤ 1/γ^2`。推广到一般范数 `D` 即得到 `M ≤ D^2 / γ^2`。

这个证明的巧妙之处在于，它不依赖于数据的任何分布假设，适用于任意序列。

## 推论一：单次遍历与随机超平面

上一节我们证明了感知机在任意序列上的错误界限，本节中我们来看看这个结论在独立同分布数据下的第一个应用。

假设数据 `(X1, Y1), ..., (Xn, Yn)` 独立同分布于某个分布 `P`。我们让感知机对数据**进行一次遍历**（单次通过所有数据点）。

一个自然的问题是：我们如何从这次运行中获得一个在未知数据上表现良好的分类器？直接使用最终的 `W_n` 可能并不理想。

一个简单的策略是：**从所有迭代产生的超平面 `{W1, ..., Wn}` 中随机均匀地选取一个**，记作 `W_τ`。

**推论**：这样随机选取的超平面 `W_τ`，其**期望的样本外错误率**（即在新数据上犯错的概率）满足：
```
E[L_01(W_τ)] ≤ (E[D^2 / γ^2]) / n
```
其中期望是对训练数据集 `S` 的抽取而言的，`D` 和 `γ` 是基于数据集 `S` 计算的直径和间隔。

**直观理解**：如果数据分布具有真实的间隔 `γ`（即存在一个“无人区”），那么期望错误率大约为 `1/(n γ^2)`。这意味着随着数据量 `n` 增大，我们随机选择的分类器犯错的概率会降低。

这个推论的证明利用了感知机的错误界限定理，并通过交换期望和利用 `W_τ` 与对应数据点 `(X_τ, Y_τ)` 的独立性来完成。

## 推论二：循环遍历与最终超平面

推论一使用了随机选择的超平面。一个更直观的想法是：使用感知机**循环遍历数据直到不再犯任何错误**后得到的最终超平面 `W_T`。这个超平面必然完美分类了所有训练数据。

**推论**：对于这样得到的最终超平面 `W_T`，其期望的样本外错误率满足：
```
E[L_01(W_T)] ≤ (E[D^2 / γ^2]) / (n+1)
```
这里的 `D` 和 `γ` 是基于 `n+1` 个数据点（`n` 个训练点 + 1个测试点）计算的，但期望上这与基于 `n` 个点的量级相同。

**证明思路（留一法分析）**：
这个证明非常精妙，采用了“留一法”的思想。
1.  考虑一个包含 `n+1` 个 IID 数据点的扩展集。
2.  在这个扩展集上运行感知机直至收敛，记录所有曾被错误分类（参与更新）的数据点索引，设其数量为 `M`，且 `M ≤ D^2/γ^2`。
3.  对于任何一个**未被记录**的索引 `t`，数据点 `Z_t` 从未引发错误。这意味着，无论 `Z_t` 是否在训练集中，最终的分类器 `W` 都会正确分类它，并且 `W` 与从数据集中移除 `Z_t` 后训练得到的分类器 `W_{-t}` 相同。
4.  因此，在留一法误差的和式中，只有那些曾被错误分类的点（最多 `M` 个）对应的项可能非零。
5.  通过期望计算和放缩，即可得到样本外错误率的界限。

这是分析经验风险最小化器样本外性能的最简洁证明之一。

## 与优化视角的联系

最后，我们可以从优化角度理解感知机的两种模式：

*   **单次遍历**：可以看作是对期望损失函数 `E[max(0, -Y*(W·X))]` 的**随机梯度下降**。每次更新使用一个新鲜样本的梯度。
*   **循环遍历直至收敛**：可以看作是对经验损失函数 `(1/n) Σ max(0, -Y_i*(W·X_i))` 的**（循环）梯度下降**，直至经验风险为零。

前者在大数据时代很流行，后者是经典统计学习理论分析的经验风险最小化器。实际应用中，我们通常折中，多次遍历数据。这两种方式代表了理解学习算法的不同路径，而感知机为我们分析后者（经验风险最小化）的性能提供了一个绝佳的简单案例。

## 总结

本节课中我们一起学习了感知机算法及其理论分析。我们首先回顾了感知机的基本更新规则，然后定义了间隔的概念并证明了感知机在任意序列上的错误次数上限。基于这个核心定理，我们推导出了两个重要推论：在IID数据下，单次遍历后随机选择超平面、以及循环遍历直至收敛后使用最终超平面，都能保证良好的样本外性能，其错误率以 `O(1/(n γ^2))` 的速率下降。最后，我们将感知机与随机梯度下降和经验风险最小化联系起来，为后续更复杂的学习理论分析奠定了基础。
