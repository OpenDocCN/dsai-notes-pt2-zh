## 数据表示：核心概念与术语

数据表示是将数据从一个空间（输入空间）映射到另一个空间（表示空间或特征空间）的过程。其目标是找到一种变换，使得数据更容易处理，或者使后续任务（如分类）更容易执行。

在不同的领域中，这个映射 **Φ** 有不同的名称：
*   **机器学习/统计学**：特征、特征映射。
*   **信号处理**：变换（如傅里叶变换、小波变换）、分析算子。
*   **信息论**：编码器。
*   **计算几何**：嵌入。

这些方法的共同点是，**Φ** 通常是通过设计原则预先定义的，而不是从数据中学习得到的。然而，在许多应用中，**Φ** 的选择对最终性能的影响往往比学习算法本身更大。因此，一个重要的问题是：**我们能否从数据中学习 Φ？**

---

## 无监督学习表示的原则

我们考虑无标签数据 **X₁, ..., Xₙ**，假设它们独立同分布地来自某个概率分布 **P(X)**。我们的目标是学习一个好的特征映射 **Φ**，使其对未来的任务有益，尽管我们并不知道这些任务具体是什么。

我们需要一个原则来指导无监督表示的学习。在监督学习中，我们使用概率分布和损失函数。在这里，我们只有概率分布，那么损失函数应该如何定义？

以下是两种主要思路：

### 1. 距离/相似性保持

一个直观的想法是，希望表示能够保持数据点之间的某种相似性或距离关系。

**核心思想**：如果两个数据点 **x** 和 **x̄** 在原始空间中相似（例如，距离很小），那么经过映射 **Φ** 后，它们在表示空间中也应该相似。

**一个简单（非学习）的例子**：随机投影（如Johnson-Lindenstrauss引理）。定义 **Φ(x) = Sᵀx**，其中 **S** 是一个随机矩阵（例如，元素服从高斯分布）。可以证明，以高概率，投影后的距离与原始距离成比例，即存在常数 **C₁, C₂** 使得：
**C₁ ||x - x̄||² ≤ ||Φ(x) - Φ(x̄)||² ≤ C₂ ||x - x̄||²**

这属于“度量学习”的范畴，但通常需要一些初始的相似性信息（例如，通过阈值判断哪些点对是“相似的”），这又回到了某种形式的监督学习。

### 2. 重构误差最小化

另一种更流行的思路是**重构**。我们不仅学习一个将数据映射到表示空间的编码器 **Φ**，还学习一个将表示映射回数据空间的重构器（解码器）**Ψ**。

**核心思想**：我们希望数据经过编码再解码后，能够尽可能接近原始数据。

**流程**：
1.  编码：**z = Φ(x)** （将数据映射到表示空间）。
2.  解码：**x̂ = Ψ(z)** （将表示重构回数据空间）。
3.  最小化重构误差：**L(x, x̂) = ||x - x̂||²**

这种由编码器 **Φ** 和解码器 **Ψ** 组成的结构被称为**自动编码器**。其经验风险最小化问题可以形式化为：
**min_{Φ, Ψ} (1/n) Σ_{i=1}^n ||x_i - Ψ(Φ(x_i))||²**

然而，这里存在一个明显的问题：如果取 **Φ** 和 **Ψ** 均为恒等映射，重构误差为零，但这显然不是一个有用的表示。因此，与监督学习不同，在无监督表示学习中，**对模型（Φ, Ψ）施加约束是先验且至关重要的**。不同的约束定义了不同的问题。

---

## 字典学习：一种具体的自动编码器框架

为了进行下去，我们引入一个具体的框架。我们假设：
1.  **解码器 Ψ 是线性的**。这意味着重构过程是表示向量的线性组合。我们可以将 **Ψ** 视为一个由列向量（称为“原子”）组成的**字典矩阵 D**（尺寸为 **d × p**）。重构公式为：**x̂ = Dβ**，其中 **β** 是系数向量。
2.  **编码器 Φ 是“最近邻编码器”**。对于输入 **x**，其表示 **z = Φ(x)** 定义为能最好地线性重构 **x** 的系数向量 **β**。即：
    **Φ(x) = argmin_{β ∈ Λ} ||x - Dβ||²**
    其中 **Λ** 是对系数向量 **β** 的约束集合（例如，稀疏性约束）。

在这个框架下，学习自动编码器就变成了**字典学习**：我们同时学习字典 **D** 和每个数据点对应的系数 **{β_i}**，以最小化总重构误差：
**min_{D ∈ 𝒟, {β_i} ∈ Λ} (1/n) Σ_{i=1}^n ||x_i - Dβ_i||²**

其中 **𝒟** 是对字典 **D** 的约束集合（例如，列向量归一化）。

通过为 **𝒟** 和 **Λ** 选择不同的约束，我们可以得到多种经典的无监督学习算法。

以下是几个关键示例：

### 示例1：主成分分析

*   **约束**：
    *   字典 **D** 的列向量是标准正交的（即 **DᵀD = I**），且列数 **p = k < d**。
    *   系数空间 **Λ = ℝᵏ**（无额外约束）。
*   **推导**：在此约束下，最优系数 **β* = Dᵀx**。问题简化为：
    **min_{D: DᵀD=I} (1/n) Σ_{i=1}^n ||x_i - DDᵀx_i||²**
    这等价于寻找一个 **k** 维子空间，使得数据到该子空间的投影误差最小。这正是PCA的公式。其几何假设是数据近似位于一个低维线性子空间附近。

### 示例2：稀疏编码

*   **约束**：
    *   字典 **D** 的列向量通常要求归一化（即 **||d_j||₂ = 1**），且列数 **p** 可能大于 **d**（过完备字典）。
    *   系数空间 **Λ = {β: ||β||₁ ≤ λ}**，即施加L1范数约束以促进稀疏性。
*   **推导**：学习一个过完备字典，使得每个数据点都能用字典中少数几个原子的线性组合来稀疏表示。其几何假设是数据近似位于**多个低维子空间的并集**上。优化通常采用交替最小化：固定 **D** 优化 **{β_i}**（套索问题），固定 **{β_i}** 优化 **D**（带约束的二次规划）。

### 示例3：K均值聚类

*   **约束**：
    *   字典 **D** 的列数 **p = k**，代表 **k** 个聚类中心。
    *   系数空间 **Λ** 是标准基向量的集合 **{e₁, e₂, ..., eₖ}**，其中 **e_j** 是第 **j** 个元素为1、其余为0的向量。这意味着每个数据点只能由一个原子（即一个聚类中心）完全表示，系数是“超级稀疏”的（只有一个1，其余为0）。
*   **推导**：编码步骤是将每个点分配给最近的中心（one-hot编码），解码步骤是用该中心重构数据。优化过程（交替更新分配和中心）正是K均值算法。它可以被视为稀疏编码的一种极端形式。

## 从线性到非线性：神经网络的连接

上述框架中的解码器是线性的。我们可以将其推广到非线性情况。例如，考虑以下形式的自动编码器：
**Φ(x) = σ(W₁x)** （编码，带非线性激活函数σ）
**Ψ(z) = W₂ᵀz** （解码，线性）
其中 **W₂** 可能与 **W₁** 存在联系（如权重绑定）。这已经是一个简单的前馈神经网络结构。

更一般地，我们可以使用更深层的网络同时作为编码器和解码器，这就是**深度自动编码器**。下一节课中，我们将引入神经网络和深度学习，并将其视为一种（可以是监督或无监督的）学习数据表示的方法。

## 总结

本节课中我们一起学习了无监督数据表示学习的基本原理。我们首先回顾了特征表示在不同领域的名称和重要性，然后探讨了无监督学习表示的两个核心原则：保持相似性和最小化重构误差。我们重点介绍了基于重构误差的**自动编码器**框架，并通过引入**线性解码器**和**最近邻编码器**的假设，将其具体化为**字典学习**问题。通过选择不同的约束条件，我们展示了**主成分分析**、**稀疏编码**和**K均值聚类**都可以统一在这个框架之下。最后，我们指出了通过引入非线性，可以过渡到神经网络模型，为下一节学习深度学习与表示学习做好了准备。

无监督表示学习的目标是找到能够捕捉数据本质结构的特征，从而减少对昂贵标签数据的依赖。虽然这是一个充满挑战且尚未完全解决的问题，但字典学习和自动编码器等思想为我们提供了强大的工具和清晰的视角。
