在本节课中，我们将学习卷积神经网络（CNN）如何应用于图像处理，特别是如何利用它们来实现端到端的自动驾驶任务。我们将从基础的图像分类概念开始，逐步深入到如何训练一个神经网络来根据前方道路视频控制车辆转向。

---

## 📊 概述：计算机视觉与机器学习

上一节我们介绍了基础的神经网络。本节中，我们来看看如何将神经网络应用于图像数据，即计算机视觉领域。

计算机视觉如今主要由数据驱动的方法和机器学习主导。图像本质上是一系列像素值的集合，每个像素对于彩色图像（RGB）包含三个0到255的数值。因此，我们可以将图像视为数字矩阵，并应用之前学过的相同机器学习方法。

计算机视觉任务主要分为两类：
*   **回归**：输入是图像，输出是一个实数值。例如，根据前方道路图像预测方向盘转角。
*   **分类**：输入是图像，输出是一个离散的类别标签。例如，判断图像中是猫还是狗。分类任务底层通常也是一个回归问题，网络输出属于各个类别的概率，我们选取概率最高的类别作为最终分类结果。

然而，计算机视觉非常困难，因为图像会受多种因素影响，例如：
*   视角变化
*   尺度变化
*   遮挡
*   形变
*   背景干扰
*   光照变化

我们需要构建对所有这些变化都鲁棒的模型。

---

## 🏆 图像分类流程与数据

图像分类的流程与我们之前讨论的监督学习流程一致。我们需要带有标签的图像数据来训练模型。

以下是计算机视觉领域一些常用的数据集：
*   **MNIST**：手写数字数据集，包含0-9十个类别。
*   **ImageNet**：大型图像数据集，包含按WordNet层次结构组织的类别。
*   **CIFAR-10 / CIFAR-100**：包含10个或100个类别的小图像数据集，常用于快速验证算法。
*   **Places**：自然场景图像数据集。

以CIFAR-10为例，它包含10个类别（如飞机、汽车、鸟、猫等）。我们的目标是构建一个分类器，能够识别输入图像属于哪个类别。

---

## 📏 简单的图像比较：K-最近邻算法

在深入神经网络之前，我们先看一个简单的基线方法：K-最近邻（K-NN）算法。其核心思想是比较两幅图像的相似度。

一种比较方法是计算两幅图像之间每个像素的绝对差值之和，这称为 **L1距离**。更常用的距离度量是 **L2距离**（欧几里得距离）。

以下是K-NN分类器的工作原理：
1.  给定一张查询图像。
2.  在数据集中找到与它最相似的K张图像（“最近邻”）。
3.  将这K张图像的标签进行统计，将出现次数最多的标签作为查询图像的预测结果。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_6.png)

以下是K值选择的影响：
*   **K=1**：仅考虑最近的一个邻居。
*   **K增大**：考虑更多邻居，决策边界更平滑，但可能出现平局区域。

对于CIFAR-10数据集，通过交叉验证调整超参数K，发现K=7时效果最好，准确率约为30%。作为对比，人类在该数据集上的准确率约为94%，而当前最先进的卷积神经网络准确率已超过95%，甚至略高于人类水平。

## 🧠 从全连接网络到卷积神经网络

上一节我们介绍了基础的神经元和全连接网络。本节中，我们来看看专门为图像数据设计的卷积神经网络有何不同。

在全连接网络中，如处理28x28的MNIST图像，我们会将784个像素展平作为输入层，连接到隐藏层。每个神经元都与上一层的所有神经元相连。

卷积神经网络（CNN）则不同，它专为具有空间结构（如图像、音频）的数据设计。其关键特性在于**参数共享**和**局部连接**。

**卷积层**的输入和输出通常是三维体积（宽度 x 高度 x 深度）。对于图像，深度就是通道数（例如，RGB图像深度为3）。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_8.png)

卷积层通过**滤波器**（或卷积核）在输入图像上滑动来工作。每个滤波器学习检测一种特定的空间特征（如边缘、纹理）。关键点在于：
*   **局部连接**：输出层中的一个神经元只与输入层中一个局部区域（感受野）相连。
*   **参数共享**：同一个滤波器在不同空间位置使用相同的权重。这意味着无论特征出现在图像的左上角还是右下角，同一个滤波器都能检测到它。这极大地减少了参数量，并赋予了模型**平移不变性**。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_10.png)

![](img/a3ea2eb7da7b24a9c2c132cab561a980_12.png)

![](img/a3ea2eb7da7b24a9c2c132cab561a980_14.png)

卷积操作的输出尺寸由三个参数控制：
1.  **滤波器数量**：决定输出体积的深度。
2.  **步长**：滤波器滑动的步长，影响输出尺寸。
3.  **填充**：在输入图像边缘填充零，以控制输出尺寸。

一个滤波器可以表示为一个小型权重矩阵 `W`，在输入图像 `I` 上滑动进行卷积计算（实际上是互相关计算），公式可简化为：
`输出特征图[x, y] = sum( I[局部区域] * W ) + bias`

---

## 🔽 池化层

在卷积层之后，通常会添加**池化层**（如最大池化）。池化层用于降低特征图的空间尺寸（宽度和高度），从而减少计算量和参数数量，并一定程度上提供平移不变性。

最大池化的操作很简单：在一个小区域（如2x2窗口）内，仅保留最大值。这是一个固定操作，没有可学习的参数。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_16.png)

关于池化层的使用程度是一个需要权衡的问题：过多的池化会丢失过多空间信息，但能提升训练效率和模型紧凑性。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_18.png)

## 🏗️ 卷积神经网络的整体架构

一个典型的用于图像分类的CNN架构如下所示：

```
输入图像 -> [卷积层 -> 激活函数 -> 池化层] x N -> 全连接层 x M -> 输出层
```

*   **前端（卷积部分）**：一系列卷积层和池化层交替堆叠，用于从图像中提取多层次的特征（从简单边缘到复杂物体部件）。
*   **后端（全连接部分）**：将卷积部分提取的特征图展平，连接到一个或多个全连接层。这些层负责根据高级特征进行最终的分类或回归决策。
*   **输出层**：对于分类任务，通常使用Softmax函数输出每个类别的概率；对于回归任务（如预测方向盘角度），则直接输出一个实数值。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_20.png)

这种架构使得CNN在ImageNet等大型图像分类竞赛中取得了突破性成果。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_22.png)

## 🚦 CNN在自动驾驶任务中的应用

![](img/a3ea2eb7da7b24a9c2c132cab561a980_24.png)

卷积神经网络的能力不仅限于图像分类。通过修改网络结构，它可以应用于各种自动驾驶子任务。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_26.png)

![](img/a3ea2eb7da7b24a9c2c132cab561a980_28.png)

![](img/a3ea2eb7da7b24a9c2c132cab561a980_30.png)

**1. 语义分割**
通过将网络末端的全连接层替换为卷积层，可以使网络输出与输入图像尺寸相同的分割图。每个像素都被分类为不同的语义类别（如道路、车辆、行人）。这帮助车辆理解场景中各部分的含义。

**2. 目标检测**
结合区域提议算法和CNN分类器，可以在图像中定位并识别多个目标。例如，先找出可能包含物体的区域，然后用CNN判断每个区域内的物体类别。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_32.png)

**3. 端到端驾驶**
这是本节课的核心应用。其思想是“绕过”中间复杂的感知、规划模块，直接学习从传感器输入（如前方道路图像）到控制输出（如方向盘转角）的映射。

**NVIDIA** 的一项开创性工作展示了这种可能性：他们使用一个相对简单的CNN，输入66x200的RGB图像，直接输出方向盘转角指令。该网络通过大量人类驾驶员的图像-转向角配对数据进行训练，成功学会了在简单道路环境下保持车道行驶。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_34.png)

这种方法的优势在于：
*   **减少人工特征工程**：网络自动从数据中学习相关特征。
*   **拥有海量数据**：人类驾驶员每天都在产生天然的“图像-动作”配对数据作为训练素材。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_36.png)

当然，端到端方法也面临挑战，例如对罕见“边缘情况”的泛化能力、可解释性差以及需要极高的准确率。

## 🎮 实践项目：DeepTesla

为了让大家亲身体验端到端驾驶，课程提供了 **DeepTesla** 项目。它有两种实现：
1.  **浏览器版本 (ConvNetJS)**：在浏览器中使用JavaScript进行训练和演示，方便快速上手。
2.  **TensorFlow版本**：功能更强大，可以利用GPU进行更复杂、更快速的训练。

项目提供了从装备摄像头的特斯拉车辆上采集的真实高速公路驾驶视频片段，以及同步的方向盘转角数据。你的任务是设计并训练一个CNN，使其能够根据单帧图像预测方向盘转角。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_38.png)

**网络输入**：裁剪并缩放后的前方道路图像（如66x200像素）。
**网络输出**：方向盘转角值（一个连续值，回归任务）。
**目标**：使网络预测的转角曲线尽可能接近真实驾驶员（或自动驾驶系统）的转角曲线。

![](img/a3ea2eb7da7b24a9c2c132cab561a980_40.png)

![](img/a3ea2eb7da7b24a9c2c132cab561a980_42.png)

通过调整网络结构（卷积层数、滤波器数量、全连接层大小等），你可以观察模型性能的变化，并直观地在视频回放中看到网络的驾驶效果。

## 🔧 其他驾驶子任务中的CNN

![](img/a3ea2eb7da7b24a9c2c132cab561a980_52.png)

除了端到端驾驶，CNN在自动驾驶的其他环节也发挥着重要作用：

*   **场景理解**：用于交通灯检测、车辆检测、行人检测等。课程提供了一个**交通灯分类（红/黄/绿）**的TensorFlow教程，这是一个典型的三分类问题，非常适合初学者练习。
*   **驾驶员状态监控**：使用车内摄像头，CNN可以估计驾驶员的头部姿态、视线方向，甚至检测情绪（如沮丧/满意），这对于理解驾驶员是否分心至关重要。
*   **路面状况分析**：通过分析车辆麦克风采集的音频频谱图（可视为图像），CNN可以判断路面是干燥还是湿滑。
