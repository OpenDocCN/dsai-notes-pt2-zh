## 间隙填充与回归均值 📊

![](img/4274851f6307fbf1eb585c5a01413525_8.png)

![](img/4274851f6307fbf1eb585c5a01413525_10.png)

![](img/4274851f6307fbf1eb585c5a01413525_12.png)

![](img/4274851f6307fbf1eb585c5a01413525_14.png)

上一节我们介绍了课程概述，本节中我们来看看一个关于旋律线条的有趣现象：间隙填充。这是指在旋律进行中，一个大跳（例如向上跳进）之后，旋律线倾向于朝相反方向运动以“填补”音高间隙。

然而，一项由 Von Hippel 和 Huron 进行的研究提出了一个关键问题：这种“间隙填充”效应是否可能只是**回归均值**（Regression to the Mean）这一统计现象的结果？回归均值是指，当一个变量处于极端位置时，它更可能在下一次观测中向平均值靠拢。

![](img/4274851f6307fbf1eb585c5a01413525_16.png)

![](img/4274851f6307fbf1eb585c5a01413525_18.png)

为了验证这一点，他们分析了 Palestrina 的作品。以下是他们分析的核心逻辑：

![](img/4274851f6307fbf1eb585c5a01413525_20.png)

1.  **定义音高中位数**：计算整首作品的音高中位数。
2.  **分类跳进**：根据跳进起点和终点相对于中位数的位置，将向上跳进分为四类：
    *   起点高于中位数，终点更高。
    *   起点低于中位数，终点高于中位数。
    *   起点低于中位数，终点达到中位数。
    *   起点低于中位数，终点仍低于中位数。
3.  **比较预测**：对于每一类，比较“间隙填充”（预测向下运动）和“回归均值”（预测向中位数运动）两种假设所指示的方向是否一致。

![](img/4274851f6307fbf1eb585c5a01413525_22.png)

他们的研究发现，在许多情况下，观察到的旋律运动方向与回归均值的预测相符，而非传统的间隙填充规则。例如，当一个音从中位数以下跳进到中位数以上后，它更可能向下运动（回归中位数），而不是继续向上。

为了进一步测试，研究者将 Palestrina 原曲的音符顺序打乱，创造了一个随机旋律。有趣的是，在这个随机序列中，“间隙填充”规则也完全成立。这强烈暗示，在某些音乐数据集中，间隙填充现象可能只是回归均值这一更普遍统计规律的表现。

**公式**：回归均值可以概念化地表示为，在时间 `t` 的一个极端观测值 `X_t` 之后，时间 `t+1` 的观测值 `X_{t+1}` 更可能满足：`|X_{t+1} - μ| < |X_t - μ|`，其中 `μ` 是序列的均值。

![](img/4274851f6307fbf1eb585c5a01413525_24.png)

![](img/4274851f6307fbf1eb585c5a01413525_26.png)

![](img/4274851f6307fbf1eb585c5a01413525_28.png)

后续使用更大数据集（包括中国民歌、欧洲民歌等）的研究，部分支持了 Von Hippel 和 Huron 的结论。但值得注意的是，在对巴赫之前的音乐（即物种对位法所基于的时期）进行分析时，发现了更多真实“间隙填充”规则的证据。这表明，在某些特定历史时期的音乐实践中，作曲家确实有意运用了这一规则。

## 超越西方：跨文化计算音乐理论的挑战 🌍

上一节我们讨论了基于西方音乐传统的分析，本节中我们来看看将计算音乐学方法应用于全球多元音乐文化时所面临的挑战。

目前大多数计算音乐理论工具（如 **music21**、**Humdrum**）主要围绕**西方记谱法**和音乐概念构建。这带来了一个“先有鸡还是先有蛋”的问题：我们因为缺乏非西方音乐的数据集和分析工具而较少研究它们，同时又因为缺乏研究需求而无人去构建这些工具和数据集。

核心挑战在于，许多音乐传统并非以乐谱为中心，而是口传心授。即使对于有记谱传统的文化（如印度古典音乐），其记谱系统和音乐概念（如音程、节奏周期、调式）也与西方体系有根本不同。

![](img/4274851f6307fbf1eb585c5a01413525_30.png)

![](img/4274851f6307fbf1eb585c5a01413525_32.png)

![](img/4274851f6307fbf1eb585c5a01413525_34.png)

一个理想的计算音乐理论框架应该能够：
*   将不同文化中的基本音乐元素（如印度音乐中的 *shruti*）视为**原生对象**，而非西方音高的变体。
*   允许用户为分析对象应用不同的**音乐语境**。例如，在分析13世纪的法国音乐时，大三度不应被自动视为协和音程；在分析冲绳 pentatonic 音阶时，需要重新定义什么是“级进”和“跳进”。

![](img/4274851f6307fbf1eb585c5a01413525_36.png)

![](img/4274851f6307fbf1eb585c5a01413525_37.png)

![](img/4274851f6307fbf1eb585c5a01413525_39.png)

**代码概念**：理想的系统可能允许这样的操作：
```python
# 伪代码示例
interval = Interval(‘M3’)
interval.applyContext(‘13thCenturyFrance’)
print(interval.isConsonance()) # 可能返回 False
interval.applyContext(‘Carnatic’)
print(interval.nameInShruti()) # 返回对应的 shruti 数量
```

尽管像 **Tons** 这样的项目尝试为世界音乐分析提供工具包，但构建真正通用、尊重不同音乐体系内在逻辑的计算工具，仍然是该领域一个重要的未来方向。

## 光学音乐识别中的音乐理论语义 🖼️➡️🎼

![](img/4274851f6307fbf1eb585c5a01413525_52.png)

上一节我们探讨了理论框架的扩展，本节中我们来看看一个非常实际的应用：利用音乐理论知识提升光学音乐识别（OMR）的准确性。OMR 旨在将乐谱图像转换为计算机可理解的符号格式（如 MusicXML）。

![](img/4274851f6307fbf1eb585c5a01413525_54.png)

![](img/4274851f6307fbf1eb585c5a01413525_55.png)

![](img/4274851f6307fbf1eb585c5a01413525_57.png)

当前 OMR 软件的准确率仍有提升空间。常见的错误包括：
*   音符或休止符的误识别或遗漏。
*   调号、临时记号错误。
*   谱号或乐器标识错误。
*   **节奏时值不匹配**，导致小节内总时值与拍号不符。

![](img/4274851f6307fbf1eb585c5a01413525_59.png)

![](img/4274851f6307fbf1eb585c5a01413525_60.png)

![](img/4274851f6307fbf1eb585c5a01413525_62.png)

节奏错误是一个特别好的切入点，因为拍号提供了一个天然的“校验和”：一个小节内所有音符的时值之和必须等于拍号规定的时值。如果OMR结果不满足这一点，那么几乎可以肯定存在识别错误。

![](img/4274851f6307fbf1eb585c5a01413525_64.png)

![](img/4274851f6307fbf1eb585c5a01413525_66.png)

我们可以利用音乐作品的**内在重复性和上下文信息**来纠正这些节奏错误。具体方法涉及两个核心概率：

![](img/4274851f6307fbf1eb585c5a01413525_68.png)

1.  **位置概率**：基于乐曲中其他部分或相邻小节的节奏型，来推测当前错误小节的正确节奏。
    *   **纵向（同一时间点）**：查看其他声部在同一小节的节奏型。
    *   **横向（前后小节）**：查看同一声部在前后的节奏型。由于乐曲通常小节数多于声部数，横向信息往往更丰富。
2.  **错误转换概率**：基于已知的 OMR 软件常见错误类型统计（例如，附点被识别为单独音符的概率），估算将当前识别结果转换为另一种可能节奏模式所需的修改概率。

![](img/4274851f6307fbf1eb585c5a01413525_70.png)

通过计算每个可能修正方案的联合概率（位置概率 × 转换概率），并选择概率最高的方案，可以有效纠正大量节奏错误。实验表明，这种方法能在测试曲目上平均减少约19%的节奏错误。

**工作流程总结**：
```
原始乐谱PDF → OMR软件 → 可能包含错误的MusicXML → 音乐理论纠正模块（应用概率模型） → 修正后的MusicXML
```

当前 OMR 研究的一个问题是，常用的测试数据集往往包含大量不寻常、缺乏重复性的现代乐谱片段。这可能导致开发的模型优化方向偏离了处理常见、规律性强的古典或流行音乐的需求。未来的挑战在于如何将音乐理论的语义知识更好地整合到基于深度学习的现代 OMR 模型中，而不仅仅是将其视为一个纯粹的图像识别问题。
