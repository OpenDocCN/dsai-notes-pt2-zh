![](img/855a434c8ea4bcf4ee332bece4a382f4_5.png)

## 两种时间序列建模方法 📈

上一节我们介绍了循环神经网络，本节中我们来看看时间序列建模的两种不同范式。

![](img/855a434c8ea4bcf4ee332bece4a382f4_7.png)

### 潜在状态方法（如RNN）

在时间序列预测任务中，我们有一系列输入 `x1, x2, x3, ...` 和对应的输出 `y1, y2, y3, ...`。我们的目标是基于到当前时刻为止的所有输入，预测输出序列 `y1:T`。形式化地，我们希望找到一个函数 `F`，使得 `y1:T = F(x1:T)`，并且 `yt` 只能依赖于 `x1:t`。

![](img/855a434c8ea4bcf4ee332bece4a382f4_9.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_11.png)

循环神经网络采用了一种**潜在状态**的方法。其核心思想是，网络维护一个隐藏状态 `Ht`，该状态总结了从序列开始到当前时刻 `t` 的所有历史信息。每个时间步，网络根据当前输入 `xt` 和上一个隐藏状态 `Ht-1` 来更新当前隐藏状态 `Ht`，并基于 `Ht` 预测输出 `yt`。

![](img/855a434c8ea4bcf4ee332bece4a382f4_13.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_15.png)

**公式：**
```
Ht = f(Ht-1, xt)
yt = g(Ht)
```

这种方法的优势在于，理论上它可以捕获无限长的历史依赖。然而，其挑战也源于此：信息需要通过很长的计算链（`H1 -> H2 -> ... -> Ht`）进行传递，这容易导致梯度消失或爆炸问题，使得网络难以学习并有效地将遥远过去的信息整合到当前状态中。

### 直接预测方法

![](img/855a434c8ea4bcf4ee332bece4a382f4_17.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_19.png)

与潜在状态方法不同，**直接预测方法**旨在设计一个架构，能够直接将任意长度的输入序列 `x1:t` 映射到固定大小的输出 `yt`。在这种范式中，没有维护一个总结历史的单一状态，而是让函数 `F` 本身具备处理变长序列并直接产生预测的能力。

![](img/855a434c8ea4bcf4ee332bece4a382f4_21.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_23.png)

这种方法的优势在于，从历史输入到未来预测的路径可以设计得更直接、更高效。例如，预测 `y4` 时，从所有输入 `x1:4` 到 `y4` 的计算路径长度可以是对称的，这有助于更好地捕获时间信息。

其缺点则是缺乏一个紧凑的状态表示。每次进行预测时，都需要将整个历史序列 `x1:t` 作为输入，无法像RNN那样仅通过更新一个隐藏状态来增量地处理新数据。

## 从卷积网络到自注意力 🔄

![](img/855a434c8ea4bcf4ee332bece4a382f4_25.png)

在深入Transformer之前，我们先看一个直接预测方法的经典例子：**时序卷积网络**。

![](img/855a434c8ea4bcf4ee332bece4a382f4_27.png)

一种实现直接预测的直观方法是使用卷积神经网络，并施加**因果卷积**约束。这意味着网络中的连接只能从过去的时间步指向当前或未来的时间步，而不能反向。这样，`t` 时刻第 `i+1` 层的隐藏状态 `H_{t}^{i+1}` 只能依赖于第 `i` 层中 `t` 及 `t` 之前时刻的隐藏状态。

![](img/855a434c8ea4bcf4ee332bece4a382f4_29.png)

**挑战：感受野有限**
这种架构的一个核心挑战是**感受野**受限于卷积核大小和网络深度。简单的因果卷积中，每个状态只依赖于当前和前一时刻的输入，因此感受野大小大致等于网络深度。为了捕获长程依赖，需要增大卷积核、使用空洞卷积或增加网络深度，但这些方法会分别带来参数量增加、跳过部分输入信息或训练困难等问题。

![](img/855a434c8ea4bcf4ee332bece4a382f4_31.png)

这引出了我们需要的一种新方法，它能在单层内混合整个序列的信息，且不随序列长度增加参数量。这就是**自注意力**机制，也是**Transformer**架构的基础。

## 自注意力机制详解 ⚙️

自注意力是注意力机制的一种特定形式，而注意力机制在深度学习中的一般定义是：对网络中的各个组件或状态进行加权，然后将它们组合在一起的任何机制。

![](img/855a434c8ea4bcf4ee332bece4a382f4_33.png)

### 基本自注意力操作

自注意力块通常被视为一个接收三个矩阵参数 `K`（键）、`Q`（查询）、`V`（值）的函数。这三个矩阵形状均为 `T x D`（`T` 为序列长度，`D` 为特征维度）。在实践中，`K, Q, V` 通常是通过对同一输入 `X`（`T x D` 矩阵）进行不同的线性变换得到的：

![](img/855a434c8ea4bcf4ee332bece4a382f4_35.png)

**公式：**
```
K = X * W_K
Q = X * W_Q
V = X * W_V
```

![](img/855a434c8ea4bcf4ee332bece4a382f4_37.png)

自注意力操作定义为：

![](img/855a434c8ea4bcf4ee332bece4a382f4_39.png)

**公式：**
```
SelfAttention(K, Q, V) = softmax( (K * Q^T) / sqrt(D) ) * V
```

![](img/855a434c8ea4bcf4ee332bece4a382f4_41.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_43.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_45.png)

让我们分解这个公式：
1.  `K * Q^T` 得到一个 `T x T` 的矩阵，其 `(i, j)` 元素是 `K_i` 和 `Q_j` 的点积，可以理解为 `i` 位置和 `j` 位置的相似度。
2.  对该矩阵的每一行应用 `softmax` 操作，使其每一行成为一组和为1的权重。这得到了注意力权重矩阵 `A`。
3.  将权重矩阵 `A` 与 `V` 相乘。输出矩阵的第 `i` 行，是 `V` 的所有行以 `A` 的第 `i` 行权重进行加权组合的结果。

![](img/855a434c8ea4bcf4ee332bece4a382f4_47.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_49.png)

因此，自注意力的输出中，每个位置都包含了序列中所有位置信息的混合。

### 自注意力的关键特性

1.  **排列等变性**：如果对输入序列（即 `K, Q, V` 的行）施加相同的排列，输出也会被施加相同的排列。这意味着其本身不包含顺序信息。
2.  **全序列混合**：单层自注意力即可让序列中任意两个位置直接交互，提供了完整的“感受野”。
3.  **参数量不随序列长度增长**：作为操作的 `K, Q, V` 投影有参数，但自注意力混合整个序列的过程本身没有额外参数。这与需要增大卷积核来扩大感受野的CNN形成对比。
4.  **计算复杂度**：朴素自注意力需要计算并存储 `T x T` 的矩阵，其时间和空间复杂度为 `O(T^2 * D)`。对于超长序列，这是一个挑战。

![](img/855a434c8ea4bcf4ee332bece4a382f4_51.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_53.png)

## Transformer架构 🏗️

![](img/855a434c8ea4bcf4ee332bece4a382f4_55.png)

Transformer是由多个**Transformer块**堆叠而成的架构。每个块将输入序列 `Z_i`（`T x D`）映射到输出序列 `Z_{i+1}`。

![](img/855a434c8ea4bcf4ee332bece4a382f4_57.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_59.png)

一个标准的Transformer块包含以下步骤：

![](img/855a434c8ea4bcf4ee332bece4a382f4_61.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_63.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_64.png)

**公式与步骤：**
1.  **自注意力层**：`Z_tilde1 = SelfAttention(Z_i * W_K, Z_i * W_Q, Z_i * W_V)`
2.  **残差连接与层归一化**：`Z_tilde2 = LayerNorm(Z_i + Z_tilde1)`
3.  **前馈网络**：对 `Z_tilde2` 的每一行独立应用一个两层前馈网络（例如：`FFN(x) = ReLU(x*W1) * W2`）。
4.  **再次残差连接与层归一化**：`Z_{i+1} = LayerNorm(Z_tilde2 + FFN(Z_tilde2))`

高层面理解：Transformer块先用自注意力混合不同时间步的信息，然后对每个时间步独立应用一个前馈网络进行非线性变换，并用层归一化和残差连接来稳定训练。

### 使Transformer适用于时间序列

基本的Transformer块有两个问题使其不适合时间序列预测：
1.  **非因果性**：输出 `y_t` 可能依赖于未来的输入 `x_{>t}`。
2.  **无顺序信息**：因其排列等变性，模型无法感知输入的顺序。

![](img/855a434c8ea4bcf4ee332bece4a382f4_66.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_68.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_69.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_70.png)

以下是解决方案：

**1. 掩码自注意力（实现因果性）**
为了确保 `t` 时刻的输出只依赖于 `1:t` 时刻的输入，我们在计算注意力权重时，将 `softmax` 输入矩阵中未来位置（`j > i`）的值设置为负无穷大（在实际操作中是一个很大的负数）。这样，经过 `softmax` 后，这些位置的权重就为0。

![](img/855a434c8ea4bcf4ee332bece4a382f4_72.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_74.png)

**公式：**
```
MaskedSelfAttention(K, Q, V) = softmax( (K * Q^T) / sqrt(D) + M ) * V
```
其中掩码矩阵 `M` 是一个 `T x T` 的上三角矩阵，上三角部分为负无穷大，下三角和对角线部分为0。

![](img/855a434c8ea4bcf4ee332bece4a382f4_76.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_77.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_79.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_80.png)

**2. 位置编码（注入顺序信息）**
为了让模型知道每个元素在序列中的位置，我们在输入（或输入投影后）添加一个**位置编码**矩阵 `P`。常用的一种编码是使用不同频率的正弦和余弦函数：

**公式（对于位置 `pos` 和维度 `i`）：**
```
PE(pos, 2i) = sin(pos / 10000^(2i/D))
PE(pos, 2i+1) = cos(pos / 10000^(2i/D))
```
然后将位置编码与输入嵌入相加：`Z = InputEmbedding(X) + P`。这样，不同位置就有不同的、可区分的表示。

![](img/855a434c8ea4bcf4ee332bece4a382f4_82.png)

通过结合**掩码自注意力**和**位置编码**，Transformer既保留了全序列混合、长程依赖建模的强大能力，又满足了时间序列预测的因果性要求，并感知了序列顺序。

## Transformer的广泛应用 🌐

尽管Transformer最初是为机器翻译等序列任务设计的，但其“自注意力混合信息 + 逐位置前馈网络”的范式已成为当前深度学习多个领域的**主导架构**。

![](img/855a434c8ea4bcf4ee332bece4a382f4_84.png)

-   **计算机视觉（Vision Transformer）**：将图像分割成小块（patch），每个块视为序列的一个元素，然后应用Transformer。在大规模数据集上，其性能可超越传统的卷积网络。
-   **图神经网络（Graph Transformer）**：将图中的节点视为序列元素。可以通过掩码矩阵来编码图的连接结构（例如，只让相连的节点互相注意），或将节点间的距离等信息融入注意力计算。
-   **音频处理、蛋白质结构预测等**：Transformer在这些领域也取得了突破性成果。

![](img/855a434c8ea4bcf4ee332bece4a382f4_86.png)

![](img/855a434c8ea4bcf4ee332bece4a382f4_88.png)

其成功源于架构的简洁性和强大性：自注意力提供了一种灵活的方式来建模序列元素间（或图像块间、图节点间）的依赖关系，并能通过掩码轻松融入领域知识（如因果性、图结构）。

## 总结 📝

本节课中我们一起学习了：
1.  **时间序列建模的两种思路**：潜在状态方法（如RNN）和直接预测方法。后者避免了RNN的长程依赖学习难题。
2.  **自注意力机制**：一种通过计算序列元素间相似度权重，并对值进行加权组合来混合全序列信息的操作。它具有排列等变性、全序列感受野、参数量不随序列长度增长等特性，但计算复杂度为 `O(T^2)`。
3.  **Transformer架构**：由多个Transformer块堆叠而成，每个块包含自注意力层、前馈网络层以及残差连接和层归一化。
4.  **Transformer用于时间序列**：通过**掩码自注意力**确保预测的因果性，通过**位置编码**为输入注入顺序信息。
5.  **Transformer的普适性**：这一架构范式因其强大的表示能力，已扩展到视觉、图结构等多个领域，成为当前许多高性能模型的基础。
