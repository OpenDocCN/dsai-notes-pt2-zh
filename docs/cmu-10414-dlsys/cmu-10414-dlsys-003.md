神经网络作为机器学习算法的一部分，主要指定了三个核心要素：假设类别、损失函数和优化过程。与之前的Softmax回归相比，神经网络的主要区别在于其假设类别更为复杂。然而，损失函数（交叉熵损失）和优化过程（随机梯度下降）保持不变。因此，实现神经网络的关键在于如何高效地计算损失函数关于所有网络参数（如权重矩阵W1, W2, ...）的梯度。

首先，计算关于第二层权重 \(W_2\) 的梯度相对简单，因为它与Softmax回归中的形式类似。

![](img/d274b065e30e493185aa38c4a0cbb160_211.png)

![](img/d274b065e30e493185aa38c4a0cbb160_213.png)

以下是计算步骤：
1.  计算网络的最终输出经过Softmax后的结果 \(S\)：
    \[
    S = \text{softmax}(\sigma(x W_1) W_2)
    \]
2.  计算损失函数关于其输入的导数，这与Softmax回归相同，为 \(S - I_Y\)，其中 \(I_Y\) 是标签的独热编码。
3.  应用链式法则，乘以 \(\sigma(x W_1)\) 的项。
4.  通过维度匹配，得到最终的矩阵形式梯度：
    \[
    \nabla_{W_2} L = (\sigma(x W_1))^T (S - I_Y)
    \]

以下是推导的核心步骤：
1.  应用链式法则，将梯度分解为多个部分的乘积。
2.  识别出每个部分：
    *   损失关于网络输出的梯度：\(S - I_Y\)
    *   输出关于中间激活值的梯度：\(W_2\)
    *   激活函数关于其输入的梯度：\(\sigma'(x W_1)\)，其中 \(\sigma'\) 是激活函数的导数。
    *   中间激活值关于 \(W_1\) 的梯度：\(x\)
3.  将所有项按正确顺序组合，并通过维度分析确定矩阵乘法和逐元素乘法的正确形式。
4.  最终得到关于 \(W_1\) 的梯度公式：
    \[
    \nabla_{W_1} L = x^T \left[ (S - I_Y) W_2^T \odot \sigma'(x W_1) \right]
    \]
    其中 \(\odot\) 表示逐元素乘法（Hadamard积）。

**2. 反向传播：**
从输出层开始，反向计算并传播梯度。
*   **初始化：** 计算损失关于最终输出的梯度 \(G_{L+1}\)。对于交叉熵损失，\(G_{L+1} = S - I_Y\)。
*   **迭代计算：** 从 \(i = L\) 到 \(1\)，递归计算损失关于第 \(i\) 层激活值的梯度 \(G_i\)：
    \[
    G_i = (G_{i+1} \odot \sigma_i'(Z_i W_i)) W_i^T
    \]
*   **计算参数梯度：** 在反向迭代的过程中，同时计算损失关于参数 \(W_i\) 的梯度：
    \[
    \nabla_{W_i} L = Z_i^T (G_{i+1} \odot \sigma_i'(Z_i W_i))
    \]

观察参数梯度的计算公式 \(\nabla_{W_i} L = Z_i^T (G_{i+1} \odot \sigma_i'(Z_i W_i))\)，可以发现一个通用模式：每一层都需要完成两项任务：
1.  执行其前向计算（计算 \(Z_{i+1}\)）。
2.  接收来自后一层的梯度 \(G_{i+1}\)，并将其与**该层操作自身的局部导数**相乘。

这个“局部梯度”与“传入梯度”相乘的操作，在数学上称为**向量-雅可比积**。这是使反向传播变得模块化和通用的关键。每一层只需要知道如何计算自己的前向传播和向量-雅可比积，就可以无缝地嵌入到整个反向传播框架中。
