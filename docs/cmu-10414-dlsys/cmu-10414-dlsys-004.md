![](img/1602e205bab7500955e1fd97cc7097a3_5.png)

## 为什么需要自动微分？

![](img/1602e205bab7500955e1fd97cc7097a3_6.png)

首先，让我们回顾一下自动微分在机器学习工作流中的位置。机器学习通常包含三个基本要素：

![](img/1602e205bab7500955e1fd97cc7097a3_8.png)

1.  **假设类**：对于给定的输入 `X`，如何得到预测。例如多层感知机。
2.  **损失函数**：衡量在给定参数 `θ` 下，预测结果的好坏。
3.  **优化方法**：寻找能最小化损失函数的参数 `θ`。

在深度学习中，最常用的优化算法是**随机梯度下降**。其核心更新规则是：
`θ = θ - α * ∇L(θ)`
其中，`α` 是学习率，`∇L(θ)` 是损失函数 `L` 关于参数 `θ` 的梯度。

因此，计算梯度 `∇L(θ)` 是几乎所有机器学习算法的核心问题。手动推导复杂模型的梯度几乎是不可能的，这就是我们需要自动微分的原因。

## 微分方法概览

有多种方法可以计算梯度，让我们逐一了解。

### 数值微分

![](img/1602e205bab7500955e1fd97cc7097a3_10.png)

根据微积分定义，函数 `f(θ)` 关于参数 `θ_i` 的偏导数可以定义为：
`∂f/∂θ_i = lim_{ε→0} (f(θ + ε*e_i) - f(θ)) / ε`
其中 `e_i` 是第 `i` 个分量为1的单位向量。

在实践中，我们使用更精确的中心差分公式来近似：
`∂f/∂θ_i ≈ (f(θ + ε*e_i) - f(θ - ε*e_i)) / (2ε)`

![](img/1602e205bab7500955e1fd97cc7097a3_12.png)

**数值微分的优缺点：**
*   **优点**：实现简单，易于验证正确性。
*   **缺点**：
    1.  存在数值误差，尤其是当 `ε` 选择不当时。
    2.  计算成本高。对于 `n` 个参数，需要 `2n` 次函数评估。

尽管数值微分不常用于实际训练，但它是一个非常重要的工具，用于**梯度检查**，即验证我们实现的自动微分算法是否正确。

以下是进行梯度检查的通用方法：随机选取一个扰动向量 `v`，检查以下近似是否成立：
`v^T * ∇f(θ) ≈ (f(θ + ε*v) - f(θ - ε*v)) / (2ε)`

### 符号微分

![](img/1602e205bab7500955e1fd97cc7097a3_14.png)

符号微分即手动或利用计算机代数系统，根据微积分规则（如链式法则、乘积法则）推导出梯度函数的解析表达式。

例如，对于函数 `f(θ) = θ1 * θ2 * ... * θn`，关于 `θ_k` 的偏导数为：
`∂f/∂θ_k = Π_{i≠k} θ_i`

![](img/1602e205bab7500955e1fd97cc7097a3_16.png)

![](img/1602e205bab7500955e1fd97cc7097a3_17.png)

![](img/1602e205bab7500955e1fd97cc7097a3_19.png)

![](img/1602e205bab7500955e1fd97cc7097a3_21.png)

**符号微分的局限性：**
直接根据符号表达式计算梯度可能导致大量重复计算。对于上述例子，计算所有 `n` 个偏导数，朴素方法需要 `O(n^2)` 次乘法，而通过共享中间结果，只需 `O(n)` 次。

符号微分可以看作是自动微分的一种特例，即先进行符号推导，再进行简化优化。

![](img/1602e205bab7500955e1fd97cc7097a3_23.png)

在深入自动微分之前，我们需要一个关键工具：**计算图**。计算图是一种有向无环图，用于表示计算过程。

![](img/1602e205bab7500955e1fd97cc7097a3_35.png)

*   **节点**：代表中间计算值或输入/输出。
*   **边**：代表数据流向，表示一个节点的输出是另一个节点的输入。

计算图不仅定义了计算逻辑，也隐含了执行顺序。通过按照拓扑顺序遍历计算图并计算每个节点的值，我们可以完成整个函数求值。

## 前向模式自动微分

![](img/1602e205bab7500955e1fd97cc7097a3_45.png)

前向模式自动微分的核心思想是：在正向计算函数值的同时，计算**所有中间变量对某一个输入变量**的导数。

![](img/1602e205bab7500955e1fd97cc7097a3_47.png)

![](img/1602e205bab7500955e1fd97cc7097a3_49.png)

我们定义 `v_i_dot = ∂v_i/∂x`，其中 `x` 是我们关心的输入变量。计算过程如下：
1.  从输入节点开始，已知 `∂输入/∂x`（对于 `x` 自身为1，对于其他输入为0）。
2.  按照计算图的拓扑顺序，对于每个节点 `v_i`，利用其输入节点的 `v_dot` 和该节点的局部导数，通过链式法则计算 `v_i_dot`。
3.  遍历到输出节点时，即得到 `∂输出/∂x`。

![](img/1602e205bab7500955e1fd97cc7097a3_51.png)

![](img/1602e205bab7500955e1fd97cc7097a3_52.png)

![](img/1602e205bab7500955e1fd97cc7097a3_54.png)

![](img/1602e205bab7500955e1fd97cc7097a3_56.png)

![](img/1602e205bab7500955e1fd97cc7097a3_58.png)

![](img/1602e205bab7500955e1fd97cc7097a3_60.png)

![](img/1602e205bab7500955e1fd97cc7097a3_62.png)

**前向模式的特点：**
*   **优点**：一次正向传播可以计算**所有输出**关于**单个输入**的导数。
*   **缺点**：对于深度学习常见的场景（单输出、多输入），要计算梯度 `∇f(θ)`（包含 `n` 个分量），需要运行 `n` 次前向传播，效率低下。

![](img/1602e205bab7500955e1fd97cc7097a3_64.png)

![](img/1602e205bab7500955e1fd97cc7097a3_66.png)

![](img/1602e205bab7500955e1fd97cc7097a3_68.png)

![](img/1602e205bab7500955e1fd97cc7097a3_72.png)

![](img/1602e205bab7500955e1fd97cc7097a3_74.png)

## 反向模式自动微分

![](img/1602e205bab7500955e1fd97cc7097a3_86.png)

![](img/1602e205bab7500955e1fd97cc7097a3_88.png)

反向模式自动微分（Reverse-Mode AD）是深度学习框架的基石。它高效地解决了单输出、多输入场景的梯度计算问题。

![](img/1602e205bab7500955e1fd97cc7097a3_90.png)

![](img/1602e205bab7500955e1fd97cc7097a3_92.png)

### 核心概念：伴随变量

我们定义伴随变量 `v_bar` 为标量输出 `y` 对中间变量 `v` 的导数：
`v_bar = ∂y/∂v`

我们的目标是计算所有输入 `x_i` 的伴随变量 `x_i_bar`，它们共同构成了梯度 `∇f(θ)`。

![](img/1602e205bab7500955e1fd97cc7097a3_94.png)

### 计算过程：反向传播

计算过程从输出节点开始，反向遍历计算图：
1.  **初始化**：输出节点 `y` 的伴随变量 `y_bar = ∂y/∂y = 1`。
2.  **反向迭代**：对于每个节点 `v_i`（按逆拓扑顺序访问）：
    *   从 `v_i` 出发，找到所有以 `v_i` 作为输入的后续节点 `v_j`。
    *   对于每个这样的边 `(v_i -> v_j)`，计算其“局部梯度贡献”：`∂y/∂v_j * ∂v_j/∂v_i`。第一部分是 `v_j_bar`，第二部分是节点 `v_j` 关于输入 `v_i` 的局部偏导数。
    *   将所有从 `v_i` 出发的边的局部梯度贡献**求和**，即得到 `v_i_bar`。
3.  **得到梯度**：当反向传播进行到输入节点时，得到的伴随变量 `x_i_bar` 就是梯度 `∂y/∂x_i`。

**多路径处理**：如果一个节点 `v_i` 被多个后续节点使用，在计算 `v_i_bar` 时，需要将所有来自不同输出路径的梯度贡献累加起来，这正是链式法则在多变量情况下的体现。

![](img/1602e205bab7500955e1fd97cc7097a3_96.png)

![](img/1602e205bab7500955e1fd97cc7097a3_98.png)

![](img/1602e205bab7500955e1fd97cc7097a3_100.png)

### 实现方式：构建梯度计算图

![](img/1602e205bab7500955e1fd97cc7097a3_102.png)

![](img/1602e205bab7500955e1fd97cc7097a3_104.png)

![](img/1602e205bab7500955e1fd97cc7097a3_106.png)

一个优雅的实现反向模式自动微分的方法，不是直接计算具体的梯度值，而是**构建另一个计算图**来描述梯度计算本身。

![](img/1602e205bab7500955e1fd97cc7097a3_108.png)

![](img/1602e205bab7500955e1fd97cc7097a3_110.png)

**算法步骤：**
1.  首先完成原始函数的前向计算，并记录计算图。
2.  初始化一个字典，为每个计算图节点维护一个列表，用于累积来自不同输出边的“局部梯度贡献”。
3.  从输出节点开始（`output_bar = 1`，表示一个常数节点 `1`）。
4.  逆序遍历节点。对于当前节点 `v_i`：
    *   将其累积的所有局部梯度贡献相加，构建一个“加法”节点，得到 `v_i_bar` 节点。
    *   遍历 `v_i` 的所有输入节点 `v_k`。
    *   计算边 `(v_k -> v_i)` 的局部梯度贡献：`v_i_bar * ∂v_i/∂v_k`。根据 `v_i` 的运算类型，构建相应的计算节点（如乘法节点）。
    *   将这个新节点加入到输入节点 `v_k` 的累积列表中。
5.  遍历结束后，输入节点对应的 `v_bar` 节点就构成了梯度计算图。

![](img/1602e205bab7500955e1fd97cc7097a3_112.png)

![](img/1602e205bab7500955e1fd97cc7097a3_114.png)

![](img/1602e205bab7500955e1fd97cc7097a3_116.png)

![](img/1602e205bab7500955e1fd97cc7097a3_118.png)

![](img/1602e205bab7500955e1fd97cc7097a3_120.png)

![](img/1602e205bab7500955e1fd97cc7097a3_121.png)

**这种方法的优势：**
1.  **高阶导数**：梯度计算图本身也是一个计算图。我们可以对其再次应用自动微分，轻松获得二阶导数（Hessian）或更高阶导数。
2.  **优化便利**：框架可以对生成的梯度计算图进行优化（如算子融合、常量折叠），可能得到比原始计算图逐层对应更高效的梯度计算代码。
3.  **泛化能力**：该算法不依赖于具体的输入值。一旦构建好梯度计算图，它可以像普通函数一样对不同的输入参数进行求值和梯度计算。

### 反向模式自动微分 vs. 误差反向传播

在之前的课程中，我们学习了误差反向传播算法。它与反向模式自动微分密切相关：
*   **误差反向传播**：通常指在特定神经网络结构（如全连接层）上，手动推导并实现梯度计算公式。它是在**原始计算图上“就地”** 进行反向计算，不显式构建新图。
*   **反向模式自动微分**：一个通用的、与模型无关的算法。通过构建**新的梯度计算图**来实现。现代框架（如PyTorch、TensorFlow）主要采用这种方式，因为它更通用、更灵活，且易于扩展和优化。

![](img/1602e205bab7500955e1fd97cc7097a3_123.png)

可以说，误差反向传播是反向模式自动微分在神经网络领域的一个特例和应用。

## 扩展到张量与复杂数据结构

![](img/1602e205bab7500955e1fd97cc7097a3_125.png)

![](img/1602e205bab7500955e1fd97cc7097a3_127.png)

以上例子使用标量进行说明，但深度学习涉及的是张量（多维数组）。将反向模式自动微分扩展到张量是直接的。

![](img/1602e205bab7500955e1fd97cc7097a3_129.png)

![](img/1602e205bab7500955e1fd97cc7097a3_131.png)

**核心**：重新定义伴随变量。对于张量 `Z`，其伴随变量 `Z_bar` 是一个与 `Z` 形状相同的张量，其中每个元素 `Z_bar[i]` 是输出 `y` 对 `Z[i]` 的偏导数 `∂y/∂Z[i]`。

局部梯度传播规则也相应变为张量运算。例如，对于矩阵乘法 `Z = X @ W`：
*   前向计算：`Z_ij = Σ_k X_ik * W_kj`
*   反向传播（梯度）：`X_bar = Z_bar @ W.T` 和 `W_bar = X.T @ Z_bar`

![](img/1602e205bab7500955e1fd97cc7097a3_133.png)

这和我们手动推导的反向传播公式一致。

**可微编程**：自动微分的思想可以进一步推广到更复杂的数据结构，如字典、元组等。只要能为该数据结构定义其“伴随变量”的形态以及其操作的反向传播规则，就可以实现对该结构包含的张量进行自动微分。这开启了**可微编程**的大门。

![](img/1602e205bab7500955e1fd97cc7097a3_135.png)

![](img/1602e205bab7500955e1fd97cc7097a3_137.png)

## 总结

本节课中，我们一起学习了自动微分的关键知识。我们首先了解了梯度计算在机器学习中的核心地位。然后，我们探讨了数值微分和符号微分这两种基础方法及其局限性。接着，我们引入了计算图这一重要工具。

课程的重点是深入讲解了**反向模式自动微分**的原理：
*   它通过定义和反向传播**伴随变量**，高效计算标量输出对多输入的梯度。
*   一种强大的实现方式是**构建显式的梯度计算图**，这带来了计算高阶导数和进行底层优化的便利。
*   该方法可以自然地扩展到张量运算，是现代深度学习框架的基石。
