## 基础注意力机制实现

![](img/1254f50969ad49569342440a51acfa2b_10.png)

上一节我们介绍了Transformer的宏观概念，本节中我们来看看其核心组件——自注意力机制的具体实现。

![](img/1254f50969ad49569342440a51acfa2b_12.png)

自注意力层的输出 **y** 由以下公式计算：

**y = softmax( (K Q^T) / √d ) V**

![](img/1254f50969ad49569342440a51acfa2b_14.png)

![](img/1254f50969ad49569342440a51acfa2b_16.png)

然而，更常见的实现方式（例如PyTorch采用的方式）是将生成K、Q、V的线性变换权重（WK, WQ, WV）以及一个额外的输出投影权重（W_out）都作为自注意力层本身的参数。因此，我们实际实现的公式更接近于：

![](img/1254f50969ad49569342440a51acfa2b_18.png)

![](img/1254f50969ad49569342440a51acfa2b_19.png)

![](img/1254f50969ad49569342440a51acfa2b_21.png)

**y = softmax( ( (X W_K) (X W_Q)^T ) / √d ) (X W_V) W_out**

![](img/1254f50969ad49569342440a51acfa2b_23.png)

![](img/1254f50969ad49569342440a51acfa2b_25.png)

以下是实现步骤：

![](img/1254f50969ad49569342440a51acfa2b_27.png)

![](img/1254f50969ad49569342440a51acfa2b_28.png)

![](img/1254f50969ad49569342440a51acfa2b_30.png)

首先，我们需要一个softmax函数。我们将实现在最后一个维度上进行softmax。

![](img/1254f50969ad49569342440a51acfa2b_31.png)

![](img/1254f50969ad49569342440a51acfa2b_33.png)

![](img/1254f50969ad49569342440a51acfa2b_35.png)

![](img/1254f50969ad49569342440a51acfa2b_37.png)

![](img/1254f50969ad49569342440a51acfa2b_38.png)

```python
def softmax(z):
    exp_z = np.exp(z - np.max(z, axis=-1, keepdims=True))
    return exp_z / np.sum(exp_z, axis=-1, keepdims=True)
```

![](img/1254f50969ad49569342440a51acfa2b_40.png)

接下来，我们实现自注意力层。为了提高计算效率，我们不会分别计算三个矩阵乘法（X W_K, X W_Q, X W_V），而是将WK、WQ、WV拼接成一个大的权重矩阵W_KQV，通过一次矩阵乘法得到拼接后的结果，然后再进行分割。

![](img/1254f50969ad49569342440a51acfa2b_42.png)

![](img/1254f50969ad49569342440a51acfa2b_43.png)

![](img/1254f50969ad49569342440a51acfa2b_44.png)

```python
def self_attention(x, mask, w_kqv, w_out):
    # x: 输入序列 (T, d) 或 (B, T, d)
    # 一次矩阵乘法得到 K, Q, V 的拼接
    kqv = x @ w_kqv
    # 将结果在最后一个维度上分割成三份
    d = x.shape[-1]
    k, q, v = np.split(kqv, 3, axis=-1)
    # 计算注意力分数
    attn_scores = (k @ q.swapaxes(-1, -2)) / np.sqrt(d)
    # 应用掩码（例如，用于解码器的因果掩码）
    if mask is not None:
        attn_scores = attn_scores + mask
    # 应用softmax得到注意力权重
    attn_weights = softmax(attn_scores)
    # 计算加权和并应用输出投影
    output = attn_weights @ v
    output = output @ w_out
    return output, attn_weights
```

## 处理批量数据

![](img/1254f50969ad49569342440a51acfa2b_54.png)

![](img/1254f50969ad49569342440a51acfa2b_55.png)

![](img/1254f50969ad49569342440a51acfa2b_57.png)

![](img/1254f50969ad49569342440a51acfa2b_59.png)

上一节我们实现了单样本的自注意力，本节中我们来看看如何高效地处理批量数据。

对于RNN，我们通常使用 `(T, B, d)` 的维度顺序，以便按时间步顺序处理。但对于Transformer，其核心是矩阵乘法，为了保持内存连续性并获得最佳性能，我们应采用 `(B, T, d)` 的格式（批量维度在前）。PyTorch的 `batch_first=True` 参数就是为此目的。

![](img/1254f50969ad49569342440a51acfa2b_61.png)

实现批处理版本时，关键点在于我们需要进行**批量矩阵乘法（Batch Matrix Multiplication）**。这与之前网络层（如全连接层、卷积层）的批处理不同。在那些层中，批处理可以通过将输入重塑为一个大矩阵，进行单次矩阵乘法，然后再重塑回来实现。但在自注意力中，我们需要为批次中的每个样本独立计算 `K_i @ Q_i^T`，这无法通过单次矩阵乘法完成。

![](img/1254f50969ad49569342440a51acfa2b_63.png)

![](img/1254f50969ad49569342440a51acfa2b_64.png)

![](img/1254f50969ad49569342440a51acfa2b_66.png)

![](img/1254f50969ad49569342440a51acfa2b_67.png)

![](img/1254f50969ad49569342440a51acfa2b_68.png)

![](img/1254f50969ad49569342440a51acfa2b_69.png)

幸运的是，NumPy的 `@` 运算符和 `np.matmul` 已经支持批量矩阵乘法。对于形状为 `(B, T, d)` 的输入，我们之前的 `self_attention` 函数代码无需修改即可工作，因为所有操作都是基于最后两个维度进行的，批次维度 `B` 会被自动广播处理。

## 实现多头注意力

![](img/1254f50969ad49569342440a51acfa2b_87.png)

单一的注意力头可能限制了模型的表达能力。多头注意力机制将输入在特征维度上分割成多个“头”，让每个头独立学习关注输入的不同方面，最后将结果拼接起来。

假设有 `h` 个头，我们将输入维度 `d` 分割为 `h` 份，每份大小为 `d_h = d / h`。对每个头 `i`，我们计算：

![](img/1254f50969ad49569342440a51acfa2b_89.png)

**y_i = softmax( (K_i Q_i^T) / √d_h ) V_i**

![](img/1254f50969ad49569342440a51acfa2b_91.png)

最终输出是所有头输出的拼接：**y = concat(y_1, ..., y_h)**，然后再通过 `W_out` 进行投影。

![](img/1254f50969ad49569342440a51acfa2b_93.png)

以下是实现步骤：

我们需要调整输入张量的形状，以分离出“头”的维度，然后进行批量矩阵乘法，最后再合并回来。

```python
def multi_head_attention(x, mask, w_kqv, w_out, num_heads):
    B, T, d = x.shape
    d_head = d // num_heads
    # 1. 计算 K, Q, V 的拼接并分割
    kqv = x @ w_kqv  # (B, T, 3*d)
    k, q, v = np.split(kqv, 3, axis=-1)  # 每个都是 (B, T, d)
    # 2. 重塑以分离出头维度: (B, T, num_heads, d_head)
    k = k.reshape(B, T, num_heads, d_head)
    q = q.reshape(B, T, num_heads, d_head)
    v = v.reshape(B, T, num_heads, d_head)
    # 3. 交换维度以便于矩阵乘法: (B, num_heads, T, d_head)
    k = k.swapaxes(1, 2)
    q = q.swapaxes(1, 2)
    v = v.swapaxes(1, 2)
    # 4. 计算缩放点积注意力 (批量矩阵乘法)
    attn_scores = (k @ q.swapaxes(-1, -2)) / np.sqrt(d_head)  # (B, num_heads, T, T)
    if mask is not None:
        attn_scores = attn_scores + mask
    attn_weights = softmax(attn_scores)
    # 5. 应用注意力到 V
    output = attn_weights @ v  # (B, num_heads, T, d_head)
    # 6. 合并头: (B, T, d)
    output = output.swapaxes(1, 2).reshape(B, T, d)
    # 7. 输出投影
    output = output @ w_out
    return output, attn_weights
```

## 构建完整的Transformer块

![](img/1254f50969ad49569342440a51acfa2b_113.png)

![](img/1254f50969ad49569342440a51acfa2b_114.png)

现在，我们将多头注意力机制整合到一个完整的Transformer编码器层中。一个标准的Transformer编码器层包含以下组件：
1.  多头自注意力层
2.  残差连接和层归一化
3.  前馈网络（两个线性层加激活函数）
4.  另一个残差连接和层归一化

![](img/1254f50969ad49569342440a51acfa2b_118.png)

![](img/1254f50969ad49569342440a51acfa2b_119.png)

首先，我们实现层归一化和ReLU激活函数。

```python
def layer_norm(z, eps=1e-5):
    mean = z.mean(axis=-1, keepdims=True)
    var = z.var(axis=-1, keepdims=True)
    return (z - mean) / np.sqrt(var + eps)

![](img/1254f50969ad49569342440a51acfa2b_121.png)

def relu(z):
    return np.maximum(z, 0)
```

![](img/1254f50969ad49569342440a51acfa2b_123.png)

现在，我们可以组装Transformer块。

![](img/1254f50969ad49569342440a51acfa2b_125.png)

![](img/1254f50969ad49569342440a51acfa2b_127.png)

```python
def transformer_block(x, mask, num_heads, w_kqv, w_out, w_ff1, w_ff2, eps=1e-5):
    d = x.shape[-1]
    # 1. 多头自注意力 + 残差 & 层归一化
    attn_output, attn_weights = multi_head_attention(x, mask, w_kqv, w_out, num_heads)
    z = layer_norm(x + attn_output, eps)
    # 2. 前馈网络 + 残差 & 层归一化
    # 前馈网络: Linear -> ReLU -> Linear
    ff_output = relu(z @ w_ff1) @ w_ff2
    output = layer_norm(z + ff_output, eps)
    return output, attn_weights
```

关键要点包括：
*   自注意力的核心是 **softmax( (K Q^T) / √d ) V** 计算。
*   批量处理时，应采用 `(B, T, d)` 的维度顺序，并利用批量矩阵乘法。
*   多头注意力通过将特征分割到多个头中来增加模型的表达能力。
*   Transformer块通过残差连接和层归一化来稳定深层网络的训练。
*   整个Transformer的底层操作主要是大型矩阵乘法，这使得它在GPU上非常高效。
