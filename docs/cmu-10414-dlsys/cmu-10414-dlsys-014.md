## 存储顺序 📦

上一节我们介绍了卷积的数学原理和自动微分实现。本节中，我们来看看如何在内存中存储卷积运算所涉及的多维张量。

对于输入图像，我们通常将其存储为四维张量。我们采用 **NHWC** 格式：
*   **N**: 批次大小
*   **H**: 图像高度
*   **W**: 图像宽度
*   **C**: 输入通道数

![](img/0932df05d90d8e157333d6981e19c846_10.png)

![](img/0932df05d90d8e157333d6981e19c846_12.png)

这种格式将通道维度放在最后，对于后续转换为矩阵乘法非常自然。

对于卷积核权重，我们将其存储为四维张量，顺序为：
*   卷积核高度
*   卷积核宽度
*   输入通道数
*   输出通道数

![](img/0932df05d90d8e157333d6981e19c846_14.png)

这与PyTorch的默认存储顺序不同，但我们认为NHWC格式在实现卷积运算时更为合理。

```python
def conv_reference(Z, weight):
    # 将输入从NHWC转换为PyTorch的NCHW格式
    Z_torch = torch.tensor(Z).permute(0, 3, 1, 2)
    # 将权重从我们的格式转换为PyTorch的格式 (输出通道, 输入通道, 高, 宽)
    W_torch = torch.tensor(weight).permute(3, 2, 0, 1)
    # 执行卷积运算（无填充）
    out = F.conv2d(Z_torch, W_torch)
    # 将输出转换回NHWC格式并返回NumPy数组
    return out.permute(0, 2, 3, 1).contiguous().numpy()
```

```python
def conv_naive(Z, weight):
    N, H, W, C_in = Z.shape
    K, _, _, C_out = weight.shape
    out = np.zeros((N, H-K+1, W-K+1, C_out))

    for n in range(N):
        for c_in in range(C_in):
            for c_out in range(C_out):
                for y in range(H-K+1):
                    for x in range(W-K+1):
                        for i in range(K):
                            for j in range(K):
                                out[n, y, x, c_out] += Z[n, y+i, x+j, c_in] * weight[i, j, c_in, c_out]
    return out
```

## 方法二：基于矩阵乘法的实现 ➕

我们可以利用卷积运算的本质是局部矩阵乘法这一事实来优化。考虑一个极端情况：如果卷积核是1x1，那么卷积就退化为一个标准的矩阵乘法。

![](img/0932df05d90d8e157333d6981e19c846_102.png)

![](img/0932df05d90d8e157333d6981e19c846_103.png)

对于1x1卷积核，实现非常简单：

![](img/0932df05d90d8e157333d6981e19c846_105.png)

![](img/0932df05d90d8e157333d6981e19c846_107.png)

```python
# 假设 weight_1x1 是一个形状为 (1, 1, C_in, C_out) 的张量
# 其 [0,0] 位置是一个 C_in x C_out 的矩阵
out = Z @ weight_1x1[0,0]
# 或者显式地进行重塑和矩阵乘法
out_matrix = Z.reshape(-1, C_in) @ weight_1x1[0,0]
out = out_matrix.reshape(N, H, W, C_out)
```

```python
def conv_matrix_mul(Z, weight):
    N, H, W, C_in = Z.shape
    K, _, _, C_out = weight.shape
    out = np.zeros((N, H-K+1, W-K+1, C_out))
    out_H, out_W = H-K+1, W-K+1

    for i in range(K):
        for j in range(K):
            # 对输入图像的切片与权重矩阵的[i,j]位置进行矩阵乘法
            out += Z[:, i:i+out_H, j:j+out_W, :] @ weight[i, j]
    return out
```

```python
def conv_im2col(Z, weight):
    N, H, W, C_in = Z.shape
    K, _, _, C_out = weight.shape
    out_H, out_W = H-K+1, W-K+1

    # 获取输入张量的步长（以元素为单位，而非字节）
    sN, sH, sW, sC = Z.strides
    # 使用as_strided创建重叠块的视图
    # 新形状: (批次, 输出高, 输出宽, 核高, 核宽, 输入通道)
    shape = (N, out_H, out_W, K, K, C_in)
    strides = (sN, sH, sW, sH, sW, sC)
    Z_strided = as_strided(Z, shape=shape, strides=strides)
    # 重塑为二维矩阵以便矩阵乘法: (N*out_H*out_W, K*K*C_in)
    Z_col = Z_strided.reshape(-1, K*K*C_in)
    # 将权重重塑为二维矩阵: (K*K*C_in, C_out)
    W_col = weight.reshape(-1, C_out)
    # 执行矩阵乘法
    out_col = Z_col @ W_col
    # 将结果重塑回输出图像格式: (N, out_H, out_W, C_out)
    out = out_col.reshape(N, out_H, out_W, C_out)
    return out
```

1.  **朴素循环实现**：逻辑简单，但效率极低，仅适用于理解和验证。
2.  **基于矩阵乘法的实现**：将卷积分解为多个小矩阵乘法，利用现有库的优化，性能大幅提升。
3.  **Im2col与跨步技巧实现**：通过 `as_strided` 函数巧妙地将输入数据重组为一个大矩阵，从而将整个卷积运算转化为**单个矩阵乘法**。这是概念上最优雅、实践中往往最高效的方法之一，充分展示了通过操作数据视图（而非数据本身）来进行高效计算的力量。
