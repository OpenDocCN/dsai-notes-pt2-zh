## 🏗️ 为什么需要函数？

![](img/8645963f9187232ae2590185c1f36e26_23.png)

到目前为止，我们编写程序的方式是：打开一个文件，输入一系列指令（如赋值、循环、条件判断等）来解决特定问题。所有代码都写在一个文件中。

![](img/8645963f9187232ae2590185c1f36e26_25.png)

这对于我们目前遇到的小型问题来说是可以的。但是，当你开始编写大型程序时，代码会迅速变得混乱不堪。

例如，如果你在代码的某处使用了一个 `for` 循环，并发现它在另一处也很有用。将来调试时，如果你想修改这个循环的逻辑，就必须找出所有使用了类似循环的地方。

![](img/8645963f9187232ae2590185c1f36e26_27.png)

随着代码规模扩大，跟踪这些细节会变得越来越困难。这就是函数发挥作用的地方。

![](img/8645963f9187232ae2590185c1f36e26_29.png)

![](img/8645963f9187232ae2590185c1f36e26_31.png)

好的编程风格不在于添加大量代码行，而在于为程序增加更多功能。函数是实现这一目标的关键机制。

### 分解

![](img/8645963f9187232ae2590185c1f36e26_47.png)

![](img/8645963f9187232ae2590185c1f36e26_49.png)

分解是指在代码中创建结构。在编程中，为了实现分解，你需要将代码划分为更小的模块。这些模块是自包含的，可以看作是小型程序：你输入一些数据，它们执行一个小任务，然后返回结果。

![](img/8645963f9187232ae2590185c1f36e26_51.png)

![](img/8645963f9187232ae2590185c1f36e26_53.png)

这些模块可以用来拆分你的代码，并且重要的是它们是**可重用**的。你只需编写和调试一次模块，就可以在不同的地方、使用不同的输入多次调用它。

![](img/8645963f9187232ae2590185c1f36e26_55.png)

这样做的好处是保持代码的组织性和连贯性。函数就是用来实现分解、创建代码结构的。在后续关于面向对象编程的课程中，你将看到如何通过类来实现分解。

![](img/8645963f9187232ae2590185c1f36e26_57.png)

### 抽象

![](img/8645963f9187232ae2590185c1f36e26_59.png)

![](img/8645963f9187232ae2590185c1f36e26_61.png)

![](img/8645963f9187232ae2590185c1f36e26_63.png)

抽象是指隐藏细节。在编程中，一旦你编写了一段执行特定任务的代码，你不需要多次重写它。你只需编写一次，并为其创建一个**函数规范**或**文档字符串**。

文档字符串是一段文本，它告诉未来任何想使用这个函数的人（包括你自己）三件事：
1.  函数接受什么输入？
2.  函数应该做什么？
3.  函数会输出什么？

![](img/8645963f9187232ae2590185c1f36e26_65.png)

使用者无需知道函数内部是如何实现的，他们只需要知道输入、功能和输出。这就是抽象。

![](img/8645963f9187232ae2590185c1f36e26_67.png)

![](img/8645963f9187232ae2590185c1f36e26_69.png)

函数是可重用的代码块。在今天的课程中，我们将通过几个例子来学习如何编写和调用函数。

## 📝 函数的组成部分

![](img/8645963f9187232ae2590185c1f36e26_81.png)

当我们思考函数时，可以从两个角度出发：
1.  **编写者**：你需要知道如何让函数工作。
2.  **使用者**：你假设函数已被正确实现，只需使用它来完成某项任务。

![](img/8645963f9187232ae2590185c1f36e26_83.png)

函数具有以下特征：
*   **名称**：你需要为函数命名，名称应具有描述性。
*   **参数**：函数的输入，可以有零个或多个。
*   **文档字符串**：实现抽象的关键，用于说明函数的使用方法（可选但强烈推荐）。
*   **函数体**：函数的核心部分，包含执行任务的代码。
*   **返回值**：函数执行完毕后返回的结果。

## ✍️ 函数定义与调用

![](img/8645963f9187232ae2590185c1f36e26_97.png)

以下是函数定义和函数调用的示例：

![](img/8645963f9187232ae2590185c1f36e26_99.png)

![](img/8645963f9187232ae2590185c1f36e26_101.png)

```python
def is_even(i):
    """
    输入：整数 i
    功能：判断 i 是否为偶数
    输出：若 i 为偶数返回 True，否则返回 False
    """
    remainder = i % 2
    return remainder == 0

![](img/8645963f9187232ae2590185c1f36e26_103.png)

# 函数调用
result = is_even(3)
print(result)  # 输出：False
```

*   `def` 关键字告诉 Python 你要定义一个函数。
*   `is_even` 是函数名。
*   `(i)` 中的 `i` 是参数（形式参数）。
*   三引号内的文本是文档字符串。
*   缩进的代码块是函数体。
*   `return` 语句指定函数的返回值。

![](img/8645963f9187232ae2590185c1f36e26_105.png)

函数调用时，传入的值称为**实际参数**。例如，`is_even(3)` 中的 `3` 就是实际参数。

![](img/8645963f9187232ae2590185c1f36e26_107.png)

## 🔍 作用域详解

![](img/8645963f9187232ae2590185c1f36e26_109.png)

![](img/8645963f9187232ae2590185c1f36e26_111.png)

作用域是另一个表示“环境”的词。你可以将函数视为小型程序，函数的作用域与主程序的作用域是**完全分离**的。

![](img/8645963f9187232ae2590185c1f36e26_113.png)

当你进行函数调用时，Python 会：
1.  离开当前主程序环境。
2.  进入一个新的函数作用域环境，并创建一套全新的、仅存在于该环境中的变量。
3.  执行函数体内的计算。
4.  遇到 `return` 语句时，获取返回值。
5.  退出该函数作用域环境，携带返回值回到主程序。

![](img/8645963f9187232ae2590185c1f36e26_115.png)

![](img/8645963f9187232ae2590185c1f36e26_117.png)

从一个作用域进入另一个作用域时，你通过参数传递值给函数，函数通过返回值将结果传递回来。

![](img/8645963f9187232ae2590185c1f36e26_119.png)

让我们逐步分析一个程序，看看作用域背后发生了什么：

![](img/8645963f9187232ae2590185c1f36e26_121.png)

![](img/8645963f9187232ae2590185c1f36e26_123.png)

![](img/8645963f9187232ae2590185c1f36e26_125.png)

```python
def f(x):
    x = x + 1
    print('in f(x): x =', x)
    return x

![](img/8645963f9187232ae2590185c1f36e26_127.png)

![](img/8645963f9187232ae2590185c1f36e26_129.png)

x = 3
z = f(x)
```

![](img/8645963f9187232ae2590185c1f36e26_131.png)

![](img/8645963f9187232ae2590185c1f36e26_133.png)

1.  程序启动，创建全局作用域。Python 看到 `def f(x): ...`，知道有一个名为 `f` 的函数，但暂时不关心其内部代码。
2.  执行 `x = 3`，在全局作用域中创建变量 `x`，其值为 `3`。
3.  执行 `z = f(x)`，这是一个函数调用。Python 创建一个新的作用域（`f` 的作用域）。
4.  将实际参数 `x`（值为 `3`）映射给形式参数 `x`。在 `f` 的作用域内，`x` 现在是 `3`。
5.  执行 `f` 的函数体：`x = x + 1`，`f` 作用域内的 `x` 变为 `4`。打印消息。
6.  执行 `return x`，将值 `4` 返回给调用者。
7.  函数调用结束，`f` 的作用域被销毁。回到全局作用域，`f(x)` 被替换为返回值 `4`。
8.  因此，`z` 被赋值为 `4`。

![](img/8645963f9187232ae2590185c1f36e26_135.png)

![](img/8645963f9187232ae2590185c1f36e26_137.png)

**关于返回值的警告**：每个函数都应该返回一些内容。如果你没有显式地使用 `return` 语句，Python 会为你隐式地添加 `return None`。`None` 是一个特殊类型（`NoneType`）的值，表示“没有值”。

![](img/8645963f9187232ae2590185c1f36e26_139.png)

![](img/8645963f9187232ae2590185c1f36e26_141.png)

## 🖨️ `print` 与 `return` 的区别

![](img/8645963f9187232ae2590185c1f36e26_143.png)

![](img/8645963f9187232ae2590185c1f36e26_145.png)

理解 `print` 和 `return` 的区别至关重要。

![](img/8645963f9187232ae2590185c1f36e26_147.png)

![](img/8645963f9187232ae2590185c1f36e26_149.png)

*   `print`：将信息输出到控制台，用于**显示**。
*   `return`：指定函数的**输出结果**，用于将值传递回调用者。

![](img/8645963f9187232ae2590185c1f36e26_151.png)

例如，在问题集中，如果你看到输出了很多 `None`，请检查是否在函数中只使用了 `print` 而忘记了 `return`。

## ♻️ 函数的强大之处：可重用性

![](img/8645963f9187232ae2590185c1f36e26_169.png)

![](img/8645963f9187232ae2590185c1f36e26_171.png)

函数的主要优势之一是可重用性。一旦你编写并调试好一个函数，就可以在代码中多次使用它，使代码看起来非常简洁。

![](img/8645963f9187232ae2590185c1f36e26_173.png)

例如，使用我们之前定义的 `is_even` 函数：

```python
for i in range(20):
    if is_even(i):
        print(i, 'even')
    else:
        print(i, 'odd')
```

如果不使用函数，你就需要将判断偶数的逻辑（`i % 2 == 0`）直接写在循环里。使用函数使得代码更清晰、更易读，并且逻辑只在一处定义，便于维护。

## 🧠 函数也是对象

![](img/8645963f9187232ae2590185c1f36e26_183.png)

在 Python 中，**一切都是对象**。整数、浮点数是对象，字符串是对象，函数也是对象。

![](img/8645963f9187232ae2590185c1f36e26_185.png)

![](img/8645963f9187232ae2590185c1f36e26_187.png)

既然我们可以将对象作为参数传递给函数，那么我们也可以将**其他函数**作为参数传递。这为编程提供了极大的灵活性。

![](img/8645963f9187232ae2590185c1f36e26_189.png)

让我们分析一个传递函数作为参数的例子：

![](img/8645963f9187232ae2590185c1f36e26_191.png)

```python
def func_a():
    print('inside func_a')
    # 隐式返回 None

![](img/8645963f9187232ae2590185c1f36e26_193.png)

def func_b(y):
    print('inside func_b')
    return y

![](img/8645963f9187232ae2590185c1f36e26_195.png)

def func_c(z):
    print('inside func_c')
    return z()

![](img/8645963f9187232ae2590185c1f36e26_197.png)

![](img/8645963f9187232ae2590185c1f36e26_199.png)

print(func_a())        # 输出：inside func_a \n None
print(5 + func_b(2))   # 输出：inside func_b \n 7
print(func_c(func_a))  # 输出：inside func_c \n inside func_a \n None
```

![](img/8645963f9187232ae2590185c1f36e26_201.png)

![](img/8645963f9187232ae2590185c1f36e26_203.png)

![](img/8645963f9187232ae2590185c1f36e26_204.png)

![](img/8645963f9187232ae2590185c1f36e26_206.png)

对于 `func_c(func_a)` 的调用：
1.  进入 `func_c` 的作用域，将实际参数 `func_a`（函数对象）映射给形式参数 `z`。
2.  打印 `'inside func_c'`。
3.  执行 `return z()`。此时 `z` 是 `func_a`，所以这相当于 `return func_a()`。
4.  这引发了对 `func_a` 的调用，创建新作用域。
5.  `func_a` 执行，打印 `'inside func_a'`，并隐式返回 `None`。
6.  `func_a()` 调用结束，返回值 `None` 传回给 `func_c`。
7.  `func_c` 返回这个 `None`。
8.  最终，`print(func_c(func_a))` 打印出 `None`。

逐步跟踪变量、形式参数和实际参数是理解作用域的关键。强烈建议在纸上画图或使用 **Python Tutor** 工具来可视化执行过程。

## ⚠️ 作用域与变量访问

![](img/8645963f9187232ae2590185c1f36e26_212.png)

在函数内部访问变量时，有三种常见情况：

1.  **局部变量**：在函数内部定义并使用同名变量。这是最典型的情况，局部变量与外部全局变量互不干扰。
    ```python
    x = 5
    def f():
        x = 1  # 这是局部变量 x
        x += 1
        return x
    print(f()) # 输出 2
    print(x)   # 输出 5，全局 x 未改变
    ```

2.  **访问外部变量**：函数内部使用了一个未在内部定义的变量，Python 会向外层作用域查找。
    ```python
    x = 5
    def g():
        print(x) # 找到全局变量 x
    g() # 输出 5
    ```

3.  **在赋值前访问外部变量（错误）**：尝试在函数内部修改一个未在内部定义的变量会导致错误。
    ```python
    x = 5
    def h():
        x += 1  # UnboundLocalError! 尝试在赋值前使用局部变量 x
    ```
    可以使用 `global` 关键字声明使用全局变量，但通常不鼓励这样做，因为它会破坏函数的封装性，使代码难以维护。

## 🎓 总结

![](img/8645963f9187232ae2590185c1f36e26_224.png)

本节课我们一起学习了函数的核心概念：

![](img/8645963f9187232ae2590185c1f36e26_225.png)

![](img/8645963f9187232ae2590185c1f36e26_227.png)

*   **分解**：通过将代码划分为更小、可重用的模块（函数）来创建程序结构。
*   **抽象**：通过文档字符串隐藏函数实现细节，只暴露其用途、输入和输出。
*   **函数的定义与调用**：使用 `def` 定义函数，通过函数名和参数进行调用。
*   **作用域**：函数拥有独立的作用域，变量在其中生命周期有限，这是理解函数如何工作的关键。
*   **`return` 与 `print`**：`return` 用于输出函数结果，`print` 仅用于显示信息。
*   **函数作为对象**：函数可以像其他数据一样被传递，这增加了代码的灵活性。

![](img/8645963f9187232ae2590185c1f36e26_229.png)

![](img/8645963f9187232ae2590185c1f36e26_231.png)

![](img/8645963f9187232ae2590185c1f36e26_232.png)

掌握函数是成为高效程序员的重要一步。它们能帮助你编写出更清晰、更模块化、更易于调试和维护的代码。
