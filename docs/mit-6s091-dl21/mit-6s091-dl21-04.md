# 4：深度生成建模 🧠

![](img/512533c0d2e766b201730493800ee5a8_1.png)

在本节课中，我们将学习如何使用深度神经网络构建不仅能识别数据模式，还能基于这些模式生成全新合成数据的系统。深度生成建模是深度学习的一个强大子领域，近年来取得了巨大成功，并在众多应用场景中展现出巨大潜力。

## 什么是生成建模？ 🤔

上一节我们介绍了生成建模的目标。本节中，我们来看看其核心原理。

![](img/512533c0d2e766b201730493800ee5a8_3.png)

在监督学习中，我们拥有带标签的数据，目标是学习从数据到标签的映射。而生成建模属于**无监督学习**，我们只拥有数据本身，没有标签。目标是让模型理解数据中隐藏的内在结构和分布。

生成建模的核心任务是：给定一组训练数据样本，学习一个能够描述该数据分布的模型。这可以通过两种主要方式实现：
1.  **密度估计**：学习一个概率密度函数 `p_model(x)`，使其尽可能接近真实数据分布 `p_data(x)`。
2.  **样本生成**：利用学到的分布 `p_model(x)`，生成新的、看起来属于原始数据分布的合成样本 `x_new`。

因此，生成建模的关键在于如何让模型学到的分布 `p_model(x)` 与真实数据分布 `p_data(x)` 相似。

![](img/512533c0d2e766b201730493800ee5a8_5.png)

## 生成建模的应用 💡

理解了生成建模的原理后，我们来看看它为何如此强大和有影响力。

![](img/512533c0d2e766b201730493800ee5a8_7.png)

以下是生成模型的一些关键应用：

*   **自动发现数据偏差**：模型可以自动学习数据集中特征的分布（例如人脸数据中的肤色、姿势），揭示哪些特征被过度代表或代表不足。这有助于我们调整训练过程，构建更公平、更具代表性的数据集和模型。
*   **异常检测**：在诸如自动驾驶等安全关键领域，生成模型可以识别训练分布之外的异常或罕见情况，帮助模型更好地处理极端事件。

## 潜在变量模型与自动编码器 🕵️♂️

![](img/512533c0d2e766b201730493800ee5a8_9.png)

![](img/512533c0d2e766b201730493800ee5a8_11.png)

为了学习数据中的隐藏结构，我们引入**潜在变量模型**。潜在变量是无法直接观测、但能解释可观测数据的变量。

![](img/512533c0d2e766b201730493800ee5a8_13.png)

![](img/512533c0d2e766b201730493800ee5a8_15.png)

一个经典的比喻是“洞穴寓言”：囚犯只能看到墙上物体的影子（可观测变量），而看不到背后真正的物体（潜在变量）。生成模型的目标就是找到这些隐藏的潜在变量。

我们首先看一个基础的潜在变量模型：**自动编码器**。

自动编码器通过学习一个低维的**潜在空间**来表示原始数据。其结构如下：
1.  **编码器**：将高维输入数据 `x` 压缩成一个低维的潜在向量 `z`。
2.  **解码器**：将潜在向量 `z` 重建回原始数据空间，得到重建输出 `x_hat`。

我们通过最小化原始输入 `x` 与重建输出 `x_hat` 之间的差异（如均方误差）来训练网络。这个过程是**自监督的**，不需要任何数据标签。

```python
# 自动编码器训练的核心思想（伪代码）
reconstruction_loss = mean_squared_error(x, x_hat)
train_model_to_minimize(reconstruction_loss)
```

这种强制将数据压缩到低维空间再重建的过程，形成了一个**信息瓶颈**，迫使模型在潜在空间 `z` 中学习数据最核心、最紧凑的特征表示。

## 变分自编码器 🔄

上一节我们介绍了基础的自动编码器。本节中，我们来看看一个重要的改进——**变分自编码器**。

标准自动编码器的潜在层 `z` 是确定性的。VAE 对其进行了关键改造：它不直接输出潜在向量 `z`，而是输出定义 `z` 分布的参数——均值 `μ` 和方差 `σ²`。然后，我们从该分布中采样得到 `z`。

这使得潜在空间变成了**概率分布**。编码器学习给定数据 `x` 时潜在变量 `z` 的分布 `q_φ(z|x)`，解码器学习给定 `z` 时数据 `x` 的分布 `p_θ(x|z)`。

VAE 的损失函数包含两部分：
1.  **重建损失**：确保解码器能忠实地从潜在空间重建输入。
2.  **正则化损失（KL散度）**：迫使编码器学到的潜在分布 `q_φ(z|x)` 接近一个我们预设的先验分布 `p(z)`，通常选择标准正态分布 `N(0, 1)`。

**公式：VAE 损失函数**
`Loss = Reconstruction_Loss + β * KL( q_φ(z|x) || p(z) )`
其中 `β` 是控制正则化强度的超参数。

![](img/512533c0d2e766b201730493800ee5a8_17.png)

这种正则化带来了两个理想特性：
*   **连续性**：潜在空间中相近的点，解码后内容也相似。
*   **完整性**：从潜在空间采样得到的点，解码后都能生成有意义的内容。

![](img/512533c0d2e766b201730493800ee5a8_19.png)

## 重参数化技巧与解纠缠 🎲

VAE 在训练时面临一个难题：无法通过随机采样层进行反向传播。解决方案是**重参数化技巧**。

![](img/512533c0d2e766b201730493800ee5a8_21.png)

我们不直接从 `N(μ, σ²)` 采样 `z`，而是改为：
`z = μ + σ * ε`，其中 `ε` 采样自标准正态分布 `N(0, 1)`。

![](img/512533c0d2e766b201730493800ee5a8_23.png)

![](img/512533c0d2e766b201730493800ee5a8_25.png)

这样，随机性被转移到了常数 `ε` 上，而 `z` 可以表示为 `μ` 和 `σ` 的确定性函数，从而支持梯度反向传播。

![](img/512533c0d2e766b201730493800ee5a8_27.png)

```python
# 重参数化技巧（伪代码）
epsilon = sample_from_standard_normal()
z = mu + sigma * epsilon  # 现在可以通过 mu 和 sigma 对 z 求导
```

对潜在变量施加正态先验还有一个好处：我们可以通过单独扰动某个潜在变量 `z_i`，并保持其他变量不变，来观察生成内容的变化，从而**解释**该变量控制的语义特征（如改变头部姿势、笑容）。

![](img/512533c0d2e766b201730493800ee5a8_29.png)

为了得到更清晰、独立的特征表示，我们可以使用 **β-VAE**。通过增大损失函数中的 `β` 值，可以加强对潜在变量的正则化，鼓励模型学习**解纠缠**的表示，即每个潜在变量尽可能独立地控制一个生成特征。

![](img/512533c0d2e766b201730493800ee5a8_31.png)

## 生成对抗网络 ⚔️

![](img/512533c0d2e766b201730493800ee5a8_33.png)

前面我们学习了显式建模数据分布的 VAE。现在，我们转向另一种强大的生成模型——**生成对抗网络**，它不显式建模分布，而是专注于学习生成逼真样本。

GAN 的核心思想是让两个神经网络——**生成器 G** 和**判别器 D**——相互对抗、共同进步。
*   **生成器 G**：接收随机噪声 `z`，生成假数据 `G(z)`，目标是让假数据看起来像真的。
*   **判别器 D**：接收真实数据 `x` 和假数据 `G(z)`，判断输入是真实的还是生成的，目标是准确区分真假。

两者在训练中玩一个“最小最大”游戏：
*   判别器试图最大化将真实数据判为真、假数据判为假的概率。
*   生成器试图最小化判别器将其假数据判为假的概率（即最大化判别器犯错的概率）。

**公式：GAN 的对抗损失**
`min_G max_D V(D, G) = E[log D(x)] + E[log(1 - D(G(z)))]`

训练完成后，我们可以丢弃判别器，使用生成器从噪声中合成新的数据样本。通过在噪声空间中进行插值，我们可以在生成的数据流形上实现平滑过渡。

## GAN 的进阶与应用 🚀

![](img/512533c0d2e766b201730493800ee5a8_39.png)

![](img/512533c0d2e766b201730493800ee5a8_41.png)

基础的 GAN 已经很强大了，但研究人员提出了许多改进和扩展，使其能生成更高质量、更可控的样本。

![](img/512533c0d2e766b201730493800ee5a8_43.png)

以下是几个重要的进阶方向：

![](img/512533c0d2e766b201730493800ee5a8_45.png)

![](img/512533c0d2e766b201730493800ee5a8_47.png)

*   **渐进式增长 GAN**：训练时逐步增加生成器和判别器的层数，从低分辨率开始，逐渐提升到高分辨率，从而生成极其清晰、细节丰富的图像。
*   **StyleGAN**：结合渐进式增长和风格迁移的思想，能够将粗粒度风格（如年龄、面部结构）从源图像应用到生成过程中，实现高度可控和逼真的图像合成。
*   **条件 GAN**：为生成器和判别器提供额外的条件信息 `c`（如类别标签、分割图），从而实现**图像到图像的翻译**。例如，根据语义分割图生成街道场景，或将白天的照片转换为夜晚。
*   **CycleGAN**：实现**无配对**的图像到图像翻译。它使用两个生成器和两个判别器，并引入**循环一致性损失**，确保将一个域的图像转换到另一个域再转换回来时，能基本还原原图。经典应用包括将马转换成斑马，或将一种绘画风格转换成另一种。

![](img/512533c0d2e766b201730493800ee5a8_49.png)

![](img/512533c0d2e766b201730493800ee5a8_51.png)

![](img/512533c0d2e766b201730493800ee5a8_53.png)

CycleGAN 的思想甚至可以应用于语音转换，通过将音频转换为频谱图（一种图像表示），训练模型学习不同说话人声音之间的转换。

![](img/512533c0d2e766b201730493800ee5a8_55.png)

## 总结 📚

![](img/512533c0d2e766b201730493800ee5a8_57.png)

在本节课中，我们一起深入探讨了深度生成建模的世界。

![](img/512533c0d2e766b201730493800ee5a8_59.png)

我们首先了解了生成建模的核心目标：学习数据分布并生成新样本。然后，我们学习了两种主流的潜在变量模型：
1.  **变分自编码器**：通过概率编码和解码，学习数据的压缩潜在表示，并可通过重参数化技巧进行训练。它擅长发现数据结构和特征解纠缠。
2.  **生成对抗网络**：通过生成器和判别器的对抗训练，直接学习生成逼真样本的转换函数。它在图像合成、风格转换等任务上表现出色，并衍生出条件GAN、CycleGAN等多种强大变体。
