## 什么是数据集偏差？🤔

深度学习在计算机视觉等应用中取得了巨大成功。模型可以准确检测和识别各种场景中的物体，甚至包括人脸识别和情感分析。

![](img/e777f37231a2d4102afe165994bf89e2_3.png)

然而，存在一个关键问题：数据集偏差。当在一个数据集上训练的模型，被应用到另一个不同环境的数据集时，其性能往往会显著下降。

![](img/e777f37231a2d4102afe165994bf89e2_5.png)

例如，在加州晴朗天气下收集的行人数据上训练的自动驾驶模型，可能无法有效识别新英格兰雪天中穿着厚外套的行人。这种训练数据与测试数据之间的差异，被称为**数据集偏移**或**领域偏移**。

### 数据集偏差何时发生？

![](img/e777f37231a2d4102afe165994bf89e2_7.png)

以下是数据集偏差发生的几种常见情况：

*   **跨地域**：在一个城市收集数据，在另一个城市测试模型。
*   **跨媒介**：从网络图片训练模型，在机器人实际环境图像上测试。
*   **模拟到现实**：在模拟环境中训练，在现实世界中部署。
*   **人群偏差**：训练数据主要包含特定肤色或文化背景的人群，测试时遇到其他群体。

实际上，任何收集的数据集都可能存在偏差，因为视觉世界极其复杂，很难收集到覆盖所有可能情况的数据。

## 为什么数据集偏差是个严重问题？⚠️

上一节我们介绍了数据集偏差的概念，本节中我们来看看它为何会导致模型失效。

![](img/e777f37231a2d4102afe165994bf89e2_13.png)

性能下降主要有两个原因：
1.  **训练和测试数据分布不同**：模型在源领域（训练数据）学习到的特征分布，与目标领域（测试数据）的特征分布不匹配。
2.  **学到的特征缺乏判别性**：模型在源领域学到的、用于区分类别的特征，在目标领域中可能无法有效地区分数据点。

让我们通过一个经典例子来量化这个问题：手写数字识别。

![](img/e777f37231a2d4102afe165994bf89e2_15.png)

*   在MNIST数据集上训练一个神经网络，并在MNIST上测试，准确率可达 **99%**。
*   如果在另一个视觉风格不同的数字数据集（如街景门牌号数据集SVHN）上训练同一个模型，然后在MNIST上测试，准确率会**大幅下降**。
*   即使是在两个视觉上非常相似的数据集（如USPS和MNIST）之间进行跨领域测试，性能也会显著降低。

这证明，一旦发生领域偏移，模型在新领域上的性能几乎崩溃。

### 现实世界的影响

![](img/e777f37231a2d4102afe165994bf89e2_17.png)

数据集偏差在现实世界中具有重要且有时是严重的含义：

*   **人脸识别偏差**：研究表明，一些商用的人脸识别和性别识别算法在识别非裔美国人和亚洲人面孔时，准确性远低于识别高加索人面孔。这是因为训练这些模型的数据集存在偏差。
*   **自动驾驶事故**：有分析认为，某些致命的自动驾驶事故，是由于训练数据中缺乏“行人不在人行横道上”的样本，导致算法未能检测到此类场景下的行人。

## 解决方案：领域适应 🛠️

既然收集和标注海量、多样化的数据成本高昂，我们能否利用大量**未标注**的目标领域数据来提升模型性能呢？这就是**领域适应**的目标。

### 领域适应的定义

在领域适应中，我们假设：
*   有一个**源领域**，包含大量**有标签**的数据：`{ (x_i^s, y_i^s) }`。
*   有一个**目标领域**，包含大量**无标签**的数据：`{ x_i^t }`。
*   我们的目标是学习一个分类器，使其在目标数据分布 `D_t` 上实现较低的预期损失。

核心思想是：利用源领域的标签和两个领域共享的未标注数据，让模型学会在目标领域上也表现良好。

## 方法一：对抗性领域对齐 ⚔️

![](img/e777f37231a2d4102afe165994bf89e2_25.png)

本节我们将深入探讨一种非常标准且有效的技术——对抗性领域对齐。

其核心思想是：训练模型的特征提取器（编码器），使其生成的特征在源领域和目标领域上无法被区分。这样，在源特征上训练的分类器就能直接用于目标特征。

![](img/e777f37231a2d4102afe165994bf89e2_27.png)

![](img/e777f37231a2d4102afe165994bf89e2_29.png)

以下是实现步骤：

1.  **训练领域鉴别器**：在编码器提取的源特征和目标特征上，训练一个神经网络（领域鉴别器）来区分数据来自哪个领域。其损失函数是二分类交叉熵损失。
    ```python
    # 伪代码示意
    domain_loss = CrossEntropyLoss(domain_predictor(features), domain_labels)
    ```
2.  **“欺骗”领域鉴别器**：固定领域鉴别器，更新特征编码器，使其生成的特征能让领域鉴别器无法做出正确判断（即最大化领域鉴别器的错误率）。这通常通过梯度反转层实现。
    ```python
    # 伪代码示意：通过最小化领域分类损失来“对齐”特征分布
    # 实际实现时，会对领域鉴别器的梯度进行反转
    feature_encoder_loss = -CrossEntropyLoss(domain_predictor(features), domain_labels)
    ```
3.  **保持分类能力**：同时，我们仍需在源领域的有标签数据上训练分类器，确保对齐后的特征仍然具有类别判别性。这防止了编码器将所有特征映射到同一个点。

### 效果如何？

![](img/e777f37231a2d4102afe165994bf89e2_31.png)

在之前的数字识别例子中，应用对抗性领域对齐后：
*   源特征（蓝点）和目标特征（红点）的分布变得高度重合。
*   目标领域上的分类准确率得到了显著提升。

**要点**：领域适应可以在完全不使用目标领域标签的情况下，提高模型在目标数据上的准确性。这可以看作是一种**无监督微调**。

## 方法二：像素空间对齐 🎨

上一节我们在特征空间中对齐了两个领域。本节我们来看看另一种思路：能否直接在图像像素层面进行对齐？

![](img/e777f37231a2d4102afe165994bf89e2_35.png)

![](img/e777f37231a2d4102afe165994bf89e2_37.png)

像素空间对齐的想法是：利用图像到图像的翻译技术（如CycleGAN），将源领域的图像“翻译”成看起来像是来自目标领域的图像。

**具体做法**：
1.  训练一个图像翻译模型，将源图像 `x^s` 转换为具有目标领域风格的图像 `G(x^s)`。
2.  由于我们拥有 `x^s` 的标签 `y^s`，我们现在就得到了一批“伪目标”图像 `{G(x^s), y^s}`。
3.  在这批风格与目标一致、且带有标签的数据上重新训练分类器，有望提升其在真实目标数据上的性能。

这种方法可以与特征空间对齐结合使用，获得更好的效果。

![](img/e777f37231a2d4102afe165994bf89e2_39.png)

### 实际案例：从游戏到现实

一个典型应用是从游戏（GTA）合成数据，适应到真实城市街道（Cityscapes）数据。
*   **任务**：语义分割（为每个像素分类，如道路、汽车、行人）。
*   **方法**：将GTA中的游戏画面翻译成具有Cityscapes真实感的图像。
*   **结果**：在目标域（Cityscapes）上的像素准确率从 **54.6%** 大幅提升至 **83.6%**。

在困难的SVHN到MNIST的数字识别任务中，像素空间对齐将准确率从特征对齐的 **76.0%** 提升到了 **90.4%**。

**要点**：无监督的图像到图像翻译能够发现并对齐两个领域之间对应的结构（如数字的形状、物体的轮廓），从而实现有效的领域适应。

## 进阶：少样本像素对齐 🎯

![](img/e777f37231a2d4102afe165994bf89e2_43.png)

![](img/e777f37231a2d4102afe165994bf89e2_45.png)

之前的方法假设我们拥有大量未标注的目标数据。但如果目标领域只有很少（如1-5张）图像怎么办？

![](img/e777f37231a2d4102afe165994bf89e2_47.png)

![](img/e777f37231a2d4102afe165994bf89e2_49.png)

我们提出了一种少样本图像翻译方法。目标是：给定一张源内容图像和一张目标风格图像，生成一张既保留源图像内容（如物体姿态），又具有目标图像风格（如物种、纹理）的新图像。

![](img/e777f37231a2d4102afe165994bf89e2_51.png)

![](img/e777f37231a2d4102afe165994bf89e2_53.png)

![](img/e777f37231a2d4102afe165994bf89e2_54.png)

**关键技术**：我们设计了一个**内容条件化的风格编码器**。它不仅接收风格图像，还接收内容图像的信息，从而能更好地根据内容来调整风格，生成更逼真、更准确的翻译结果。

![](img/e777f37231a2d4102afe165994bf89e2_56.png)

![](img/e777f37231a2d4102afe165994bf89e2_58.png)

![](img/e777f37231a2d4102afe165994bf89e2_59.png)

例如，我们可以将一张猫的图片（内容），翻译成具有特定品种狗的风格，同时完美保留猫原有的姿态。

![](img/e777f37231a2d4102afe165994bf89e2_60.png)

![](img/e777f37231a2d4102afe165994bf89e2_62.png)

**要点**：通过让风格编码同时依赖于内容和风格图像，我们可以在仅有极少数目标样本的情况下，实现高质量的跨领域图像翻译，为后续的模型适应提供了可能。

## 超越对齐：自监督与一致性学习 🔄

领域对齐并非唯一途径，尤其当源领域和目标领域的类别集合不完全相同时（称为“类别偏移”），强行对齐所有特征可能导致灾难性后果。

![](img/e777f37231a2d4102afe165994bf89e2_68.png)

本节我们介绍两种不依赖严格对齐的方法。

### 1. 自监督学习处理类别偏移

当目标领域存在源领域没有的“未知类别”时，我们采用自监督学习策略：
*   **邻域聚类**：利用预训练模型初始化，通过自监督损失（如让相似特征靠近，不相似特征远离），将目标数据中属于“已知类别”的点聚类到源领域对应类别附近，而将“未知类别”的点推远。
*   **熵分离**：进一步鼓励模型对可能属于未知类别的点（预测概率分布熵值高）产生低置信度，实现异常值拒绝。

![](img/e777f37231a2d4102afe165994bf89e2_70.png)

这种方法在存在类别偏移的适应任务上，性能显著优于传统的领域对齐方法。

### 2. 自监督预训练与一致性训练

![](img/e777f37231a2d4102afe165994bf89e2_72.png)

![](img/e777f37231a2d4102afe165994bf89e2_74.png)

另一种强大的组合是：
*   **自监督预训练**：在源和目标数据上执行自监督任务（如图像旋转预测），让模型学习对领域变化更鲁棒的特征表示。
    ```python
    # 例如，预测图像被旋转了0°, 90°, 180°, 还是270°
    rotation_loss = CrossEntropyLoss(predict_rotation(features), rotation_label)
    ```
*   **一致性训练**：对未标注的目标图像进行数据增强（裁剪、颜色抖动等），要求模型对原始图像和增强后图像的预测保持一致。这鼓励模型学习更本质的、不受低级视觉变化影响的特征。

**要点**：在领域适应中，除了对齐特征分布，我们还可以利用自监督学习和一致性训练等策略，来学习更具判别性和鲁棒性的表示，从而提升模型在目标领域的性能。

## 总结 📚

本节课中，我们一起学习了如何驯服数据集偏差。

![](img/e777f37231a2d4102afe165994bf89e2_78.png)

1.  **问题认知**：我们认识到数据集偏差是深度学习在实际部署中的主要挑战，它源于训练与测试数据分布的不同。
2.  **核心方案**：领域适应技术利用未标注的目标数据，实现知识的迁移，可视为一种无监督微调。
3.  **经典方法**：
    *   **对抗性领域对齐**：在特征空间对齐两个领域的分布。
    *   **像素空间对齐**：通过图像翻译，在像素层面使数据风格一致。
4.  **前沿进展**：
    *   **少样本适应**：即使目标样本极少，也能实现高质量的图像翻译。
    *   **超越对齐**：采用自监督学习、一致性训练等方法处理更复杂的场景（如类别偏移），学习更鲁棒的特征表示。
