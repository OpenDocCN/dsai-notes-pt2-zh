## 🏗️ 神经网络的基本构建块：感知器

![](img/6c4cc81a285b92ef56246e277232f320_15.png)

![](img/6c4cc81a285b92ef56246e277232f320_17.png)

上一节我们介绍了深度学习的宏观概念，本节中我们来看看其最基本的构建单元——感知器（或称神经元）。

![](img/6c4cc81a285b92ef56246e277232f320_19.png)

![](img/6c4cc81a285b92ef56246e277232f320_21.png)

感知器的概念非常简单。信息通过单个神经元进行前向传播的过程如下：每个输入数字乘以对应的权重，然后求和，再加上一个偏置项，最后通过一个非线性激活函数，产生最终输出 `y`。

![](img/6c4cc81a285b92ef56246e277232f320_23.png)

我们可以用数学公式更简洁地表示这个过程。设大写 `X` 为输入向量 `[x1, x2, ..., xn]`，大写 `W` 为权重向量 `[w1, w2, ..., wm]`，则输出 `y` 的计算公式为：

![](img/6c4cc81a285b92ef56246e277232f320_25.png)

**`y = g(W · X + b)`**

![](img/6c4cc81a285b92ef56246e277232f320_27.png)

![](img/6c4cc81a285b92ef56246e277232f320_29.png)

其中，`·` 表示点积，`b` 是偏置项，`g` 是非线性激活函数。

以下是激活函数的一个常见例子——Sigmoid函数，它在TensorFlow中的代码表示如下：

```python
# Sigmoid 激活函数示例
import tensorflow as tf
tf.keras.activations.sigmoid(x)
```

Sigmoid函数将任意实数输入映射到0到1之间，这使其非常适合处理概率问题。另一个流行的激活函数是ReLU（修正线性单元），它是一个分段线性函数。

![](img/6c4cc81a285b92ef56246e277232f320_31.png)

**为什么我们需要激活函数？** 激活函数的核心作用是在网络中引入非线性。现实世界的数据几乎总是非线性的。如果只使用线性激活函数，无论神经网络有多深，其决策边界只能是一条直线，这将无法处理复杂的非线性数据。非线性激活函数使得神经网络能够逼近任意复杂的函数，这是其强大能力的关键。

## 🧱 从感知器到神经网络

理解了单个感知器后，我们可以开始构建神经网络。一个多输出的神经网络可以简单地通过添加更多感知器来实现。

![](img/6c4cc81a285b92ef56246e277232f320_33.png)

![](img/6c4cc81a285b92ef56246e277232f320_35.png)

以下是构建一个“密集层”（或全连接层）的步骤。密集层由多个感知器组成，每个感知器都与所有输入相连。

1.  **初始化参数**：初始化该层的权重和偏置。
2.  **前向传播**：计算输入与权重的点积，加上偏置，然后应用激活函数。

![](img/6c4cc81a285b92ef56246e277232f320_37.png)

在TensorFlow中，我们无需从头实现，可以直接调用预定义的层。例如，创建一个具有两个输出的密集层：

![](img/6c4cc81a285b92ef56246e277232f320_39.png)

![](img/6c4cc81a285b92ef56246e277232f320_41.png)

```python
import tensorflow as tf
dense_layer = tf.keras.layers.Dense(units=2)
```

## 🏢 构建深度神经网络

![](img/6c4cc81a285b92ef56246e277232f320_47.png)

![](img/6c4cc81a285b92ef56246e277232f320_49.png)

现在，让我们将层堆叠起来，构建更深的网络。在输入层和输出层之间的层被称为“隐藏层”。

在TensorFlow中，我们可以使用 `Sequential` API 轻松堆叠多个密集层来创建深度神经网络：

```python
model = tf.keras.Sequential([
    tf.keras.layers.Dense(n, activation='relu'),  # 第一隐藏层，n个神经元
    tf.keras.layers.Dense(2, activation='sigmoid') # 输出层，2个输出
])
```

通过不断添加层，我们可以创建任意深度的神经网络，使其能够学习更复杂、更抽象的数据表示。

## 📊 训练神经网络：损失与优化

![](img/6c4cc81a285b92ef56246e277232f320_55.png)

拥有了网络结构后，我们需要教会它如何学习。这涉及到两个核心概念：损失函数和优化算法。

**损失函数**衡量模型预测值与真实值之间的差距。差距越大，损失越大。我们的目标是找到一组网络权重，使损失最小化。对于不同任务，需使用不同的损失函数：
*   **二元分类**（如判断是否通过考试）：常用 **二元交叉熵损失**。
*   **回归**（如预测具体分数）：常用 **均方误差损失**。

**优化算法**（如梯度下降）则用于寻找使损失最小化的权重。其核心思想是计算损失函数相对于每个权重的梯度（即导数），梯度指明了损失上升最快的方向。因此，我们沿着梯度的反方向更新权重，从而降低损失。

![](img/6c4cc81a285b92ef56246e277232f320_57.png)

![](img/6c4cc81a285b92ef56246e277232f320_59.png)

权重更新公式为：
**`W_new = W_old - η * ∇J(W_old)`**

其中，`η` 是**学习率**，它控制着每次更新的步长。

## 🔁 反向传播：计算梯度的引擎

![](img/6c4cc81a285b92ef56246e277232f320_63.png)

梯度下降告诉我们“往哪个方向走”，而**反向传播**算法则告诉我们“如何计算这个方向（梯度）”。

![](img/6c4cc81a285b92ef56246e277232f320_65.png)

反向传播通过递归地应用链式法则，从输出层开始，反向计算损失函数对网络中每一个权重的梯度。它解释了网络中每一个微小变化如何最终影响总的损失。虽然原理上只是微积分中链式法则的应用，但它是高效训练深度神经网络的关键。

## ⚙️ 实践训练技巧

![](img/6c4cc81a285b92ef56246e277232f320_69.png)

理论听起来简单，但在实践中训练神经网络充满挑战。以下是几个至关重要的技巧：

![](img/6c4cc81a285b92ef56246e277232f320_71.png)

![](img/6c4cc81a285b92ef56246e277232f320_73.png)

### 1. 自适应学习率与优化器
固定的学习率 `η` 可能带来问题：太小会导致收敛缓慢且易陷入局部最小值；太大会导致优化过程震荡甚至发散。解决方案是使用**自适应优化器**（如Adam、RMSprop），它们能根据当前的梯度情况动态调整每个参数的学习率。

在TensorFlow中，可以轻松更换优化器：
```python
optimizer = tf.keras.optimizers.Adam(learning_rate=0.001) # 使用Adam优化器
# optimizer = tf.keras.optimizers.SGD(learning_rate=0.01) # 或使用随机梯度下降
```

![](img/6c4cc81a285b92ef56246e277232f320_75.png)

### 2. 批处理
在每次迭代中计算整个数据集的梯度（批量梯度下降）计算成本太高；而仅使用一个样本（随机梯度下降）又太嘈杂。折中的方法是使用**小批量梯度下降**：每次随机选取一小批（batch）数据计算梯度。这既提高了计算效率（利于GPU并行），又得到了比单样本更稳定的梯度估计。

![](img/6c4cc81a285b92ef56246e277232f320_77.png)

### 3. 正则化：防止过拟合
过拟合是指模型在训练数据上表现很好，但在未见过的测试数据上表现很差。正则化技术用于约束模型，防止其过度复杂化。

*   **Dropout（随机失活）**：在训练期间，随机将网络中一部分神经元的输出置零。这迫使网络不能过度依赖某些特定的路径，必须学习更鲁棒的特征，从而提升泛化能力。
    ```python
    model.add(tf.keras.layers.Dropout(rate=0.5)) # 添加Dropout层，失活率为50%
    ```
*   **早停**：在训练过程中，持续监控模型在验证集上的性能。一旦验证集上的损失停止下降并开始上升，就立即停止训练。这样可以获取到泛化性能最佳的模型，避免其继续“记忆”训练数据。

## 🎯 总结

本节课中我们一起学习了深度学习的基石：
1.  **神经网络的基本单元**：我们从感知器的工作原理出发，理解了其“点积、加偏置、非线性激活”的三步计算过程。

4.  **实战技巧**：我们探讨了自适应优化器、批处理以及Dropout和早停等正则化技术，这些都是成功训练现代深度神经网络的关键。
