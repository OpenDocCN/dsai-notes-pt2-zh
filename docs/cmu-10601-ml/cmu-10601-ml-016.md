## 回顾AdaBoost

上一节我们介绍了Boosting。Boosting是一种通用方法，旨在提升任何给定学习算法的准确率。其核心思想是：通过创建一系列具有挑战性的数据集，使得在这些数据集上即使表现平平的弱学习器，也能被组合成一个整体上高度准确的预测器。

我们详细讨论了AdaBoost算法。该算法在实践中被广泛使用，并且已从理论角度得到了深入分析。我们证明了随着Boosting轮数的增加，AdaBoost的训练误差会迅速趋近于0。本节我们将进一步探讨其泛化保证。

### AdaBoost算法简述

AdaBoost是一个Boosting过程。其输入包括：
*   一个带标签的训练样本集 `(x1, y1), ..., (xm, ym)`，其中 `x` 是特征，`y` 是标签。
*   一个弱学习算法，该算法能够生成弱学习器或“规则”，例如朴素贝叶斯或决策树桩。

算法的目标是输出一个高精度的分类器。

以下是AdaBoost的工作流程：

1.  **初始化**：为每个训练样本赋予初始权重 `D1(i) = 1/m`。
2.  **迭代轮次**：对于每一轮 `t = 1, ..., T`：
    *   基于当前权重分布 `Dt`，运行弱学习算法，得到一个弱分类器 `ht`。
    *   计算 `ht` 在分布 `Dt` 下的加权错误率 `εt`。
    *   根据 `εt` 计算该弱分类器的权重 `αt = 1/2 * ln((1-εt)/εt)`。
    *   **更新样本权重**：这是算法的关键步骤。对于每个样本 `i`：
        *   如果 `ht` 分类错误，则增加其权重：`Dt+1(i) ∝ Dt(i) * exp(αt)`。
        *   如果 `ht` 分类正确，则减少其权重：`Dt+1(i) ∝ Dt(i) * exp(-αt)`。
    *   对所有权重进行归一化，使 `Dt+1` 成为一个概率分布。
3.  **输出最终分类器**：经过 `T` 轮后，输出一个加权多数投票的强分类器：
    `H(x) = sign( Σ_{t=1}^{T} αt * ht(x) )`

简单来说，AdaBoost在每一轮中，都会根据当前弱分类器的表现，**增加被错误分类样本的权重，减少被正确分类样本的权重**，从而在下一轮迫使弱学习器更关注那些“难”的样本。

一个便于记忆的性质是：更新后的分布 `Dt+1` 会确保**恰好一半的权重**落在当前弱分类器 `ht` 预测错误的样本上，另一半落在预测正确的样本上。

### 关于“一半权重”的技术原因

有同学可能会问，为什么更新分布时要设定为“一半对一半”的权重分配，而不是将所有权重都放在错误样本上？

原因如下：如果将所有权重都放在错误样本上，弱学习器可能会简单地返回 `ht` 的相反分类，这并不能提供新的信息。因此，我们需要一个更平衡的策略。

从理论分析的角度，我们上一节证明了最终分类器的训练误差上界为 `m * Π_{t=1}^{T} Zt`，其中 `Zt` 是每一轮权重更新的归一化因子。为了使这个上界（即训练误差）尽可能快地下降至0，我们需要最小化每个 `Zt`。数学推导表明，当我们将 `Dt+1` 设置为在 `ht` 正确和错误的样本上各分配一半权重时，得到的 `Zt` 是最小的，从而使得训练误差以**最快**的指数速度下降。

具体而言，如果每个弱分类器 `ht` 都比随机猜测好一个“边缘” `γt`，那么最终分类器的训练误差上界可以进一步简化为：
`训练误差 ≤ exp(-2 * Σ_{t=1}^{T} γt²)`
如果所有 `γt` 都至少为一个常数 `γ`，那么这个上界就是 `exp(-2γ²T)`，呈指数级下降。而“一半对一半”的更新规则正是实现这一最快下降速度的关键。

## 总结

本节课我们一起学习了：
1.  **AdaBoost算法**：我们回顾了这个强大而简单的集成学习算法。它通过迭代调整样本权重，聚焦于之前被错误分类的样本，并将多个弱分类器组合成一个强分类器。
2.  **核心机制**：我们深入理解了算法中更新样本权重的公式，以及其确保“一半权重”在错误样本上的设计。

AdaBoost算法易于实现、无需复杂调参、运行高效，并且能作为黑盒与任何基础学习器结合使用，这些特点使其成为机器学习中一个非常实用的工具。
