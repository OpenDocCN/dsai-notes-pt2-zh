# 024：神经网络 🧠

在本节课中，我们将要学习神经网络的基本概念。我们将从最基础的阈值单元（或称逻辑单元）开始，了解其工作原理，并探讨如何通过组合这些单元来构建能够学习复杂非线性决策边界的多层网络。

## 概述

神经网络是一种受生物神经系统启发的计算模型。它由大量相互连接的简单处理单元（称为神经元或节点）组成。本节课我们将重点介绍一种经典的神经网络单元——逻辑函数（或称S型函数）单元，并解释如何通过组合这些单元来构建强大的函数逼近器和分类器。

## 逻辑函数单元

上一节我们介绍了神经网络的基本思想，本节中我们来看看构成网络的基本单元。一个神经网络单元的功能类似于逻辑函数。逻辑函数的一个同义词是S型函数。

该单元的输出由输入X1到Xn决定。输出是输入加权线性组合的熟悉逻辑函数，再加上一个偏置权重，这与逻辑回归非常相似。

但与逻辑回归不同的是，在神经网络中，我们通常会将一系列这样的阈值单元组合在一起。正如我们稍后在课程中会看到的，一些神经网络（尤其是现代的变体）并不严格使用这个精确的函数。如今，研究人员正在探索各种用于定义神经网络单个单元的函数。有些人使用线性函数，有些人使用这种阈值逻辑函数，还有些人使用简单的线性函数，当输出为负时直接输出零。目前研究领域正在进行各种各样的实验。

但在大多数情况下，以及过去几十年里，这就是神经网络的定义。我们将从这里开始，并在后续课程中了解一些其他替代方案。

## 训练神经网络

就像任何函数逼近器一样，如果要求你开发一个机器学习算法来训练它，你应该想到，或许可以通过最大化数据的似然来训练它，即求参数的最大似然估计。参数是什么？就是这些权重。实际上，网络中的每个单元都有一组权重。

我们可以通过最小化网络输出的平方误差和来实现这一点。我们也可以使用最大后验概率方法进行训练。

## 网络结构与示例

以下是神经网络的一个具体示例。这个网络有两个输入节点，代表语音信号。在本例中，它们是频率共振峰，捕捉了语音信号功率谱中两个不同部分的能量。因此，这里的每个点都是一个语音数据点。这些特定的数据点代表某人发出元音时的声音，例如“head”、“hi”、“had”、“ho”、“hoard”、“heed”、“hood”、“who”等，这些都是不同的元音，每个都会产生不同的频率成分，并绘制在这里。

这些就是输入，任何一个声音都可以表示为这两个数字组成的向量。图中的每个圆圈代表一个阈值单元的输出。例如，这里是一个阈值单元，它有两个输入，并根据逻辑函数产生输出。你可以将这些边视为代表权重W1和W2，用于计算该逻辑函数的输出。当然，我们知道逻辑函数的输出值在0到1之间。

然后，这个介于0到1之间的值，连同所有其他中间值，成为第二层阈值单元的输入。例如，这个单元以所有这些中间值作为输入，并使用其自己的逻辑函数（与这些边关联的参数）输出一个同样介于0到1之间的数字。因此，可以将其视为逻辑函数的逻辑函数。这样可以得到非常非线性的函数。

这个网络经过训练，用于识别当捕捉到这些频率成分时，发出的是哪个元音。其编码方式是在训练网络时，我们在每个输出上放置一个0，只在对应于目标元音的那个单一输出上放置一个1。例如，如果目标是“e”或“ee”，我们会将这个输出设为1，其他所有输出设为0。这样，我们就得到了一个接收向量输入并产生向量输出的网络。

我们可以尝试训练它。这意味着训练所有与这些黑线关联的权重。训练完成后，如果我们观察它学习到的决策面，实际上就显示在这里。图上的点对应于不同的数据点。如果你视力很好，可能会看到那是一个空心圆，代表“heed”，那是一个加号，代表“hod”。

这个网络学习到的决策面，即它实际学习到的函数，将这个空间划分成不同的区域。每个区域由一组点定义，对于这些点，网络在给定F1和F2输入时，会输出其中一个标签作为预测。例如，这里是上方的区域，网络会在此预测“heed”。

## 输出解码与非线性来源

训练时，我们使用全零向量和一个单一的一。训练完成后，网络可以输出任何向量。我们通过选择输出值最高的那个输出单元来解码它。因此，输出实际上是一个连续向量，我根据该分类规则将其离散化，以划分这个区域。

这就像是不同分隔符的组合。但它不是线性分隔符的组合，因为事实证明，线性函数的线性函数仍然是线性函数。所以，如果每个函数都是线性的，你无法通过组合线性函数来产生非线性函数。虽然逻辑回归确实产生线性决策面，但它产生的是0到1之间的连续输出。

理解为什么这个网络如此非线性的关键在于，要认识到这里的输入是实数值，中间层的输出也是介于0和1之间的实数值。在这一层，我们并不将其固定为0或1，它可能是0.7。然后这个0.7将作为输入传递给下一阶段。因此，我们最终得到的是对连续变量进行非线性函数的非线性函数运算。当我应用分类规则（即通过选择最大输出来离散化）时，就得到了这个非线性的决策面。

人们也尝试过使用线性单元而非逻辑单元。但仔细想想，线性函数的线性函数仍然是线性函数，所以这并不能带来任何额外的好处。

## 总结

本节课中，我们一起学习了神经网络的基础知识。我们从基本的逻辑函数（S型函数）单元开始，了解了其作为网络构建块的作用。我们探讨了如何通过组合这些单元形成多层网络，并看到了一个用于元音分类的具体示例。这个示例简洁地展示了由S型单元构成的多层网络，以及能够通过学习得到的非常有趣的非线性函数分类器。关键在于，中间层的连续值输出使得网络能够学习复杂的非线性决策边界，这是简单线性组合无法实现的。
