# 061：支持向量机与核函数

在本节课中，我们将要学习支持向量机的基本原理、核函数的概念，以及拉格朗日乘数法的简要介绍。支持向量机是一种用于解决分类问题的强大工具，其核心思想是寻找一个能够最大化分类间隔的决策边界。

## 支持向量机的基本概念

上一节我们介绍了分类问题的基本目标。本节中我们来看看支持向量机如何解决这个问题。

分类问题的目标是预测一个离散的标签。例如，在支持向量机中，我们有一些被标记为正类的点，和一些被标记为负类的点。我们希望找到一个通用的规则，以便预测未来新数据点的类别。

如果我们想绘制一个线性的决策边界，可以画出多种可能的直线。例如，一条决策边界可能将所有正类点放在一侧，所有负类点放在另一侧。但这并非唯一的决策边界。

在两条可能的边界中，实线边界通常被认为是更好的选择。直觉上，实线边界与最近的数据点之间有更大的距离。这意味着，如果数据点发生微小的移动，使用实线边界进行分类的结果会更加稳定。

这种追求最大间隔的思想，正是支持向量机名称的由来。位于间隔边界上的点被称为**支持向量**。分类结果实际上只依赖于这些支持向量的值。即使有成千上万个数据点，可能也只有少数几个是支持向量，这使得模型表示更加高效。

我们可以用数学公式来描述这个决策边界。决策边界由以下方程定义：
`W^T x + b = 0`
其中，`W` 是法向量，`b` 是偏置项。在预测时，我们使用以下规则：
`y_pred = sign(W^T x + b)`
如果结果大于0，则预测为正类；如果小于0，则预测为负类。

## 优化问题与间隔

上一节我们定义了决策边界。本节中我们来看看如何通过优化找到最佳的 `W` 和 `b`。

假设正类标签为 +1，负类标签为 -1。我们希望所有正类点满足 `W^T x + b >= 1`，所有负类点满足 `W^T x + b <= -1`。这两个超平面之间的区域被称为“间隔”。

间隔的宽度（记为 γ）与向量 `W` 的范数成反比关系。因此，最大化间隔等价于最小化 `||W||`。于是，我们可以将支持向量机构建为以下优化问题：

**目标**：最小化 `(1/2) * ||W||^2`
**约束条件**：对于每一个训练样本 `(x_i, y_i)`，满足 `y_i (W^T x_i + b) >= 1`

这是一个带有线性约束的凸二次规划问题，可以通过标准优化方法求解。

## 处理非线性可分数据：软间隔与核函数

上一节讨论的是“硬间隔”支持向量机，它要求数据必须严格线性可分。本节中我们来看看当数据无法完美分离时该怎么办。

在现实世界中，数据往往不是线性可分的，或者存在噪声点。为了处理这种情况，我们引入“软间隔”概念，允许一些数据点违反间隔约束，但会施加惩罚。优化目标变为：
最小化 `(1/2) * ||W||^2 + C * Σ ξ_i`
其中，`ξ_i` 是松弛变量，表示第 `i` 个样本违反约束的程度；`C` 是一个超参数，用于控制对误分类的惩罚力度。

然而，即使使用软间隔，模型仍然是线性的。为了处理复杂的非线性决策边界，我们需要使用**核技巧**。

核函数允许我们在不显式计算高维特征映射的情况下，在高维空间中计算数据点的内积。这相当于将数据映射到一个更高维的空间，使其在该空间中线性可分。常用的核函数包括：

以下是几种常见的核函数：
*   **线性核**：`K(x, z) = x^T z`
*   **多项式核**：`K(x, z) = (x^T z + c)^d`
*   **径向基函数核**：`K(x, z) = exp(-γ * ||x - z||^2)`

通过使用核函数，支持向量机可以学习非常复杂的非线性决策边界，同时其优化问题的形式基本保持不变，只是将内积 `x_i^T x_j` 替换为核函数 `K(x_i, x_j)`。

## 拉格朗日乘数法简介

上一节我们提到了优化问题。本节中我们简要了解一下求解支持向量机优化问题的关键数学工具——拉格朗日乘数法。

拉格朗日乘数法是一种用于求解带约束优化问题的方法。对于支持向量机的原始优化问题，我们可以构造其拉格朗日函数：
`L(W, b, α) = (1/2) * ||W||^2 - Σ α_i [y_i (W^T x_i + b) - 1]`
其中，`α_i >= 0` 是拉格朗日乘子。

通过对 `W` 和 `b` 求导并令导数为零，我们可以将原始问题转化为其对偶问题。对偶问题通常更容易求解，并且其形式自然地引入了核函数。最终，决策函数可以完全由支持向量（即 `α_i > 0` 对应的样本）和核函数表示：
`f(x) = sign( Σ α_i y_i K(x_i, x) + b )`

## 总结

本节课中我们一起学习了支持向量机的核心思想。我们从寻找最大间隔的线性分类器开始，介绍了其数学 formulation 为一个优化问题。接着，我们探讨了通过引入软间隔和核函数来处理非线性可分数据，这极大地扩展了支持向量机的应用范围。最后，我们简要了解了拉格朗日乘数法在推导支持向量机对偶形式中的作用。支持向量机因其坚实的理论 foundation 和良好的实践效果，在机器学习中占有重要地位。
