# 025：强化学习入门 🎯

在本节课中，我们将学习一种与之前讨论过的监督学习截然不同的机器学习范式——强化学习。我们将了解其核心思想、应用场景以及用于形式化描述此类问题的马尔可夫决策过程模型。

## 概述

强化学习与我们目前讨论过的主题有很大不同。其核心区别在于，我们做出的决策是关于**采取何种行动**。例如，这在机器人导航中非常有用，机器人必须采取特定行动才能朝特定方向移动。在强化学习中，我们做出的决策就是关于采取何种行动。

这些决策会影响我们接下来看到的示例，或者说我们所处的**状态**。我们的决策会影响我们接下来看到的内容，这与我们之前讨论的监督分类等任务有所不同。

此外，我们的目标不一定只是进行预测（例如，判断我们面前是否有门），我们还希望**能够决定采取什么行动**。例如，机器人应该朝哪个方向走。这是一种有趣且不同类型的机器学习问题。

为了讨论这类问题，我们将使用一种称为**马尔可夫决策过程**的模型，稍后将给出定义。

## 延迟反馈

强化学习的一个非常酷的方面是，我们拥有**延迟反馈**。具体来说，我们行动的主要影响不会立即显现，而是可能在很久之后才出现。

以下是一个简单的示例。想象一个机器人在一个6单元格的网格中移动。假设机器人位于左上角的单元格。在这个位置，机器人只能向下或向右移动。无论选择哪个动作，它立即获得的奖励都是0。然而，如果它决定向右移动，这实际上是好的，因为它更接近目标了。我们假设目标状态是右下角的单元格G。一旦进入该状态，机器人将获得100的奖励。

这个简单的例子表明，在强化学习中，我们行动的主要影响不会立即显现，而是可能在未来。另一个例子是，如果强化学习用于直升机飞行，一个坏的操作不一定会立即导致直升机坠毁，但它可能导致一系列非常糟糕的操作，最终导致坠毁。

## 核心思想与价值函数

正因为有上面的图示，强化学习中的一个主要思想是：我们如何决定下一步的行动？应该采取什么策略？这里的一个核心思想是定义一种称为**价值函数**的东西，以便能够为每个状态计算出一个数值。这个数值可以理解为从该状态开始，在未来采取最优行动所能获得的**最优折扣奖励**的总和。

事实证明，通过尊重这些价值，我们将能够采取最优行动。我们稍后会具体讨论，这里只是给你一个关于我们将要讨论的算法类型的高层次概念。

## 应用领域

在应用方面，一些明显的例子包括机器人导航。但另一个非常酷、并使强化学习真正流行起来的应用是**游戏**。

特别是，IBM的杰拉尔多·特索罗在1995年将强化学习应用于西洋双陆棋游戏。在这种情况下，状态就是棋盘的局面。学习任务是决定下一步走哪个状态。显然，我们不会因为走了一步棋而立即获得奖励，真正的奖励在游戏结束时才出现，即我们赢或输的时候。这再次体现了延迟反馈。

事实证明，杰拉尔多·特索罗将强化学习与神经网络结合，为这个游戏开发了一种游戏玩法技术，并且该技术与人类玩家水平相当。这显示了这些技术的强大能力。

## 形式化定义

形式化定义如下：我们有一个**智能体**，它存在于某个**环境**中。智能体在任何时刻都处于某个**状态**中。

例如，一个机器人，它可能有一个目标，比如开门或送披萨。那么，机器人可能处于的状态就是它的位置、朝向等。在玩棋盘游戏时，状态就是我们所在的棋盘局面。

基本上，我们有一个智能体，它处于世界的某个状态中。在任何给定状态下，智能体可以采取一个**行动**。

例如，如果我在状态S0，智能体可以采取行动A0。当它采取行动时，机器人会获得一些**奖励**，比如奖励R0，然后进入另一个状态，比如状态S1。

接着，它决定执行另一个行动，比如行动A1，获得奖励R1，然后可能以概率方式进入另一个状态。因为当它采取一个行动时，下一个状态可能是当前状态和当前行动的**概率函数**。

然后，机器人可能处于状态S2，依此类推。这就是智能体在世界中运作的方式。

## 目标

那么，我们进行学习的目标是什么？目标是让机器人学会选择行动，以最大化**长期折扣奖励**。目标有多种，但在今天的课程中，我们将专注于一个特定的目标：让机器人选择行动，以最大化其长期折扣奖励。

这是什么意思呢？我们有一个**折扣参数 γ**。可以认为γ是0.9，这将是贯穿本讲座的例子。所以，让γ等于0.9。机器人想要最大化长期折扣奖励。

长期折扣奖励具体指什么？如果我们执行的行动序列如上所述：在状态S0执行行动A0，获得奖励R0，进入状态S1，执行行动A1，获得奖励R1，依此类推。如果我们观察到的奖励序列是R0, R1, R2,...，那么由折扣因子γ折扣后的长期奖励就是以下表达式：

**总奖励 = R₀ + γR₁ + γ²R₂ + γ³R₃ + ...**

其中，我们在第一时间点立即获得的奖励是R₀，时间点1的奖励乘以γ，时间点2的奖励乘以γ²，依此类推。一般来说，我们在时间T获得的奖励要乘以γ^T。

之所以这样定义，是因为γ是一个介于0和1之间的数。随着时间的推移，我对在时间T获得的奖励赋予的权重越来越小（因为γ^T随着T增大而减小）。可以这样理解这个目标：我们确实希望获得很多奖励，但我们更关心立即获得的奖励。这就是长期折扣奖励所捕捉的含义。

## 总结

本节课中，我们一起学习了强化学习的基本概念。我们了解到强化学习关注的是智能体通过与环境交互、采取行动来最大化长期累积奖励。其核心特点包括**延迟反馈**，以及通过定义**价值函数**来指导行动决策。我们介绍了形式化描述强化学习问题的**马尔可夫决策过程**框架，并明确了智能体的目标是最大化**长期折扣奖励**。最后，我们还看到了强化学习在机器人导航和游戏等领域的强大应用。
