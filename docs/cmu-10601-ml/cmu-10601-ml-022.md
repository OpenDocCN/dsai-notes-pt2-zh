# 022：学习数据表示

在本节课中，我们将学习一个关于“学习数据表示”的有趣话题。讨论将完全基于无监督学习，并涵盖几种学习数据表示的具体技术，即主成分分析和核主成分分析。此外，我们还会在一张幻灯片中提及独立成分分析，这是另一种常见但更高级的技术，因此不会深入细节。

## 动机：为何需要学习数据表示？

正如我们现在所知，当今数据已进入“大数据”时代，我们拥有海量的数据点。不仅如此，在许多有趣的应用中，数据维度也非常高。

以下是几个例子：
*   **文档分类**：经典的“词袋”表示法为英语词典中的每个单词创建一个特征，这就有数千个特征。在自然语言处理中，人们甚至使用“二元语法”特征，即考虑所有可能的相邻单词对，这会导致特征数量激增至数百万。如果再加入其他上下文信息，数据维度会更高。
*   **推荐系统**：以Netflix为例，我们可能有约50万用户和2万部电影。这可以看作是约50万个样本，每个样本在2万维空间中。
*   **脑成像**：通过光谱仪测量大脑不同区域的活动，并以毫秒级频率采样，会产生非常高维的数据。
*   **图像数据**：高分辨率图像数据同样具有非常高的维度。

因此，在现代背景下，我们拥有海量且通常维度极高的数据。在这种情况下，尝试学习数据中具有洞察力但低维度的表示就变得非常有用。这正是我们今天要讨论的主题。

## 学习数据表示的益处

上一节我们介绍了高维数据的普遍性，本节中我们来看看学习低维表示为何有益。以下是几个主要原因：

1.  **计算效率**：低维表示可以更有效地利用计算资源，包括运行时间、内存和存储空间。在分布式计算系统中，通信开销也可能降低。
2.  **可视化与理解**：高维稀疏数据难以可视化或理解。降维有助于我们更好地观察和把握数据的本质。
3.  **统计与泛化性能**：粗略地说，所需的样本数量可能与数据维度相关。通过提取捕获数据本质的少数维度，我们可能获得更好的统计性质和泛化能力。
4.  **降噪**：数据可能存在于一个低维流形或子空间中，并带有噪声。一些降维或表示学习技术可以帮助我们消除这些噪声，使数据结构更清晰。
5.  **提升其他算法效率**：获得数据的低维表示后，我们可以更高效地运行后续的机器学习算法，如聚类、回归或有监督分类。

总而言之，学习数据的低维表示在许多场景下都大有裨益。

## 主成分分析概述

在了解了动机和益处之后，本节我们来看看最基础的技术之一：主成分分析。

主成分分析是一种无监督学习技术，仅使用未标记的样本来从高维数据集中提取结构。这里所说的“结构”特指方差结构。PCA的目标是找到一个低维子空间，该子空间能够尽可能多地捕获高维数据的方差或变化信息。

另一种表述方式是：PCA是一种正交投影或变换，将数据映射到一个可能更低维的子空间，使得投影数据在该子空间中的方差在所有可能同维度的子空间选择中是最大的。

## PCA简单示例

上一节我们介绍了PCA的核心思想，本节中我们通过一个简单例子来具体看看。考虑一个数据点生活在二维空间中的例子，即每个数据点有两个特征。

假设我们有一组二维数据点，它们大致沿着一条斜线分布。虽然数据点在两个维度上都有取值，但大部分的变化（方差）都集中在某个特定的方向上。PCA的目标就是找到这个方向（即第一主成分），并将数据投影到这个方向上，从而实现从二维到一维的降维，同时最大程度地保留原始数据的方差信息。

通过这个例子，我们可以直观理解PCA如何通过寻找数据中方差最大的方向来提取主要特征，并实现有效的降维。

## 总结

本节课中，我们一起学习了数据表示学习的基本概念。我们首先讨论了在大数据和高维数据背景下，学习低维表示的动机和多种益处，包括提升计算效率、改善可解释性、增强泛化能力和降噪等。接着，我们重点介绍了主成分分析这一基础且强大的无监督技术，它通过寻找数据中方差最大的方向进行正交投影，以实现降维并保留关键信息。最后，我们通过一个简单的二维示例直观展示了PCA的工作原理。
