# 010：学习理论 III

在本节课中，我们将深入探讨机器学习中的学习理论，重点关注计算与数据量之间的关系。我们将回顾样本复杂度的概念，并详细分析在“可实现”与“不可知”两种不同假设下，学习算法性能的理论保证有何不同。

## 分布学习模型

上一节我们介绍了学习理论关心的核心问题。本节中，我们来看看用于形式化分析这些问题的框架——分布学习模型。该模型也是PAC模型和统计学习理论框架的基础。

在此框架下，我们假设存在一个未标记样本空间上的分布。例如，在垃圾邮件检测任务中，这个分布就是所有可能的电子邮件消息的分布。在训练阶段，我们首先从该分布中随机抽取样本，并由专家（例如，人工判断是否为垃圾邮件）进行标记，然后将这些带标签的样本作为输入提供给学习算法。算法输出一个假设函数，该函数随后将用于预测来自同一数据源的新样本。

我们显然关心获得具有良好泛化误差的分类器。我们讨论的误差概念是0-1损失或分类误差损失，其定义为：对于一个从相同底层源中随机抽取的样本，我们的分类器预测结果与目标函数不同的概率。

鉴于本幻灯片内容已多次讨论，此处不再赘述细节。

## 归纳偏置与假设空间

在此分布学习场景中，我们通常通过固定一个假设空间 **H**（例如线性分类器或决策树）来讨论归纳偏置，并尝试与假设空间 **H** 中的最佳分类器竞争。

这里的隐含假设是，我们相信目标函数接近我们假设空间或类别中的某个分类器。

我们讨论过两种不同的情况：
*   **可实现情况**：我们假设目标函数确实属于假设空间 **H**。这简化了底层的数学分析，但仍抓住了许多关键问题的本质。
*   **不可知情况**：目标函数不一定属于概念空间，但接近概念空间中的某个函数。

## 样本复杂度回顾

在过去的几讲中，我们在此框架下了解了样本复杂度。我们想知道，为了确信在训练集上优化得到的模型在未来也能表现良好，我们需要多少样本。

在可实现情况下，即目标函数确实属于我们的概念空间时，我们已为一个非常通用的算法类型证明了样本复杂度界限。

对于一个**一致学习器**，其本质是输出一个与训练集中所有数据都一致的分类器，即完美正确分类训练集的分类器。这是一个通用算法，逻辑回归或决策树等都可以是其具体实例，唯一要求是输出的分类器能完美标记训练数据。

对于此类一致学习器，我们得到了非常通用的样本复杂度界限，无论 **H** 是有限还是无限。
*   当 **H** 有限时，所需样本数仅与 **H** 大小的对数相关，这非常有利，意味着即使假设空间很大，也只需要相对较少的训练样本。
*   当 **H** 无限时，显然不能使用 **H** 的大小。但我们上次看到，可以用一个称为**VC维**的良好组合量来代替，它量化了概念空间的有效维度。

我们得到了类似的样本复杂度界限，基本上就是用 **H** 的VC维替换了 **log|H|**，并增加了一个 **log(1/ε)** 因子。虽然不确定这是否绝对最优，但原则上我们得到了非常通用且良好的复杂度界限。

## VC维简要回顾

VC维是概念空间的一个组合性质。对于无限假设空间（例如线性分类器、决策树等大多数有趣案例），VC维定义为：假设空间 **H** 能够“打散”的最大样本集的大小。所谓“打散”，是指 **H** 中的假设能够对该样本集的所有可能标记方式都实现完美分类。

例如，在 **R^d** 空间中的线性分类器，其VC维就是 **d**。这使我们能立即得到输出线性分类器的算法的样本复杂度界限。

需要强调的是，在可实现情况下，样本复杂度对 **1/ε** 的依赖是**线性**的。具体界限形式如下：如果训练阶段看到的样本数 **m** 满足：
```
m ≥ O( (1/ε) * (VC(H) + log(1/δ)) )
```
那么我可以确信，以至少 **1-δ** 的概率，所有完美标记数据的假设的真实误差最多为 **ε**。

## 不可知情况下的学习

可实现情况的理论较为简洁，但从实践角度看，**不可知情况**——即目标函数不一定属于我们的假设空间——更为重要。今天我们将更仔细地探讨这种情况，因为它非常重要。

考虑目标函数不一定属于我们概念空间的情况，这被称为不可知情况。显然，在这种情况下，我们可能无法找到一个与数据完全一致的分类器。

我们可以采取的策略是，尝试找到一个在训练集上经验误差最小的分类器，然后输出它。这被称为**经验风险最小化**算法，是一种非常自然的算法。

问题是，我们能否为此类算法提供样本复杂度界限？答案是肯定的，其界限形式与可实现情况非常相似，但有一个关键区别：样本复杂度对 **1/ε** 的依赖从线性变成了**平方**关系。

具体来说，我们得到的样本复杂度界限是**一致收敛**界。其含义是：如果训练样本数 **m** 满足：
```
m ≥ O( (1/ε²) * (VC(H) + log(1/δ)) )
```
那么以高概率，对于我概念空间中的所有分类器，其真实误差与经验误差的差异都在 **ε** 之内。

显然，如果我们能保证这样的界限成立，那么执行经验风险最小化就是合理的。

## 总结

本节课中，我们一起学习了机器学习理论的核心框架。我们回顾了分布学习模型、归纳偏置与假设空间的概念，并重点比较了可实现情况与不可知情况下的学习理论。关键点在于：
1.  在可实现情况下，一致学习器的样本复杂度与 **1/ε** 呈线性关系。
2.  在更一般的不可知情况下，经验风险最小化算法的样本复杂度与 **1/ε²** 呈平方关系，这通过一致收敛界来保证。
理解这些理论界限有助于我们认识机器学习算法所需的数据量及其泛化性能的理论基础。
