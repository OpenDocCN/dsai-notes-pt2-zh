# 006：逻辑回归 🧠

在本节课中，我们将学习逻辑回归模型。上一节我们介绍了高斯朴素贝叶斯分类器，它是一种基于概率生成模型的分类方法。本节中，我们将探讨一种与之紧密相关但属于判别模型的强大算法——逻辑回归。我们会从高斯朴素贝叶斯的决策边界出发，自然地推导出逻辑回归的形式，并详细解释其核心概念、训练方法以及优缺点。

## 概述

逻辑回归是一种广泛用于二分类问题的线性模型。尽管名字中有“回归”，但它实际上是一种分类算法。其核心思想是直接对给定输入特征后属于某个类别的**条件概率**进行建模，而不是像朴素贝叶斯那样先对联合概率分布建模。我们将看到，在特定假设下，高斯朴素贝叶斯的决策函数可以转化为逻辑回归的形式。

## 从高斯朴素贝叶斯到逻辑回归

上一节我们介绍了高斯朴素贝叶斯。回忆一下，对于一个二分类问题（Y=1 或 Y=0），我们假设每个连续特征 $X_i$ 在给定类别 Y 的条件下服从高斯分布。在预测时，我们计算后验概率 $P(Y=1|X)$ 和 $P(Y=0|X)$，并选择概率更大的类别。

这个决策过程可以通过比较一个“分数”来实现。具体来说，我们预测 Y=1 当且仅当：
$$P(Y=1|X) > P(Y=0|X)$$

利用贝叶斯定理和朴素贝叶斯假设，这个不等式可以转化为：
$$\frac{P(Y=1) \prod_{i} P(X_i|Y=1)}{P(X)} > \frac{P(Y=0) \prod_{i} P(X_i|Y=0)}{P(X)}$$

两边取对数并消去分母 $P(X)$，我们得到一个线性决策函数。预测 Y=1 当且仅当：
$$\log \frac{P(Y=1)}{P(Y=0)} + \sum_{i} \log \frac{P(X_i|Y=1)}{P(X_i|Y=0)} > 0$$

如果我们进一步假设**所有特征 $X_i$ 在给定 Y 的条件下的方差都相等**（即 $\sigma^2$ 相同），那么这个求和项会大大简化。此时，$\log \frac{P(X_i|Y=1)}{P(X_i|Y=0)}$ 会变成一个关于 $X_i$ 的线性函数。最终，整个决策条件可以写成：
$$w_0 + \sum_{i=1}^{n} w_i X_i > 0$$
其中 $w_i$ 是由高斯分布的参数（均值、方差）和先验概率计算得到的权重。

这个不等式定义了一个**线性决策边界**：一个在特征空间中的超平面，将预测为类别1和类别0的区域分开。

## 逻辑回归模型

从上面的推导我们看到，在高斯朴素贝叶斯满足等方差假设时，其决策函数是线性的。逻辑回归直接对这个决策函数背后的**条件概率** $P(Y=1|X)$ 进行建模，并将其形式设定为输入特征的线性函数的 sigmoid 变换。

以下是逻辑回归模型的核心定义：

1.  **线性得分**：首先，计算输入特征 $X = [X_1, X_2, ..., X_n]$ 的线性组合。
    $$z = w_0 + w_1X_1 + w_2X_2 + ... + w_nX_n$$
    这里，$w_0$ 是偏置项（截距），$w_1, ..., w_n$ 是特征权重。

2.  **Sigmoid 函数**：然后，将线性得分 $z$ 通过 sigmoid 函数（也称为 logistic 函数）进行映射。sigmoid 函数将任意实数 $z$ 压缩到 (0, 1) 区间内，这个值正好可以解释为概率。
    $$\sigma(z) = \frac{1}{1 + e^{-z}}$$

3.  **条件概率**：逻辑回归模型直接定义正例（Y=1）的条件概率为：
    $$P(Y=1 | X; w) = \sigma(w^T X) = \frac{1}{1 + e^{-(w_0 + \sum_i w_i X_i)}}$$
    相应地，$P(Y=0 | X; w) = 1 - P(Y=1 | X; w)$。

因此，模型的预测规则是：如果 $P(Y=1|X) > 0.5$（即 $z > 0$），则预测为 Y=1，否则预测为 Y=0。

## 模型训练：最大似然估计

逻辑回归的参数（权重向量 $w$）需要通过训练数据来学习。与许多机器学习模型一样，我们采用**最大似然估计（MLE）** 的方法。

以下是最大似然估计的步骤：

1.  **定义似然函数**：对于一组包含 $m$ 个独立同分布样本的训练数据 ${(x^{(1)}, y^{(1)}), ..., (x^{(m)}, y^{(m)})}$，其似然函数是每个样本预测概率的乘积。
    $$L(w) = \prod_{j=1}^{m} P(Y=y^{(j)} | X=x^{(j)}; w)$$
    利用 $P(Y=1|X)=h_w(X)$ 和 $P(Y=0|X)=1-h_w(X)$ 的表示，可以将上式写为：
    $$L(w) = \prod_{j=1}^{m} (h_w(x^{(j)}))^{y^{(j)}} (1 - h_w(x^{(j)}))^{1-y^{(j)}}$$

2.  **转为对数似然**：乘积形式不便于优化，我们通常取对数得到对数似然函数，这不会改变最优解的位置。
    $$l(w) = \log L(w) = \sum_{j=1}^{m} [ y^{(j)} \log h_w(x^{(j)}) + (1-y^{(j)}) \log (1 - h_w(x^{(j)})) ]$$
    我们的目标是找到参数 $w$，使得这个对数似然函数 $l(w)$ 的值最大。

3.  **优化求解**：不幸的是，对于逻辑回归，令梯度 $\nabla_w l(w) = 0$ 得到的方程没有解析解（闭式解）。因此，我们必须使用**迭代优化算法**来寻找最优的 $w$。
    *   **梯度上升**：一种直接的方法是梯度上升。我们初始化一组权重 $w$，然后反复沿对数似然函数的梯度方向更新权重，直到收敛。
        $$w := w + \alpha \nabla_w l(w)$$
        其中 $\alpha$ 是学习率。
    *   **高级优化方法**：在实践中，更常用的是更高效的优化算法，如**牛顿法（Newton‘s Method）** 或**拟牛顿法（如L-BFGS）**。这些方法利用了目标函数（对数似然）的二阶导数（海森矩阵）信息，通常收敛速度比梯度上升快得多。许多机器学习库（如 scikit-learn）在训练逻辑回归时默认使用这类算法。

## 逻辑回归的优缺点

在深入应用之前，了解逻辑回归的特性非常重要。

以下是逻辑回归的主要优点：

*   **实现简单，高效**：模型形式简单，训练和预测的计算开销通常较低。
*   **可解释性强**：权重 $w_i$ 的大小和正负直接反映了特征 $X_i$ 对预测结果的影响方向和程度。
*   **输出概率**：它不仅给出分类结果，还输出属于各类别的概率，这对于需要不确定性估计的任务非常有用。
*   **易于扩展**：可以通过正则化（如L1、L2）来防止过拟合，也可以很容易地扩展到多分类问题（如使用 softmax 回归）。

以下是逻辑回归的一些局限性：

*   **线性决策边界**：本质上是线性模型，无法直接捕捉特征间复杂的非线性关系。虽然可以通过引入特征交叉项或多项式特征来增加非线性能力，但这需要人工设计或可能导致维度爆炸。
*   **对特征相关性和异常值敏感**：如果特征之间存在高度相关性，可能会影响权重估计的稳定性。异常值也可能对模型产生较大影响。
*   **数据假设**：当数据线性不可分时，性能可能不佳。

## 总结

本节课中我们一起学习了逻辑回归。我们从高斯朴素贝叶斯的等方差假设出发，推导出了线性决策边界，并引出了直接对条件概率 $P(Y|X)$ 进行建模的逻辑回归。其核心是将特征的线性组合通过 **sigmoid 函数** 映射为概率。我们使用 **最大似然估计** 来训练模型，并通过**梯度上升**或更高效的**牛顿法**等迭代算法来优化对数似然函数。逻辑回归以其简单、高效和可解释性强的优点，成为分类任务中一个基础而重要的工具，尽管它受限于线性的决策边界。在接下来的课程中，我们将探索能够学习更复杂非线性决策边界的模型。
