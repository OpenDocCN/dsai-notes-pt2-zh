# 017：核方法 🧠

在本节课中，我们将要学习一种称为“核方法”的非常有趣的技术。它能以一种巧妙的方式，从计算的角度处理非线性决策边界问题。我们将从回顾几何间隔和感知机算法开始讨论核方法，并在课程末尾探讨其在支持向量机（SVM）中的应用。SVM是机器学习中另一个备受推崇、应用广泛且理论基础坚实的算法。

## 回顾：感知机算法与几何间隔

上一节我们介绍了核方法的基本概念，本节中我们来看看其理论基础——感知机算法与几何间隔。

感知机算法是机器学习中最简单的算法之一，自20世纪50年代就已为人所知。它最初是在所谓的“在线学习”模型中被分析的。

在课堂上，我们讨论的大多是“批量学习”场景，即假设所有样本都是预先一次性提供给我们的。例如，在进行垃圾邮件检测或图像分类时，我们假设所有电子邮件或图像都是预先给定的。

相比之下，在在线学习模型中，我们假设样本是顺序到达的。例如，每天都会有一封电子邮件到达，我们必须立即预测它是否是垃圾邮件。然后在一天结束时，我们会被告知该样本的真实标签。这就是在线学习模型：样本是顺序出现的，我们进行多轮（可能非常多轮）学习。在每一轮中，算法首先看到一个未标记的样本 **X_i**，然后必须立即预测该样本的标签。之后，在当天结束时，真实标签被揭示，在线学习算法可以利用这些信息更新其内部状态，以便在未来处理样本时表现得更好。

从分析角度看，在这种模型中，我们对输入序列不做任何分布假设。实际上，输入序列可以是一个任意选择的、甚至可能是对抗性的序列。正因如此，我们无法真正讨论算法的“收敛”。相反，从分析的角度，我们能做的是尝试最小化错误的数量。我们希望设计出能最小化错误数量的在线算法。如果真实标签与某个目标函数一致，并且该函数属于某个结构化的函数类，我们就有可能做到这一点。

为什么我们不能讨论收敛呢？假设你认为算法在若干轮后会以某种方式收敛。但由于序列是对抗性选择的，对手可以在前三轮给出同一个样本。我们反复看到同一个样本，并没有学到新信息。三轮之后，对手可以开始给出完全不同的样本集。因此，由于是在线且序列是对抗性选择的，我们无法期望收敛到一个固定函数。但我们能做的，正如在感知机学习线性分类器的背景下所展示的，是去最小化错误的数量。

这就是在线学习模型。我们上次分析了感知机算法在该模型下学习线性分类器的表现。我们详细讨论过，但可以再次强调：我们总可以假设分类器经过原点，这并不失一般性，因为我们总可以平移原点。

该算法非常简单，是一种贪心算法。在任意时刻，我们有一个对目标函数的当前猜测，并用它来预测新样本的标签。当新样本到达时，我们使用当前猜测来预测标签：如果样本在分类器的正侧，则预测为正；否则预测为负。然后，如果我们犯了错误，就按如下方式简单更新：
*   如果对正样本犯了错误（即我们的点积为负，但本应为正），我们就将该样本加到当前猜测中，这可以增加当前猜测对该样本正确的可能性。
*   如果对负样本犯了错误，我们就从当前猜测中减去该样本。

该算法非常简单：你只需维护一个代表当前猜测的权重向量 **W**。新样本到达时，用它来预测。只有当犯错时才更新，因此它是一个保守的算法。如果犯错，就根据情况加上或减去该样本。

关于这个算法的一个有趣事实是，在任何给定时刻，我们对目标函数的当前猜测的法向量 **W_T**，都是我们迄今为止犯错样本的加权和。这很容易证明：我们从零向量开始，只有犯错时才更新权重向量（对正样本犯错则加，对负样本犯错则减）。例如，假设犯错的样本是 **X_i1, X_i2, ..., X_ik**（它们是不同的向量）。如果第一个错误是对正样本 **X_i1** 犯的，那么其对应的系数就是 +1；如果最后一个错误是对负样本 **X_ik** 犯的，那么其系数就是 -1。因此，在任何时刻，当前猜测的法向量 **W_T** 都是误分类样本的加权和。

为什么这很有趣？原因有二：
1.  为了预测一个新样本的标签，我们需要能够使用当前的目标猜测来得出标签预测。为此，我们需要计算新样本与那些犯错样本之间的点积。因为 **W_T · X** 本质上就是这个加权和与新样本的点积（根据点积的线性性质）。
2.  我们上次详细证明了感知机算法所犯错误的数量，完全取决于样本序列的几何间隔。具体来说，算法将犯的错误数量仅取决于样本序列的几何间隔，而与序列的长度（序列可以非常长）以及样本所在空间的维度（样本可以存在于非常高维的特征空间中）无关。

接下来，我们快速回顾一下几何间隔的概念，因为它对理解核方法至关重要。

几何间隔是最自然的间隔概念。如果我们的样本存在于欧几里得空间中，给定一个分类器（例如一条线或一个超平面），几何间隔就是所有样本点到该分类器边界的最小距离。它衡量了分类器将不同类别样本分开的“清晰”程度。

本节课中我们一起学习了核方法的引入及其重要的理论基础：感知机算法和几何间隔。我们回顾了在线学习模型的特点，感知机算法如何在该模型下工作并更新权重，以及一个关键性质——权重向量是误分类样本的加权和。最后，我们强调了感知机的错误界限仅依赖于样本序列的几何间隔，这为后续引入核方法以高效处理非线性问题奠定了基础。
