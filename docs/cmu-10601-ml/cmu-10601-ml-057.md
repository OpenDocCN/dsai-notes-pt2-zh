# 057：概率图模型基础

在本节课中，我们将学习概率图模型的基础知识。我们将从概率论的核心概念开始，逐步理解什么是概率模型，并介绍联合分布、先验、似然和后验等关键术语。这些知识是理解更复杂图模型并进行推理的基础。

## 概率论基础回顾

要理解图模型，首先需要扎实的概率论基础。本节我们将回顾一个核心概念：如何从联合概率分布计算单个变量的概率。

假设我们有两个骰子，我们掷出它们。我们知道两个骰子点数组合的联合概率。例如，我们知道第一个骰子为1且第二个骰子为3的概率，也知道第一个骰子为6且第二个骰子为1的概率。我们可以将所有组合的概率列在一个表格中。

如果我们想计算**单个骰子**取某个特定值的概率，例如计算第一个骰子（`die1`）等于4的概率，我们需要对另一个骰子（`die2`）所有可能的取值进行求和。

用公式表示如下：
`P(die1 = 4) = Σ_{v} P(die1 = 4, die2 = v)`
其中，`v` 代表第二个骰子所有可能的取值（1到6）。

这个“边缘化”或“求和”操作是概率论中的一个基本事实，我们将在图模型的推理中反复使用它。

## 什么是概率模型？

在深入图模型之前，我们先明确一个更基本的概念：概率模型。

一个概率模型是一组**相互关联的随机变量**的集合。例如，第一个骰子和第二个骰子就构成了一个简单的概率模型。

我们可以直观地将其视为**确定性模型**的扩展。在确定性模型中，例如方程 `y = 2x + 1`，给定 `x` 的值，我们可以精确计算出 `y` 的值。这类方程在描述物理现象等确定性世界时非常有用。

概率模型则将变量视为**随机**的。例如，如果 `x` 服从均值为0、方差为1的正态分布，那么 `y` 的分布会是什么样？这在描述具有**不确定性或随机性**的现象时非常有用。现实世界中的许多现象（也是机器学习试图建模的对象）都充满了不确定性，概率模型能帮助我们捕捉并处理这种不确定性。

## 关键概率术语定义

在概率模型中，我们经常使用联合、先验、似然和后验这些术语。以下是它们的精确定义：

假设我们的概率模型包含两类随机变量：
*   **观测变量**：`x1, x2, ..., xn`（在机器学习中，这通常是你的数据）。
*   **未知（隐藏）参数**：`θ1, θ2, ..., θm`。

以下是基于这些变量的关键定义：

*   **联合分布**：这是模型中**所有**随机变量（包括观测变量和未知参数）的分布。
    *   公式：`P(x1, ..., xn, θ1, ..., θm)`。
    *   为简洁起见，常写作 `P(X, θ)`，其中 `X` 代表所有 `x`，`θ` 代表所有 `θ`。

现在，假设我们已经观测到了所有 `x` 变量的值，我们可以定义以下三个重要概念：

*   **先验**：这是**未知**随机变量（即参数 `θ`）的概率分布，它表达了我们在看到任何数据之前对参数的信念。
    *   公式：`P(θ)`。
    *   我们也可以讨论单个参数 `θi` 的先验 `P(θi)`。

*   **似然**：这是在给定未知参数 `θ` 的特定取值时，观测到当前数据 `X` 的概率。
    *   公式：`P(X | θ)`。
    *   当我们进行**最大似然估计**时，我们试图找到能使这个 `P(X | θ)` 值最大的参数 `θ`。

*   **后验**：这是在已经观测到数据 `X` 之后，未知参数 `θ` 的概率分布。它结合了先验信念和观测到的数据证据。
    *   公式：`P(θ | X)`。
    *   根据贝叶斯定理，后验与先验和似然的乘积成正比：`P(θ | X) ∝ P(X | θ) * P(θ)`。
