# 053：Octave基础与矩阵操作

在本节课中，我们将学习Octave编程环境的基础知识，包括基本运算、变量赋值、矩阵与向量的创建、随机数生成以及矩阵索引操作。Octave是一款免费开源的数值计算软件，语法与Matlab高度相似。

## 基本运算与变量赋值

首先，我们可以执行基本的数学运算，例如加法、减法、乘法和除法。输入运算后，结果会直接显示。

```octave
3 + 5
7 - 2
4 * 6
8 / 2
```

变量赋值通过等号 `=` 完成。若语句末尾不加分号，变量值会立即显示；添加分号则抑制输出，这在编写脚本时非常有用。

```octave
x = 3;
y = 5.0;
z = 'hello';
```

要打印变量值，可以直接输入变量名，或使用 `disp` 函数。`disp` 函数输出更简洁。

```octave
x
disp(x)
disp(z)
```

结合 `sprintf` 函数可以格式化输出字符串。`sprintf` 使用占位符如 `%d`（整数）、`%f`（浮点数）和 `%s`（字符串）来插入变量值。

```octave
disp(sprintf('x 的值是 %d， y 的值是 %f', x, y))
```

## 创建矩阵与向量

上一节我们介绍了基本操作，本节中我们来看看如何创建矩阵和向量。

矩阵可以通过手动输入元素来创建。空格或逗号分隔同一行的元素，分号 `;` 表示新的一行。

```octave
A = [1 2 3; 4 5 6; 7 8 9];
```

以下是其他创建特殊矩阵的函数：

*   `zeros(m, n)`： 创建 `m` 行 `n` 列的全零矩阵。
*   `ones(m, n)`： 创建 `m` 行 `n` 列的全一矩阵。要创建全为其他值的矩阵，可以将其与 `ones` 相乘，例如 `3 * ones(2,2)`。
*   `eye(n)`： 创建 `n` 阶单位矩阵（主对角线为1，其余为0）。

向量是只有一行或一列的矩阵。

```octave
row_vector = [1, 2, 3, 4, 5]; % 行向量
column_vector = [1; 2; 3; 4; 5]; % 列向量
```

使用冒号 `:` 运算符可以创建等间距的向量。

```octave
v = 0:1:10; % 从0开始，步长为1，到10结束
```

矩阵可以通过方括号进行拼接。逗号 `,` 或空格实现水平拼接，分号 `;` 实现垂直拼接。

```octave
B = [1 2; 3 4];
C = [5 6; 7 8];
D_horizontal = [B, C]; % 水平拼接
D_vertical = [B; C];   % 垂直拼接
```

## 生成随机数

在数据处理和模型初始化中，经常需要生成随机数。Octave提供了相关的函数。

`rand(m, n)` 函数生成 `m` 行 `n` 列的矩阵，其元素服从 `[0, 1)` 区间上的均匀分布。

```octave
R_uniform = rand(2, 3);
```

通过线性变换可以改变均匀分布的范围。

```octave
R_uniform_scaled = 1 + rand(2, 3); % 范围变为 [1, 2)
```

`randn(m, n)` 函数生成 `m` 行 `n` 列的矩阵，其元素服从标准正态分布（均值为0，标准差为1）。

```octave
R_normal = randn(5, 1);
```

同样，可以通过线性变换改变正态分布的参数。

```octave
mu = 5;
sigma = 2;
R_normal_scaled = mu + sigma * randn(5, 1); % 均值5，标准差2
```

## 矩阵尺寸与索引

了解矩阵的尺寸和如何提取其中的元素至关重要。

`size(A)` 函数返回矩阵 `A` 的行数和列数。`size(A, 1)` 和 `size(A, 2)` 分别返回行数和列数。

```octave
rows = size(A, 1);
cols = size(A, 2);
```

`length(v)` 函数返回向量 `v` 中元素的个数。对矩阵使用会返回行数和列数中较大的一个，通常不推荐。

矩阵索引使用圆括号 `()`，格式为 `A(row, column)`。

```octave
element = A(2, 3);       % 获取第2行第3列的元素
whole_row = A(2, :);     % 获取第2行所有元素
whole_col = A(:, 3);     % 获取第3列所有元素
```

可以使用范围或特定索引来获取子矩阵。

```octave
sub_matrix1 = A(1:5, :);           % 获取第1到5行，所有列
sub_matrix2 = A(1:5, 1:2);         % 获取第1到5行，第1到2列
sub_matrix3 = A([1, 5, 10], :);    % 获取第1, 5, 10行，所有列
sub_matrix4 = A(:, [2, 4]);        % 获取所有行，第2, 4列
```

提取的子矩阵可以赋值给新的变量，这在数据分割（如分为训练集和测试集）时非常有用。

```octave
training_set = A(1:80, :);
test_set = A(81:end, :);
```
