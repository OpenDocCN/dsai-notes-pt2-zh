# 021：聚类分析

在本节课中，我们将要学习聚类分析的基本概念、目标以及应用。我们将从非正式定义开始，探讨聚类的目的，并介绍几种常见的聚类目标，特别是K均值目标。最后，我们将讨论解决K均值问题的计算复杂性。

---

## 什么是聚类？

聚类的目标是在无标签数据中，自动将数据点划分为相似的组。我们只有未标记的示例，希望以某种方式自动进行划分，使得相似的点归入相似的组。

为什么需要这样做？原因有多种。例如，我们可能希望自动组织大量数据，或者理解数据中隐藏的结构，这更像是无监督学习。第二个目标，或者我们可能只是将聚类作为进一步数据分析的预处理步骤。例如，在数据可视化之前进行聚类，可能使可视化更容易。

## 应用领域

聚类在许多领域都有应用。以下是几个例子：
*   按主题聚类新闻文章、网页或搜索结果。
*   按功能聚类蛋白质序列，或根据表达谱聚类基因。
*   根据兴趣聚类社交网络中的用户，这通常被称为社区检测。
*   根据购买历史聚类客户，以便后续做出更好的定价等决策。
*   使用斯隆数字巡天等数据聚类星系和邻近恒星。

这些只是应用示例，实际上聚类出现在所有领域。因此，理解这类算法很重要。

之前已经讨论过一种通过EM算法进行聚类的特定方法，但该算法没有近似保证，并且其推导非常特定于混合高斯等概率模型。今天，我们将更多地从算法角度讨论聚类，包括基于目标的聚类算法和层次聚类。

## 基于目标的聚类概述

现在，让我们开始介绍基于目标的聚类。首先，对基于目标的聚类进行一个总体概述，然后重点介绍K均值算法或K均值目标，这可能是机器学习中最流行的目标。

更一般的问题定义如下：我们有一个包含n个点的集合S，以及一个距离或相异性度量，用于指定点对之间的距离。这个度量可以由领域给出，例如蛋白质序列的编辑距离，也可以是基于共同关键词数量（用于新闻文章聚类），或基于相关系数（用于图像聚类）。如果点嵌入在R^D中，它也可以是欧几里得空间中的自然距离概念。

传统上，基于目标的聚类的目标是输出数据的一个划分。人们考虑过多种类型的目标。最流行的是K均值目标。这里的定义可以写得更一般化，稍后将专门讨论机器学习中最流行的欧几里得情况。

更一般地，我们只有点集S和满足三角不等式的距离函数作为输入。目标是输出中心点C1, C2, ..., CK（这些中心点本身可能不是数据点）以及将点划分为簇的方式，以最小化所有点到其最近中心点的平方距离之和。

这就是K均值目标。我们想输出一个划分，并为每个部分输出一个中心点。每个点被分配到最近的中心点，我们希望最小化所有点到其对应中心点的平方距离之和。

## 其他聚类目标

除了K均值，还有其他目标。以下是另外两个常见目标：
*   **K中位数**：与K均值类似，但目标是最小化点到其对应中心点的距离之和（线性惩罚而非二次惩罚）。这在运筹学和理论计算机科学中更常用。
*   **K中心点**：目标是找到数据的划分，以最小化最大半径。

还有许多其他目标，例如最小化簇内距离之和等。但在机器学习中，最流行的是K均值目标。

## 欧几里得K均值问题

现在，让我们专门讨论机器学习中最常见的欧几里得K均值问题。在这种情况下，我们假设数据点是R^D维欧几里得空间中的点。

输入是一个点集（在n维空间中）和目标簇的数量K。K的来源可以稍后讨论。目标是输出K个中心点C1, C2, ..., CK，它们可以是R^D中的任意点，不一定是输入点。

然后，我们希望将每个点分配到离它最近的中心点，并找到一组中心点，使得所有点到其对应中心点的平方距离之和最小。

这就是K均值目标。我相信你们都见过，因为它非常流行且被广泛使用。

## 沃罗诺伊划分

实际上，如果我们考虑一下，这里所做的就是给定一组中心点C1, C2, ..., CK，我们将每个点分配到其最近的中心点。这非常自然。

给定一组中心点，例如这里有三个中心点C1、C2和C3，如果我们进行这种分配，将每个点分配到离它最近的中心点，那么得到的就是这种沃罗诺伊划分。这样就将空间划分成紧凑的单元，每个单元称为一个沃罗诺伊胞腔。
