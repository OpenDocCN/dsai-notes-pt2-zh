# 004：朴素贝叶斯分类器

在本节课中，我们将要学习第一个贝叶斯分类器——朴素贝叶斯。我们将从回顾概率估计的基本原理开始，然后详细探讨如何构建和应用朴素贝叶斯模型。

---

上一节我们回顾了从数据中估计概率参数的两种核心方法。本节中，我们来看看如何将这些原理应用于构建一个具体的分类器。

## 概率估计回顾

估计概率参数是任何概率化机器学习方法的核心。主要有两个原则：

1.  **最大似然估计**：选择参数 `θ`，使得在给定 `θ` 的条件下，观测到数据的概率 `P(数据 | θ)` 最大。
    *   **公式**：`θ_MLE = argmax_θ P(数据 | θ)`

2.  **最大后验估计**：选择在给定数据条件下，自身概率最大的参数 `θ`。
    *   根据贝叶斯规则：`P(θ | 数据) = [P(数据 | θ) * P(θ)] / P(数据)`
    *   由于分母 `P(数据)` 与 `θ` 无关，最大化后验概率等价于最大化分子。
    *   **公式**：`θ_MAP = argmax_θ P(数据 | θ) * P(θ)`

我们以伯努利分布（如抛硬币）为例进行了说明。若观测到 `α1` 次正面和 `α0` 次反面，则：
*   **数据似然**：`P(数据 | θ) = θ^(α1) * (1-θ)^(α0)`
*   **MLE估计**：`θ_MLE = α1 / (α1 + α0)`
*   **MAP估计**：若选择 **Beta分布** 作为先验 `P(θ)`，其形式为 `P(θ) ∝ θ^(β1-1) * (1-θ)^(β0-1)`，则 MAP 估计结果为：
    *   **公式**：`θ_MAP = (α1 + β1 - 1) / (α1 + β1 + α0 + β0 - 2)`
    *   这相当于在观测计数中加入了“虚构”的 `β1-1` 次正面和 `β0-1` 次反面。Beta分布是二项分布的**共轭先验**，这使得数学形式非常简洁。

理解了参数估计的基础后，我们现在可以将其扩展到分类任务。本节将介绍朴素贝叶斯分类器的核心思想。

## 朴素贝叶斯分类器 😊

朴素贝叶斯是一个基于贝叶斯定理的简单但强大的分类算法。它的目标是：给定一个具有多个特征的数据点，预测它最可能属于哪个类别。

以下是其工作原理的核心步骤：

1.  **贝叶斯定理应用**：对于类别 `y` 和特征向量 `x = (x1, x2, ..., xn)`，我们想找到使后验概率 `P(y | x)` 最大的 `y`。
    *   **公式**：`y_pred = argmax_y P(y | x) = argmax_y [P(x | y) * P(y) / P(x)]`
    *   由于对于所有类别 `y`，分母 `P(x)` 是相同的，因此可以简化为：
    *   **公式**：`y_pred = argmax_y P(x | y) * P(y)`

2.  **“朴素”条件独立性假设**：直接估计 `P(x | y)` 非常困难，因为特征组合的空间太大。朴素贝叶斯做出了一个关键假设：**在给定类别 `y` 的条件下，所有特征 `x1, x2, ..., xn` 彼此独立**。
    *   基于此假设，联合概率可以分解为单个特征概率的乘积：
    *   **公式**：`P(x | y) = P(x1 | y) * P(x2 | y) * ... * P(xn | y)`

3.  **分类决策规则**：将独立性假设代入，我们得到最终的分类决策公式：
    *   **公式**：`y_pred = argmax_y P(y) * Π_i P(xi | y)`
    *   其中 `Π_i` 表示对所有特征 `i` 求乘积。

---

要实际构建一个朴素贝叶斯分类器，我们需要从训练数据中估计模型中的所有参数。以下是需要估计的组成部分。

## 模型参数估计

根据分类决策规则，我们需要估计两类参数：

1.  **类先验概率 `P(y)`**：这很简单，就是每个类别在训练数据中出现的频率。
    *   **公式**：`P(y = c) = (训练集中类别为 c 的样本数) / (训练集总样本数)`

2.  **条件概率 `P(xi | y)`**：这取决于特征 `xi` 的类型（离散或连续）。
    *   **对于离散特征**：使用计数并平滑（如拉普拉斯平滑，相当于MAP估计中的先验），以避免出现零概率。
        *   **公式**：`P(xi = v | y = c) = (类别 c 中特征 i 取值为 v 的样本数 + α) / (类别 c 的样本数 + α * k)`
        *   其中 `k` 是特征 `i` 可能取值的数量，`α` 是平滑参数（通常为1，即拉普拉斯平滑）。
    *   **对于连续特征**：通常假设其服从某种分布，如高斯分布，然后估计该分布的参数。
        *   **公式**：假设 `P(xi | y=c) ~ N(μ_c,i, σ²_c,i)`，则用类别 `c` 中特征 `i` 的样本均值和方差来估计 `μ` 和 `σ²`。

现在我们已经了解了模型的原理和参数估计方法。最后，我们来总结一下朴素贝叶斯的优缺点和应用流程。

## 总结与应用

本节课中我们一起学习了朴素贝叶斯分类器。

**总结**：
*   **核心思想**：基于贝叶斯定理，利用“特征条件独立”的朴素假设，将复杂的联合概率估计分解为简单的单个特征概率估计的乘积。
*   **训练过程**：从训练数据中估计类先验 `P(y)` 和每个特征的条件概率 `P(xi | y)`。
*   **预测过程**：对于新样本，计算其属于每个类别的“未归一化后验概率” `P(y) * Π_i P(xi | y)`，并选择概率最大的类别作为预测结果。

**优点**：
*   简单、高效，训练和预测速度都非常快。
*   对于小规模数据集和高维数据（如文本分类）表现良好。
*   对缺失数据不太敏感。

**缺点**：
*   “特征条件独立”的假设在现实中往往不成立，这可能会影响模型精度。
*   当特征之间存在强相关性时，模型性能可能下降。

**典型应用**：
*   垃圾邮件过滤
*   文本分类（如新闻分类）
*   情感分析
*   简单推荐系统

尽管其假设“朴素”，但朴素贝叶斯因其简单性和实效性，至今仍是机器学习中一个重要的基础分类算法。
