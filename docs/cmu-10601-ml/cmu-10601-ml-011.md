## 偏差与方差：统计学视角

在之前的课程中，我们多次提及“偏差”与“方差”，但并未给出精确定义。现在，让我们从统计学家的角度来审视它们。

假设我们有一个待估计的参数 θ。例如，在抛硬币的例子中，θ 代表硬币正面朝上的真实概率。我们有一个算法或**估计器**，它会根据数据产生一个对 θ 的估计值，记作 θ̂。

### 偏差的定义

一个估计器的**偏差**，定义为该估计值的**期望**与参数**真实值**之间的差异。

**公式**：`Bias(θ̂) = E[θ̂] - θ`

其中，`E[θ̂]` 表示估计值 θ̂ 的期望（即多次重复实验所得估计值的平均值）。

### 方差的定义

一个估计器的**方差**，衡量的是单次实验得到的估计值围绕其期望值的波动程度。

**公式**：`Var(θ̂) = E[(θ̂ - E[θ̂])²]`

### 一个具体例子：最大似然估计

让我们以抛硬币为例。假设我们的估计器是：抛掷硬币 n 次，记录正面朝上的次数 k，然后用 **最大似然估计** 来估计 θ，即 θ̂_MLE = k / n。

对于这个 MLE 估计器：
*   **偏差**：可以证明，`E[θ̂_MLE] = θ`。因此，`Bias(θ̂_MLE) = θ - θ = 0`。我们称这个估计器是**无偏**的。
*   **方差**：对于伯努利分布，该估计器的方差为 `Var(θ̂_MLE) = θ(1-θ) / n`。

这个结果非常直观：**方差与抛掷次数 n 成反比**。这意味着，为了获得更可靠（即更接近真实值）的估计，我们需要更多的数据（更大的 n）。

上一节我们回顾了偏差与方差的统计学定义，并看到了数据量如何影响估计的精度。这为我们理解机器学习模型的复杂度与泛化能力之间的关系奠定了基石。现在，让我们正式进入本节课的核心主题——图模型。

## 图模型简介

图模型是概率论与图论相结合的强大框架。它使用图（由**节点**和**边**构成）来直观地表示一组随机变量之间的复杂依赖关系。

### 为什么使用图模型？

图模型主要有两大优势：
1.  **直观表示**：图提供了一种人类可理解的“语言”，来可视化变量间如何相互影响。这极大地帮助了模型的设计、理解和沟通。
2.  **高效计算**：图结构揭示了变量之间的条件独立性。利用这些独立性，我们可以将复杂的联合概率计算分解为更小的、更易处理的局部因子的乘积，从而进行高效的概率推理。

### 图模型的核心组件

一个图模型包含两个部分：
*   **图结构**：这是一个由节点和边组成的图。每个节点代表一个随机变量。边则表示变量之间的依赖或关联关系。
*   **参数**：这些是与图结构相关联的数字，用于量化变量之间的依赖强度。例如，在贝叶斯网络中，参数就是条件概率表。

以下是图模型中常见的两种基本图结构及其含义：

*   **有向边**：通常用箭头 `→` 表示。`A → B` 表示变量 A 对变量 B 有直接的因果或影响关系。基于有向图的模型称为**贝叶斯网络**。
*   **无向边**：用连线 `—` 表示。`A — B` 表示变量 A 和 B 之间存在某种关联或相互作用，但不指定方向。基于无向图的模型称为**马尔可夫随机场**。

在本节课中，我们将重点介绍**贝叶斯网络**。

## 贝叶斯网络

贝叶斯网络是一种使用**有向无环图**来表示一组变量及其条件依赖关系的概率图模型。

### 关键概念：条件独立性

贝叶斯网络的核心是编码变量之间的**条件独立性**假设。图中的缺失边（即没有箭头连接）通常意味着在给定某些其他变量的条件下，这两个变量是独立的。

例如，考虑三个变量：下雨、洒水器开启、草地湿滑。
*   下雨会导致草地湿滑。
*   洒水器开启也会导致草地湿滑。
*   但下雨和洒水器开启通常彼此独立（除非有人看到草地湿滑才关掉洒水器，但这在简单模型中可以忽略）。

在贝叶斯网络中，这可以表示为：`Rain → GrassWet` 和 `Sprinkler → GrassWet`。`Rain` 和 `Sprinkler` 之间没有边，表示它们（在未观测到`GrassWet`时）是独立的。

### 联合概率的分解

贝叶斯网络最强大的特性是，它允许我们将整个系统的联合概率分布分解为一系列局部条件概率的乘积。

对于一个有 n 个变量 `X1, X2, ..., Xn` 的贝叶斯网络，其联合概率可以分解为：

**公式**：`P(X1, X2, ..., Xn) = Π_i P(X_i | Parents(X_i))`

其中，`Parents(X_i)` 表示在图中直接指向 `X_i` 的所有父节点。这个公式意味着，每个变量的概率只依赖于其直接的父节点，而与图中的其他非后代节点条件独立（在给定父节点的条件下）。

### 一个简单例子

假设我们有三个布尔变量：`A`， `B`， `C`。图结构为 `A → B ← C`。即，`B` 有两个父节点 `A` 和 `C`，而 `A` 和 `C` 之间没有边（相互独立）。

根据贝叶斯网络的分解规则，这个模型的联合概率为：

**公式**：`P(A, B, C) = P(A) * P(C) * P(B | A, C)`

这里：
*   `P(A)` 和 `P(C)` 是**先验概率**，因为 A 和 C 没有父节点。
*   `P(B | A, C)` 是**条件概率表**，它列出了在 `A` 和 `C` 所有可能取值组合下，`B` 为真或为假的概率。

这个分解比直接定义 `P(A, B, C)` 的 2³=8 个概率值要紧凑得多，因为它利用了 `A` 和 `C` 独立的假设。

## 总结

本节课中我们一起学习了两个关键部分。

首先，我们精确地定义了统计学中的**偏差**与**方差**。偏差衡量估计值的期望与真实值的系统误差，而方差衡量估计值自身的波动性。我们通过抛硬币的最大似然估计例子，看到其偏差为0（无偏），方差随数据量增加而减小，这量化了“更多数据带来更可靠估计”的直觉。

其次，我们开始了**图模型**的学习，并重点介绍了**贝叶斯网络**。贝叶斯网络使用有向无环图来表示变量间的依赖关系，其核心思想是通过条件独立性假设，将复杂的联合概率分布分解为一系列简单的局部条件概率的乘积。这种分解既使模型更易于理解和构建，也为高效的概率推理计算奠定了基础。在接下来的课程中，我们将继续深入探讨图模型的推理与学习算法。
