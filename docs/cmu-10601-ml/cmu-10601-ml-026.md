## 现代机器学习的关键挑战

以下是当前机器学习面临的一些关键挑战与研究方向。

### 挑战一：高效利用海量数据

大多数课程中讨论的技术都假设人类可以标注或注释所有可用于训练的数据。然而，许多现代机器学习应用拥有海量数据（如脑成像数据、监控数据、信息提取和计算生物学数据），让人工标注所有数据变得不切实际。因此，当前一个很酷的研究方向是设计学习技术，以最有效地利用可用数据，同时最小化对专家或人工干预的需求。课程中涉及的一些半监督学习和主动学习技术，只是非常程式化的初步方法，目前有更激动人心的研究致力于更好地解决这一问题。

**核心目标**：设计能在**最小化专家干预**的前提下，最有效利用**海量可用数据**的技术。

### 挑战二：分布式学习与通信效率

在许多现代应用中，海量数据分布在多个站点或位置。例如，大量的科学数据或视频/图像数据集。为了在这些组合数据集上进行学习，各个节点必须进行通信。在这种情况下，**通信成为一种新的关键资源**。除了我们在课程中一直讨论的两个关键资源——**计算能力**和**（标注）数据**——我们现在还必须考虑**通信效率**。

### 挑战三：新型资源与约束

上述的交互式学习（最小化人工标注）和分布式学习（考虑通信效率）只是现代机器学习方法的两个例子。此外，还有许多其他激动人心的新方向，例如：
*   **持续学习**：系统能够持续不断地学习新知识。
*   **深度学习**：使用多层神经网络进行表征学习。
*   **多任务/迁移学习**：利用来自多个相关任务的数据，以更高效地学习新任务。

这些方法中的每一种都带来了一系列新的挑战和**资源约束**。经典机器学习主要考虑**计算**和**（标注）数据量**这两个约束。而现在，我们突然需要考虑更多：
*   **人工标注成本**：在交互式学习中需最小化。
*   **通信效率**：在分布式学习中需考虑。
*   **隐私**：当数据涉及敏感信息时（如医疗数据、基因组序列数据），隐私成为必须认真对待的关键约束。
*   **激励机制**：当数据实际归属于某个实体（如研究机构、Google、Microsoft）时，他们分享数据的动机以及分享何种数据成为问题。

与10年前的经典机器学习相比，我们现在需要考虑的资源与约束要多得多，这非常令人兴奋。当然，**可扩展性**问题即使在经典约束（如计算）下也依然存在。

## 聚焦主题：深度学习 🤖

上一节我们介绍了机器学习面临的现代挑战，本节中我们来看看其中一个最重要的当前发展趋势：**深度学习**。它已经在众多商业企业中得到应用。

上周我们讨论了神经网络。今天我想在此基础上补充两点。

### 深度网络的关键特性

1.  **多层结构**：深度网络拥有许多阶段（层）来计算其结果。
2.  **表征学习**：深度网络训练的一个关键价值在于，当你给它一个输入时，它试图输出相同的东西（例如在自编码器中）。其核心思想是，网络可以学习生成一种非常有趣且复杂的内部编码，来捕捉你给它的输入数据的分布。**学习这种数据表征**，正是深度网络如此强大的原因之一。
