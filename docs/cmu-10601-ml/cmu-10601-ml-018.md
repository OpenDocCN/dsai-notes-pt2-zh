# 018：支持向量机 (SVM) 🎯

在本节课中，我们将要学习支持向量机。支持向量机是机器学习中一个经典且强大的算法。它基于我们之前讨论过的间隔和核函数概念，并在完全监督学习的设定下运行，即假设所有训练数据都已预先标注好。本节课我们将详细介绍SVM的推导和原理，并简要介绍更现代的、数据仅部分标注的半监督学习方法。

## 间隔概念回顾 📏

上一节我们介绍了间隔的概念，本节中我们来看看它的具体定义，这对于理解SVM至关重要。

我们讨论的是几何间隔。为简化问题，我们考虑穿过原点的分离超平面。如果一个分离超平面由法向量 **w** 指定，那么一个样本点 **x** 相对于该超平面的间隔，就是该点到超平面的距离。

如果我们对法向量进行归一化，使其模长为1（这总是可以做到的），那么这个间隔在数学上就等于该样本点与法向量点积的绝对值。

**公式**：`间隔(点x) = | w · x |`

对于一个样本点集 **S**，其相对于某个分离超平面的间隔，是集合中所有样本点间隔的最小值。而点集 **S** 本身的间隔，是所有可能分离超平面中，能得到的最大间隔值。

## 大间隔的理论优势 🛡️

间隔是机器学习中的一个重要概念，具有算法和理论上的意义。

我们之前看到，如果数据本身具有（或接近）一个大间隔，那么感知机算法的错误次数完全由这个间隔决定，而与数据所在空间的维度无关。这意味着大间隔对感知机算法非常有利。

在更经典的、数据全部预先标注的分布学习场景中，研究也证明了：如果一个分布确实具有大间隔，并且存在一个算法能够根据样本产生一个大间隔分类器，那么为了获得良好的泛化保证所需的标注样本数量是很少的，且仅依赖于间隔大小，同样与空间维度无关。

**核心结论**：寻找具有大间隔的分类器是一个好主意，因为从形式化理论上看，它们更不容易过拟合。这也为支持向量机算法提供了直接动机。

## 支持向量机算法推导 ⚙️

基于上述理论，支持向量机算法的核心目标就是：给定一组样本，寻找具有最大间隔的分类器。

为了简化推导，让我们从最简单的情况开始：假设数据是线性可分的。

我们的目标是找到一个分离超平面，使得所有样本点到该超平面的最小间隔（即“间隔”）最大化。这可以表述为一个优化问题：在保证所有样本点都被正确分类的前提下，最大化这个最小间隔。

用数学形式表达，对于每个样本点 `(x_i, y_i)`，其中 `y_i ∈ {+1, -1}`，我们希望：
`y_i (w · x_i) >= γ` 对于所有 `i` 成立，其中 `γ` 是我们想要最大化的间隔。

同时，我们约束法向量 **w** 的模长 `||w|| = 1` 以保持几何解释。因此，优化问题是：
最大化 `γ`
满足约束：`y_i (w · x_i) >= γ`， 对于所有 `i`， 且 `||w|| = 1`。

然而，直接处理 `||w|| = 1` 这个约束在优化中不太方便。我们可以进行一个巧妙的变换。注意到，间隔 `γ` 可以写为 `γ = ^γ / ||w||`，其中 `^γ` 是函数间隔（未归一化的点积值）。我们可以固定函数间隔 `^γ = 1`，因为等比例缩放 **w** 不会改变超平面本身。这样，最大化几何间隔 `γ = 1 / ||w||` 的问题，就等价于最小化 `||w||`，或者更简便地，最小化 `(1/2) ||w||^2`。

于是，优化问题转化为一个更易处理的形式：
最小化 `(1/2) ||w||^2`
满足约束：`y_i (w · x_i) >= 1`， 对于所有 `i`。

这是一个凸二次规划问题，存在高效的求解方法。这就是线性可分支持向量机的基本型。

## 关键要点总结 ✨

本节课中我们一起学习了支持向量机的核心思想。

*   **核心动机**：基于大间隔理论，寻找具有最大几何间隔的分类器可以提升泛化能力，减少过拟合风险。
*   **问题转化**：将“最大化间隔”的直观目标，通过固定函数间隔，转化为一个“最小化权重向量模长平方”的凸优化问题。
*   **基本形式**：对于线性可分数据，SVM的求解是一个在线性约束下最小化二次目标函数的优化问题。

这个推导为我们理解SVM奠定了坚实基础。在后续课程中，我们将探讨当数据非线性可分时如何引入松弛变量，以及如何利用核函数将线性方法扩展到非线性分类任务中。
