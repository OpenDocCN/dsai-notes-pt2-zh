# 007：线性回归

在本节课中，我们将要学习线性回归，这是一种用于预测连续数值型输出的学习算法。在此之前，我们首先回顾并深入探讨上一讲末尾提出的一个重要问题：在实践中，何时应使用朴素贝叶斯，何时应使用逻辑回归？这两者有何区别？我们将以此作为起点，然后引入线性回归的核心概念。

## 回顾：逻辑回归

上一讲我们介绍了逻辑回归。其核心思想是：如果我们做出与高斯朴素贝叶斯相同的初始假设（即特征在给定类别下条件独立），那么我们可以推导出条件概率 **P(y|x)** 的形式。

我们推导出的形式是逻辑函数：
```
P(y=1 | x; w) = 1 / (1 + exp(-(w0 + Σ wi*xi)))
```

为了从训练数据中学习权重 **w**，我们采用了**条件最大似然估计**原则。即，我们寻找能使给定观测特征 **x** 时，训练集中标签 **y** 的条件概率最大化的权重 **w**。基于此，我们得到了一个**梯度上升**更新规则来优化权重。

此外，我们提到可以通过引入参数的**先验分布**来进行最大后验估计。例如，若假设每个权重 **wi** 服从均值为0、方差为σ²的正态分布（即 **wi ~ N(0, σ²)**），那么在梯度更新规则中就会增加一个正则化项 **-λwi**。这个项会在每次更新时将权重 **wi** 向0收缩，有助于防止过拟合。

## 生成式分类器 vs. 判别式分类器

上一节我们回顾了逻辑回归，本节中我们来看看一个更普遍的机器学习概念：生成式分类器与判别式分类器的区别。朴素贝叶斯和逻辑回归正是这对概念的一个典型案例。

以下是两者的核心区别：

*   **生成式分类器（如朴素贝叶斯）**：这类方法首先对联合概率分布 **P(x, y)** 进行建模。它假设数据由某个生成过程产生，并分别估计**类先验 P(y)** 和**似然 P(x|y)** 的参数（例如，假设 **P(x|y)** 服从高斯分布）。然后，在预测时使用贝叶斯规则计算 **P(y|x)**。
*   **判别式分类器（如逻辑回归）**：这类方法直接对条件概率分布 **P(y|x)** 进行建模。它假设 **P(y|x)** 具有某种函数形式（如逻辑函数），并直接估计该条件分布的参数。

一个关键区别在于它们对**条件独立性假设**的依赖程度。朴素贝叶斯在**训练**（估计参数）和**预测**（应用贝叶斯规则）时都严格依赖此假设。而逻辑回归虽然在推导 **P(y|x)** 的形式时用到了该假设，但其**训练过程**（梯度上升）本身并不强制要求此假设成立。它只是寻找能最好拟合数据的权重 **w**。

因此，当条件独立性假设成立时，两种方法在数据量充足时都能取得良好性能。若该假设不成立，逻辑回归因其训练过程更具灵活性，通常对违背此假设的情况更具鲁棒性。

关于过拟合，一个经验法则是参数越多越容易过拟合。朴素贝叶斯通常需要估计更多参数（例如，对于N个布尔特征，需要估计4N+1个参数），而逻辑回归只需估计N+1个权重。然而，由于两者建模方式不同，不能简单依据参数数量判断过拟合倾向。

## 引入线性回归

在讨论了用于离散输出的分类算法后，本节我们将目光转向回归问题。线性回归是一种用于预测**连续实数值 y** 的经典学习算法。

线性回归的目标是学习一个从输入特征 **x** 到连续输出 **y** 的线性映射函数。其假设函数为：
```
h_w(x) = w0 + w1*x1 + w2*x2 + ... + wn*xn
```
也可以写作向量形式：**h_w(x) = w^T x**（其中 **x** 包含了常数项1）。

为了学习最优的权重 **w**，我们需要一个标准来衡量预测值 **h_w(x)** 与真实值 **y** 之间的差距。最常用的标准是**最小化平方误差损失**。

我们定义**代价函数 J(w)** 为所有训练样本上预测误差的平方和：
```
J(w) = (1/2) * Σ (h_w(x^(i)) - y^(i))^2
```
这里的 1/2 是为了后续求导方便而添加的常数。

我们的目标是找到使 **J(w)** 最小化的权重向量 **w**。对于线性回归，这是一个凸优化问题，存在解析解（通过求解正规方程）。同时，我们也可以使用**梯度下降**这种迭代优化算法。

对代价函数 **J(w)** 关于某个权重 **wj** 求偏导，得到梯度：
```
∂J(w)/∂wj = Σ (h_w(x^(i)) - y^(i)) * xj^(i)
```

基于此，我们可以得到批量梯度下降的权重更新规则（对于每个权重 **wj**）：
```
wj := wj - α * Σ (h_w(x^(i)) - y^(i)) * xj^(i)
```
其中 **α** 是学习率，控制每次更新的步长。

## 总结

本节课中我们一起学习了以下内容：
1.  回顾了逻辑回归及其与最大似然估计、最大后验估计的关系。
2.  深入探讨了生成式分类器（以朴素贝叶斯为例）与判别式分类器（以逻辑回归为例）的核心区别，特别是它们对条件独立性假设的依赖以及各自的优缺点。
3.  引入了线性回归算法，用于解决连续值预测问题。我们定义了线性假设函数、平方误差代价函数，并推导了用于优化权重的梯度下降更新规则。

线性回归为理解更复杂的回归和机器学习模型奠定了重要基础。
