# 005：高斯朴素贝叶斯

在本节课中，我们将学习朴素贝叶斯分类器的核心概念，特别是如何处理连续值属性。我们将回顾朴素贝叶斯的基本原理，探讨其在文本分类中的应用，并介绍如何将其扩展至处理连续特征，为后续学习逻辑回归打下基础。

## 课程概述

上一节我们介绍了条件独立性假设，它帮助我们大幅减少了需要从训练数据中估计的概率参数数量，从而构建了朴素贝叶斯分类器。本节中，我们来看看如何将朴素贝叶斯应用于文本分类，以及当属性不是离散值（例如布尔值）而是连续值（例如温度）时，我们需要做出哪些调整。

## 朴素贝叶斯核心原理

朴素贝叶斯分类器的核心思想基于贝叶斯规则。对于任何给定的由特征 **x** 描述的例子，我们预测其类别 **y** 的概率 **P(y|x)** 如下：

**公式：P(y|x) = [P(y) * Π_i P(x_i|y)] / P(x)**

其中，条件独立性假设允许我们将联合概率 **P(x|y)** 替换为各个特征条件概率的乘积 **Π_i P(x_i|y)**。这使得模型参数估计变得可行。

在实际分类一个新例子时，我们通常不需要计算分母 **P(x)**，因为它对所有可能的 **y** 值都是相同的常数。因此，我们只需选择使分子 **P(y) * Π_i P(x_i|y)** 最大的 **y** 值作为预测类别。

以下是分类决策的简化过程：

**代码：y_pred = argmax_y [ P(y) * Π_i P(x_i|y) ]**

## 分类新示例

为了具体说明，让我们通过一个例子来演示如何使用训练好的朴素贝叶斯模型进行分类。假设我们有一个新的测试样本，其特征值为：D=1（驾车），G=0（未在特定商店购物），B=0（生日不在特定日期前）。

我们的目标是计算该样本属于类别 **S=1**（例如，住在松鼠山）的概率。根据贝叶斯规则和条件独立性，该概率计算如下：

**公式：P(S=1|D=1,G=0,B=0) = [P(S=1) * P(D=1|S=1) * P(G=0|S=1) * P(B=0|S=1)] / 分母**

分母是归一化常数，等于 **P(D=1,G=0,B=0)**。在比较 **S=1** 和 **S=0** 的概率时，我们可以忽略分母，直接比较分子部分。

从上一节的训练数据中，我们可以获取所有必要的概率值：
*   **P(S=1)** = 26/102
*   **P(D=1|S=1)** = 3/26
*   **P(G=0|S=1)** = 0/26
*   ...（其他概率同理可得）

将这些值代入公式即可计算出最终概率。值得注意的是，当某个条件概率为0时（如 **P(G=0|S=1)=0**），会导致整个分子为0，这就是所谓的“零概率”问题，在实际应用中通常需要使用平滑技术（如拉普拉斯平滑）来处理。

## 处理连续值属性

到目前为止，我们讨论的特征都是离散的（如布尔值）。但在现实中，很多特征是连续的，例如温度、身高或收入。

为了将朴素贝叶斯应用于连续特征，我们需要对 **P(x_i|y)** 的建模方式进行改变。我们不再使用离散的概率表，而是假设每个连续特征在给定类别 **y** 下服从某种概率分布，最常用的假设是高斯分布（正态分布）。

这意味着，对于每个类别 **y** 和每个连续特征 **x_i**，我们假设：

**公式：P(x_i|y) ~ N(μ_{i,y}, σ²_{i,y})**

其中，**μ_{i,y}** 是类别 **y** 下特征 **x_i** 的均值，**σ²_{i,y}** 是其方差。在训练阶段，我们从训练数据中估计这些参数：

**代码：μ_{i,y} = (1/N_y) * Σ_{x in class y} x_i**
**代码：σ²_{i,y} = (1/N_y) * Σ_{x in class y} (x_i - μ_{i,y})²**

这里，**N_y** 是属于类别 **y** 的样本数量。在分类时，我们使用高斯分布的概率密度函数来计算 **P(x_i|y)** 的值：

**公式：P(x_i|y) = (1 / √(2πσ²_{i,y})) * exp( - (x_i - μ_{i,y})² / (2σ²_{i,y}) )**

然后将这个值代入朴素贝叶斯的分子公式中进行计算。这种假设特征服从高斯分布的朴素贝叶斯，被称为**高斯朴素贝叶斯**。

## 课程总结

本节课中我们一起学习了朴素贝叶斯分类器的深入应用与扩展。我们回顾了其基于贝叶斯规则和条件独立性假设的核心分类原理，并通过一个具体例子演示了分类过程。更重要的是，我们探讨了如何将朴素贝叶斯从处理离散特征扩展到处理连续特征，关键步骤是假设连续特征在给定类别下服从高斯分布，并通过估计均值和方差来建模 **P(x_i|y)**。这为解决更广泛的现实世界分类问题提供了工具。下一讲，我们将学习与朴素贝叶斯有紧密联系且广泛使用的逻辑回归算法。
