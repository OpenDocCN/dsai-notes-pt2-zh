# 020：主动学习

在本节课中，我们将学习主动学习。这是一种旨在减少专家标注工作量，同时最大化学习效率的技术。我们将首先回顾半监督学习，然后深入探讨主动学习的核心思想。

## 半监督学习回顾

上一节我们介绍了半监督学习，它旨在利用大量未标注数据来辅助少量标注数据的学习。本节中，我们来快速总结一下其核心思想。

在典型的半监督学习场景中，我们有一个数据源分布。在训练阶段，除了从该分布中独立同分布抽取的一小部分标注样本外，算法还能看到来自同一分布的大量未标注样本。

**目标**是找到一个分类器，使其在来自该数据源分布的新样本上具有较低的误差率。

未标注数据能在特定情况下极大地帮助我们，即当我们的归纳偏置不仅涉及目标函数的形式（例如，它是一个线性分隔器），还涉及目标函数与底层数据分布之间的关系时。

以下是三种基于不同假设的半监督学习技术：

*   **转导支持向量机**：其核心假设是目标函数不仅能分隔正负样本，而且是以**大间隔**进行分隔。对应的优化问题是：在给定少量标注数据和大量未标注数据的情况下，找到一个分类器，使其正确分类标注数据，并最大化在所有数据（包括未标注数据）上的间隔。
*   **协同训练**：其核心假设是问题存在两个**自然特征视图**，每个视图本身都足以进行正确分类，并且两个视图的预测结果**相互一致**。通过迭代或联合优化的方式，可以利用未标注数据在两个视图间传播学习信息。
*   **基于相似性的半监督学习**：其核心假设是存在一个**相似性函数**，并且我们相信**相似的点必须具有相似的标签**。算法上，可以构建一个图（节点是所有数据点，边代表相似性），然后以最小化切割边权重的方式对图进行划分，使得相同标签的点位于同一分区。

## 主动学习简介

了解了如何被动利用未标注数据后，本节我们来看看主动学习。主动学习的关键区别在于，学习算法能够**主动选择**哪些数据点最值得由专家进行标注。

在许多现代机器学习应用中，数据量巨大，让人工专家标注所有数据是不现实的。主动学习的目标是，通过智能地选择最有信息量的样本进行标注，从而用尽可能少的标注成本达到最佳的学习效果。

主动学习的典型流程如下：
1.  从一个包含少量已标注数据和大量未标注数据的池开始。
2.  使用当前已标注数据训练一个模型。
3.  模型评估未标注数据池，并**选择**一个或多个它认为“最有价值”或“最不确定”的数据点。
4.  将这些选中的数据点提交给专家（**查询**）以获取其真实标签。
5.  将新标注的数据加入训练集，并重复步骤2-5。

以下是几种常见的查询策略：

*   **不确定性采样**：选择模型预测最不确定的数据点。例如，在二分类中，可以选择预测概率最接近0.5的点。公式可表示为：`argmax_x (1 - |P(y=1|x) - 0.5|)`。
*   **查询委员会**：训练多个模型（委员会），选择那些委员会成员之间**分歧最大**的数据点进行标注。
*   **基于模型变化**：选择那些如果被标注并加入训练集，最可能导致当前模型发生最大改变的数据点。
*   **期望误差减少**：选择那些在标注后，能最大程度减少模型在未标注数据上**期望误差**的数据点。

## 总结

本节课中，我们一起学习了主动学习。我们首先回顾了半监督学习如何利用未标注数据，然后重点介绍了主动学习的核心思想：算法不再被动接受数据，而是主动选择最有信息量的样本请求标注。我们探讨了主动学习的流程以及几种关键的查询策略，如不确定性采样和查询委员会。主动学习是一种在数据标注成本高昂时，显著提升学习效率的强大范式。
