## 回顾贝叶斯规则 📐

上次课程结束时，我们介绍了贝叶斯牧师及其著名的贝叶斯规则。这是后续许多内容的关键。

我们通过链式法则以两种方式分解事件 **A** 和 **B** 的联合概率，从而推导出该规则。

以下是贝叶斯规则的几种形式。第一种形式与黑板角落的公式相同，只是分母 **P(B)** 被替换成了一个等价的表达式。

根据概率论的一个规则，如果我们有两个随机变量 **A** 和 **B**，想知道 **B** 的概率，我们可以简单地对 **A** 的所有可能值求和：**P(B|A)P(A) + P(B|¬A)P(¬A)**。因此，这只是公式的简单重写。实际上，我们会更频繁地使用这种形式。

最下面的形式则指出，在条件侧，除了 **B**，我们通常还可以加入其他条件。这是更一般形式的贝叶斯规则。同样，从概率的基本规则出发，不难证明这一点。

## 贝叶斯规则实例分析 🤒

为了确保理解，我们来看一个具体例子。

假设有人咳嗽了。你可能会想，他们是否得了流感？作为一个贝叶斯主义者，你可以这样思考这个问题。

定义两个随机变量：
*   **A**：是否得流感。
*   **B**：是否咳嗽。

我们可以使用贝叶斯规则来提问：在已知咳嗽的条件下，得流感的概率是多少？即求 **P(流感 | 咳嗽)**。

我们需要定义一些概率：
*   **P(A)**：先验概率，即一个人得流感的概率。假设现在是流感季，概率为 **5%**（0.05）。
*   **P(B|A)**：如果得了流感，咳嗽的概率。假设几乎所有人都会咳嗽，概率为 **80%**（0.8）。
*   **P(B|¬A)**：如果没有得流感，咳嗽的概率。假设很多人没流感也会咳嗽，概率为 **20%**（0.2）。

现在，我们可以使用贝叶斯规则计算：

**P(A|B) = [P(B|A) * P(A)] / [P(B|A)P(A) + P(B|¬A)P(¬A)]**

代入数值：
*   分子：**0.8 * 0.05 = 0.04**
*   分母：**0.8 * 0.05 + 0.2 * 0.95 = 0.04 + 0.19 = 0.23**

因此：
**P(流感 | 咳嗽) = 0.04 / 0.23 ≈ 0.174**

计算结果表明，在已知咳嗽的条件下，得流感的概率约为 **17.4%**。

所以，即使咳嗽了，也无需过度担心。虽然先验概率只有 **5%**，但咳嗽这个证据将后验概率提升到了约 **17%**，但这个概率仍然不算很高。
