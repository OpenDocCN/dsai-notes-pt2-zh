# 019：半监督学习 🧠

在本节课中，我们将要学习半监督学习。这是一种在拥有大量未标记数据和少量标记数据时使用的学习范式。我们将探讨其基本概念、几种主要方法，并理解未标记数据如何以及为何能帮助我们提升模型性能。

## 学习范式回顾

上一节我们介绍了监督学习的基本框架。本节中我们来看看半监督学习如何扩展这一框架。

到目前为止，在课程的大部分时间里，我们讨论的都是从**完全标记的数据**中学习。这属于**分布监督学习**设定。

我们假设存在一个数据源，即一个在数据空间（如垃圾邮件分类中的邮件空间）上的潜在分布。在训练阶段：
1.  首先从该数据分布中随机抽取一个样本。
2.  假设这个随机样本由专家根据某个目标函数进行标记。
3.  学习算法获得一个**完全标记的样本**，即从潜在分布中随机抽取的独立同分布样本，并根据目标函数进行了标记。
4.  学习算法输出一个假设函数（如决策树或线性分类器）。
5.  我们的目标是确保输出的假设函数对于来自同一潜在分布的新样本具有较小的错误率。

这是我们分析朴素贝叶斯、逻辑回归、支持向量机等算法的基础框架。我们讨论了算法本身和样本复杂度的基本问题，例如需要多少数据才能确信应用这些算法后得到的分类器对新样本有效。

这是典型的、经过长期研究的监督学习范式。

## 半监督学习的动机

然而，正如上次提到的，这种完全监督学习的范式虽然取得了巨大进步和广泛应用，但对于现代大数据时代来说，其成本可能过高。

大多数现代机器学习应用都拥有**海量的原始或未标记数据**。例如：
*   网络上有海量的图像集合。
*   存在大量的视频序列数据。
*   在信息提取领域，网络上有数十亿的网页。
*   计算生物学中，数据也在不断积累。

让人类坐下来为所有这些数据根据我们的学习任务进行标注，是不现实或希望渺茫的。

因此，在过去大约15年里，机器学习领域投入了大量努力来设计能够最有效利用可用数据、并最小化专家或人工干预需求的学习技术。这是一个主要的研究主题。

今天和下次课，我们将讨论两种试图实现这一目标的范式：**半监督学习**和**交互式学习**。这些是算法层面和理论层面都取得了巨大进展的学习范式。

## 半监督学习模型

我们上次已经接触过这个模型，现在让我们更具体地定义它。

在半监督学习场景中，除了典型的监督学习通道外，算法还能看到大量来自同一潜在分布的**未标记样本**。

形式化地描述该模型：
*   **输入**：算法除了获得一个从潜在分布中随机抽取的**小型标记样本集**外，还能看到一个从同一数据源抽取的**大型独立同分布未标记样本集**。
*   **目标**：与之前相同，输出一个分类器，使其对于给定的未知目标函数和未知潜在分布，具有较小的错误率。

简而言之，该模型在随机标记样本的基础上，增加了对底层数据源的大型未标记样本的访问，而目标保持不变。

正如我们所讨论的，这是过去15年机器学习研究的一个主要课题，产生了大量活动、许多著名论文、研讨会，甚至出版了相关书籍和综述。这是一个兼具广泛应用和良好理论基础的主题。

## 主要技术方法概述

在今天的课程中，我将介绍三种主要类型的半监督学习技术。

以下是本节课将涵盖的三种主要技术：
1.  **直推式支持向量机**：我们上次已经讨论过，它是一种特定的半监督算法。
2.  **协同训练法**：由Blum和Mitchell提出。
3.  **基于图的方法**：另一类不同的半监督学习技术。

这些技术以非常不同且有趣的方式利用未标记数据。因此，我将逐一讲解它们。在课程最后，我会退一步，从更高层次思考：未标记数据在何种意义上帮助我们？因为所有这些技术都非常不同，我们可以思考是否存在更基本的潜在原则。

我认为这是三种用于整合未标记数据进行半监督学习的主要技术类型。当然还有其他方式，但今天的讲座主要聚焦于这些能够非常积极地利用未标记数据的方法。

## 未标记数据的价值

在深入具体方法之前，让我们先思考一个基本问题。在半监督学习场景中，起初你可能会感到困惑或不清楚未标记数据能为我们做什么。

因为从某种意义上说，它缺少了最重要的信息——我们想要预测的**标签**。因此，未标记数据在何种意义上、以及为何能帮助我们，尤其是在实质性的帮助上，起初并不明确。你可能会想到一些次要的用途，但若希望它能带来实质性帮助，则有点难以理解。

事实证明，从非常高的层次上讲，未标记数据在以下情况下能帮助我们：如果我们关于问题的信念或归纳偏差不仅仅关乎目标函数的形式（例如它是线性分隔器或决策树等），还关乎数据的**分布特性**。

我们将在介绍完每种技术后再回到这个观点进行更深入的讨论。

本节课中我们一起学习了半监督学习的基本概念、动机和模型框架，并概述了三种主要的技术方法：直推式SVM、协同训练和基于图的方法。我们初步探讨了未标记数据的潜在价值，为后续深入每种具体技术奠定了基础。
