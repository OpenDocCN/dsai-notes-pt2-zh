#  007：导论与多单元组织

在本节课中，我们将探讨一个核心问题：一个大型组织内部能否存在多个独立的“组织”？我们将通过分析课程论坛中的讨论，并结合社会学概念，来理解组织内部部门与独立组织之间的界限。

本周我将回应的第一个屏幕旁聊问题，是由去年的一位学员提出的，这个问题在上一期课程中也引起了广泛讨论。问题是：**一个巨型组织内部是否可能存在多个组织？**

他谈到组织在规模和结构上可以不同，但想知道在一个巨型组织内部是否可能存在多个组织。他提出了一个难题：组织内部的部门或子单元，有时似乎也开始像一个独立的组织。那么，我们还能称其为组织的一部分吗？我们如何决定一个部门是一个组织，还是一个单一组织的一部分？这关乎**边界**问题：即一个部门何时是在独立行动，何时是在为整个公司的利益服务？

论坛上对此有许多精彩的回复。其中一些观点认为，**不可能**在一个组织内部存在多个组织。

以下是持“不可能”观点的部分论述：

*   **基兰·丁扎**认为，要成为一个组织，各单元之间必须有目标一致性。一个组织是为了满足市场需求而实现共同目标的集合。组织内的每个业务单元、部门或板块，都致力于实现这个由更大组织设定的共同目标。它们并非独立于业务目标运作，一切都与组织战略保持一致。
*   **吉姆·道德**也对此表示怀疑。根据我在讲座中提供的定义，他认为组织不能由其他组织构成。任何组织都可以分解为其各自的职能（如工程、人力资源、销售等）。从本质上讲，每个职能都像是一个不同的组织，拥有自己的特点、文化，以及既针对本小组也针对整个组织的共同目标。

这里出现了一点概念上的模糊：协调是发生在部门内部还是组织内部？我们开始看到这个难题：一个组织内部能否有多个组织？

然而，也有相当多的学员认为**可以**。以下是持“可以”观点的部分论述：

*   **唐·麦克尤恩**（很抱歉，我可能念错了你的名字）以通用电气这样的 conglomerate organization 为例。它拥有不同行业领域的各种业务，这些业务可以且必须作为独立的实体运营，并且可以被出售或剥离成为一个完整的组织。他认为部门拥有独立的身份，但最终作为一个整体运作。这里的关键转变似乎是：一旦你成为一个组织，你就独立于那个整体而行动。
*   **丽贝卡·梅恩斯**和**戴尔·霍尔沃森**也提出了这一点。他们关于公司内部差异化的论点很精妙：单元可以拥有不同的参与者和目标，但它们的任务和功能是相互依存的，必须跨角色和功能进行协调，才能使公司运转。
*   戴尔以及后来的**米格尔·苏博萨**补充道，随着规模扩大，组织内部的变异或差异化也会增加。戴尔鼓励人们向上看一层，从一个部门看到管理多个部门的层级。他认为组织试图培养**级联目标**，使得下层制定出某种程度上独特但又与上层保持**一致**的目标。

通过阅读这些讨论，我开始感觉到，即使是“不可能”和“可以”两派，如果我们开始明确“协调发生在何处”这个概念，他们实际上是有共识的。这涉及到**差异化**与**协调**。将一个实体称为部门还是独立组织，暗示着当它成为自己的组织时，单元间的协调就不再存在了。

**辛娜·哈通**很好地总结了这一点，她提出了一个折中的方案：子单元是**部分组织**。这意味着它们具备组织的特征，但依赖于母组织。此外，部分组织中的个体可能仍认为自己是母组织的一部分。

从我的角度来看，社会学家**埃米尔·涂尔干**对组织（或社会）类型有一个很好的描述。随着规模扩大，社会会从一种类型转变为另一种类型。他谈到，在古代社会，整合的发生是因为每个人都做相同的事情，就像在一个部门或一群人内部，他们有着相似的目标和活动，内部容易产生协作与协调。

但随着社会（就像组织一样）的成长，它们变得更加专业化和差异化。大公司开始出现职能差异化的单元。涂尔干问道：社会如何保持团结？我们也可以对组织提出同样的问题：是什么让它们保持为一个实体？他认为，随着差异化而来的是**相互依赖**，他称之为**有机团结**。这指的是，尽管存在差异化，但跨单元的协调仍然存在于一个公司或组织内部。一旦这种跨单元的协调不复存在，我们便开始失去对“一个组织”的定义，它变得更像一个独立的实体。

因此，思考什么是部门、什么是组织，是一个富有成果的问题。我认为通过讨论，我们已经取得了一些进展，并可以开始以有用的方式来阐述它。

**本节课总结**

本节课我们一起探讨了组织内部结构的边界问题。通过分析“一个组织内部能否存在多个组织”的辩论，我们理解了**目标一致性**、**差异化**、**协调**和**相互依赖**等核心概念。关键区别在于，一个**部门**虽然可能有独特的目标和文化，但其运作仍与整体组织战略紧密协调；而一个独立的**组织**则意味着这种跨单元的协调关系已经减弱或消失，它能够作为自主的实体运作。涂尔干的“有机团结”概念帮助我们理解，大型组织正是通过各差异化部分之间的相互依赖和协调来维持其整体性的。
