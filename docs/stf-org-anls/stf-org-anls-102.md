#  102：种群生态学（第三部分）🏢

![](img/821ec1f9f6346af4d19e6c481e222228_0.png)

在本节课中，我们将要学习种群生态学中关于企业生存与死亡的另外三个核心理论：**新生劣势理论**、**生态位理论**和**资源分割理论**。我们将探讨企业年龄、环境变化类型以及市场竞争结构如何影响不同类型企业的命运。

上一节我们讨论了密度依赖理论，本节中我们来看看另一个关于企业死亡的理论——**新生劣势理论**。该理论的核心观点是，新成立的组织最有可能失败。

![](img/821ec1f9f6346af4d19e6c481e222228_8.png)

新组织之所以脆弱，主要有内外两方面原因。**内部**，成员需要学习新组织形态所要求的所有新角色和关系，这阻碍了他们专注于核心业务，即投入与产出的流程。**外部**，它们缺乏老牌企业所拥有的合法性和稳定的关系网络，因此难以吸引外部支持。

![](img/821ec1f9f6346af4d19e6c481e222228_10.png)

因此，随着企业年龄的增长，其生存几率会提高。这一观点有时与企业规模效应相混淆，许多人认为规模才是决定性因素。其核心逻辑可以概括为：**企业越老，其内部结构越完善、越稳定，从而能更专注于生产**。而新企业则需要解决所有这些问题。

虽然新企业最终可能发展出更适应环境的形式，但在此之前需要完成大量的基础建设工作。

新生劣势理论同样适用于危机时期。在这些时期，许多组织试图进行内部变革。当这种情况发生时，内部结构不再反映企业积累的历史，从而丧失了先前的生存价值。

![](img/821ec1f9f6346af4d19e6c481e222228_17.png)

![](img/821ec1f9f6346af4d19e6c481e222228_18.png)

![](img/821ec1f9f6346af4d19e6c481e222228_19.png)

![](img/821ec1f9f6346af4d19e6c481e222228_20.png)

以下是组织核心特征变化带来的问题：
*   **使命与价值观**：当组织的使命和价值观发生变化时，会给企业带来各种问题，并可能解释组织的死亡。
*   **适应性**：因环境压力而试图突然改变的企业很容易死亡。

然而，短期策略和**外围特征**的变更，与种群生态学的适应性观点更为一致。这些非核心、外围的特征更容易改变以适应环境。而改变内部核心特征则困难得多，因为当你改变技术和社会结构等核心特征时，还需要其他许多方面的配合，才能使转型可行并与环境保持一致。因此，这类变革往往弊大于利。

接下来，我们介绍第三个关于企业生存与死亡的理论——**生态位理论**。其基本思想是：不同的环境条件有利于**专业化**或**通用化**的组织形态。

![](img/821ec1f9f6346af4d19e6c481e222228_27.png)

![](img/821ec1f9f6346af4d19e6c481e222228_28.png)

*   **专业化企业**：专注于特定技术，并承担风险以最大化利用某一环境，同时充分意识到该环境可能轻易改变。
*   **通用化企业**：以较低水平利用多个环境或一个广泛的生态位，因此在面对环境变化时具有更高的安全性。

![](img/821ec1f9f6346af4d19e6c481e222228_29.png)

以葡萄酒行业为例：
*   **通用化企业**：如嘉露酒庄这样的大规模生产商，生产勃艮第、夏布利等多种以地理区域命名的葡萄酒。它们通常以低得多的成本生产品质较低的大众餐酒。
*   **专业化企业**：如农场酒庄，生产以特定葡萄品种命名的单一品种葡萄酒，并带有原产地命名标签，其关注点非常不同。

通用化生产占据了葡萄酒销售的大部分，利润丰厚，小型酒庄难以与之竞争。

![](img/821ec1f9f6346af4d19e6c481e222228_31.png)

---

![](img/821ec1f9f6346af4d19e6c481e222228_32.png)

现在我们对通用化和专业化企业有了一些概念，可以开始讨论生态位如何偏向其中一方的理论。第一个是**生态位宽度理论**，由汉南和弗里曼于1977年提出。

生态位宽度理论关注环境变异性的两个方面，以解释专业化或通用化企业的不同生存率。在我的早期讨论中，我描述了组织种群如何占据相同的生态位或相同的独特环境资源领域，并依赖相同的环境资源。

如果两个组织种群占据相同的生态位，但在某些组织特征上存在差异，那么选择论的论点是，具有较不适应环境特征的种群将被淘汰。

![](img/821ec1f9f6346af4d19e6c481e222228_34.png)

![](img/821ec1f9f6346af4d19e6c481e222228_35.png)

然而，当涉及通用化和专业化企业时，组织对环境反应不同，情况变得更复杂。通用化组织拥有更多冗余，它们汲取不同的资源或实现多个生态位，因此如果某一环境的资源枯竭，它们可以在另一环境中生存。相比之下，专业化组织更为精简，它们试图在单一环境或单一实现的生态位中开发资源。

通用化企业并非在任何单一情况下都处于最优状态。因此，它们拥有一个广泛的**基础生态位**，其中许多**实现生态位**更有利于通用化企业。一个跨越环境两个或更多不同部分的企业（如通用化企业）将能够应对环境，无论环境如何变化。然而，它需要承担覆盖更广泛环境部分的成本。

![](img/821ec1f9f6346af4d19e6c481e222228_37.png)

因此，通用化企业受益于**广泛的生态位**和**不稳定的市场**，因为它们拥有多样化的利益和努力，能够应对波动。

相比之下，专业化企业适合**狭窄、稳定的生态位**。狭窄的生态位拥有适合小范围产品的资源，因此专业化企业在这种背景下获得优势。它们在**狭窄的生态位**和**稳定的市场**中取得成功。在这些情境下，它们可以利用自身与一个实现生态位的契合度，并忽略其他生态位。在稳定的环境中，它们可以击败通用化企业，因为通用化企业无法像专业化企业那样最大化其市场份额。专业化企业无需支付从事多项业务的间接成本。

因此，专业化企业能更好地满足特殊利益，而通用化企业则准备好应对环境兴趣的变化，以防一个生态位规模超过另一个。

![](img/821ec1f9f6346af4d19e6c481e222228_39.png)

![](img/821ec1f9f6346af4d19e6c481e222228_40.png)

![](img/821ec1f9f6346af4d19e6c481e222228_41.png)

第二种生态位理论对“通用化企业在不稳定环境中更成功，而专业化企业在稳定环境中更成功”的观点提出了质疑。该理论实际上与人们所关注的变化和变异水平有关。

![](img/821ec1f9f6346af4d19e6c481e222228_44.png)

因此，生态位对专业化或通用化企业生存的影响，实际上取决于人们将环境变异视为**细粒度**还是**粗粒度**。

表面上看，稳定的环境鼓励专业化组织，而不稳定的环境导致通用化组织激增，这似乎合乎逻辑。但汉南和弗里曼认为，这并非适用于所有情况。

环境变化是偏向通用化还是专业化企业，取决于两种企业之间的**距离**（即企业当前的专业化或通用化程度）以及环境变异的**粒度**。

![](img/821ec1f9f6346af4d19e6c481e222228_46.png)

![](img/821ec1f9f6346af4d19e6c481e222228_47.png)

![](img/821ec1f9f6346af4d19e6c481e222228_48.png)

![](img/821ec1f9f6346af4d19e6c481e222228_50.png)

*   **细粒度变异**：通常指相对于组织生命周期而言，持续时间短、增量小的变化。
*   **粗粒度变异**：指长期变化，例如国家政治结构的变化或某些法律的通过。

![](img/821ec1f9f6346af4d19e6c481e222228_52.png)

![](img/821ec1f9f6346af4d19e6c481e222228_53.png)

当环境变化**迅速且为细粒度**时，最好成为**专业化**企业，因为你更灵活，能适应这些变化。

当环境变化**迅速且为粗粒度**时，最好成为**通用化**企业，因为专业化企业可能无法存活足够长的时间以逐步调整到最优状态。

![](img/821ec1f9f6346af4d19e6c481e222228_55.png)

![](img/821ec1f9f6346af4d19e6c481e222228_56.png)

![](img/821ec1f9f6346af4d19e6c481e222228_57.png)

企业类型之间的**距离**在这里也起作用。如果你是一个远离通用化状态的专业化企业，那么由于政治结构和法律等变化导致的快速粗粒度环境变化，可能使你无法生存。

这是对先前生态位宽度理论的一种细化，现在它涉及更具体的变化案例（细粒度或粗粒度），以偏向专业化或通用化类型的企业。

在知识共享资源中搜索时，我最终找到了一些图像，分析师试图识别股票市场的结构性变化，这些变化揭示了市场在何处容易系统性崩溃。有趣的是，他们发现稳定的正常市场行为包含强烈的残差相关性，股票以分段方式相关。这就像一个实现的空间或狭窄的生态位，专业化企业可以在其中获胜，这看起来像是正常市场行为或稳定环境。

在异常市场的情况下（如2001年9/11事件后市场下跌时），我们看到股票突然之间在整个市场上彼此之间具有非常强的相关性，并且与残差失去相关性。这意味着存在巨大的波动性和相互依赖性。

在这种情况下，我们面临一个实现生态位相互碰撞的环境，企业争夺相同的资源。在这里，如果生态位不适合，专业化企业可能会消亡，而通用化企业则倾向于存活下来。

第三种生态位理论被称为**资源分割理论**，由格伦·卡罗尔提出。该理论描述了生态位宽度动态，以解释专业化和通用化企业的不同生存能力。

先前的理论主张适应一组条件，并预测对于一个给定的种群，存在一种最优策略。相比之下，卡罗尔提出，大型通用化组织为占据市场中心而进行的竞争，释放了**边缘**的资源，这些资源可以被小型专业化企业利用，而无需与通用化企业直接竞争。

阿南德·斯瓦米纳坦有一篇关于这在葡萄酒行业如何发生的论文，而他的合作者格伦·卡罗尔也描述了这在啤酒行业如何发生。正如你所想象的，在这两个行业中，都存在通用化和专业化企业。

格伦·卡罗尔和斯瓦米纳坦都发现，通用化企业的**拥挤**反而为专业化企业创造了机会。当百威、康胜、安海斯-布希、喜力、米勒等所有通用化啤酒公司开始相互竞争时，它们为精酿啤酒的出现和生存创造了空间。葡萄酒行业也是如此，罗伯特·蒙达维、嘉露、舒特家族等所有大规模生产餐酒和盒装酒的酒庄相互竞争，这为农场酒庄的生产开辟了空间。

我提到的这两篇论文都发现，在过去的二十年里，随着这些通用化企业之间大规模葡萄酒产量的增加，这些专业化企业的数量也在增长。这是一个关于生态位内**资源分割**的论点，使得这些专业化企业有生存空间。

![](img/821ec1f9f6346af4d19e6c481e222228_65.png)

正如我所说，卡罗尔和斯瓦米纳坦认为，通用化企业之间的竞争以及市场空间的开放，为专业化企业提供了一个**资源分割**的过程。这个资源分割过程具有独特的机制，催生了专业化企业能够成功的环境，这在某种程度上与密度依赖理论形成对比。

![](img/821ec1f9f6346af4d19e6c481e222228_67.png)

以下是资源分割的具体机制：

第一个机制是**定制化过程**。这里的论点是，小企业更灵活，可以将其产品定制为特定的消费者口味。相比之下，大企业行动缓慢，无法快速适应这些不断变化的口味。

![](img/821ec1f9f6346af4d19e6c481e222228_69.png)

![](img/821ec1f9f6346af4d19e6c481e222228_71.png)

但这又产生了另一个问题，即**身份认同**。你是谁很重要。即使大型酿酒商可以复制精酿啤酒厂的技术方面，它们也不会非常成功，因为它们是大型企业，而非独立拥有的企业。在这种情况下，**你是谁**似乎比**你做什么**更重要。

事实上，合同酿酒商在进入这个市场分割部分时也失败了。因此，消费者似乎非常信任小生产商来制造这些优质产品。消费者可能是在对大众社会及其生产技术做出反应，甚至是对电视上不断播放的百威与米勒的广告做出反应。消费者购买可能是一种自我表达的形式，他们厌倦了一成不变的东西，想要这些个性化的本地啤酒和葡萄酒。他们可能将购买视为一种地位生成和表达，通过购买来彰显独特性。

因此，如果通用化啤酒和葡萄酒泛滥，就会出现针对质量的**专业市场**，这个细分市场似乎会蓬勃发展。专业化市场细分似乎通过定制化和身份认同而茁壮成长。

---

表面上，资源分割理论似乎与密度依赖理论相悖。例如，密度依赖理论认为，随着环境中企业密度或数量的增加，合法性变得不那么突出，竞争决定了企业的命运。

![](img/821ec1f9f6346af4d19e6c481e222228_73.png)

![](img/821ec1f9f6346af4d19e6c481e222228_74.png)

相比之下，资源分割理论认为环境是**分割**的，对于通用化企业，竞争确实存在，但对于分割环境中的专业化企业则不然。在这种分割的环境中，专业化企业可以蓬勃发展，诉诸于质量和身份认同，这更像是合法化过程。

![](img/821ec1f9f6346af4d19e6c481e222228_75.png)

![](img/821ec1f9f6346af4d19e6c481e222228_76.png)

因此，资源分割理论实际上有助于解释，为何密度依赖理论总体上适用于主要行业，但在**边缘市场**等较小的生态位中，合法性机制仍然可以适用。

一个尚不明确的问题是，通用化企业最终是否能够开发出精酿啤酒或农场葡萄酒产品并赢得客户。例如，罗伯特·蒙达维甚至米勒啤酒都开发了挑战专业化葡萄酒和啤酒质量的产品，它们甚至赢得了评论界的赞誉和奖项。这可能只是需要更长时间才能获得这种关键赞誉并进入那些专业市场。但可能发生的情况是，通用化企业可以通过在资源分割的两个部分都取得成功，从而进入这两个细分市场。

在本节课中，我们一起学习了种群生态学的三个重要理论。我们探讨了**新生劣势理论**如何解释新企业的脆弱性，**生态位理论**（包括生态位宽度和粒度变异）如何分析专业化与通用化企业在不同环境下的生存策略，以及**资源分割理论**如何揭示市场竞争结构如何为小型专业化企业创造生存空间。这些理论为我们理解组织种群在复杂环境中的动态演变提供了多角度的分析框架。
