#  087：小道消息与职场人际关系 📊

在本节中，我们将探讨组织中的非正式网络与亲密关系。我们将分析管理者如何了解并影响非正式网络，并讨论同事间亲密关系对工作效率的潜在影响。

上一节我们介绍了组织中的正式结构，本节中我们来看看非正式网络与职场人际关系。

## 如何监测与管理非正式网络？

非正式网络，常被称为“小道消息”或“葡萄藤沟通”，是组织内自发形成的社会关系。管理者能否监测并控制这些网络，同时让成员不易察觉？自1939年乃至更早，学者如切斯特·巴纳德在《经理人员的职能》中，以及罗特利斯伯格和迪克森在《管理与工人》中，便开始研究企业，并思考：我们能否设计或引导非正式组织？因为人们形成的非正式群体似乎会影响工作效率。巴纳德曾设想，若能引导这些自发形成的关系，使其有利于工作、提升工作规范和生产力，那将非常理想。这种思路延续至今。

以下是研究非正式网络的几种可行且符合伦理的方法：

*   **问卷调查**：可以询问员工向谁寻求建议。了解网络结构、哪些人感到建议匮乏或工作困难并无大碍。
*   **观察与记录**：观察员工与谁共进午餐等活动。共餐通常是与喜欢的人进行的私人活动，能揭示亲密关系。通过观察记录，可以快速绘制公司的社交网络图。
*   **物理空间安排**：可以通过座位安排等方式改变网络形态，将有建议需求或工作困难的员工安排在能提供更多建议或更核心的员工旁边。

在研究中需注意伦理。大学通常设有人类受试者委员会以确保不伤害参与者。一个很重要但研究不足的领域是“负面关系”，即人们厌恶或回避的对象。直接询问员工不喜欢谁可能带来伤害，因此我们通常更关注积极的情感关系。

关于其他研究手段，许多公司拥有员工的电子邮件。在未经许可的情况下分析邮件互动模式是可行的，但这可能引发隐私担忧。对于情报或军事类机构或许适用，但对其他组织可能构成冒犯。Facebook和LinkedIn等社交平台的数据或许有帮助，但数据所有权在平台方，且需获得许可。此外，这些平台上的关系网络可能存在虚假信息，如粉丝主页、招聘人员甚至已故人士，不一定反映真实的友谊或工作关系。因此，问卷调查能提供更一致的定义（如“建议关系”）。

研究方式多种多样，甚至对幼儿园儿童也可行。例如，研究者在孩子面前放两个篮子和一堆照片，请他们将喜欢一起玩的朋友照片放入左边篮子，以此了解他们的社交网络。公司记录也是公开信息的来源，合理汇编可以揭示大量活动。

关键在于，非正式网络是资产还是问题？它可能在这两者间转换。在许多工作场所，家人朋友介入工作可能成为问题，但如果他们带来外部信息，则可能有益。同样，内部非正式关系可以促进沟通、建立社群，但如果这些关系与工作无关，或涉及欺凌等，则可能有害。信息网络在收集信息方面很棒，但也可能泄露你不希望传播的信息。

总而言之，管理者应尝试引导或至少促进非正式网络。最常用且可行的方法是调整团队、角色和员工所处位置，并思考你希望优化什么，以及你希望员工围绕哪些与工作相关的内容来建立关系。

## 同事间的亲密关系：是助力还是阻力？

接下来我们看看同事间的亲密关系或家庭纽带。它们是有害的吗？会分散注意力、降低效率，还是有所帮助？

我认为论坛上的大多数评论都有道理。有人认为这是问题，也有人认为它能带来信任和亲密感。这里存在一个关键矛盾：**不同的关系规范与义务会发生碰撞**。

众所周知，企业家历史上常有将一种关系转化为另一种的做法。例如，文艺复兴时期，婚姻常被用于商业目的。那时，家庭这种亲密关系被用于商业而无须顾虑。但私人关系遵循的是**特殊主义**规范，意味着你偏爱、喜欢并常常无条件支持某些特定个体。

而在工作场所，工作关系通常基于**普遍主义**和/或**任人唯贤**的规范。当这两种规范结合时，就可能产生争议。一个当前的例子是某将军的案件，但这种冲突不仅限于浪漫关系，也可能涉及家庭纽带或友谊——任何将偏袒带入职场的行为都可能被视为冲突。

另一方面，我也理解人们所说的相反观点：如果不在工作中，你还能在哪里遇到未来的伴侣？大多数人确实如此。我们在工作上花费大量时间，作为人类，我们形成纽带、产生浪漫关系是自然而然的。

此外，还有一种理想化的观点：如果能与伴侣共享深刻的纽带，包括智力、工作和个人生活，那该多好？就像萨特和西蒙娜·德·波伏娃那样。但不幸的是，我们是人，工作中会出现问题，这两个世界的规范可能发生冲突。

例如，我听说某个大学院系的故事：教授娶了自己的学生，后来聘其为教员，之后离婚，该教授再婚后又把新伴侣聘为教员。结果导致了一个功能失调的环境，没人能和睦相处。最终，他们不得不聘请外部系主任来管理该系，因为内部人员互不喜欢、互不听从。这清楚地显示了功能失调的潜在可能。

因此，最终我认为，**清晰区分这些关系及其规范至关重要**。关键在于，能否说服员工不使用其配偶作为项目线人，并解释这对他人可能不公，且违反公司行为准则？如果可以，那很好。也可以将相关人员安排在不同的办公楼或分支机构，以避免利益冲突或不同关系规范之间的冲突发生。

本节课中我们一起学习了非正式网络的监测与管理方法，以及职场亲密关系的双重影响。核心在于认识到非正式网络的双刃剑特性，并通过合乎伦理的方式进行引导；同时理解亲密关系可能带来的规范冲突，并通过明确规则和物理隔离来管理潜在风险。
