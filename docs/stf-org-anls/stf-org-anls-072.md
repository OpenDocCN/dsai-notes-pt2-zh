#  072：内外聚焦的驱动因素 🧭

在本节中，我们将探讨组织在何时以及为何会将其关注点转向内部或外部环境。通过分析论坛中的讨论，我们将识别出影响组织注意力分配的几个关键因素。

上一节我们介绍了组织与环境互动的基本概念，本节中我们来看看具体是什么因素驱动了组织对内或对外的聚焦。

## 概述

论坛上的第一个问题由我提出：“你的组织有多重视环境及其中的外部关系？” 迄今为止，我们的理论主要聚焦于公司的内部流程，而现在我们开始关注这些外部环境或组织间的特征。我希望了解你们是否观察到公司向内或向外聚焦的时刻及其原因，并获取你们的实际经验。这个问题没有标准答案，但通过你们的回答，我看到了很多差异，你们也可以看到这些差异，并开始反思为何有些公司这样做而有些则不，以及我们如何分类这些向内或向外聚焦的经验，本质上是什么在驱动这些组织内部的注意力。

我想花点时间梳理一下大家的观点、我的解读以及一些反思。

## 影响聚焦方向的因素

以下是大家提出的几个主要影响因素。

### 1. 领导与个性

一些人认为，对内部与外部环境的关注因领导者而异。这取决于领导者的个性，他们会决定采取哪种边界或关注点。这是一个有趣的视角，表明这取决于个体。

### 2. 职位与角色

类似地，一些人认为关注点因职位或角色而异。例如，组织的行政单位（如管理层）或特定部门可能专门关注外部边界。这很有道理，在具有劳动分工并关注外部环境的公司中是可行的。

### 3. 行业特性

另一些人认为这会因行业而异。例如，有人认为娱乐业总是向外聚焦，或者航空公司等任何以营利为目的、服务客户的业务都可能高度关注环境。

相反，一些人认为政府机构甚至教育机构在关注点上高度向内。这主要是因为组织间的交换法律合同已经建立，例如联邦系统自动分配资金。

但对此存在一些争论。我注意到一些非营利组织或政府组织实际上确实关注环境，特别是像国家公园这样的机构。值得注意的是，政府内部的这类关注实际上处于需求或关注层级的边缘。例如，在加州预算危机期间，州立公园被关闭，员工减少。但即使在财政紧缩时期，为了维持运营，它们仍然没有足够的资金做想做的事，不得不采取联盟和网络组织形式来获取资源、提供服务，例如将餐饮或清洁服务外包，而护林员则负责公园维护。这表明，即使在政府机构内部，不同的结构也可能向外或向内导向。我认为，如果你处于需求或关注层级的边缘（如环境问题），向外聚焦是一种在可能不被充分重视的情况下获取资源或汇集资源并拥有发言权的方式。反之，对于高度官僚化的公司，向内聚焦对这些政府机构可能是可行的。因此，行业很重要，但这些行业所处的具体环境也很重要。

### 4. 环境与情境

向外聚焦的另一个原因是**环境背景**。我想详细阐述这一点，因为在某些情况下，公司处于竞争环境中，这可能迫使它们高度关注环境中的外部关系。例如，有人提到波音和空客的双寡头飞机制造业，这两家公司争夺大量相同业务，它们高度关注对方的行动。竞争环境会导致这一点，这很有道理。

环境背景的另一个特征可能是经济衰退或萧条。资源的丧失使公司非常关注在环境中寻找新的资源来源。因此，环境本身可以决定你的关注点。

### 5. 时间动态

最后一个主题也出现在你们的答案中：关注点如何随时间变化，注意力如何随时间转移，这取决于每个公司的问题和复杂性存在于何处。

我喜欢采矿业的例子。有人说，在获取土地准入权的阶段，环境非常重要；而在另一个阶段，为了建设像连接多个矿场的道路这样昂贵的基础设施，他们会联合起来。因此，他们在环境中相互合作以实现目标。但在其他时候，你可以想象内部注意力会转移到仅仅完成特定矿场的采矿作业上。

## 总结

从某种意义上说，我认为所有这些答案都适用且重要。关键在于，你会根据问题所在、对公司而言最重要的行动领域、最突出的问题以及存在模糊性和问题的地方来切换关注点。在许多情况下，这将是组织关注和聚焦的环境边界（无论是内部还是外部）。

恰巧，接下来的三周，我们将重点学习主要关注环境的组织理论。

本节课中，我们一起学习了驱动组织向内或向外聚焦的多种因素，包括领导个性、职位角色、行业特性、环境背景以及时间动态。理解这些因素有助于我们分析组织行为背后的逻辑。
