#  008：社会运动与临时性组织 🏃‍♂️💨

在本节课中，我们将探讨一个核心问题：大规模抗议活动能否被视为一种组织？我们将通过分析学生们的讨论，来辨析社会运动与正式组织之间的异同，并理解“组织化”这一动态过程的重要性。

上一节我们介绍了组织的基本定义，本节中我们来看看社会运动这种特殊形式是否契合该定义。

---

一位前学生维维安娜·塞鲁蒂提出了一个问题：“大规模抗议活动能否被视为组织？”她以埃及革命和西班牙“愤怒者”运动为例，指出这些活动似乎具备组织的某些特征：它们有目标、有参与者为实现目标而工作、能完成任务，甚至最终会出现代表整个群体发言的社会结构。

许多同学对此表示赞同并加入了讨论。例如，辛西娅·柯林斯提到了“占领华尔街”运动。这场运动被认为开启了美国关于经济不平等的对话，但其决策基于共识，难以形成明确的信息、目标和目的。那么，它仅仅是一场运动，还是某种形式的组织？

金·考纳也认同维维安娜的观点，认为它们可能是**临时性组织**。他指出，占领者们仍属于同一场运动，拥有共同目标，只是对目标的理解和实现方式有不同解读。这实质上指出，运动是**临时性的组织化努力**。

不仅如此，在某些案例中，如“占领”运动，其目标本身就不清晰，或者共识是短暂易逝的。新闻报道常指出，“占领华尔街”运动最终包含了各种不同的声音和参与者想要达成的目标。

迈克尔·彼得斯·罗贝尔则认为，由此产生的群体行为往往**类似于组织**。它们深受周围环境影响，拥有共同目标、演进中的社会结构、亚文化，并利用社交媒体等技术手段推进事业。这符合将输入转化为输出的过程。他进而提出，从这个意义上讲，帮派是否也算一种临时组织？

**“临时性”** 似乎是关键。共识也是临时的。这些形态并非总能自我维持。当我们把组织视为一个名词或实体时，其**自我维持与再生产的能力**就非常重要。这也正是“运动”这个概念与之有所区别的原因。当然，社会运动与组织确有相似之处。

巴勃罗·安德烈斯·穆索指出，大规模抗议活动并非高度组织化，个体可以根据自身便利加入或退出；有时没有明确的领导者，即使有，其利益也可能与参与者不同，甚至参与者彼此之间的利益和目标也不一致。

这里揭示的核心是**偏好与目标的不一致性**。这种现象在组织内部也可能发生，但关键在于，组织通常存在一个**共识点**。社会运动似乎可以被理解为**临时性的联盟**：它们达成共识、采取行动，然后解散。不过，联盟作为一种组织形式，往往会反复聚合以采取行动。

法尔贡·库马尔提出了一个有趣的观点，他开始提及“人群”。这引出了一个值得思考的区分：**人群**、**抗议活动**与**组织**三者之间存在一个连续谱。例如，一场无意识的骚乱是组织吗？并非如此，它缺乏我们在课程中讨论的组织定义性特征。抗议活动在某种程度上是协调的。如果它缺乏共同目标和协调性，其“组织性”就减弱；如果不可重复、无法持续，那么它就更不符合“组织”一词的严格定义。

当然，许多抗议活动最终会演变为**社会运动组织**。例如绿色和平组织，许多我们熟知的激进主义组织，很可能都起源于社会运动，随后被制度化，成为具有重复性、持续性等特征的正式组织。

莉莉·玛丽亚提出了一个深刻的见解：大规模抗议是一种**有组织的事件**，而非一个组织。有组织的事件包括会议、抗议活动、派对等，由一群人为特定目的而组织。它可能是一次性的、偶尔的、零星的或周期性重复的。她进一步列举了这类事件与组织的不同之处：

以下是其与典型组织的一些区别：
*   事件的目的可能并非被所有参与者共享或认同（例如，有些人可能只是为了陪伴朋友而参与）。
*   特别是抗议活动，有时包含大量随机参与者，并非所有参与者的贡献都相同。
*   重复出现的模式并非总是预期和计划好的。
*   边界不一致，角色未明确界定，参与者可以变换角色。

当然，其中许多特征在临时组织甚至内部存在冲突的模糊组织中也可能出现。

通过讨论，我们认识到，组织的定义可以从非常清晰、理想的“强”形式（即一个能自我再生产、拥有社会结构、并能持续围绕共同目标达成共识和协调的实体），延伸到临时性的形式，甚至到缺乏上述大部分特征的形式。

实际上，我们是否拥有一个关于“组织”的完美定义，或许并不那么重要。对我而言，更有趣的是**组织化的过程**。我甚至对这门课程的名称“组织分析”略有不满，更倾向于称之为“**组织化分析**”，因为我们关注的许多内容都是临时的，或者关乎组织化的过程。我认为这更能涵盖社会运动和社会运动组织。

如果我们能掌握**组织化的过程**及其**再生产**，那么我们也就学会了如何构建更理想形式的组织。同学们在这个讨论中提出的许多概念，将在后续关于“联盟”的章节中出现，届时我们将探讨如何在不一致的偏好下通过妥协来维持协调行动。此外，还有“组织无政府状态”的概念，即组织由短暂的时刻或事件驱动，人们聚集起来做出决策并行动，进而对整个组织产生影响。这些都是对组织更为流动的理解。

我认为，这类主题将在课程中反复出现。社会运动的例子很好地呼应了我们后续课程中将遇到的一些关于组织的核心议题。

---

本节课中我们一起学习了社会运动与正式组织的异同。核心在于理解“临时性”、“共识的短暂性”以及“自我维持能力”是区分运动与强定义组织的关键。更重要的是，我们应将焦点从静态的“组织”实体，转向动态的“组织化”过程，这为我们理解各种形式的集体行动提供了更灵活的框架。
