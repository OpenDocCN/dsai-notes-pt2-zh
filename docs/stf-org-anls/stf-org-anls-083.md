#  083：文化与网络

在本节中，我们将探讨组织学习与网络化组织形式中“文化”概念的具体内涵与差异。我们将看到，尽管两者都涉及文化，但其焦点、范围和表现形式各不相同。

上一节我们介绍了网络化组织的基本形式，本节中我们来看看文化在其中扮演的角色。许多学习者对“文化”的概念感到困惑，尤其是在讨论信任、寻找合适的网络伙伴等议题时。实际上，不同的组织理论隐含着对文化的不同理解。

## 组织学习中的文化

当我们思考组织内部的网络时，通常会联想到组织学习理论。这种理论所设想的组织文化有其特定性。

它并非指与技术相关的“科技文化”，或那种提供免费食物、充满自由精神的氛围。这种文化的核心是**对实践及其改进的反思**。它关注的是**改进工作的过程**。

这是一种非常具体的文化，不涉及风格、颜色或主题等表象。文化本身聚焦于与核心技术相关的实践和仪式，并围绕以下方面展开：
*   反思与改进
*   记住有效的方法
*   持续评估和讨论改进方案

这种特定的文化也对应着特定的网络形态。例如，**实践社群** 指的是组织内部密集的非正式关系网络，而**实践网络**则可能延伸到组织外部。网络确实存在，并且有一种专注于实践及改进实践的特定文化与之整合。

除此之外，文化才可能在风格上有所区分，例如着装要求等。因此，这是一种聚焦于文化核心功能的观点。

## 网络化组织中的文化

在网络化组织形式中，隐含的是一种**组织间文化**。相关理论常常讨论联盟或家族，并强调确保所有成员拥有共享的价值观。

但这与组织学习中的文化是同一种吗？并不完全是。例如，假设你是一个协调多个群体的管理组织，试图在学校或大学的不同群体（如食堂员工、保洁、保安、教师、辅导员）之间建立共享的价值观，而这些群体可能都已外包给不同的公司。

作为协调这些群体的组织间网络管理者，你需要的并非一种高度整合、如同特许经营那样要求所有人信奉“苹果方式”或“赛百味方式”的文化。因为许多职能已经外包，这种深度整合并非必需。

然而，你确实需要某种文化来促进：
*   整合与开放：关于每个人在做什么
*   信息共享
*   协调工作
*   建立互补性：确保每个人都履行自己的职责

这是一种关于**维持网络**并以学习的方式**改进网络**的文化，但它是组织间的。思考这一点很有趣，它不一定需要像组织学习文化那样丰富、密集和层次分明。

这里确实存在文化的成分，我们需要理解，这种文化真正关注的是**网络维护的过程**以及对**联盟支持**的关切。可以想象，当多个群体从事类似工作时，由于需要大规模提供服务，单个组织无法独立完成，此时文化可能会有所不同。

在联盟内部，这些群体之间可能存在更多的文化镜像和一致性，这是合理的。但是，在垂直方向上，例如保险公司和实际的医疗服务提供者（医生团体）之间，整合不同单元的努力可能会面临文化上的巨大差异。

在各单元或各公司内部，它们可以拥有专注于改进自身实践的组织文化。但在网络内部，即网络化组织形式中，还存在这种共享的开放性、脆弱性，以及一种协同工作、通过加入网络提供更具价值服务的共识。

甚至存在一种共享的治理观念，即有一种共同的责任感来维持这个网络。这种集体规范需要通过某种组织间文化来强化，无论是通过公司野餐、会议、数据共享、集成网站还是各种媒体技术来实现组织间的沟通。

这具有一种学习的感觉，像是组织间的文化，但它不是组织学习文化。它是一种**维持网络的组织间网络文化**，并且比组织内部那种类似“信仰”的文化更“薄”。

## 核心对比与总结

组织学习和网络化组织形式似乎都拥有一种文化，这种文化关乎**维持网络**或**改进实践**。这是一个比风格或更广义的身份认同更窄的概念，后者可能出现在科技公司或专业服务公司中。那些组织的文化甚至可以超越实际的工作流程及其强化的仪式与实践。

之所以提出这一点，是因为组织文化和文化的概念会不断重现。事实上，下周我们将讨论**新制度理论**，它关乎环境中的文化。那将不是网络化组织形式所暗示的关于过程的特定文化，也不是组织学习所暗示的组织内部文化，而可能是关于环境中信仰的更广泛概念。

让我们持续关注这一点。理解这些理论在某些方面是相互关联的非常重要，本节旨在阐明这种关联。

本节课中我们一起学习了组织学习与网络化组织两种理论背景下“文化”概念的差异。组织学习文化聚焦于组织内部实践与改进的反思，而网络化组织文化则是一种更“薄”的、用于维持跨组织联盟协作与开放共享的组织间文化。理解这种区别有助于我们更精准地运用不同理论分析实际问题。
