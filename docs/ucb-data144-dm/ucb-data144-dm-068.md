# 68：如何选择样本量 📊

在本节课中，我们将学习如何为特定的分析任务选择合适的样本量。样本量的大小会影响样本均值的变异性，进而影响我们进行统计推断（如假设检验和构建置信区间）的精确度。我们将从回顾置信区间的概念开始，并探讨如何根据对置信区间宽度的要求，反过来确定所需的样本量。

## 置信区间回顾 📈

上一节我们介绍了样本量对分析的影响，本节中我们来看看如何利用置信区间的概念来确定样本量。首先，让我们快速回顾一下置信区间的核心思想。

根据中心极限定理，对于一个样本量为 `n` 的样本，其样本均值 `X̄` 的分布近似服从正态分布。该分布的均值等于总体均值 `μ`，标准差（即标准误）等于总体标准差 `σ` 除以样本量的平方根。

**公式：**
`标准误 (SE) = σ / √n`

这意味着，大约95%的样本，其样本均值 `X̄` 与总体均值 `μ` 之间的距离，不会超过两个标准误。

**公式：**
`P( |X̄ - μ| < 2 * SE ) ≈ 0.95`

这个关系是构建置信区间的基础。

## 从理论到实践：构建置信区间 🛠️

理解了样本均值的分布后，我们来看看如何在实际中构建置信区间。关键在于视角的转换。

在理论上，我们知道样本均值 `X̄` 的分布以总体均值 `μ` 为中心。但在实践中，我们只有一个样本，我们知道计算出的 `X̄`，却不知道 `μ`。

由于上述距离关系是对称的，因此“从 `μ` 出发，在 ±2SE 范围内能找到 `X̄`” 等价于 “从我们已知的 `X̄` 出发，在 ±2SE 范围内能捕获未知的 `μ`”。

因此，总体均值 `μ` 的一个近似95%置信区间可以构建为：

**公式：**
`95% 置信区间 = [ X̄ - 2*(σ/√n), X̄ + 2*(σ/√n) ]`

该区间以样本均值 `X̄` 为中心，向两侧各延伸约两个标准误的距离。

## 置信区间的宽度与样本量 📏

现在，我们进入核心部分：样本量的选择。置信区间的总宽度直接取决于样本量。

根据上面的公式，95%置信区间的总宽度 `W` 是上下界之间的距离：

**公式：**
`W = [X̄ + 2*(σ/√n)] - [X̄ - 2*(σ/√n)] = 4 * (σ/√n)`

以下是关于区间宽度 `W` 的几个关键点：
*   `W` 与总体标准差 `σ` 成正比。总体变异性越大，要达到相同精度所需的区间越宽（或需要更大的样本量）。
*   `W` 与样本量 `n` 的平方根成反比。这是最重要的关系：**增加样本量可以缩小置信区间的宽度，从而提高估计的精确度**，但改善的速度会随着 `n` 增大而减慢。

## 如何根据精度要求确定样本量 🎯

在实际研究设计（如设计调查问卷）时，我们常常希望估计值能达到某个预设的精度。例如，我们可能希望置信区间的宽度不超过某个值 `W`。

根据公式 `W = 4σ / √n`，我们可以解出所需的样本量 `n`：

**公式：**
`n = (4σ / W)²`

**应用步骤：**
1.  **确定精度要求**：设定你希望置信区间达到的总宽度 `W`。
2.  **估计总体变异性**：利用历史数据、预实验或专业知识，对总体标准差 `σ` 进行一个合理的估计。
3.  **代入公式计算**：将 `W` 和估计的 `σ` 代入上述公式，即可计算出理论上所需的样本量 `n`。

**示例**：假设根据以往研究，我们估计总体标准差 `σ` 约为 10。如果我们希望95%置信区间的总宽度 `W` 不超过 4 个单位，那么所需的样本量至少为：
`n = (4 * 10 / 4)² = (10)² = 100`

本节课中我们一起学习了如何根据统计推断的精度要求来选择样本量。我们从回顾置信区间的概念和构建原理出发，明确了置信区间宽度 `W` 与总体标准差 `σ` 及样本量 `n` 的关系（`W = 4σ / √n`）。最终，我们掌握了通过预设可接受的置信区间宽度 `W` 和估计的总体标准差 `σ`，来反推所需样本量 `n` 的方法（`n = (4σ / W)²`）。这是在研究设计阶段确保分析结果具有足够精确度的关键一步。
