---

## 回归模型回顾

上一节我们介绍了回归诊断，本节中我们来看看如何利用回归模型进行推断。线性回归模型的核心在于拟合一条直线，使其能够最好地描述数据点之间的关系。

线性回归模型包含以下关键部分：
*   **回归线（信号）**：我们试图拟合的最佳直线，代表了数据中的系统性模式。
*   **残差（噪声）**：观测值与回归线预测值之间的差异，代表了模型未能解释的随机波动。

模型的假设是，这些残差服从一个均值为零的正态分布。我们通过**最小二乘法**来找到这条回归线，该方法的目标是**最小化所有残差的平方和**，其公式表示为：

`最小化 Σ(观测值 - 预测值)²`

---

## 真实回归线与拟合回归线

为了理解回归估计的可靠性，我们需要区分两个概念：
*   **真实回归线**：描述了变量之间潜在的真实关系，其参数（斜率和截距）是固定但未知的。
*   **拟合回归线**：我们根据样本数据计算得到的回归线，是对真实回归线的最佳估计。

我们的目标是通过样本数据得到的拟合回归线，尽可能接近未知的真实回归线。

---

## 样本量的重要性：一个模拟演示

以下将通过一个模拟函数来展示样本量如何影响拟合回归线与真实回归线的接近程度。该函数的核心逻辑是：
1.  设定真实的斜率 (`true_slope`) 和截距 (`true_intercept`)。
2.  根据设定的样本量 (`sample_size`) 随机生成预测变量 `x` 和服从正态分布的误差项 `error`。
3.  根据公式 **`y = true_slope * x + true_intercept + error`** 生成响应变量 `y`。
4.  基于生成的 `(x, y)` 样本数据，拟合出一条回归线。
5.  比较真实回归线与拟合回归线。

我们将运行两次模拟，分别使用小样本量和大样本量，观察其差异。

### 演示一：小样本量 (n=10)

```
# 模拟参数设置
true_slope = 2
true_intercept = -5
sample_size = 10
```

运行模拟后，我们得到四个图形，其中最关键的是对比图。可以观察到，当样本量仅为10时，基于数据拟合出的回归线（红色虚线）与真实的回归线（蓝色实线）存在显著差异。拟合线的斜率和截距估计都可能有较大偏差。

**结论**：当样本量过小时，即使我们知道存在一个真实的线性关系，基于有限数据得到的估计也可能严重偏离真实情况。这并非线性回归方法本身的缺陷，而是由于数据信息不足导致的估计不确定性增大。

### 演示二：大样本量 (n=100)

```
# 模拟参数设置
true_slope = 2
true_intercept = -5
sample_size = 100
```

再次运行模拟。当样本量增加到100时，拟合回归线与真实回归线几乎重合。基于大样本数据得到的估计非常接近真实参数。

**结论**：足够大的样本量可以显著提高回归估计的准确性和稳定性。拟合线能够更可靠地反映潜在的真实关系。

---

## 核心要点总结

本节课中我们一起学习了回归推断的起点：
1.  **回归模型** 旨在分离数据中的**信号**（回归线）与**噪声**（残差）。
2.  我们通过**最小二乘法**拟合数据，得到**拟合回归线**，作为**未知真实回归线**的估计。
3.  **样本量**是影响估计质量的关键因素。**小样本可能导致估计严重偏离真实值**，而**大样本则能提供更可靠、更准确的估计**。
