![](img/5f821fd1d8af50bc8b92f953f33456d1_0.png)

在本节课中，我们将学习如何衡量数据集的离散程度，特别是标准差这一核心概念。我们将从理解“变异性”开始，探讨不同衡量方法的优缺点，并详细讲解标准差的计算过程及其意义。

## 概述：什么是离散度？

上一节我们介绍了衡量数据中心趋势的方法，如均值。本节中我们来看看如何衡量数据的离散程度，即数据点围绕中心值的分散情况。一个简单的衡量方法是**极差**。

## 极差及其局限性

极差是数据集中最大值与最小值之差。其计算公式为：
**极差 = 最大值 - 最小值**

以下是使用极差的一个主要缺点：
它无法描述数据分布的形状。例如，一个对称的钟形分布（数据在1到5之间，中心在3）和一个两极分布（一半数据点在1，另一半在5）可能拥有相同的极差（4），但两者的形状截然不同。因此，仅凭极差无法充分了解数据的内在分布。

## 围绕均值的变异性

一个更常用的方法是测量数据围绕均值的变异性。这要求我们先计算出数据的均值，然后量化每个数据点与均值的差异。

我们通过一个示例来演示这个过程。假设我们有一个数据集：`[2, 3, 3, 9]`。

首先，我们计算其均值：
**均值 = (2 + 3 + 3 + 9) / 4 = 4.25**

## 计算偏差

接下来，我们计算每个数据点的**偏差**，即每个值减去均值。

以下是计算步骤：
1.  列出所有数据值。
2.  对每个值，计算 `值 - 均值`。

对于我们的数据集，偏差计算如下：
*   2 - 4.25 = -2.25
*   3 - 4.25 = -1.25
*   3 - 4.25 = -1.25
*   9 - 4.25 = 4.75

直接使用偏差的一个问题是，所有偏差之和必然为0。这是因为均值的定义就是使偏差之和为零的那个值。因此，单纯加总偏差无法有效衡量整体的离散程度。

## 方差与标准差

为了解决偏差正负抵消的问题，我们引入**方差**。计算方差的步骤如下：
1.  计算每个数据点的偏差。
2.  将每个偏差**平方**（从而消除负号，专注于距离）。
3.  计算这些平方偏差的**平均值**。

方差的计算公式可以表示为：
**方差 = (Σ(每个值 - 均值)²) / 数据点数量**

对我们的例子进行平方计算：
*   (-2.25)² = 5.0625
*   (-1.25)² = 1.5625
*   (-1.25)² = 1.5625
*   (4.75)² = 22.5625

然后计算这些平方值的平均值（方差）：
**方差 = (5.0625 + 1.5625 + 1.5625 + 22.5625) / 4 ≈ 7.6875**

由于方差是平方后的结果，其单位与原数据单位不同。为了得到一个与原数据单位一致的离散度度量，我们取方差的平方根，即得到**标准差**。

**标准差 = √方差 ≈ √7.6875 ≈ 2.77**

在Python中，你可以使用 `numpy.std()` 函数直接计算标准差。

## 总结

本节课中我们一起学习了衡量数据离散度的核心方法。我们首先指出了极差的局限性，然后深入讲解了围绕均值衡量变异性的方法。通过计算偏差、平方偏差得到方差，最终取平方根获得标准差，我们得到了一种能够有效反映数据分布形状且单位一致的离散度统计量。下节课我们将继续探讨标准差的性质和理解方式。
