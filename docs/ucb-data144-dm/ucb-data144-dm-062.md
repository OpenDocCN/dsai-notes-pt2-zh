## 概述
在本节课中，我们将要学习如何衡量数据分布的中心。我们将重点介绍一个最常用的中心度量指标：**均值**。我们将了解它的定义、特性、计算方法，并探讨它与另一个中心度量指标——中位数——之间的关系。

## 什么是均值？
均值，也称为算术平均数，是衡量数据分布中心位置的一种方法。它的计算方式是将所有数据点的值相加，然后除以数据点的总个数。

### 均值的计算公式
对于一个包含 `n` 个数据点的数据集，其均值 `mean` 的计算公式为：
`mean = (x1 + x2 + ... + xn) / n`

## 均值的特性
以下是均值的一些重要特性，理解它们有助于我们更好地应用这个指标。

1.  **均值不一定是数据集中的值**。例如，对于数据集 `[2, 3, 3, 9]`，其均值为 `4.25`，这个值并不存在于原始数据中。
2.  **均值不一定是整数**。即使所有数据点都是整数，其均值也可能是一个小数。
3.  **均值介于最小值和最大值之间**。但它不一定正好位于中间位置，位于中间位置的是中位数。
4.  **均值与数据具有相同的单位**。例如，如果数据是年龄（年），那么均值的单位也是年。
5.  **均值可以看作是一个平滑算子**。它将所有数据点的信息汇总成一个单一的代表值，尤其在大数据集中，这种“平滑”效果更为明显。

## 如何计算均值？
上一节我们介绍了均值的概念和特性，本节中我们来看看如何在实践中计算它。我们将使用一个简单的例子，并通过Python代码进行演示。

### 示例数据集
假设我们有一个包含四个数据点的数组：`[2, 3, 3, 9]`。

### 计算方法
以下是几种计算该数据集均值的方法：

*   **手动计算**：根据公式，`(2 + 3 + 3 + 9) / 4 = 4.25`。
*   **使用Python的`numpy`库**：`numpy`库提供了高效计算均值的函数。
    ```python
    import numpy as np
    values = np.array([2, 3, 3, 9])
    mean_value = np.mean(values)  # 或者使用 np.average(values)
    print(mean_value)  # 输出：4.25
    ```
*   **加权平均视角**：均值也可以视为一种加权平均。每个数据点的权重是其出现频率除以总数据量。对于我们的例子：
    `mean = 2 * (1/4) + 3 * (2/4) + 9 * (1/4) = 4.25`

### 在对称分布中
当数据的直方图分布是**对称**的时候，均值和中位数会相等。因为数据在中心两侧均匀分布，算术平均的平衡点正好落在排序后的中间位置上。

![](img/2f602f78d754fc2e89c64b2bad9983a2_7.png)

![](img/2f602f78d754fc2e89c64b2bad9983a2_8.png)

### 在偏态分布中
当数据的直方图分布是**偏态**（不对称）时，均值和中位数通常不相等。

*   **中位数**：由于它是数据排序后的中间点，受极端值的影响较小，相对稳健。
*   **均值**：它会受到数据中所有值的影响，包括远离中心的极端值。因此，在偏态分布中，均值会被“拉向”分布的长尾方向。

**核心结论**：对于偏态分布，均值会偏离中位数，偏离的方向与分布的长尾方向一致。

## 实例分析
为了巩固对均值和中位数关系的理解，我们来看几个具体的图形化例子。

以下是三个数据分布的直方图，请思考它们的均值和中位数之间的关系。

![](img/2f602f78d754fc2e89c64b2bad9983a2_3.png)

**问题1**：对于这个对称分布，均值和中位数分别是多少？它们有什么关系？
> **答案**：均值和中位数都等于3。因为分布完全对称，平衡点（均值）与中间点（中位数）重合。

![](img/2f602f78d754fc2e89c64b2bad9983a2_7.png)

**问题2**：现在，将上图（左）中所有值为5的数据点改为10，得到右图。这个新分布的均值和中位数与原来相比有何变化？
> **答案**：
> *   **中位数**：保持不变，仍然是3。因为改变的是右侧的极端值，并未影响数据排序后的中间位置。
> *   **均值**：会增大。因为原本与左侧较低值平衡的“5”变成了更大的“10”，导致算术平均的平衡点向右移动。

![](img/2f602f78d754fc2e89c64b2bad9983a2_10.png)

**问题3**：这是2013年NBA球员身高的分布，它呈左偏态。请问均值和中位数哪个更大？
> **答案**：**中位数更大**。根据“均值被拉向长尾方向”的规律，在左偏分布（长尾在左）中，均值会向左偏离中位数，因此均值小于中位数。实际数据也证实了这一点：平均身高约为79英寸，而中位数身高为80英寸。

## 总结
本节课中我们一起学习了数据中心的度量指标——均值。

我们首先给出了**均值**的**定义**和**计算公式**，并阐述了它的几个关键**特性**。接着，我们演示了如何使用**Python**计算均值。然后，我们重点比较了**均值**和**中位数**的关系：在**对称分布**中两者相等；在**偏态分布**中，均值会受极端值影响而偏离中位数，且偏离方向指向分布的长尾。最后，通过几个实例分析，我们加深了对这一重要概念的理解。

![](img/2f602f78d754fc2e89c64b2bad9983a2_10.png)

记住，选择使用均值还是中位数来描述数据的中心，需要根据数据的具体分布情况来决定。当数据分布对称时，两者皆可；当数据分布偏斜或存在极端值时，中位数通常是更稳健的选择。
