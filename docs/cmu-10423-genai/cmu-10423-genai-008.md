## 扩散模型回顾 📚

上一节我们介绍了U-Net作为一种可能的骨干网络，它可以接收特定尺寸的图像，并输出一个具有相同空间维度（但可能通道数不同）的特征层。这对于我们构建扩散模型至关重要。

![](img/85bf00ca226b74abfc1f53da9723031a_1.png)

在无监督学习中，我们的目标是学习一个参数化的分布 **P_θ**，使其尽可能接近真实的数据分布 **Q**。这可以是一个语言模型（针对文本）或一个生成模型（针对图像）。

对于语言模型，如果选择自回归模型，通常可以使用基于梯度的优化器来最大化观测数据的似然，这是可处理的。然而，对于简单的图像生成架构（如我们见过的GANs），直接最大化似然通常并不可行。

![](img/85bf00ca226b74abfc1f53da9723031a_11.png)

对于GANs和扩散模型，我们实际上是在最大化一个包含隐变量 **Z** 的**边缘似然**。具体形式为：

![](img/85bf00ca226b74abfc1f53da9723031a_19.png)

**P(x) = ∫ P(x|z) P(z) dz**

扩散模型包含一个**前向过程**和一个**反向过程**。

![](img/85bf00ca226b74abfc1f53da9723031a_21.png)

![](img/85bf00ca226b74abfc1f53da9723031a_23.png)

![](img/85bf00ca226b74abfc1f53da9723031a_17.png)

前向过程（图中红色箭头）是一个向数据 **x₀** 逐步添加噪声的马尔可夫链，直到最终变为纯高斯噪声 **x_T**。其条件分布定义为：

![](img/85bf00ca226b74abfc1f53da9723031a_25.png)

![](img/85bf00ca226b74abfc1f53da9723031a_26.png)

**q(x_t | x_{t-1}) = N(√α_t * x_{t-1}, (1-α_t)I)**

![](img/85bf00ca226b74abfc1f53da9723031a_28.png)

![](img/85bf00ca226b74abfc1f53da9723031a_30.png)

反向过程（图中蓝色箭头）则是从噪声 **x_T** 开始，逐步去噪以恢复数据 **x₀** 的生成过程。我们试图学习一个参数化的分布 **p_θ(x_{t-1} | x_t)** 来近似这个真实但难以计算的反向过程。

![](img/85bf00ca226b74abfc1f53da9723031a_32.png)

![](img/85bf00ca226b74abfc1f53da9723031a_34.png)

![](img/85bf00ca226b74abfc1f53da9723031a_35.png)

**时间步数 T 的选择**：T 通常需要足够大（几十到几百步）以确保生成质量，但这会导致采样时计算成本高昂，因为每一步都需要运行神经网络。训练时，我们可以高效地随机采样一个时间步 t 来更新参数。

![](img/85bf00ca226b74abfc1f53da9723031a_37.png)

![](img/85bf00ca226b74abfc1f53da9723031a_39.png)

## 前向过程与精确反向过程 🔄

![](img/85bf00ca226b74abfc1f53da9723031a_41.png)

![](img/85bf00ca226b74abfc1f53da9723031a_43.png)

![](img/85bf00ca226b74abfc1f53da9723031a_45.png)

![](img/85bf00ca226b74abfc1f53da9723031a_46.png)

![](img/85bf00ca226b74abfc1f53da9723031a_47.png)

以下是前向过程 **q** 和精确反向过程 **q** 的一些关键性质。

**性质一：任意时间步的分布**
给定初始数据 **x₀**，我们可以直接采样得到任意时间步 **t** 的加噪图像 **x_t**，而无需逐步迭代：

**q(x_t | x₀) = N(√ᾱ_t * x₀, (1-ᾱ_t)I)**

其中 **ᾱ_t = ∏_{i=1}^{t} α_i**。这意味着我们可以通过一步计算得到 **x_t**：

**x_t = √ᾱ_t * x₀ + √(1-ᾱ_t) * ε**, 其中 **ε ~ N(0, I)**

![](img/85bf00ca226b74abfc1f53da9723031a_49.png)

通过精心设计 **α_t** 的调度表（如线性或余弦调度），可以确保最终分布 **q(x_T)** 接近标准高斯分布 **N(0, I)**。

**性质二：精确反向条件分布**
精确的反向条件分布 **q(x_{t-1} | x_t, x₀)** 是**可处理的**，并且也是一个高斯分布：

**q(x_{t-1} | x_t, x₀) = N(μ̃_t(x_t, x₀), σ̃_t² I)**

其方差 **σ̃_t²** 是一个仅与调度表相关的标量。其均值 **μ̃_t** 是 **x₀** 和 **x_t** 的线性组合：

**μ̃_t(x_t, x₀) = (√ᾱ_{t-1} * β_t / (1-ᾱ_t)) * x₀ + (√α_t * (1-ᾱ_{t-1}) / (1-ᾱ_t)) * x_t**

其中 **β_t = 1 - α_t**。为简化，我们将 **x₀** 和 **x_t** 前的系数分别记为 **α_t⁰** 和 **α_t^t**。

然而，**q(x_{t-1} | x_t)**（不依赖于 **x₀**）是难以直接计算的，这正是我们需要用神经网络去近似的部分。

## 参数化学习到的反向过程 🧠

![](img/85bf00ca226b74abfc1f53da9723031a_57.png)

![](img/85bf00ca226b74abfc1f53da9723031a_59.png)

![](img/85bf00ca226b74abfc1f53da9723031a_61.png)

我们的目标是学习一个参数化的反向过程 **p_θ**，其每一步也是一个高斯分布：

![](img/85bf00ca226b74abfc1f53da9723031a_63.png)

![](img/85bf00ca226b74abfc1f53da9723031a_65.png)

**p_θ(x_{t-1} | x_t) = N(μ_θ(x_t, t), Σ_θ(x_t, t))**

![](img/85bf00ca226b74abfc1f53da9723031a_67.png)

![](img/85bf00ca226b74abfc1f53da9723031a_69.png)

![](img/85bf00ca226b74abfc1f53da9723031a_71.png)

我们希望 **p_θ(x_{t-1} | x_t)** 尽可能接近 **q(x_{t-1} | x_t, x₀)**。这引出了两个核心思想：

**核心思想一：固定方差**
我们不学习方差 **Σ_θ**，而是直接使用精确反向过程中已知的方差：

![](img/85bf00ca226b74abfc1f53da9723031a_73.png)

![](img/85bf00ca226b74abfc1f53da9723031a_75.png)

![](img/85bf00ca226b74abfc1f53da9723031a_77.png)

**Σ_θ(x_t, t) = σ̃_t² I**

![](img/85bf00ca226b74abfc1f53da9723031a_79.png)

**核心思想二：学习均值 μ_θ**
我们训练一个神经网络（如U-Net）来预测均值。关键在于，精确均值 **μ̃_t** 依赖于未知的 **x₀**，而我们的网络 **μ_θ** 只能看到加噪图像 **x_t** 和时间步 **t**。

![](img/85bf00ca226b74abfc1f53da9723031a_81.png)

![](img/85bf00ca226b74abfc1f53da9723031a_83.png)

因此，我们有以下三种参数化 **μ_θ** 的策略：

![](img/85bf00ca226b74abfc1f53da9723031a_85.png)

![](img/85bf00ca226b74abfc1f53da9723031a_87.png)

以下是三种主要的参数化策略：

![](img/85bf00ca226b74abfc1f53da9723031a_89.png)

![](img/85bf00ca226b74abfc1f53da9723031a_91.png)

1.  **选项A：直接预测均值**
    让网络 **μ_θ(x_t, t)** 直接输出对 **μ̃_t** 的估计。损失函数是预测均值与真实均值之间的均方误差（MSE）。

![](img/85bf00ca226b74abfc1f53da9723031a_93.png)

2.  **选项B：预测原始图像 x₀**
    让网络 **f_θ⁰(x_t, t)** 预测原始图像 **x̂₀**。然后利用公式 **μ̃_t = α_t⁰ * x̂₀ + α_t^t * x_t** 来构造估计的均值 **μ̂_t**。

3.  **选项C：预测噪声 ε**
    让网络 **ε_θ(x_t, t)** 预测在前向过程中添加到 **x₀** 上以得到 **x_t** 的噪声 **ε̂**。由于 **x_t = √ᾱ_t * x₀ + √(1-ᾱ_t) * ε**，我们可以反推出：
    **x̂₀ = (x_t - √(1-ᾱ_t) * ε̂) / √ᾱ_t**
    再将 **x̂₀** 代入选项B的公式，即可得到 **μ̂_t**。经验表明，**选项C（预测噪声）通常效果最好**，因为它更直接地利用了前向过程的噪声特性，并且训练更稳定。

![](img/85bf00ca226b74abfc1f53da9723031a_95.png)

**训练流程**：
在实际训练中，我们不会遍历所有时间步。对于每个训练样本 **x₀**，我们：
1.  随机采样一个时间步 **t ~ Uniform(1, T)**。
2.  采样噪声 **ε ~ N(0, I)**，根据性质一计算 **x_t**。
3.  根据选择的参数化策略（如选项C），计算网络预测值（如 **ε_θ(x_t, t)**）与真实值（如 **ε**）之间的损失（如MSE）。
4.  通过梯度下降更新网络参数 **θ**。

![](img/85bf00ca226b74abfc1f53da9723031a_97.png)

![](img/85bf00ca226b74abfc1f53da9723031a_99.png)

这种随机采样时间步的方式类似于随机梯度下降，能提供更多样的梯度信号。

![](img/85bf00ca226b74abfc1f53da9723031a_101.png)

## 从学习到的模型中进行采样 🎨

训练完成后，我们可以从学习到的反向过程 **p_θ** 中采样生成新图像：

1.  从标准高斯分布采样初始噪声：**x_T ~ N(0, I)**。
2.  从 **t = T** 到 **t = 1** 循环：
    *   根据参数化策略，利用 **μ_θ(x_t, t)** 和固定方差 **σ̃_t²**，定义当前步的条件高斯分布 **p_θ(x_{t-1} | x_t)**。
    *   从该分布中采样：**x_{t-1} = μ_θ(x_t, t) + σ̃_t * ε**，其中 **ε ~ N(0, I)**。
3.  最终得到的 **x₀** 即为生成的图像。

**为什么采样时要添加噪声？** 因为我们的目标是**从分布 p_θ(x₀) 中抽取样本**，而不仅仅是获取其均值（均值可能是一张合理的图像，但不是随机样本）。通过每一步从高斯分布中采样，我们执行的是对边缘分布 **p_θ(x₀) = ∫ p_θ(x_0:T) dx_{1:T}** 的蒙特卡洛估计。

## 变分推断与证据下界（ELBO） 📉

我们之前提到，直接最大化数据的对数似然 **log p_θ(x₀)** 是困难的。扩散模型的训练可以通过**变分推断**的框架来理解，其核心是最大化**证据下界**。

![](img/85bf00ca226b74abfc1f53da9723031a_105.png)

对于扩散模型，其ELBO可以推导并分解为以下形式（忽略常数项）：

![](img/85bf00ca226b74abfc1f53da9723031a_107.png)

**L = E_q [ log p_θ(x_0:T) / q(x_1:T | x_0) ] ≤ -log p_θ(x₀)**

进一步分解后，ELBO包含一系列KL散度项：

**L = L_T + Σ_{t=2}^{T} L_{t-1} + L_0**

其中关键的一项 **L_{t-1}** 正是我们之前试图最小化的目标：

![](img/85bf00ca226b74abfc1f53da9723031a_109.png)

![](img/85bf00ca226b74abfc1f53da9723031a_111.png)

![](img/85bf00ca226b74abfc1f53da9723031a_113.png)

**L_{t-1} = D_KL( q(x_{t-1} | x_t, x₀) || p_θ(x_{t-1} | x_t) )**

![](img/85bf00ca226b74abfc1f53da9723031a_114.png)

![](img/85bf00ca226b74abfc1f53da9723031a_116.png)

**KL散度** 是衡量两个概率分布 **Q** 和 **P** 差异的非对称性度量：

![](img/85bf00ca226b74abfc1f53da9723031a_118.png)

![](img/85bf00ca226b74abfc1f53da9723031a_120.png)

![](img/85bf00ca226b74abfc1f53da9723031a_122.png)

![](img/85bf00ca226b74abfc1f53da9723031a_124.png)

![](img/85bf00ca226b74abfc1f53da9723031a_126.png)

**D_KL(Q||P) = E_{x~Q} [ log Q(x) / P(x) ]**

它在意 **Q** 中概率高的区域是否与 **P** 匹配，但对 **Q** 中概率低、**P** 中概率高的区域惩罚较小。最小化 **D_KL(Q||P)** 就是在寻找一个尽可能接近 **P** 的分布 **Q**。

![](img/85bf00ca226b74abfc1f53da9723031a_128.png)

![](img/85bf00ca226b74abfc1f53da9723031a_130.png)

![](img/85bf00ca226b74abfc1f53da9723031a_132.png)

在扩散模型中，我们通过最小化 **L_{t-1}**（即KL散度）来训练 **p_θ**，使其接近精确的反向过程 **q**，这恰好是在优化ELBO，从而间接地最大化了数据似然。

![](img/85bf00ca226b74abfc1f53da9723031a_134.png)

## 总结 🎯

![](img/85bf00ca226b74abfc1f53da9723031a_136.png)

本节课我们一起学习了扩散模型的完整构建与训练流程：
1.  **回顾了扩散模型**的前向（加噪）与反向（去噪）过程框架。
2.  **深入分析了前向过程的性质**，使我们能够高效地采样任意时间步的加噪图像。
3.  **探讨了参数化反向过程的三种策略**，并理解了预测噪声（选项C）通常是更优的选择。
4.  **描述了从训练好的扩散模型中采样生成新图像**的迭代过程。
5.  **引入了变分推断和ELBO的概念**，从理论层面解释了扩散模型训练目标（最小化KL散度）的合理性，即通过优化可处理的下界来近似难以直接计算的最大似然目标。

![](img/85bf00ca226b74abfc1f53da9723031a_138.png)

通过结合强大的神经网络架构（如U-Net）和严谨的概率图模型理论，扩散模型成为了当前最强大的生成式AI模型之一。
