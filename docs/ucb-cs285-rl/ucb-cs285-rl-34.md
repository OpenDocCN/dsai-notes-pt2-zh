# 34：连续动作空间下的 Q 学习 🎯

在本节课中，我们将要学习如何将 Q 学习算法从离散动作空间扩展到连续动作空间。我们将探讨在连续动作空间中执行 `argmax` 操作所面临的挑战，并介绍几种解决此问题的有效方法。

## 连续动作空间的问题所在 🤔

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_5.png)

上一节我们介绍了离散动作空间下的 Q 学习。本节中我们来看看当动作空间是连续时，算法会遇到什么核心问题。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_7.png)

问题在于选择动作时需要执行 `argmax` 操作。对于离散动作，这很直接：只需评估每个可能动作的 Q 值并选择最佳的一个。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_9.png)

```python
# 离散动作的 argmax
best_action = argmax_a Q(s, a) for a in discrete_action_set
```

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_11.png)

但当动作是连续的时，这个操作就变得困难得多。这主要出现在两个地方：
1.  在评估 `argmax` 策略时。
2.  在计算目标值时，这需要 `max` 操作（或在双 Q 学习中是另一个 `argmax`）。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_13.png)

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_15.png)

其中，目标值的 `max` 操作尤其成问题，因为它发生在训练的内部循环中，因此我们希望它非常快速和高效。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_17.png)

---

## 解决方案一：使用优化程序 🔍

既然连续动作的 `argmax` 本质上是一个优化问题，我们可以使用优化程序来解决。以下是几种可行的选择。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_19.png)

### 1. 梯度下降法
一种选择是使用梯度下降等连续优化程序。但这可能很慢，因为它需要多步计算，而这是在外部学习循环的内部发生的。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_21.png)

### 2. 随机优化法
我们的动作空间通常是低维的，这使问题变得相对简单。因此，一个非常好的选择是使用无导数的随机优化程序。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_23.png)

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_25.png)

让我们具体讨论一下这个问题。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_27.png)

一个非常简单的解决方案是，将连续动作集上的最大值近似为在随机采样动作集上的离散最大值。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_29.png)

以下是具体步骤：
1.  从有效的动作集中均匀随机采样 N 个动作。
2.  计算这些动作的 Q 值。
3.  取其中 Q 值最大的动作作为近似最优动作。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_31.png)

这种方法不会给出精确的最大值，但如果动作空间维度较低且采样足够多，近似效果可能相当好。它的优势在于非常简单和高效，因为可以像处理小批量数据一样同时评估所有采样动作。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_33.png)

然而，它的缺点是不准确，尤其是在动作空间维度增加时，随机采样无法给出非常准确的最大值。但如果过度估计是我们的主要问题，一个不那么精确的最大值或许是可以接受的。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_35.png)

### 3. 交叉熵方法 (CEM)
如果想要更精确的解决方案，可以使用基于相同原理但更好的算法，例如交叉熵方法。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_37.png)

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_39.png)

交叉熵方法是一种简单的迭代随机优化方案。其直觉是：
1.  首先像简单方案一样采样一组动作。
2.  然后，不是简单地取最好的一个，而是根据表现好的动作来 refine（精炼）采样分布。
3.  从 refined 后的分布（即更好的区域）采样更多样本。
4.  重复此过程。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_41.png)

这也可以是一个非常快速的算法，特别是在可以并行化且动作空间维度较低的情况下。对于大约 40 维的动作空间，这些方法效果良好。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_43.png)

**核心思想**：在使用这些解决方案时，只需将 `argmax` 操作替换为这种优化过程来找到最佳动作，Q 学习过程的其余部分基本保持不变。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_45.png)

## 解决方案二：使用易于优化的函数类 📐

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_49.png)

上一节我们介绍了通过优化程序求解 `argmax`。本节我们来看看另一种思路：设计本身易于执行 `argmax` 的 Q 函数。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_51.png)

我们可以使用一类具有闭式极值解的函数。任意神经网络相对于其输入并不容易优化，但其他函数类有闭式解。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_53.png)

这种函数类的一个例子是二次函数。例如，可以将 Q 函数表示为关于动作的二次函数：

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_55.png)

```python
Q(s, a) = -1/2 * (a - μ(s))^T P(s) (a - μ(s)) + V(s)
```
其中，`μ(s)` 是向量，`P(s)` 是正定矩阵，`V(s)` 是标量。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_57.png)

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_59.png)

二次函数的最优解有闭式解。因此，可以构建一个输出三个量的神经网络：
*   一个标量 `V_φ(s)`。
*   一个向量 `μ_φ(s)`。
*   一个矩阵 `P_φ(s)`。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_61.png)

向量和矩阵共同定义了关于动作的二次函数。这样，Q 函数在状态上是完全非线性的，可以表示状态的任何函数，但对于给定状态，关于动作的 Q 值形状总是二次的。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_63.png)

当总是二次时，你总是可以找到最大值。在这种情况下，最大值就是 `μ_φ(s)`（只要 `P_φ(s)` 是正定矩阵）。这被称为 **归一化优势函数 (Normalized Advantage Function, NAF)**。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_65.png)

被称为“归一化”的原因是，如果你对它取指数，会得到一个归一化的概率分布。因此，`argmax_a Q_φ(s, a) = μ_φ(s)`，而 `max_a Q_φ(s, a) = V_φ(s)`。

**优势与代价**：现在，我们使最大化操作变得非常容易，但代价是牺牲了 Q 函数的表示能力。如果真实的 Q 函数不是关于动作二次的，那么我们就无法精确表示它。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_67.png)

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_69.png)

算法本身没有改变，它像 Q 学习一样高效，但失去了一些表示能力。

## 解决方案三：学习一个近似最大化器 🧠

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_71.png)

本节我们将介绍最后一种方法：通过训练一个单独的神经网络来近似执行最大化操作，这与深度确定性策略梯度算法密切相关。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_73.png)

其核心思想是：既然 `max_a Q_φ(s, a)` 就是在 `argmax` 处评估的 `Q_φ`，那么我们可以训练另一个网络 `μ_θ(s)`，使得 `μ_θ(s)` 近似等于 `argmax_a Q_φ(s, a)`。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_75.png)

你也可以将 `μ_θ(s)` 视为一种策略，因为它观察状态并输出动作（特别是近似最优动作）。我们如何训练它？只需要求解 `θ` 以最大化 `Q_φ(s, μ_θ(s))`。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_77.png)

这可以通过梯度上升实现。梯度可以通过链式法则计算：
`dQ_φ / dθ = (da / dθ) * (dQ_φ / da)`
其中 `a = μ_θ(s)`。我们可以通过 Q 函数反向传播得到 `dQ_φ / da`，再通过 `μ` 网络反向传播得到 `da / dθ`。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_79.png)

现在，我们的目标值 `y` 将变为：
`y = r_j + γ * Q_φ‘(s’_j, μ_θ‘(s’_j))`
这里使用目标网络 `Q_φ‘` 和 `μ_θ‘` 来稳定训练。只要 `μ_θ` 是对 `argmax` 的良好估计，这就是一个有效的近似。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_81.png)

以下是该算法的基本步骤：

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_83.png)

1.  **收集数据**：执行动作 `a_i`，观察转移 `(s_i, a_i, s’_i, r_i)` 并存入经验回放缓冲区。
2.  **采样批次**：从缓冲区中均匀随机采样一个小批量的转移 `(s_j, a_j, s’_j, r_j)`。
3.  **计算目标**：使用目标网络计算目标值 `y_j = r_j + γ * Q_φ‘(s’_j, μ_θ‘(s’_j))`。
4.  **更新 Q 网络**：通过最小化 `Q_φ(s_j, a_j)` 与 `y_j` 之间的误差来更新 `φ`。
5.  **更新策略网络**：通过最大化 `Q_φ(s_j, μ_θ(s_j))` 来更新 `θ`（即沿 `Q` 值梯度上升）。
6.  **更新目标网络**：使用软更新（如 Polyak 平均）缓慢更新目标网络参数 `φ‘` 和 `θ‘`。
7.  **重复**：重复步骤 1-6。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_85.png)

这就是一个适用于连续动作的 Q 学习算法的基本伪代码。这个特定算法通常被称为 **深度确定性策略梯度**。它还有许多更早（如 NFQ-CA）和更近的变体。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_87.png)

## 总结 📝

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_91.png)

本节课中我们一起学习了如何将 Q 学习应用于连续动作空间。我们分析了核心挑战在于高效计算 `argmax` 操作，并介绍了三种主要解决方案：
1.  **使用优化程序**：如随机采样或交叉熵方法，在运行时进行优化。
2.  **使用易于优化的函数类**：如归一化优势函数，通过限制 Q 函数形式使 `argmax` 有闭式解。
3.  **学习一个近似最大化器**：如 DDPG 算法，训练一个独立的策略网络来近似最优动作。

![](img/ec4f7fca8a2e53a95bbb64e0bd6482f6_93.png)

每种方法都在计算效率、实现复杂度和表示能力之间有不同的权衡，可以根据具体问题选择合适的方法。
